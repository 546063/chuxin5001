## Step 1：基础指标计算

拿到数据后，先算出这几个关键数字：

```
ROAS = 销售额 ÷ 花费
CPA  = 花费 ÷ 转化数
CVR  = 转化数 ÷ 点击数（注：如果不知道点击数，跳过这步）

AOV  = 销售额 ÷ 订单数
```

---

## Step 2：好与坏的判断

### 判断标准一：ROAS 是否达标

| 目标 ROAS | ROAS 表现 | 判断 |
|-----------|---------|------|
| 2.0 | > 2.0 | ✅ 达标，继续维持 |
| 2.0 | 1.5 - 2.0 | 🟡 接近，密切关注 |
| 2.0 | < 1.5 | 🔴 不达标，需优化 |
| 2.0 | < 1.0 | 🚨 严重不达标，建议关停 |

### 判断标准二：CPA 是否盈利

```
盈利线 CPA = 客单价 × 毛利率

例：客单价 $50，毛利率 60% → 盈利线 = $30
```

| CPA vs 盈利线 | 判断 |
|--------------|------|
| CPA < 盈利线的 70% | ✅ 优秀，可以加预算 |
| CPA 在盈利线 70%-100% | 🟡 可接受，尝试优化 |
| CPA > 盈利线 | 🔴 亏损，需立即优化 |
| CPA > 盈利线 2 倍 | 🚨 严重亏损，建议关停 |

### 判断标准三：学习期状态

| 学习期状态 | 判断 |
|-----------|------|
| Active（已通过）| ✅ 正常，可做优化决策 |
| Learning（学习中）| 🟡 正常，数据波动大，不做重大决策 |
| Learning 7天+ 未通过 | 🔴 学习期失败，需排查（见诊断 SOP） |

---

## Step 3：漏斗诊断（逐层分析）

### 如果 ROAS 不达标，按以下顺序排查：

**第一步：看 CPM**
- CPM > 行业均值 30%+？→ 问题在展示层
- 解法：扩受众 / 换版位 / 换素材提质量分

**第二步：看 CTR**
- CTR < 1%？→ 问题在点击层
- 解法：换素材（换 Hook 是最快的方式）

**第三步：看 CVR**
- CVR < 2%？→ 问题在转化层
- 解法：查落地页速度/信任元素/价格/结账流程

**第四步：看 AOV**
- AOV 是否比历史下降？→ 产品/定价问题
- 解法：查 SKU 是否缺货/是否在清货/是否有折扣产品占比上升

---

## Step 4：结构诊断

### 检查广告结构是否合理

**结构问题信号**：
- 1 个 Campaign 里超过 8 个 Ad Set（预算太分散）
- 1 个 Ad Set 只放 1 条广告（没有素材测试空间）
- 只跑 1 个 Campaign（没有分层测试空间）

**好的结构特征**：
- TOF（拉新）+ BOF（再营销）分层
- 每个 Ad Set 至少 2-3 条素材
- 至少 1 个 ASC 或 Advantage+ Audience 测试

### 检查频次

| 频次 | 判断 | 动作 |
|------|------|------|
| < 2 | 正常 | 无需操作 |
| 2 - 3 | 偏高 | 关注，7 天后复查看是否继续涨 |
| 3 - 4 | 疲劳信号 | 开始换素材 |
| > 4 | 严重疲劳 | 立即换素材/扩受众 |

---

## Step 5：素材诊断

### 如果数据量足够（> 5000 impressions/素材）

| 指标 | 优秀 | 正常 | 需换 |
|------|------|------|------|
| CTR | > 2% | 1-2% | < 0.8% |
| 3s 留存率 | > 60% | 40-60% | < 40% |
| 视频完播率 | > 50% | 25-50% | < 25% |

### 素材数量是否足够

| 场景 | 最低素材数 | 建议素材数 |
|------|-----------|-----------|
| 手动 CBO | 3 条/Ad Set | 6-10 条/Campaign |
| ASC | 6 条 | 10-15 条 |
| 新品测试 | 3 条 | 5 条 |

## Step 6：复盘结论输出模板

```
## 【计划名称】复盘结论

### 整体评分
- ROAS：X （达标/接近/不达标）
- CPA：$X （盈利线$XX，占比XX%）
- 学习期：通过 / 未通过

### 漏斗表现
- CPM：$X （正常/偏高）
- CTR：X% （优秀/正常/偏低）
- CVR：X% （优秀/正常/偏低）

### 主要问题（按优先级）
1. [最严重问题]：原因 → 解法
2. [次要问题]：原因 → 解法

### 优化建议
- 立即动作（本周）：
  - [ ] 关停/保留 XXX
  - [ ] 上新素材 X 条
  - [ ] 调整预算至 $XX
- 下周观察：
  - [ ] 关注指标 X 是否改善
- 长期优化：
  - [ ] 测试 ASC
  - [ ] 重组账户结构

### 是否值得继续投放？
✅ 继续：ROAS 达标或接近，有优化空间
🟡 继续观察：数据尚不充分，需要更多时间/数据
🔴 关停：ROAS 远不达标，且无明显优化方向
```

## 常见复盘场景示例

### 场景 1：ROAS 1.2，但 CPM 高

```
判断：
- ROAS 1.2 不达标（假设目标 2.0）
- CPM $25（假设行业均值 $15），高 67%
- CTR 1.8%（正常）

结论：问题在展示层（CPM 太高）

解法：
1. 扩受众到 Broad
2. 开启 Advantage+ Placements
3. 检查 Quality Ranking 是否低
4. 换素材提高质量分

复盘结论：继续投放，但优先解决 CPM 问题
```

### 场景 2：ROAS 0.8，CTR 只有 0.5%

```
判断：
- ROAS 0.8 远不达标
- CTR 0.5% 极低（正常 > 1%）
- CPM $12（正常）

结论：问题在点击层（素材不吸引）

解法：
1. 立即换新 Hook 素材
2. 检查 3s 留存率
3. 测试不同类型的 Hook

复盘结论：素材问题为主，先解决素材再判断是否值得继续
```

### 场景 3：ROAS 3.0，但预算跑不满

```
判断：
- ROAS 3.0 远超目标（优秀！）
- 但预算每天只花出去 70%

结论：效果很好但没花够钱

解法：
1. 逐步加预算（每天 +20%）
2. 复制到新的 Campaign 扩量
3. 测试 ASC 能否承接更多预算

复盘结论：继续投放，加预算是主要动作
```

## 待用户提供的数据才能完成的判断

以下情况**需要用户提供数据**才能给出具体建议：

1. **CPM 是否正常**：需要知道用户行业/品类的平均 CPM
2. **CTR 是否正常**：需要知道用户的历史 CTR 或品类基准
3. **是否该加预算**：需要知道用户的目标 CPA 和利润率
4. **是否该关停**：需要知道用户的盈亏平衡线

## 附录 A：评分量化表

> 用打分替代模糊判断。每个维度 0-5 分，满分 30 分。

### 六维评分卡

| 维度 | 5分（优秀） | 3分（及格） | 1分（危险） | 得分 |
|------|------------|------------|------------|------|
| **ROAS** | > 目标 120% | 目标 80-120% | < 目标 50% | __/5 |
| **CPA** | < 盈利线 60% | 盈利线 60-100% | > 盈利线 | __/5 |
| **CTR** | > 2% | 1-2% | < 0.8% | __/5 |
| **CVR** | > 3% | 1.5-3% | < 1% | __/5 |
| **学习期** | Active + 稳定 7天+ | Active 或 Learning 正常 | Learning Limited > 7天 | __/5 |
| **预算消耗** | 90-100% | 70-90% | < 70% | __/5 |

**总分：__/30**

### 评分解读

| 总分区间 | 判断 | 动作 |
|---------|------|------|
| 24-30 分 | 🟢 优秀 | 加预算扩量，保持现状 |
| 18-23 分 | 🟡 及格 | 找失分维度，针对性优化 |
| 12-17 分 | 🔴 危险 | 有严重问题，按诊断 SOP 排查 |
| < 12 分 | 🚨 紧急 | 建议关停重做或大改 |
