## 打法卡1：教培·素质类

> 羽毛球馆/舞蹈班/音乐培训/书法班/游泳/轮滑/击剑

### 行业特征
- **客单价**：体验课 9.9-49.9 元，正价课 2000-8000 元/期
- **决策人**：家长（28-42岁），使用者：孩子
- **决策周期**：7-14 天（需要体验后转化）
- **转化路径**：留资表单 → 邀约体验课 → 正价课成交
- **合规等级**：中低（需教培许可证）

### 推荐工具与计划结构
| 工具 | 用途 | 预算占比 |
|------|------|---------|
| 标准推广·表单目标 | 收集家长信息 | 70% |
| 标准推广·私信目标 | 即时沟通答疑 | 20% |
| 标准推广·直播间 | 公开课/体验课直播 | 10% |

### 定向公式
```
年龄：28-42 岁
性别：女（占决策 70%+）
地域：门店周边 3-8 公里
兴趣标签：亲子教育 + 少儿培训 + 对应品类（舞蹈/体育/音乐）
DMP：附近小区 + 有孩家庭标签
```

### 素材模板（3 种）

| 模板 | 内容 | 时长 | 预期 CTR |
|------|------|------|---------|
| **效果展示** | 学员前后对比（不会跳绳→连续跳100个） | 30s | 3-5% |
| **专业背书** | 教练资质 + 教学方法展示 | 20s | 2-4% |
| **成长故事** | 家长讲述孩子变化（真实采访） | 45s | 4-6% |

### 出价基准
| 指标 | 基准值 |
|------|--------|
| 表单 CPA | 60-120 元 |
| 体验课到店率 | 40-60% |
| 体验→正价转化率 | 20-35% |
| 正价课 CAC | 800-1500 元 |

### 避坑要点
- ❌ 不要承诺"包过级""包拿奖"
- ❌ 不要使用未成年人单独出镜（需监护人同意书）
- ✅ 素材突出"体验课"降低决策门槛
- ✅ 落地页放教练资质 + 校区环境 + 家长评价

## 打法卡2：教培·学历类

> 中专/职校/技校/成人学历提升

### 行业特征
- **客单价**：3000-15000 元
- **决策人**：学生本人（16-25岁）或家长（40-55岁）
- **决策周期**：15-30 天
- **转化路径**：短视频 → 表单/私信 → 电销 → 到校参观 → 报名
- **合规等级**：极高（办学许可证 + 广告审查表）

### 推荐工具
| 工具 | 用途 |
|------|------|
| 标准推广·表单目标 | 核心转化渠道 |
| 搜索直投 | 截流"XX学校怎么样"等关键词 |

### 定向公式
```
学生人群：16-25 岁 + 教育兴趣 + 初高中标签
家长人群：40-55 岁 + 教育/升学兴趣 + 二三线城市
地域：全省（学历类可跨城）
```

### 素材模板
| 模板 | 内容 | 合规要点 |
|------|------|---------|
| **校园实景** | 宿舍/教室/实训室实拍 | 不能修图过度 |
| **就业保障** | 合作企业 + 就业率数据 | 数据需真实可查 |
| **政策解读** | 升学政策变化 + 应对方案 | 避免"包过"承诺 |

### 出价基准
| 指标 | 基准值 |
|------|--------|
| 表单 CPA | 80-200 元 |
| 表单→到校率 | 15-25% |
| 到校→报名率 | 30-50% |
| 最终 CAC | 1000-3000 元 |

### 避坑要点
- ❌ 绝对不能用"包过""包就业""包分配"
- ❌ 不能用"最便宜""学费最低"等价格比较
- ❌ 不能使用"国家级""重点"等未经审批的表述
- ✅ 所有数据标注来源，所有承诺有合同依据

> 整装/局装/旧房翻新/地板/瓷砖/门窗/橱柜

- **客单价**：整装 5-20 万，局装 1-5 万，建材单品 2000-30000 元
- **决策周期**：30-90 天
- **决策人**：夫妻双方 + 有时父母参与
- **转化路径**：内容种草 → 留资 → 免费量房/设计方案 → 签约
- **合规等级**：中高（施工资质）

### 推荐工具
| 工具 | 用途 | 预算占比 |
|------|------|---------|
| 标准推广·表单目标 | 核心转化 | 50% |
| 标准推广·私信目标 | 即时设计咨询 | 25% |
| 标准推广·直播间 | 装修案例直播讲解 | 15% |
| 搜索直投 | 截流品牌词+品类词 | 10% |

### 定向公式
```
年龄：28-48 岁
地域：同城 + 周边 20 公里（可跨城）
兴趣：房产 + 家居 + 装修 + 新房交付
DMP：新交付小区人群 + 近期搜索装修关键词
```

### 素材模板
| 模板 | 内容 | 时长 | CTR |
|------|------|------|-----|
| **工地实拍** | 施工过程透明化展示 | 30-60s | 3-5% |
| **前后对比** | 改造前 vs 改造后 | 30s | 4-6% |
| **透明报价** | 公开套餐价格明细 | 20s | 2-4% |
| **业主访谈** | 真实业主入住体验 | 45-60s | 3-5% |

### 出价基准
| 指标 | 基准值 |
|------|--------|
| 表单 CPA | 150-400 元 |
| 表单→量房率 | 20-35% |
| 量房→签约率 | 15-25% |
| 最终 CAC | 3000-8000 元 |

### 避坑要点
- ❌ 不能展示未完工工地（需客户授权）
- ❌ 价格描述必须与实际一致
- ✅ "免费量房""免费设计"是强诱饵
- ✅ 素材突出"透明报价""无增项"降低信任门槛

## 打法卡4：餐饮/生活服务

> 餐厅/咖啡厅/美甲/美发/SPA/健身/便利店

### 行业特征
- **客单价**：15-200 元
- **决策周期**：即时/当天
- **转化路径**：短视频/直播 → 团购下单 → 到店核销
- **合规等级**：低（食品经营许可证）
- **核心目标**：到店核销率

### 推荐工具
| 工具 | 用途 |
|------|------|
| 全域推广·团购成交 | 核心工具，算法自动优化 |
| 短视频带货 | 种草引流 |
| 直播带货 | 集中转化 |

### 定向公式
```
地域：门店周边 3-5 公里（餐饮）/ 5-8 公里（美业）
年龄：18-40 岁（根据品类调整）
兴趣：美食探店/网红打卡/对应品类
```

### 素材模板
| 模板 | 内容 | 预期核销率 |
|------|------|----------|
| **菜品特写** | 色香味展示 + 价格 | 60-75% |
| **环境体验** | 店内环境 + 氛围 | 50-65% |
| **优惠套餐** | 多人套餐 + 限时价 | 70-85% |
| **达人探店** | KOL 真实体验 | 55-70% |

### 出价基准
| 品类 | CPM | CTR | 团购 CPA | 核销率 |
|------|-----|-----|---------|--------|
| 餐饮 | 15-30 元 | 3-8% | 5-15 元 | 60-80% |
| 美业 | 20-40 元 | 2-5% | 15-30 元 | 50-70% |
| 健身 | 15-25 元 | 3-6% | 10-25 元 | 40-60% |

### 避坑要点
- ❌ 团购套餐必须与到店实际一致（不能"图文不符"）
- ❌ 不能虚假标原价（需有真实销售记录）
- ✅ 核销率 < 50% 时：检查团购吸引力 + 门店距离 + 预约便利性
- ✅ 周末/节假日前 2 天投放效果最好

## 打法卡5：医疗/医美

> 口腔/眼科/体检/轻医美/整形医院

### 行业特征
- **客单价**：口腔 500-20000 元，医美 1000-50000 元
- **决策周期**：7-30 天
- **合规等级**：极高（医疗机构执业许可证 + 医疗广告审查证明）
- **转化路径**：科普内容 → 留资/私信 → 到院面诊 → 治疗

### 合规前置清单
- [ ] 医疗机构执业许可证
- [ ] 医疗广告审查证明（广审表）
- [ ] 医师执业证书（出镜医生）
- [ ] 素材不能出现：疗效承诺、前后对比（医美）、"最"字

### 推荐工具
| 工具 | 用途 |
|------|------|
| 标准推广·表单目标 | 预约到院 |
| 标准推广·私信目标 | 即时咨询 |
| 短视频科普 | 建立专业信任 |

### 定向公式
```
口腔：25-55 岁 + 健康/口腔兴趣 + 同城 10 公里
医美：20-40 岁 女 + 美业/护肤兴趣 + 同城 15 公里
眼科：35-60 岁 + 健康兴趣 + 同城 15 公里
```

### 素材模板
| 模板 | 内容 | 合规要求 |
|------|------|---------|
| **科普知识** | 医生讲解常见问题 | 内容需医学准确 |
| **案例分享** | 患者讲述体验（非疗效承诺） | 需患者授权 |
| **设备展示** | 先进设备/环境实拍 | 不能夸大设备效果 |

### 出价基准
| 科室 | 表单 CPA | 到院率 | 成交率 | CAC |
|------|---------|--------|--------|-----|
| 口腔 | 80-200 元 | 30-50% | 40-60% | 300-800 元 |
| 医美 | 150-400 元 | 20-40% | 30-50% | 800-2500 元 |
| 眼科 | 100-250 元 | 25-40% | 50-70% | 400-800 元 |

### 避坑要点
- ❌ 绝对不能承诺疗效（"一次见效""根治"）
- ❌ 医美不能展示手术前后对比图
- ❌ 不能用患者治疗中的血腥/不适画面
- ✅ 用"科普"替代"推销"，建立专业信任
- ✅ 所有案例标注"个体差异，效果因人而异"
