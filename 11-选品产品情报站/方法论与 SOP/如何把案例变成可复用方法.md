# 问题：如何把案例变成可复用方法？

## 相关答案

### 提炼框架

1. **要素拆解**
   - 平台 / 行业 / 客户画像 / 投流目标
   - 广告形式 / 素材类型 / 核心钩子
   - 转化路径 / 关键数据 / 成功因素

2. **模式识别**
   - 哪些元素是行业特有的？
   - 哪些元素是跨行业通用的？
   - 成功的关键驱动因素是什么？

3. **策略抽象**
   - 将具体案例中的"做法"抽象为"方法"
   - 去除行业特定细节，保留通用框架
   - 形成可套用的模板

4. **场景适配**
   - 这个方法适合什么行业？
   - 适合什么平台？
   - 需要什么前置条件？
   - 有什么风险点？

### 案例→方法转换模板

```
案例：[具体案例描述]
→ 方法：[抽象后的方法名称]
→ 适用：[适合的行业/平台/场景]
→ 步骤：[1][2][3]
→ 风险：[需要注意的点]
→ 参考：[[相关案例]]
```
