## 0. 数据口径说明

### 0.1 为什么不用“伪精确数字”

Semrush、Ahrefs、Google Keyword Planner、DataForSEO 等工具的精确月搜索量、关键词难度、CPC 都需要账号或API导出。公开网页可以确认这些工具提供搜索量、KD、CPC、意图、趋势等指标，但无法直接批量导出你要求的全部词的精确数值。

因此本文采用：

```text
搜索量 = 区间估算
KD = 关键词难度区间估算
CPC = Google Ads点击成本区间估算
优先级 = 按商业价值 + 竞争难度 + 新站可操作性综合判断
```

# 11. 数据复核SOP

## 11.1 Semrush复核步骤

```text
1. 打开 Semrush Keyword Overview
2. 选择目标国家：US / Germany / Japan / France / Spain / Mexico / Brazil
3. 输入关键词
4. 导出：
   - Volume
   - KD%
   - CPC
   - Intent
   - Trend
   - SERP features
5. 批量导出后更新本文矩阵
```

## 11.2 Ahrefs复核步骤

```text
1. 打开 Ahrefs Keywords Explorer
2. 输入关键词列表
3. 国家选择 US / JP / DE / FR / ES
4. 导出：
   - Volume
   - KD
   - CPC
   - Parent Topic
   - Traffic Potential
   - SERP overview
5. 重点看Top10页面需要多少Referring Domains
```

## 11.3 Google Keyword Planner复核步骤

```text
1. 进入 Google Ads Keyword Planner
2. Discover new keywords
3. 输入英文和本地语言关键词
4. 设置国家和语言
5. 导出：
   - Avg. monthly searches
   - Competition
   - Top of page bid low/high
6. 注意：Google Keyword Planner常给区间，不如Semrush/Ahrefs精细
```
