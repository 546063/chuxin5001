# 问题：如何判断投流案例是否值得复用？

## 相关答案

### 判断维度

1. **行业相似度**
   - 同行业：直接参考（如食品加工机械→食品加工机械）
   - 同类型：B2B→B2B、高客单→高客单可参考
   - 跨行业：仅参考策略思路，不参考具体数据

2. **平台一致性**
   - 同平台：可直接参考定向、素材、出价
   - 跨平台：参考策略思路，但需适配平台特性

3. **预算可比性**
   - 相近预算：数据可直接对比
   - 差距较大：关注趋势和比例，不关注绝对值

4. **目标市场**
   - 同国家/地区：可直接参考
   - 不同市场：需考虑文化、消费习惯差异

5. **时间有效性**
   - 近6个月案例：参考价值高
   - 6-12个月：需确认平台规则是否变化
   - 超过1年：仅作参考，不宜直接复用

### 复用评估清单

- [ ] 行业类型是否相同或相似？
- [ ] 投放平台是否一致？
- [ ] 预算量级是否可比？
- [ ] 目标市场是否相近？
- [ ] 案例时间是否在1年内？
- [ ] 素材类型是否可复制？
- [ ] 转化路径是否相似？
