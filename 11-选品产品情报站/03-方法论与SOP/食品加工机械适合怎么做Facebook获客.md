# 问题：食品加工机械适合怎么做Facebook获客？

## 相关答案

### 适用场景

- B2B食品加工机械出口
- 目标市场：东南亚/中东/非洲/南美
- 产品线：果蔬清洗机、切菜机、肉类加工设备、包子机、饺子机

### 推荐策略

1. **受众定向**
   - 兴趣：食品加工、餐饮设备、食品工厂
   - 行为：B2B采购决策者、行业展会参观者
   - Lookalike：基于已成交客户邮箱/网站访客
   - 地理：目标出口国家

2. **素材类型**
   - 工厂实力展示：生产线实拍、厂房、团队
   - 产品演示：设备运行视频、效果展示
   - 客户案例：海外客户使用反馈、安装案例
   - 流程展示：从咨询到交付的全流程

3. **转化路径**
   - 广告 → 落地页（产品详情+询盘表单）
   - 广告 → WhatsApp直接沟通
   - 广告 → 视频预约（线上看厂）

4. **预算分配**
   - 测试期：$30-50/天，2-3周
   - Scaling期：$50-150/天
   - 再营销：总预算20-30%
