## 一、买家角色矩阵

### 9 类核心决策参与者

| 角色 | 典型头衔 | 核心关注点 | 决策权重 | 活跃平台 |
|------|---------|-----------|---------|---------|
| **工厂老板/CEO** | Owner, Founder, Managing Director | ROI、回本周期、总拥有成本、供应商可信度 | **拍板人** | LinkedIn, YouTube, 展会 |
| **采购经理** | Purchasing Manager, Procurement Director | 价格、付款条件(L/C,T/T)、Incoterms、供应商资质(SGS/TUV) | **询价人** | B2B平台, LinkedIn, RFQ门户 |
| **工厂运营负责人** | Plant Manager, Operations Manager | 产能、停机率、维护便捷度、操作员培训、备件可得性 | **参数比较人** | YouTube(教程视频), 同行网络 |
| **技术/工程总监** | Chief Engineer, Technical Director | 技术参数、认证(CE/FDA/ISO22000/HACCP)、产线集成能力、自动化水平 | **技术把关人** | 制造商官网, 工程论坛, 技术规格书 |
| **食品加工厂负责人** | Factory Director, Production Director | 产能匹配、食品安全合规、产线兼容性、招工难度倒逼自动化 | **需求发起人** | 展会, LinkedIn, 行业媒体 |
| **餐饮中央厨房采购** | Central Kitchen Manager, F&B Operations | 标准化程度、清洗/切割一致性、卫生等级、占地面积、人工替代率 | **需求发起人** | YouTube, Facebook群组, 同行推荐 |
| **冷冻食品厂采购** | Frozen Food Plant Manager | 低温环境适应性、解冻损耗、产能峰值匹配、自动化程度 | **技术把关+需求发起** | YouTube, 行业展会, 设备供应商官网 |
| **海外经销商** | Distributor, Trading Company, Agent | 利润空间、供货稳定性、沟通效率、文档完整性、售后责任划分 | **转售方** | Alibaba, Made-in-China, 展会, WeChat |
| **工程项目集成商** | EPC Contractor, System Integrator | 整线方案能力、技术文档质量、安装支持、多设备兼容性、交期可靠性 | **方案整合者** | LinkedIn, 行业展会, 技术研讨会 |

### 决策链权重分配

```
老板/CEO ──→ 拍板 (最终决定权，通常只看 ROI 和信任)
技术总监 ──→ 技术否决权 (不合格直接否决)
采购经理 ──→ 价格谈判权 (压价主力)
运营经理 ──→ 使用体验权重 (日常维护痛点)
经销商 ──→ 转售可行性 (如果走渠道)
```

## 二、他们为什么买？（购买触发事件）

### 6 大核心购买触发点

1. **现有设备故障或老化** → 紧急替换，决策快（2-4 周），对价格敏感度低
2. **产能扩张/新建产线** → 计划性采购，周期长（3-12 月），多方比价
3. **法规/食品安全标准变更** → 合规驱动，不可拖延，认证要求严格
4. **人工成本上涨/招工难** → 自动化替代，关注"省几个人"、"多久回本"
5. **新产品线/新市场进入** → 需要新设备类型，关注灵活性和适配性
6. **竞争对手升级** → 跟随性采购，"别人有了我也得有"

### 不同触发点的内容策略差异

| 触发类型 | 买家状态 | 有效内容类型 | CTA 方向 |
|---------|---------|------------|---------|
| 紧急替换 | 焦虑、时间紧 | 快速选型指南、现货库存、快速发货案例 | "3天发货"、"现货可选" |
| 计划扩容 | 理性比较 | ROI计算器、整线方案、客户案例 | "免费方案设计"、"下载规格书" |
| 合规驱动 | 刚需、怕出错 | 认证清单、合规对照表、检测报告 | "CE认证确认"、"合规咨询" |
| 自动化替代 | 算经济账 | 人工成本对比、回本周期计算 | "算算你能省多少" |

## 四、购买旅程（6-10 个利益相关者，3-12 个月）

```
阶段 1: 问题识别 (2-4周)
  → 旧机器坏了 / 产能不够 / 新产线计划
  → 内容机会: 痛点型短视频、行业趋势内容

阶段 2: 需求评估 (2-6周)
  → 技术参数确定、预算审批
  → 内容机会: 选型指南、参数对照表

阶段 3: 供应商发现 (4-8周)  ← 社媒最关键阶段
  → 展会、B2B平台、LinkedIn、推荐
  → 内容机会: 工厂实拍、客户案例、认证展示

阶段 4: RFQ 与规格对比 (4-6周)
  → 3-5 家短名单、技术评估
  → 内容机会: 产品规格书、对比表、视频演示

阶段 5: 工厂审核 (2-4周)
  → 虚拟考察或实地拜访、第三方检验
  → 内容机会: VR工厂参观、检测报告、资质文件

阶段 6: 商业谈判 (2-6周)
  → 价格、付款方式、保修、售后
  → 内容机会: 售后案例、备件库存展示、保修政策

阶段 7: 采购与物流 (4-12周)
  → 合同、生产、发货、清关
  → 内容机会: 装柜视频、物流追踪、清关协助

阶段 8: 安装与调试 (2-4周)
  → 培训、交接
  → 内容机会: 安装视频、操作教程、客户见证
```
