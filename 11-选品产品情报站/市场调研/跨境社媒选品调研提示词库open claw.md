## 提示词

```text

你现在是一名跨境电商与社媒选品分析师，请帮我评估这个产品是否适合未来1到2年做跨境社媒成交。

  

产品名称：{产品名}

  

请从以下维度输出详细分析，并尽量引用可验证的数据来源、页面、趋势、用户反馈或平台现象：

  

1. 市场需求趋势

- 过去12个月和过去3到5年的需求趋势

- 是否为长期需求还是短期热点

- 是否存在明显季节性

- 哪些国家或地区需求更强

  

2. 社交媒体成交适配度

- 这个产品是否适合通过 TikTok / Instagram / Facebook / YouTube Shorts 成交

- 是否具备“3秒看懂、15秒种草、容易转化”的特点

- 是否容易做开箱、测评、前后对比、教程、场景化视频

- 用户是否容易冲动下单

  

3. 竞争强度

- 目前主流平台上同类产品的竞争是否激烈

- 是否已经高度同质化

- 是否仍有内容差异化切入空间

- 是否有新卖家进入机会

  

4. 利润与履约可行性

- 该产品大致适合什么客单价区间

- 是否适合跨境小包、海外仓或本地仓

- 是否容易因运费、体积重、易碎、退货等问题影响利润

- 是否适合达人分销或投流成交

  

5. 风险与限制

- 是否可能涉及认证、知识产权、敏感功效宣称、平台限制或法律风险

- 是否容易产生售后纠纷或高退货率

- 是否存在侵权、仿品、商标或专利风险

  

6. 未来1到2年的持续性

- 该产品未来1到2年继续做的可能性如何

- 更像短期风口，还是可以长期经营

- 是否适合做品牌化或扩展系列产品

  

最后请输出：

A. 结论：适合 / 谨慎测试 / 不建议做

B. 核心理由

C. 最值得关注的3个数据指标

D. 建议优先测试的3个国家

E. 建议优先测试的3种内容方向

适合什么时候用

- 刚拿到一个产品
- 客户或供应链推荐了一个新品
- 你想先做第一轮筛选

你重点看什么

- 它最终给出的结论是否明确
- 有没有把“趋势、内容、竞争、风险”都覆盖到
- 它给出的国家和内容方向是否具体

  

02 判断未来 1–2 年还能不能做

使用场景

当你不是只想知道“现在能不能卖”，而是想判断这个产品未来 1–2 年值不值得持续投入时用。

提示词

请帮我判断这个产品未来1到2年是否还值得做，不要只看当前热度，要重点看持续性。

  

产品：{产品名}

  

请重点分析：

1. 过去12个月趋势

2. 过去3到5年趋势

3. 是否属于短期爆红型产品

4. 是否已经接近饱和

5. 是否有多个国家同时增长

6. 是否有复购潜力

7. 是否能延伸出系列产品

8. 是否适合长期品牌化经营

  

请最后给出明确结论：

- 长期可做

- 可以短中期测试

- 不建议作为未来1到2年的重点产品

  

并说明依据。

适合什么时候用

- 你已经发现它最近挺火
- 但你担心它只是风口
- 你不想做“昙花一现”的产品

你重点看什么

- 是不是多个国家都在增长
- 它是不是靠单一热点带起来的
- 有没有复购、系列化、品牌化可能

  

03 判断是否适合社媒成交

使用场景

重点判断这个产品是不是适合用内容带货，而不是只看搜索电商。

提示词

请分析这个产品是否适合通过社交媒体内容成交，重点不是传统电商，而是内容带货。

  

产品：{产品名}

  

请从以下角度分析：

1. 用户是否能快速理解产品价值

2. 是否适合短视频展示

3. 是否有明显的前后对比或使用效果

4. 是否有情绪价值、视觉吸引力或新奇性

5. 是否容易引发评论互动和分享

6. 是否适合达人种草、直播带货、UGC传播

7. 是否适合冲动型下单

8. 是否需要复杂教育成本

  

请最后输出：

- 社媒成交适合度评分（100分制）

- 最适合的平台排序

- 最适合的内容形式排序

- 为什么适合或不适合

适合什么时候用

- 你做的是 TikTok / Instagram / Facebook 内容成交
- 你想判断它适不适合达人或短视频
- 你担心产品太难解释

你重点看什么

- 能不能 3 秒看懂
- 能不能 15 秒展示价值
- 有没有天然的短视频表达力
- 是否适合冲动型购买

  

04 判断竞争与红海程度

使用场景

当你担心这个产品已经被做烂了，或者想看还有没有切入空间时用。

提示词

请评估这个产品当前是不是红海，以及是否还有切入机会。

  

产品：{产品名}

  

请分析：

1. 主流平台上同类内容和商家数量是否很多

2. 同类产品价格带是否已经打穿

3. 头部卖家是否垄断明显

4. 新卖家是否还有机会

5. 同类内容是否严重重复

6. 是否还能从人群、场景、功能、包装、内容角度做差异化

7. 是否存在“看起来很火但实际上利润很差”的情况

  

最后输出：

A. 红海程度：低 / 中 / 高

B. 切入难度：低 / 中 / 高

C. 可行的差异化方向

D. 是否建议进入

适合什么时候用

- 你看到很多人在卖
- 你怀疑这个产品很卷
- 你想看自己还能从哪里切进去

你重点看什么

- 是真的有市场，还是因为已经卷成红海看起来很热闹
- 内容是不是已经严重同质化
- 有没有“新卖家还能做的角度”

  

05 排查风险与限制

使用场景

做跨境外贸时，提前排查认证、侵权、平台限制、售后问题。

提示词

请帮我排查这个产品做跨境社媒和外贸销售的主要风险。

  

产品：{产品名}

目标市场：{国家或地区}

  

请重点排查：

1. 合规认证风险

2. 知识产权风险

3. 平台禁限售风险

4. 功效宣传风险

5. 物流运输限制

6. 售后与退货风险

7. 差评高发问题

8. 是否容易因为质量、尺寸、体验不一致而引发纠纷

  

最后输出：

- 高风险项

- 中风险项

- 低风险项

- 是否适合新手卖家做

- 是否适合长期放大

适合什么时候用

- 你准备进入一个新类目
- 产品可能涉及认证或敏感属性
- 你想避免刚做就踩坑

你重点看什么

- 有没有硬性合规门槛
- 有没有明显侵权风险
- 是否容易售后爆炸
- 是否适合新手测试

  

06 多产品横向对比

使用场景

你手里有多个产品，不知道先测哪个，用这条让 OpenClaw 帮你排优先级。

提示词

请把以下产品按照“未来1到2年适合做跨境社媒成交”的潜力进行横向对比和排序：

  

产品列表：

1. {产品A}

2. {产品B}

3. {产品C}

4. {产品D}

  

请统一按以下维度打分：

1. 市场趋势

2. 社媒内容适配

3. 竞争强度

4. 利润空间

5. 履约难度

6. 风险控制

7. 长期性

8. 品牌化潜力

  

请输出：

- 每个产品的详细评分

- 总分排名

- 最适合测试的产品

- 最不建议做的产品

- 每个产品的核心理由

适合什么时候用

- 你选了 3–10 个候选产品
- 你要做优先级排序
- 你不想凭感觉拍脑袋

你重点看什么

- 排名是否合理
- 它是否把“能卖”和“能长期做”区分开了
- 低分产品是不是卡在风险、利润或内容适配

  

附加建议

建议 1：不要一次问太大

不要这样问：

帮我把这个产品所有东西都查清楚

这种提示词通常会让输出很散。

更好的做法是：

- 先用 [[#01 产品可行性总分析]]
- 再针对重点问题继续追问
- 然后用 [[#05 排查风险与限制]] 补风险
- 最后再用 [[#06 多产品横向对比]] 做决策

  

建议 2：要求固定输出结构

你可以在每条提示词末尾都加这一段：

请按以下格式输出：

1. 结论

2. 核心依据

3. 关键数据

4. 风险点

5. 建议动作

这样便于你在 Obsidian 里统一整理。

  

建议 3：让它区分“事实”和“判断”

你还可以额外加一句：

请把“客观数据/事实”和“你的判断/推断”分开写。

这样更方便你复盘，不会把猜测当事实。

  

建议 4：让它指出“数据缺口”

建议每次再补一句：

如果某些结论缺少足够证据，请明确指出数据缺口，不要强行下结论。

这很重要，能避免它瞎补。

# 01 产品可行性总分析

  


## 使用场景

当你拿到一个新产品，想先做一轮整体判断时用。  

适合第一次调研，帮助你快速形成全局认知。

  


## 提示词

```text

你现在是一名跨境电商与社媒选品分析师，请帮我评估这个产品是否适合未来1到2年做跨境社媒成交。

  

产品名称：{产品名}

  

请从以下维度输出详细分析，并尽量引用可验证的数据来源、页面、趋势、用户反馈或平台现象：

  

1. 市场需求趋势

- 过去12个月和过去3到5年的需求趋势

- 是否为长期需求还是短期热点

- 是否存在明显季节性

- 哪些国家或地区需求更强

  

2. 社交媒体成交适配度

- 这个产品是否适合通过 TikTok / Instagram / Facebook / YouTube Shorts 成交

- 是否具备“3秒看懂、15秒种草、容易转化”的特点

- 是否容易做开箱、测评、前后对比、教程、场景化视频

- 用户是否容易冲动下单

  

3. 竞争强度

- 目前主流平台上同类产品的竞争是否激烈

- 是否已经高度同质化

- 是否仍有内容差异化切入空间

- 是否有新卖家进入机会

  

4. 利润与履约可行性

- 该产品大致适合什么客单价区间

- 是否适合跨境小包、海外仓或本地仓

- 是否容易因运费、体积重、易碎、退货等问题影响利润

- 是否适合达人分销或投流成交

  

5. 风险与限制

- 是否可能涉及认证、知识产权、敏感功效宣称、平台限制或法律风险

- 是否容易产生售后纠纷或高退货率

- 是否存在侵权、仿品、商标或专利风险

  

6. 未来1到2年的持续性

- 该产品未来1到2年继续做的可能性如何

- 更像短期风口，还是可以长期经营

- 是否适合做品牌化或扩展系列产品

  

最后请输出：

A. 结论：适合 / 谨慎测试 / 不建议做

B. 核心理由

C. 最值得关注的3个数据指标

D. 建议优先测试的3个国家

E. 建议优先测试的3种内容方向

适合什么时候用

- 刚拿到一个产品
- 客户或供应链推荐了一个新品
- 你想先做第一轮筛选

你重点看什么

- 它最终给出的结论是否明确
- 有没有把“趋势、内容、竞争、风险”都覆盖到
- 它给出的国家和内容方向是否具体

  

02 判断未来 1–2 年还能不能做

使用场景

当你不是只想知道“现在能不能卖”，而是想判断这个产品未来 1–2 年值不值得持续投入时用。

提示词

请帮我判断这个产品未来1到2年是否还值得做，不要只看当前热度，要重点看持续性。

  

产品：{产品名}

  

请重点分析：

1. 过去12个月趋势

2. 过去3到5年趋势

3. 是否属于短期爆红型产品

4. 是否已经接近饱和

5. 是否有多个国家同时增长

6. 是否有复购潜力

7. 是否能延伸出系列产品

8. 是否适合长期品牌化经营

  

请最后给出明确结论：

- 长期可做

- 可以短中期测试

- 不建议作为未来1到2年的重点产品

  

并说明依据。

适合什么时候用

- 你已经发现它最近挺火
- 但你担心它只是风口
- 你不想做“昙花一现”的产品

你重点看什么

- 是不是多个国家都在增长
- 它是不是靠单一热点带起来的
- 有没有复购、系列化、品牌化可能

  

03 判断是否适合社媒成交

使用场景

重点判断这个产品是不是适合用内容带货，而不是只看搜索电商。

提示词

请分析这个产品是否适合通过社交媒体内容成交，重点不是传统电商，而是内容带货。

  

产品：{产品名}

  

请从以下角度分析：

1. 用户是否能快速理解产品价值

2. 是否适合短视频展示

3. 是否有明显的前后对比或使用效果

4. 是否有情绪价值、视觉吸引力或新奇性

5. 是否容易引发评论互动和分享

6. 是否适合达人种草、直播带货、UGC传播

7. 是否适合冲动型下单

8. 是否需要复杂教育成本

  

请最后输出：

- 社媒成交适合度评分（100分制）

- 最适合的平台排序

- 最适合的内容形式排序

- 为什么适合或不适合

适合什么时候用

- 你做的是 TikTok / Instagram / Facebook 内容成交
- 你想判断它适不适合达人或短视频
- 你担心产品太难解释

你重点看什么

- 能不能 3 秒看懂
- 能不能 15 秒展示价值
- 有没有天然的短视频表达力
- 是否适合冲动型购买

  

04 判断竞争与红海程度

使用场景

当你担心这个产品已经被做烂了，或者想看还有没有切入空间时用。

提示词

请评估这个产品当前是不是红海，以及是否还有切入机会。

  

产品：{产品名}

  

请分析：

1. 主流平台上同类内容和商家数量是否很多

2. 同类产品价格带是否已经打穿

3. 头部卖家是否垄断明显

4. 新卖家是否还有机会

5. 同类内容是否严重重复

6. 是否还能从人群、场景、功能、包装、内容角度做差异化

7. 是否存在“看起来很火但实际上利润很差”的情况

  

最后输出：

A. 红海程度：低 / 中 / 高

B. 切入难度：低 / 中 / 高

C. 可行的差异化方向

D. 是否建议进入

适合什么时候用

- 你看到很多人在卖
- 你怀疑这个产品很卷
- 你想看自己还能从哪里切进去

你重点看什么

- 是真的有市场，还是因为已经卷成红海看起来很热闹
- 内容是不是已经严重同质化
- 有没有“新卖家还能做的角度”

  

05 排查风险与限制

使用场景

做跨境外贸时，提前排查认证、侵权、平台限制、售后问题。

提示词

请帮我排查这个产品做跨境社媒和外贸销售的主要风险。

  

产品：{产品名}

目标市场：{国家或地区}

  

请重点排查：

1. 合规认证风险

2. 知识产权风险

3. 平台禁限售风险

4. 功效宣传风险

5. 物流运输限制

6. 售后与退货风险

7. 差评高发问题

8. 是否容易因为质量、尺寸、体验不一致而引发纠纷

  

最后输出：

- 高风险项

- 中风险项

- 低风险项

- 是否适合新手卖家做

- 是否适合长期放大

适合什么时候用

- 你准备进入一个新类目
- 产品可能涉及认证或敏感属性
- 你想避免刚做就踩坑

你重点看什么

- 有没有硬性合规门槛
- 有没有明显侵权风险
- 是否容易售后爆炸
- 是否适合新手测试

  

06 多产品横向对比

使用场景

你手里有多个产品，不知道先测哪个，用这条让 OpenClaw 帮你排优先级。

提示词

请把以下产品按照“未来1到2年适合做跨境社媒成交”的潜力进行横向对比和排序：

  

产品列表：

1. {产品A}

2. {产品B}

3. {产品C}

4. {产品D}

  

请统一按以下维度打分：

1. 市场趋势

2. 社媒内容适配

3. 竞争强度

4. 利润空间

5. 履约难度

6. 风险控制

7. 长期性

8. 品牌化潜力

  

请输出：

- 每个产品的详细评分

- 总分排名

- 最适合测试的产品

- 最不建议做的产品

- 每个产品的核心理由

适合什么时候用

- 你选了 3–10 个候选产品
- 你要做优先级排序
- 你不想凭感觉拍脑袋

你重点看什么

- 排名是否合理
- 它是否把“能卖”和“能长期做”区分开了
- 低分产品是不是卡在风险、利润或内容适配

  

附加建议

建议 1：不要一次问太大

不要这样问：

帮我把这个产品所有东西都查清楚

这种提示词通常会让输出很散。

更好的做法是：

- 先用 [[#01 产品可行性总分析]]
- 再针对重点问题继续追问
- 然后用 [[#05 排查风险与限制]] 补风险
- 最后再用 [[#06 多产品横向对比]] 做决策

  

建议 2：要求固定输出结构

你可以在每条提示词末尾都加这一段：

请按以下格式输出：

1. 结论

2. 核心依据

3. 关键数据

4. 风险点

5. 建议动作

这样便于你在 Obsidian 里统一整理。

  

建议 3：让它区分“事实”和“判断”

你还可以额外加一句：

请把“客观数据/事实”和“你的判断/推断”分开写。

这样更方便你复盘，不会把猜测当事实。

  

建议 4：让它指出“数据缺口”

建议每次再补一句：

如果某些结论缺少足够证据，请明确指出数据缺口，不要强行下结论。

这很重要，能避免它瞎补。
