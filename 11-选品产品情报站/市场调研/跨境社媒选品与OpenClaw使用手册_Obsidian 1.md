# 一、先看核心结论

你判断一个产品值不值得做，不是先看“有没有人在卖”，而是看下面这几个核心问题：

1. 这个产品未来 1–2 年还有没有持续需求？
2. 这个产品适不适合社媒内容成交？
3. 这个产品竞争是不是已经卷烂？
4. 这个产品利润能不能覆盖试错？
5. 这个产品有没有合规、侵权、物流、售后风险？
6. 这个产品是不是适合长期做，而不是短期风口？

### 最简单的五句话判断法

1. 这个产品未来一年搜索和内容热度会不会继续存在？
2. 这个产品能不能用 15 秒视频打动用户？
3. 这个产品有没有足够利润覆盖试错？
4. 这个产品有没有合规、侵权、退货大坑？
5. 这个产品是不是能从内容种草直接走到下单成交？

> 如果这 5 句里有 3 句都很勉强，就不要重仓。

## 1）需求趋势
重点看：
- Google Trends 搜索趋势
- TikTok / Instagram / Pinterest 相关热度
- Amazon / 电商平台需求迹象
- 季节性强不强
- 多国是否都有热度

你真正要判断的是：
- 是长期需求，还是短期热点
- 是缓慢增长，还是已经见顶
- 是单一区域热，还是多国扩散
- 是节日型产品，还是全年能卖

---

## 2）内容适配度
重点看这个产品是否适合：
- 3 秒看懂
- 15 秒展示价值
- 有明显前后对比
- 有情绪价值 / 视觉冲击 / 痛点解决
- 适合达人测评 / 开箱 / 教程 / 场景视频
- 容易让用户冲动下单

### 特别适合社媒成交的产品类型
- 美妆个护
- 家居收纳
- 厨房神器
- 宠物用品
- 健身小件
- 母婴小工具
- 穿搭配饰
- 看一眼就懂的功能型小商品

### 不太适合纯社媒冷启动成交的产品
- 解释成本太高
- 使用场景不直观
- 客单太高且决策长
- 售后复杂
- 合规门槛高
- 用户需要大量对比参数才下单

---

## 3）竞争强度
重点看：
- 平台内容量多不多
- 头部是否垄断
- 同质化是否严重
- 广告和达人竞争是否激烈
- 是否已经价格战

---

## 4）利润与履约
必须查：
- 工厂拿货价
- 平台抽佣
- 广告成本 / 达人佣金
- 国际物流成本
- 退货率风险
- 包装体积重
- 是否易碎 / 带电 / 液体 / 敏感品
- 售后复杂度

---

## 5）合规与平台风险
重点查：
- 是否涉及认证（CE / FCC / FDA / CPSIA / REACH 等）
- 是否有外观专利 / 商标 / 版权风险
- 是否容易被平台限流或禁售
- 是否涉及医疗、减肥、儿童、敏感功效等高风险赛道

---

## 6）社媒成交链路
你要看这个链路是否成立：

**短视频 / 达人种草 → 点击商品卡 / 落地页 → 快速理解 → 下单**

如果这个链路成立，才更适合做跨境社媒成交。

# 四、OpenClaw 提示词库

## 01 产品可行性总分析
```text
你现在是一名跨境电商与社媒选品分析师，请帮我评估这个产品是否适合未来1到2年做跨境社媒成交。

产品名称：{产品名}

请从以下维度输出详细分析，并尽量引用可验证的数据来源、页面、趋势、用户反馈或平台现象：

1. 市场需求趋势
- 过去12个月和过去3到5年的需求趋势
- 是否为长期需求还是短期热点
- 是否存在明显季节性
- 哪些国家或地区需求更强

2. 社交媒体成交适配度
- 这个产品是否适合通过 TikTok / Instagram / Facebook / YouTube Shorts 成交
- 是否具备“3秒看懂、15秒种草、容易转化”的特点
- 是否容易做开箱、测评、前后对比、教程、场景化视频
- 用户是否容易冲动下单

3. 竞争强度
- 目前主流平台上同类产品的竞争是否激烈
- 是否已经高度同质化
- 是否仍有内容差异化切入空间
- 是否有新卖家进入机会

4. 利润与履约可行性
- 该产品大致适合什么客单价区间
- 是否适合跨境小包、海外仓或本地仓
- 是否容易因运费、体积重、易碎、退货等问题影响利润
- 是否适合达人分销或投流成交

5. 风险与限制
- 是否可能涉及认证、知识产权、敏感功效宣称、平台限制或法律风险
- 是否容易产生售后纠纷或高退货率
- 是否存在侵权、仿品、商标或专利风险

6. 未来1到2年的持续性
- 该产品未来1到2年继续做的可能性如何
- 更像短期风口，还是可以长期经营
- 是否适合做品牌化或扩展系列产品

最后请输出：
A. 结论：适合 / 谨慎测试 / 不建议做
B. 核心理由
C. 最值得关注的3个数据指标
D. 建议优先测试的3个国家
E. 建议优先测试的3种内容方向
```

# 五、OpenClaw 正确使用方法

## 不要把 6 条提示词一次性全丢进去
正确方法是：

**一条提示词 = 解决一个问题。**

如果你把 6 条全塞进去，容易出现：
- 输出太长
- 重点不清楚
- 结论混在一起
- 你最想要的信息被淹没

---

## 正确使用顺序

### 情况 A：你只有 1 个产品
顺序建议：
1. 先用“01 产品可行性总分析”
2. 再用“03 判断是否适合社媒成交”
3. 再用“05 排查风险与限制”
4. 最后用“02 判断未来 1–2 年还能不能做”

### 情况 B：你有多个产品
顺序建议：
1. 先用“06 多产品横向对比”
2. 选出前 1–2 名
3. 再分别用“01 / 03 / 05 / 02”去深挖

---

## 你每次都要加的补充要求
建议在提示词最后补一句：

```text
请不要泛泛而谈，请尽量输出可验证的信息，并按以下格式回答：
1. 结论
2. 核心依据
3. 关键数据
4. 风险点
5. 建议动作
```

还可以再补一句：

```text
请把“客观数据/事实”和“你的判断/推断”分开写。
```

以及：

```text
如果某些结论缺少足够证据，请明确指出数据缺口，不要强行下结论。
```

# 一、先看核心结论

你判断一个产品值不值得做，不是先看“有没有人在卖”，而是看下面这几个核心问题：

1. 这个产品未来 1–2 年还有没有持续需求？
2. 这个产品适不适合社媒内容成交？
3. 这个产品竞争是不是已经卷烂？
4. 这个产品利润能不能覆盖试错？
5. 这个产品有没有合规、侵权、物流、售后风险？
6. 这个产品是不是适合长期做，而不是短期风口？

### 最简单的五句话判断法

1. 这个产品未来一年搜索和内容热度会不会继续存在？
2. 这个产品能不能用 15 秒视频打动用户？
3. 这个产品有没有足够利润覆盖试错？
4. 这个产品有没有合规、侵权、退货大坑？
5. 这个产品是不是能从内容种草直接走到下单成交？

> 如果这 5 句里有 3 句都很勉强，就不要重仓。

# 二、判断一个产品值不值得做，要看哪 6 类数据


## 1）需求趋势
重点看：
- Google Trends 搜索趋势
- TikTok / Instagram / Pinterest 相关热度
- Amazon / 电商平台需求迹象
- 季节性强不强
- 多国是否都有热度

你真正要判断的是：
- 是长期需求，还是短期热点
- 是缓慢增长，还是已经见顶
- 是单一区域热，还是多国扩散
- 是节日型产品，还是全年能卖

---


## 2）内容适配度
重点看这个产品是否适合：
- 3 秒看懂
- 15 秒展示价值
- 有明显前后对比
- 有情绪价值 / 视觉冲击 / 痛点解决
- 适合达人测评 / 开箱 / 教程 / 场景视频
- 容易让用户冲动下单

### 特别适合社媒成交的产品类型
- 美妆个护
- 家居收纳
- 厨房神器
- 宠物用品
- 健身小件
- 母婴小工具
- 穿搭配饰
- 看一眼就懂的功能型小商品

### 不太适合纯社媒冷启动成交的产品
- 解释成本太高
- 使用场景不直观
- 客单太高且决策长
- 售后复杂
- 合规门槛高
- 用户需要大量对比参数才下单

---


## 3）竞争强度
重点看：
- 平台内容量多不多
- 头部是否垄断
- 同质化是否严重
- 广告和达人竞争是否激烈
- 是否已经价格战

---


## 4）利润与履约
必须查：
- 工厂拿货价
- 平台抽佣
- 广告成本 / 达人佣金
- 国际物流成本
- 退货率风险
- 包装体积重
- 是否易碎 / 带电 / 液体 / 敏感品
- 售后复杂度

---


## 5）合规与平台风险
重点查：
- 是否涉及认证（CE / FCC / FDA / CPSIA / REACH 等）
- 是否有外观专利 / 商标 / 版权风险
- 是否容易被平台限流或禁售
- 是否涉及医疗、减肥、儿童、敏感功效等高风险赛道

---


## 6）社媒成交链路
你要看这个链路是否成立：

**短视频 / 达人种草 → 点击商品卡 / 落地页 → 快速理解 → 下单**

如果这个链路成立，才更适合做跨境社媒成交。

# 五、OpenClaw 正确使用方法


## 不要把 6 条提示词一次性全丢进去
正确方法是：

**一条提示词 = 解决一个问题。**

如果你把 6 条全塞进去，容易出现：
- 输出太长
- 重点不清楚
- 结论混在一起
- 你最想要的信息被淹没

---


## 正确使用顺序

### 情况 A：你只有 1 个产品
顺序建议：
1. 先用“01 产品可行性总分析”
2. 再用“03 判断是否适合社媒成交”
3. 再用“05 排查风险与限制”
4. 最后用“02 判断未来 1–2 年还能不能做”

### 情况 B：你有多个产品
顺序建议：
1. 先用“06 多产品横向对比”
2. 选出前 1–2 名
3. 再分别用“01 / 03 / 05 / 02”去深挖

---


## 你每次都要加的补充要求
建议在提示词最后补一句：

```text
请不要泛泛而谈，请尽量输出可验证的信息，并按以下格式回答：

```

以及：

```text
