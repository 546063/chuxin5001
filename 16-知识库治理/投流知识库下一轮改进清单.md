## 依赖关系图

```
P0 真实数据 ──→ P1 质量升级（数据驱动优化）
P1 Facebook ──→ P2 跨平台对比
P1 LinkedIn ──→ P2 跨平台对比
P1 补强 ──→ P2 多语言（素材基础）
```

## 执行建议

1. **首次投流后立即执行 P0**：真实数据是最有价值的知识库资产
2. **P1 与 P0 并行**：不依赖真实数据的 P1 项目可以提前做
3. **P2 在 P1 完成后**：基础扎实后再做跨平台对比
4. **P3 按需执行**：有需求再建，不预设
