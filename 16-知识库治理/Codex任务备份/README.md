# Codex任务备份

这个目录用于保存 Codex 的任务记录、对话记录、执行摘要和备份清单。

注意：
- 不保存 API key。
- 不保存 token。
- 不保存 credentials。
- 不保存 cache。
- 不保存 node_modules。
- 不保存大体积二进制文件。
