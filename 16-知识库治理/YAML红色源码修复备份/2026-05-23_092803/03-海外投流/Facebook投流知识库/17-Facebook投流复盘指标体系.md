# Facebook投流复盘指标体系

> 本文档定义 B2B 食品机械出口 Facebook 广告的完整指标体系，包括指标定义、计算公式、行业基准阈值、诊断决策树和复盘模板。
>
> **适用场景**：食品机械出口企业（整单 >= 20,000 RMB），通过 Facebook/Instagram 获取海外 B2B 询盘（工厂老板/采购经理/经销商/中央厨房）。

---

## 一、核心指标定义与计算公式

### 1.1 漏斗层级概览

```
曝光(Impressions) → 点击(Clicks) → 线索(Leads) → 有效询盘(Qualified)
  → 报价(Quoted) → 样品(Sample) → 订单(Order) → 复购(Repeat)
```

### 1.2 指标定义表

| 指标 | 英文全称 | 计算公式 | 含义 |
|------|---------|---------|------|
| CPM | Cost Per Mille | `广告花费 / 曝光量 * 1000` | 每千次曝光的成本 |
| CTR | Click-Through Rate | `链接点击量 / 曝光量 * 100%` | 广告吸引用户点击的能力 |
| CPC | Cost Per Click | `广告花费 / 链接点击量` | 每次点击的成本 |
| CPL | Cost Per Lead | `广告花费 / 线索数` | 获取一条线索的成本 |
| CPA | Cost Per Action | `广告花费 / 转化行动数` | 每次转化行动的成本 |
| CVR | Conversion Rate | `转化行动数 / 链接点击量 * 100%` | 点击后转化为目标行动的比例 |
| ROAS | Return on Ad Spend | `广告带来的收入 / 广告花费` | 每花1元广告费带来的收入 |

### 1.3 辅助指标

| 指标 | 计算公式 | 含义 |
|------|---------|------|
| 频率 (Frequency) | `曝光量 / 触达人数` | 平均每人看到广告的次数 |
| 线索成本趋势 | `本周CPL / 上周CPL - 1` | CPL的周环比变化 |
| 线索量 | Facebook 表单提交数 + Messenger对话数 + WhatsApp点击数 | 总线索数量 |
| 触达 (Reach) | Facebook 去重后的看到广告的人数 | 广告覆盖的独立用户数 |
| 互动率 | `(点赞+评论+分享+点击) / 曝光量 * 100%` | 广告内容的整体吸引力 |

## 二、指标基准阈值（B2B 食品机械行业）

> 以下阈值基于 B2B 工业品/机械行业公开基准范围 + 跨境B2B投流经验，标注为「参考范围」的需在实际投放中验证校准。

### 2.1 CPM 基准

| 状态 | 阈值范围（USD） | 说明 |
|------|---------------|------|
| 正常 | $3 - $12 | 东南亚/南亚市场偏低（$3-6），欧美市场偏高（$8-12） |
| 预警 | $12 - $20 | CPM偏高，检查受众是否过窄、竞争是否加剧 |
| 危险 | > $20 | CPM过高，需检查：受众重叠、竞价策略、广告相关性 |

**区域差异参考**：
- 东南亚（越南/泰国/印尼/菲律宾）：$3 - $7
- 南亚（印度/巴基斯坦/孟加拉）：$2 - $5
- 中东（沙特/阿联酋/土耳其）：$6 - $12
- 欧洲（德国/法国/意大利）：$8 - $15
- 北美（美国/加拿大）：$10 - $18

### 2.2 CTR 基准（链接点击率）

| 状态 | 阈值范围 | 说明 |
|------|---------|------|
| 优秀 | > 2.0% | 素材+受众匹配极好，可考虑放量 |
| 正常 | 1.0% - 2.0% | B2B机械行业合理范围 |
| 预警 | 0.5% - 1.0% | 素材吸引力不足或受众偏差，需优化 |
| 危险 | < 0.5% | 严重不匹配，必须更换素材或调整受众 |

**注意**：B2B 机械广告的 CTR 天然低于消费品，不要用电商标准（>3%）来衡量。

### 2.3 CPC 基准

| 状态 | 阈值范围（USD） | 说明 |
|------|---------------|------|
| 正常 | $0.30 - $1.50 | 东南亚/南亚偏低，欧美偏高 |
| 预警 | $1.50 - $3.00 | 点击成本偏高，检查 CTR 和 CPM |
| 危险 | > $3.00 |  CPC 过高，需立即优化素材或调整竞价 |

### 2.4 CPL 基准

| 状态 | 阈值范围（USD） | 说明 |
|------|---------------|------|
| 优秀 | < $15 | 线索成本极低，大力放量 |
| 正常 | $15 - $50 | B2B机械行业合理范围（取决于目标市场） |
| 预警 | $50 - $100 | 线索成本偏高，检查转化路径和受众质量 |
| 危险 | > $100 | 线索成本过高，需暂停排查 |

**区域差异参考**：
- 东南亚/南亚：$5 - $25
- 中东：$15 - $40
- 欧洲：$30 - $80
- 北美：$40 - $120

### 2.5 CVR 基准（线索转化率）

| 状态 | 阈值范围 | 说明 |
|------|---------|------|
| 优秀 | > 15% | 落地页/表单体验优秀 |
| 正常 | 8% - 15% | B2B机械行业合理范围 |
| 预警 | 5% - 8% | 转化路径有摩擦，检查表单长度和加载速度 |
| 危险 | < 5% | 转化率极低，必须优化落地页或表单 |

### 2.6 ROAS 基准

| 状态 | 阈值 | 说明 |
|------|-----|------|
| 优秀 | > 10x | 极高回报（食品机械单笔订单金额大） |
| 正常 | 5x - 10x | 合理回报范围 |
| 预警 | 2x - 5x | 回报偏低，需优化线索质量和成交率 |
| 危险 | < 2x | 回报不足，需重新评估投放策略 |

> **注意**：ROAS 计算需要考虑 **成交周期长** 的特点。食品机械从线索到成交通常需要 30-90 天，因此短期 ROAS 可能偏低。建议按 **90 天滚动窗口** 计算 ROAS。

### 2.7 CPA 基准

| 状态 | 阈值范围（USD） | 说明 |
|------|---------------|------|
| 正常 | $50 - $200 | 取决于"行动"的定义（表单提交/对话发起/报价请求） |
| 预警 | $200 - $500 | 行动成本偏高 |
| 危险 | > $500 | 行动成本过高，需暂停排查 |

## 三、询盘有效率计算

### 3.1 定义

```
有效询盘 = 满足以下至少2个条件的线索：
  1. 提供了真实有效的联系方式（邮箱/电话/WhatsApp）
  2. 明确表达了采购意向（询问价格/规格/MOQ/交货期）
  3. 来自目标行业（食品加工/中央厨房/餐饮连锁/经销商）
  4. 来自目标区域（投放设定的目标国家）
  5. 在 48 小时内回复了跟进消息
```

### 3.2 计算公式

```
询盘有效率 = 有效询盘数 / 总线索数 * 100%

单条有效线索成本 = 广告花费 / 有效询盘数
                 = CPL / 询盘有效率
```

### 3.3 基准范围

| 状态 | 询盘有效率 | 说明 |
|------|----------|------|
| 优秀 | > 60% | 线索质量极高 |
| 正常 | 30% - 60% | B2B机械行业合理范围 |
| 预警 | 15% - 30% | 线索质量偏低，检查受众定向和广告内容 |
| 危险 | < 15% | 大量无效线索，需优化定向或增加表单筛选问题 |

## 四、WhatsApp 回复率基准

### 4.1 定义

```
WhatsApp 回复率 = 主动回复的用户数 / WhatsApp点击量 * 100%
```

### 4.2 基准范围

| 状态 | 回复率 | 说明 |
|------|-------|------|
| 优秀 | > 40% | 点击后主动对话意愿强 |
| 正常 | 20% - 40% | B2B机械行业合理范围 |
| 预警 | 10% - 20% | 回复率偏低，检查预设消息和开场白 |
| 危险 | < 10% | 回复率极低，点击者无意向对话 |

### 4.3 关键影响因素

- **预设消息质量**：是否有明确的采购场景引导（如"Hi, I'm interested in your [machine type]. Can you send me the price?"）
- **响应速度**：客户发消息后 5 分钟内回复 vs 2 小时后回复，成交率差异可达 3x
- **工作时间覆盖**：是否覆盖目标时区的黄金工作时间
- **自动回复设置**：是否有合理的自动回复引导下一步动作

## 五、Messenger 回复率基准

### 5.1 定义

```
Messenger 回复率 = 完成对话的用户数 / Messenger点击量 * 100%
```

### 5.2 基准范围

| 状态 | 回复率 | 说明 |
|------|-------|------|
| 优秀 | > 50% | 对话完成率高 |
| 正常 | 25% - 50% | B2B机械行业合理范围 |
| 预警 | 12% - 25% | 对话流失率高 |
| 危险 | < 12% | 点击者基本不回复 |

### 5.3 Messenger vs WhatsApp 对比

| 维度 | Messenger | WhatsApp |
|------|-----------|----------|
| 回复率 | 通常更高（25-50%） | 通常较低（20-40%） |
| 线索质量 | 中等（门槛低，误触多） | 较高（需跳转，意向更强） |
| 适合市场 | 东南亚/拉美 | 中东/欧洲/非洲 |
| 自动化工具 | 丰富（ManyChat等） | 有限（WhatsApp Business API） |
| 后续跟进 | 在平台内完成 | 可转入 WhatsApp 深度沟通 |

## 六、报价率计算

### 6.1 定义

```
报价率 = 发送正式报价单的客户数 / 有效询盘数 * 100%
```

### 6.2 基准范围

| 状态 | 报价率 | 说明 |
|------|-------|------|
| 优秀 | > 70% | 大部分有效询盘都进入了报价阶段 |
| 正常 | 40% - 70% | B2B机械行业合理范围 |
| 预警 | 25% - 40% | 报价转化偏低，检查询盘跟进流程 |
| 危险 | < 25% | 大量询盘未报价，需排查跟进SOP |

### 6.3 报价到订单转化率

```
报价→订单转化率 = 成交订单数 / 报价客户数 * 100%
```

| 状态 | 转化率 | 说明 |
|------|-------|------|
| 优秀 | > 20% | 报价策略和价格竞争力强 |
| 正常 | 10% - 20% | B2B机械行业合理范围 |
| 预警 | 5% - 10% | 报价转化率偏低 |
| 危险 | < 5% | 报价后大量流失，需检查价格和竞品对比 |

## 七、样品 / 订单转化率计算

### 7.1 样品转化率

```
询盘→样品转化率 = 寄出样品/试机的客户数 / 有效询盘数 * 100%
样品→订单转化率 = 样品后成交的客户数 / 寄出样品的客户数 * 100%
```

### 7.2 基准范围

| 指标 | 优秀 | 正常 | 预警 | 危险 |
|------|------|------|------|------|
| 询盘→样品 | > 15% | 8% - 15% | 4% - 8% | < 4% |
| 样品→订单 | > 50% | 30% - 50% | 15% - 30% | < 15% |

### 7.3 全链路转化基准

```
完整转化漏斗（参考范围）：
  1000 曝光
    → 15 点击（CTR 1.5%）
      → 1.5 线索（CVR 10%）
        → 0.75 有效询盘（有效率 50%）
          → 0.4 报价（报价率 53%）
            → 0.06 订单（报价→订单 15%）
```

> 即：约 16,000 次曝光产生 1 笔订单。实际数据因产品单价、市场竞争、销售能力而异。

## 八、广告组停启标准

### 8.1 立即暂停（Red Flag）

满足以下 **任一** 条件时，立即暂停广告组：

| 条件 | 触发标准 | 建议动作 |
|------|---------|---------|
| CPL 过高 | 连续 3 天 CPL > 危险阈值（> $100） | 暂停，检查受众+素材+落地页 |
| CTR 极低 | 连续 3 天 CTR < 0.5% | 暂停，更换素材或调整受众 |
| 零线索 | 花费 >= 3 * 目标CPL 且线索数 = 0 | 暂停，检查追踪和落地页 |
| 频率过高 | 频率 > 4.0 且 CTR 下降 > 30% | 暂停或更换素材 |
| 花费异常 | 单日花费 > 日均预算的 150% | 检查竞价策略，设日预算上限 |
| 相关性差 | 广告相关性诊断低于 4/10 | 暂停优化素材 |

### 8.2 观察期（Yellow Flag）

满足以下条件时，进入观察期（继续投放但密切关注）：

| 条件 | 触发标准 | 建议动作 |
|------|---------|---------|
| CPL 偏高 | CPL 在预警区间（$50-$100） | 观察 3-5 天，优化素材 |
| CTR 偏低 | CTR 在预警区间（0.5%-1.0%） | 准备新素材替换 |
| CVR 偏低 | CVR 在预警区间（5%-8%） | 检查落地页和表单 |
| 频率偏高 | 频率 2.5 - 4.0 | 扩大受众或更换素材 |

### 8.3 保持投放（Green Flag）

| 条件 | 标准 | 建议动作 |
|------|------|---------|
| 指标正常 | 所有核心指标在正常范围内 | 保持投放，记录数据 |
| 指标优秀 | 核心指标进入优秀区间 | 准备放量 |
| 学习期完成 | 7 天内累计 >= 50 个转化事件 | 减少编辑，保持稳定 |

## 九、素材衰减判断

### 9.1 衰减信号清单

| 信号 | 判断标准 | 严重程度 |
|------|---------|---------|
| CTR 下降 | 较素材上线首周 CTR 下降 > 30% | 高 |
| CTR 下降 | 较素材上线首周 CTR 下降 > 15% | 中 |
| CPL 上升 | 较素材上线首周 CPL 上升 > 50% | 高 |
| CPL 上升 | 较素材上线首周 CPL 上升 > 25% | 中 |
| 频率过高 | 同一受众平均看到 > 3 次 | 中 |
| 互动率下降 | 点赞/评论/分享率下降 > 40% | 中 |
| 线索质量下降 | 询盘有效率下降 > 20 个百分点 | 高 |
| 广告疲劳指数 | `1 / CTR` 持续上升且超过行业均值 2x | 高 |

### 9.2 衰减判断公式

```
素材健康指数 = (当前CTR / 首周CTR) * 0.4
             + (首周CPL / 当前CPL) * 0.3
             + (1 / 当前频率) * 0.3

健康指数 > 0.7：素材健康，继续投放
健康指数 0.5 - 0.7：素材开始衰减，准备替换
健康指数 < 0.5：素材疲劳，立即更换
```

### 9.3 素材生命周期管理

| 阶段 | 特征 | 动作 |
|------|------|------|
| 冷启动期（0-7天） | 数据波动大，CTR/CPL不稳定 | 不轻易关停，给足学习期 |
| 稳定期（7-21天） | 指标稳定在正常或优秀区间 | 保持投放，可适当加预算 |
| 衰减期（21-35天） | CTR开始下降，CPL开始上升 | 准备新素材，开始 A/B 测试 |
| 疲劳期（35天+） | 指标跌破预警线 | 替换素材，保留受众数据 |

## 十、放量标准（Scale-Up Criteria）

### 10.1 加预算条件

满足以下 **全部** 条件时，可以考虑加预算：

| 条件 | 标准 |
|------|------|
| 学习期完成 | 广告组已积累 >= 50 个转化事件 |
| CPL 稳定 | 连续 7 天 CPL 在正常或优秀区间 |
| CTR 稳定 | 连续 7 天 CTR >= 1.0% |
| 线索数量 | 日均线索数 >= 3 条 |
| 询盘有效率 | >= 30% |
| ROAS | >= 5x（按 90 天滚动窗口） |
| 素材健康 | 素材健康指数 >= 0.7 |

### 10.2 放量方式

| 方式 | 操作 | 幅度 | 适用场景 |
|------|------|------|---------|
| 垂直放量 | 在原广告组增加日预算 | 每次 +20% | 单个广告组表现优秀 |
| 水平放量 | 复制优秀广告组，扩大受众 | 新组预算 = 原组 50-100% | 受众还有扩展空间 |
| 多素材放量 | 在同一广告组增加新素材 | 新素材预算 = 原素材 30% | 素材表现好但需要多样性 |
| 多市场放量 | 将优秀广告复制到其他目标国家 | 按国家 GDP 和 CPC 调整预算 | 已验证模型可复制 |

### 10.3 放量节奏

```
第 1 次放量：预算 +20%，观察 3 天
第 2 次放量：预算再 +20%（累计 +44%），观察 3 天
第 3 次放量：预算再 +20%（累计 +73%），观察 5 天
...
每次放量后 CPL 上升 > 30%，停止放量，回退到上一个稳定水平
```

## 十一、降本标准（Cost-Reduction Criteria）

### 11.1 降本条件

满足以下 **任一** 条件时，可以考虑执行降本操作：

| 条件 | 标准 | 建议动作 |
|------|------|---------|
| CPL 连续超标 | 连续 5 天 CPL > 正常区间上限 | 优化素材+受众 |
| CPM 异常高 | CPM 连续 3 天 > 预警值 | 扩大受众、切换竞价策略 |
| CTR 持续下降 | CTR 连续 5 天下降 | 更换素材、优化文案 |
| CVR 持续下降 | CVR 连续 5 天下降 | 优化落地页、精简表单 |
| 频率过高 | 频率 > 3.0 | 扩大受众范围 |

### 11.2 降本方法

| 方法 | 操作 | 预期效果 |
|------|------|---------|
| 优化素材 | 更换前3秒hook、调整视觉风格 | CTR 提升 20-50%，CPC 下降 |
| 精简表单 | 减少表单字段至 3-5 个 | CVR 提升 15-30%，CPL 下降 |
| 扩大受众 | 放宽兴趣标签、使用 Lookalike | CPM 下降 10-30% |
| 切换竞价 | 从最低成本切换到成本上限 | 控制 CPL 在目标范围内 |
| 排除无效受众 | 排除已转化用户、排除低质量地区 | 有效率提升，有效CPL下降 |
| 优化投放时段 | 按目标时区黄金时段投放 | 减少无效曝光，降低CPM |
| A/B 测试 | 持续测试新素材/文案/受众 | 找到更优组合 |

### 11.3 降本节奏

```
第 1 步：诊断问题根源（CPM高？CTR低？CVR低？）
第 2 步：执行 1 项优化，观察 3-5 天
第 3 步：评估效果，有效则继续，无效则换方法
第 4 步：每次只改 1 个变量，避免归因混乱
第 5 步：连续 2 周无改善，考虑暂停广告组重来
```

## 十四、数据判断决策树（CPM → CTR → CVR 三步诊断）

### 14.1 诊断流程图

```
问题：广告效果不理想

第一步：检查 CPM
├── CPM 正常（< $12） → 进入第二步
├── CPM 偏高（$12-$20） → 预警
│   ├── 受众是否过窄？ → 扩大受众范围或使用 Lookalike
│   ├── 竞争是否加剧？ → 考虑切换投放时段
│   └── 竞价策略是否合理？ → 尝试成本上限竞价
└── CPM 过高（> $20） → 危险
    ├── 立即检查受众重叠度
    ├── 检查广告相关性评分
    ├── 考虑更换目标市场
    └── 如无法改善，暂停广告组

第二步：检查 CTR
├── CTR 正常（> 1.0%） → 进入第三步
├── CTR 偏低（0.5%-1.0%） → 预警
│   ├── 素材前3秒是否有吸引力？ → 更换hook
│   ├── 文案是否击中痛点？ → 重写文案
│   ├── CTA是否明确？ → 优化行动号召
│   └── 素材与受众是否匹配？ → 调整受众定向
└── CTR 过低（< 0.5%） → 危险
    ├── 立即更换素材（至少测试3个新版本）
    ├── 检查广告是否与落地页内容一致
    ├── 检查是否有竞品在同一时段大量投放
    └── 如连续3天无改善，暂停广告组

第三步：检查 CVR
├── CVR 正常（> 8%） → 广告整体健康，继续投放
├── CVR 偏低（5%-8%） → 预警
│   ├── 表单是否太长？ → 精简至3-5个字段
│   ├── 落地页加载速度是否慢？ → 优化加载速度
│   ├── 是否有信任信号？ → 添加客户案例/认证
│   └── CTA是否明确？ → 优化表单提交按钮
└── CVR 过低（< 5%） → 危险
    ├── 检查表单是否有技术问题
    ├── 检查落地页移动端适配
    ├── 检查广告承诺与落地页是否一致
    ├── 添加社会证明（客户数量/案例/认证）
    └── 如连续3天无改善，暂停广告组重做落地页
```

### 14.2 常见问题速查表

| 问题现象 | 最可能原因 | 首选解决方案 |
|---------|-----------|-------------|
| 曝光量大但点击少 | CTR 低 | 更换素材，优化前3秒hook |
| 点击多但线索少 | CVR 低 | 精简表单，优化落地页 |
| 线索多但有效率低 | 受众不准 | 增加表单筛选问题，收紧定向 |
| CPL 持续上升 | 素材衰减/竞争加剧 | 更换素材，检查竞品动态 |
| 频率高但效果差 | 受众过窄 | 扩大受众，使用 Lookalike |
| 学习期反复失败 | 预算太低/受众太窄 | 加预算或扩大受众 |
| 花费不出去 | 竞价太低/受众太小 | 提高竞价或扩大受众 |
| CPM 突然飙升 | 市场竞争变化/节假日 | 检查行业日历，调整策略 |

### 14.3 诊断优先级

```
诊断顺序（不可跳过）：
  CPM → 决定曝光成本是否合理
    → CTR → 决定素材是否吸引人
      → CVR → 决定转化路径是否顺畅
        → 询盘有效率 → 决定线索质量是否达标
          → 报价率 → 决定销售跟进是否到位
            → 成交率 → 决定整体ROI是否健康
```

每一步正常才进入下一步。如果在某一步发现问题，先解决该步骤的问题，不要跳过。

## 十五、附录

### 15.1 指标计算速查卡

```
CPM = 花费 / 曝光 * 1000
CTR = 链接点击 / 曝光 * 100%
CPC = 花费 / 链接点击
CPL = 花费 / 线索数
CVR = 转化行动 / 链接点击 * 100%
ROAS = 广告收入 / 广告花费
频率 = 曝光 / 触达
询盘有效率 = 有效询盘 / 总线索 * 100%
报价率 = 报价客户 / 有效询盘 * 100%
报价→订单转化率 = 订单 / 报价客户 * 100%
素材健康指数 = (当前CTR/首周CTR)*0.4 + (首周CPL/当前CPL)*0.3 + (1/频率)*0.3
```

### 15.2 投放检查清单（每日晨检）

```
每日晨检（5分钟）：
□ 昨日花费是否在预算范围内？
□ CPM 是否在正常区间？
□ CTR 是否 >= 1.0%？
□ CPL 是否 <= 目标值？
□ 是否有广告组触发暂停条件？
□ 是否有广告组满足放量条件？
□ 素材频率是否 > 3.0？
□ 是否有新线索需要跟进？
```

### 15.3 关键时间节点

| 时间节点 | 检查内容 |
|---------|---------|
| 广告上线后 24 小时 | 检查是否开始消耗、CPM是否正常 |
| 广告上线后 72 小时 | 检查CTR、CPC初步数据 |
| 广告上线后 7 天 | 检查CPL、CVR、学习期进度 |
| 每周一次 | 周度复盘，调整预算和素材 |
| 每月一次 | 月度复盘，策略调整和下月规划 |
| 每季度一次 | 整体策略回顾，受众和素材大换血 |

### 15.4 数据来源

- Facebook Ads Manager：曝光、点击、花费、CTR、CPC、CPM、频率、触达
- Facebook Lead Center / CRM：线索数、线索质量、跟进状态
- WhatsApp Business / CRM：WhatsApp点击、回复数、对话质量
- Messenger 后台：Messenger点击、对话数、回复率
- 销售 CRM / 表格：有效询盘、报价、样品、订单、成交金额

### 15.5 注意事项

1. **不编造真实数据**：本文档中的阈值均为行业参考范围或「待实测」基准，实际投放中需根据自身数据校准
2. **B2B 与 B2C 差异**：食品机械是典型的 B2B 高客单价产品，决策周期长（30-90天），不能用电商或消费品的指标标准来衡量
3. **区域差异**：不同目标市场的 CPM/CPC/CPL 差异巨大，应分区域设定不同的基准
4. **学习期**：Facebook 广告学习期需要约 50 个转化事件（7 天内），学习期内数据波动属正常现象
5. **ROAS 计算周期**：建议使用 90 天滚动窗口计算 ROAS，而非按天或按周
6. **数据延迟**：Facebook 广告数据可能存在 24-48 小时的延迟，当日数据不建议作为决策依据
7. **归因窗口**：建议设置 7-day click / 1-day view 归因窗口，与行业惯例一致
