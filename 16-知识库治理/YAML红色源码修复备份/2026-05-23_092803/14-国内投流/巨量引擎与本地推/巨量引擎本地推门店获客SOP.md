# 巨量引擎本地推门店获客SOP

> 适用范围：有实体门店的本地商家
> 最后更新：2026-05-22

---

## 一、门店半径设置

### 不同行业的推荐半径

| 行业 | 核心半径 | 扩展半径 | 说明 |
|---|---|---|---|
| 餐饮 | 1-3km | 3-5km | 餐饮决策半径短 |
| 美容美发 | 3-5km | 5-8km | 愿意为好的发型师多走 |
| 教培 | 3-5km | 5-10km | 家长接送范围有限 |
| 口腔门诊 | 5-8km | 8-15km | 医疗需求愿意走远 |
| 汽修保养 | 3-5km | 5-10km | 就近原则 |
| 宠物医院 | 3-5km | 5-8km | 就近+紧急需求 |

### 半径设置策略
```
测试期：门店为中心 3km（验证核心辐射圈效果）
成长期：扩展到 5km（扩量覆盖更多人群）
成熟期：根据数据调整（核销率高的方向扩展）
```

### 核销率 vs 半径关系
- 1-3km：核销率通常 40-60%（最高）
- 3-5km：核销率通常 25-40%
- 5-10km：核销率通常 10-25%（最低）
- 超过 10km：核销率 < 10%，建议关闭

## 二、区域投放策略

### 商圈分层
| 层级 | 特征 | 预算占比 |
|---|---|---|
| A类：核心商圈 | 步行可达/同小区 | 50% |
| B类：次级商圈 | 车程10分钟内 | 30% |
| C类：边缘商圈 | 车程10-20分钟 | 20% |

### 分时段投放
| 时段 | 预算比例 | 适用行业 |
|---|---|---|
| 7:00-9:00 早高峰 | 15% | 早餐/咖啡/通勤相关 |
| 11:00-14:00 午间 | 30% | 餐饮/美容/美甲 |
| 14:00-17:00 下午 | 20% | 教培/健身/医美 |
| 17:00-21:00 晚高峰 | 35% | 全行业高峰 |
| 21:00-24:00 夜间 | 0-10% | 夜宵/娱乐/应急 |

## 三、团购转化

### 团购券设计公式
```
引流款：低价（原价的30-50%）+ 高感知价值 + 低门槛
利润款：中价（原价的60-80%）+ 组合套餐 + 中门槛
形象款：高价（原价的80-90%）+ 高端服务 + 高门槛
```

### 团购券命名公式
```
[核心卖点]+[价格]+[限时/限量]
例："新客精剪体验39元-限前100名"
例："三室一厅除醛套餐1980-不达标准全额退"
```

### 团购券关键参数
| 参数 | 建议值 | 说明 |
|---|---|---|
| 有效期 | 15-30天 | 太短核销率低，太长客户遗忘 |
| 库存 | 按月投放量 × 1.5 | 避免断货影响广告效果 |
| 使用规则 | 明确不可用日期 | 节假日除外/周末加价 |
| 预约要求 | 需提前X小时预约 | 避免到店无接待能力 |

## 四、到店核销

### 核销SOP
```
客户到店
  → 验证团购券（扫码/验证码）
  → 确认服务项目
  → 提供服务
  → 引导好评/复购
  → 标记核销完成
```

### 提升核销率的6个动作
1. **购买后立即短信** — "您的团购券已生效，有效期至X月X日"
2. **到期前3天提醒** — "您的团购券还剩3天，请尽早预约使用"
3. **到期前1天电话** — "您好，您购买的XX团购券明天到期，今天过来吗？"
4. **预约确认短信** — "您已预约X月X日X点，地址：XX，电话：XX"
5. **到店路线指引** — 发送地图链接+停车指引
6. **爽约提醒** — "您未按时到店，团购券已过期，如需重新购买请联系"

### 核销率基准
| 行业 | 优秀 | 合格 | 预警 |
|---|---|---|---|
| 餐饮 | ≥ 50% | 30-50% | < 30% |
| 美容美发 | ≥ 45% | 25-45% | < 25% |
| 教培体验 | ≥ 40% | 20-40% | < 20% |
| 除虫/除醛 | ≥ 50% | 30-50% | < 30% |
| 口腔/医美 | ≥ 35% | 15-35% | < 15% |

## 五、短视频挂载

### 挂载方式
| 挂载类型 | 适用场景 | 操作 |
|---|---|---|
| POI门店 | 品牌曝光+到店 | 视频左下角显示门店 |
| 团购链接 | 直接成交 | 视频左下角显示团购券 |
| 私信组件 | 咨询留资 | 视频引导私信 |
| 电话组件 | 即时沟通 | 视频显示电话号码 |

### 挂载素材要求
- 视频前 3 秒必须出现门店/产品画面
- 第 5-10 秒出现核心卖点/价格
- 结尾必须引导点击（"点左下角"）
- 团购券价格与视频展示一致
