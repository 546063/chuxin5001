# 巨量引擎投放复盘指标体系

> 适用范围：巨量引擎/巨量本地推/千川
> 最后更新：2026-05-22

---

## 一、核心指标定义

### 漏斗指标
```
展现量（Impressions）
  → 点击量（Clicks） → CTR = 点击量/展现量
    → 表单提交/私信/购买 → CVR = 转化量/点击量
      → 有效线索 → 有效率 = 有效量/转化量
        → 到店/沟通 → 到店率 = 到店量/有效量
          → 成交 → 成交率 = 成交量/到店量
```

### 成本指标
| 指标 | 公式 | 说明 |
|---|---|---|
| CPC | 消耗/点击量 | 单次点击成本 |
| CPA | 消耗/转化量 | 单次转化成本 |
| CPL | 消耗/线索量 | 单条线索成本 |
| CPS | 消耗/成交量 | 单次成交成本 |
| ROI | 收入/消耗 | 投入产出比 |

## 二、日度复盘指标

### 每日必看7个指标
| 指标 | 怎么看 | 异常标准 |
|---|---|---|
| 消耗 | 是否在预算范围内 | 低于预算50%或超预算20% |
| CTR | 素材吸引力 | 连续2天 < 1% |
| CVR | 承接效率 | 连续2天 < 3% |
| CPA | 获客成本 | 超过目标价150% |
| 线索量 | 获客数量 | 低于日均50% |
| 有效率 | 线索质量 | 低于30% |
| ROI | 投入产出 | 连续3天 < 1:2 |

## 四、何时停计划

### 立即关停信号
| 信号 | 条件 | 原因 |
|---|---|---|
| 成本爆表 | CPA > 目标价200%，持续1天 | 算法跑偏 |
| 零消耗 | 24小时消耗 < 预算10% | 出价过低或素材太差 |
| 零转化 | 消耗>200元但0转化 | 定向/素材/承接有问题 |
| 异常波动 | 消耗量突然翻倍但线索不增 | 可能刷量/作弊 |

### 观察后关停信号
| 信号 | 条件 | 动作 |
|---|---|---|
| 成本偏高 | CPA在目标价120-150%之间 | 观察1天，第2天仍高则关停 |
| CTR下降 | CTR从3%降到1%以下 | 3天内换素材，换后仍低则关停 |
| CVR下降 | CVR从8%降到3%以下 | 检查承接链路，2天内无改善则关停 |
| 素材疲劳 | 同一条素材连续使用7天+ | 必须替换，不要关停整个计划 |

## 五、何时放量

### 放量条件（同时满足3项）
1. **学习期已通过** — 累计 ≥ 20 个转化
2. **成本稳定** — CPA ≤ 目标价，连续3天
3. **素材有效** — CTR ≥ 2%，CVR ≥ 5%
4. **承接能力够** — 客服/销售能处理更多线索

### 放量方式
| 方式 | 操作 | 幅度 |
|---|---|---|
| 加预算 | 提高计划日预算 | 每次 +30-50% |
| 加价 | 提高出价 | 每次 +5-10% |
| 扩定向 | 放宽年龄/地域/兴趣 | 逐步放宽 |
| 复制计划 | 复制有效计划，微调 | 新计划独立跑 |

### 放量节奏
```
第1次放量：预算+30%，观察2天
第2次放量：预算+50%，观察2天
第3次放量：复制计划+扩定向，观察3天
持续放量：每周新增2-3条素材
```

## 六、何时换素材

### 素材替换标准
| 指标 | 替换标准 | 紧急程度 |
|---|---|---|
| CTR | 连续2天 < 1% | 紧急 |
| 消耗量 | 连续2天下降30%+ | 紧急 |
| 转化成本 | CPA > 目标价150%，连续2天 | 紧急 |
| 衰减天数 | 使用超过衰减周期 | 正常 |
| 对比排名 | 在同计划中排名垫底 | 正常 |

### 素材衰减周期参考
| 素材类型 | 衰减周期 | 说明 |
|---|---|---|
| 实拍对比 | 5-7天 | 效果最好但衰减最快 |
| 探店体验 | 7-10天 | 中等衰减速度 |
| 口播科普 | 10-14天 | 衰减较慢 |
| 优惠活动 | 3-5天 | 活动结束后立即衰减 |
| 资质展示 | 14-21天 | 衰减最慢 |

## 七、月度复盘

### 月度分析维度
| 维度 | 内容 |
|---|---|
| 目标达成 | 月初目标 vs 实际结果 |
| 成本趋势 | 月度成本曲线 |
| 素材效率 | 素材产出TOP5和BOTTOM5 |
| 线索质量 | 月度有效率/到店率/成交率 |
| ROI趋势 | 月度ROI曲线 |
| 竞品动态 | 同行投放变化 |
| 经验总结 | 成功经验 + 失败教训 |

### ROI拆解公式
```
ROI = 客单价 × 成交率 × 到店率 × 有效率 × (1/CPA)
    = 客单价 × 成交率 × 到店率 × 有效率 / CPA

要提高ROI，可以：
1. 提高客单价（升级产品/交叉销售）
2. 提高成交率（优化销售话术/跟进）
3. 提高到店率（优化提醒/预约体验）
4. 提高有效率（优化表单/定向）
5. 降低CPA（优化素材/出价/承接）
```
