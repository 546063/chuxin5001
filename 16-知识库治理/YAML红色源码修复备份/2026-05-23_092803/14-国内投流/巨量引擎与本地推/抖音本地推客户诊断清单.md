# 抖音本地推客户诊断清单

> 适用：首次接触的客户/准备接的投流代运营客户
> 最后更新：2026-05-22

---

## 一、基础信息收集

### 客户画像
| 维度 | 问题 | 记录 |
|---|---|---|
| 行业 | 您做什么生意？ | |
| 门店数量 | 几家店？位置在哪？ | |
| 经营时长 | 做了多久？ | |
| 团队规模 | 多少人？有专人运营抖音吗？ | |

| 维度 | 问题 | 记录 |

## 二、投流适配性评估

### 1. 行业适配（权重 20%）

| 评分 | 标准 | 说明 |
|---|---|---|
| 5分 | 本地生活高频行业（餐饮/美业/教培） | 天然适配本地推 |
| 4分 | 本地服务行业（除虫/消防/汽修/家装） | 需要定制化方案 |
| 3分 | 区域性B2B（设备/招商/工厂） | 可以投，但要留资模式 |
| 2分 | 全国性电商/纯线上服务 | 不太适配，建议千川/AD |
| 1分 | 违禁行业/政策限制 | 不建议投 |

### 2. 客单价（权重 15%）

| 评分 | 标准 | 说明 |
|---|---|---|
| 5分 | 客单价 200-1000 元 | 最佳区间，转化决策快 |
| 4分 | 客单价 1000-5000 元 | 可以，但需要私信/电话承接 |
| 3分 | 客单价 5000-20000 元 | 需要深度沟通，适合留资模式 |
| 2分 | 客单价 < 100 元 | 利润不够覆盖广告成本 |
| 1分 | 客单价 > 50000 元 | 决策周期太长，不适合直接转化 |

### 3. 毛利率（权重 15%）

| 评分 | 标准 | 说明 |
|---|---|---|
| 5分 | 毛利率 ≥ 60% | 有足够空间覆盖获客成本 |
| 4分 | 毛利率 40-60% | 可以，但要控制获客成本 |
| 3分 | 毛利率 20-40% | 需要高转化率才能盈利 |
| 2分 | 毛利率 10-20% | 风险较高，不建议 |
| 1分 | 毛利率 < 10% | 不建议投流 |

### 4. 地域覆盖（权重 10%）

| 评分 | 标准 | 说明 |
|---|---|---|
| 5分 | 一二线城市 | 抖音用户基数大，竞争也激烈 |
| 4分 | 三四线城市 | 竞争少，获客成本低 |
| 3分 | 县城/乡镇 | 用户基数小，效果有限 |
| 2分 | 偏远地区 | 用户少，建议其他渠道 |
| 1分 | 无固定服务区域 | 本地推不适合 |

### 5. 预算能力（权重 15%）

| 评分 | 标准 | 说明 |
|---|---|---|
| 5分 | 月预算 ≥ 5000 元 | 可以正常测试和放量 |
| 4分 | 月预算 3000-5000 元 | 可以跑，但要精打细算 |
| 3分 | 月预算 1000-3000 元 | 只能做基础测试 |
| 2分 | 月预算 < 1000 元 | 不建议，预算不够学习 |
| 1分 | 无预算概念 | 需要先教育客户 |

### 6. 过往投放经验（权重 5%）

| 评分 | 标准 | 说明 |
|---|---|---|
| 5分 | 有抖音投流经验，知道基本数据 | 好沟通，上手快 |
| 4分 | 投过其他平台（美团/百度/朋友圈） | 有概念，需要教育差异 |
| 3分 | 只做过自然流量/团购 | 需要从0教育 |
| 2分 | 完全没做过线上推广 | 教育成本高 |
| 1分 | 之前投亏过且心态负面 | 很难建立信任 |

### 7. 素材能力（权重 5%）

| 评分 | 标准 | 说明 |
|---|---|---|
| 5分 | 有专人拍摄/剪辑，能日更 | 素材供应充足 |
| 4分 | 老板/员工能出镜拍简单视频 | 可以支撑基础投放 |
| 3分 | 只有门店照片，不会拍视频 | 需要代做素材 |
| 2分 | 完全没有素材 | 需要从零制作 |
| 1分 | 拒绝拍摄/出镜 | 素材严重受限 |

### 8. 承接能力（权重 10%）

| 评分 | 标准 | 说明 |
|---|---|---|
| 5分 | 有专人回复私信/接听电话/接待 | 线索不浪费 |
| 4分 | 老板本人回复，能做到即时响应 | 可以接受 |
| 3分 | 兼职人员回复，响应不够及时 | 需要优化 |
| 2分 | 无人回复，线索白白流失 | 必须先解决承接 |
| 1分 | 完全不知道怎么做承接 | 需要培训/代运营 |

### 9. 销售跟进能力（权重 5%）

| 评分 | 标准 | 说明 |
|---|---|---|
| 5分 | 有标准SOP，线索30分钟内跟进 | 转化率高 |
| 4分 | 当天内能跟进 | 可以接受 |
| 3分 | 24小时内跟进 | 流失率较高 |
| 2分 | 无固定跟进时间 | 大量线索浪费 |
| 1分 | 线索来了不管 | 不适合投流 |

## 三、综合评分与判断

### 评分汇总
```
行业适配（×20%）+ 客单价（×15%）+ 毛利率（×15%）+
地域覆盖（×10%）+ 预算能力（×15%）+ 过往经验（×5%）+
素材能力（×5%）+ 承接能力（×10%）+ 销售跟进（×5%）
= 综合评分（满分5分）
```

### 评分结论
| 综合评分 | 结论 | 建议 |
|---|---|---|
| 4.0-5.0 | 非常适合 | 立即启动，全量投放 |
| 3.0-3.9 | 比较适合 | 启动，但需要先补齐短板 |
| 2.0-2.9 | 一般 | 先解决关键短板再投 |
| 1.0-1.9 | 不太适合 | 建议先做基础建设，3个月后再考虑 |

### 一票否决项
| 条件 | 原因 |
|---|---|
| 毛利率 < 15% | 利润覆盖不了获客成本 |
| 月预算 < 500 元 | 预算不够算法学习 |
| 完全无人承接 | 线索全部浪费 |
| 违禁行业 | 审核不通过，白花钱 |
| 之前投亏过且不愿调整 | 心态问题，很难做好 |

## 四、客户沟通话术

### 当客户不适合投流时
```
"根据您的情况，我建议先做这几件事：
1. 先把抖音企业号搭建好，完善门店信息
2. 每周发2-3条短视频，积累自然流量
3. 把团购券设计好，上架到抖音
4. 等有一定的自然数据和内容基础，再考虑投流

这样做的好处是：
- 先低成本验证抖音能不能给你获客
- 积累素材和粉丝，后续投流效果更好
- 避免一开始就花钱但接不住"
```

### 当客户适合投流时
```
"您的条件很适合做抖音本地推：
- 行业是本地生活的高频行业
- 客单价和毛利率都有足够的利润空间
- 预算也够测试和放量

建议的投放方案是：
1. 第一阶段（1-2周）：小预算测试，找到有效素材和定向
2. 第二阶段（3-4周）：放量投放，积累客户
3. 第三阶段（5周+）：稳定投放，持续优化

预算建议：第一阶段每天XX元，第二阶段每天XX元"
```
