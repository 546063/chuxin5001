# 本地推客户首次诊断话术

> 适用：首次微信/电话/面对面沟通
> 最后更新：2026-05-22

---

## 一、开场白（建立信任）

### 微信首次通过
```
[客户称呼]您好，我是[你的名字]，专门帮[城市]本地商家做抖音获客的。

看您在抖音上发的[内容]，做得挺好的！

想跟您简单聊聊，看看能不能帮您在抖音上多获一些客~

不收费的，就当交个朋友，行就行不行也没关系~
```

### 转介绍开场
```
[客户称呼]您好，我是[介绍人]的朋友[你的名字]。

听[介绍人]说您在做[行业]，正好我这边帮不少[行业]的老板在抖音上获客。

方便聊几句吗？10分钟就行，看看有没有能帮到您的地方~
```

---

## 二、诊断提问（了解情况）

### 第一步：了解业务（2分钟）
```
Q1: "您现在店里最主要的收入来源是什么？"
Q2: "客单价大概多少？"
Q3: "老客户回头率怎么样？"
Q4: "在抖音上做过推广吗？"
```

### 第二步：了解痛点（2分钟）
```
Q5: "现在获客主要靠什么渠道？"
Q6: "最头疼的问题是什么？（客流少/成本贵/竞争大）"
Q7: "之前在美团/百度/朋友圈投过吗？效果怎么样？"
```

### 第三步：了解能力（1分钟）
```
Q8: "店里有专人负责抖音吗？"
Q9: "平时能拍视频吗？还是只有照片？"
Q10: "如果抖音上来线索了，有人能及时回复吗？"
```

### 第四步：了解预算（1分钟）
```
Q11: "如果抖音能稳定获客，您愿意一个月投多少钱？"
Q12: "之前推广花了多少？效果您能接受吗？"
```

---

## 三、快速判断（心中评估）

根据客户回答，在心中快速评估：

| 维度 | 适合 | 不适合 |
|---|---|---|
| 行业 | 本地生活/服务 | 全国性/纯线上 |
| 客单价 | 100-5000元 | <100或>5万 |
| 毛利率 | >30% | <15% |
| 预算 | >2000元/月 | <500元/月 |
| 承接 | 有人回复 | 无人回复 |
| 素材 | 能拍/愿意拍 | 拒绝拍摄 |

---

## 四、给出初步建议（展现专业）

### 如果适合投流
```
"根据您的情况，我觉得可以做抖音本地推。原因有三：

1. 您的[行业]在抖音上效果不错，同城有不少成功案例
2. 您的客单价[XX元]和毛利都够覆盖广告成本
3. 您店里有[人]能接住线索

建议的投放方案：
- 先用每天[XX元]的小预算测试1-2周
- 测试重点是找到有效的素材和定向
- 效果好再放量

如果做起来，预计每月能获[XX]条线索，按您的成交率，大概能成交[XX]单。

您觉得呢？我们可以约个时间详细聊方案~"
```

### 如果不适合投流
```
"说实话，根据您的情况，我建议您先做这几件事：

1. 先把抖音企业号搭好，每周发2-3条内容
2. 把团购券上架到抖音
3. 先跑1个月自然流量看看效果

现在投流的话，[承接/素材/预算]这块还没准备好，容易白花钱。

等这些基础做好了，再考虑投流效果会好很多。

如果您愿意，我可以免费帮您看看账号，给点建议~"
```

---

## 五、推动下一步（约方案/约见面）

### 微信约方案
```
"要不这样，我花1天时间帮您做一个详细的投放方案：
- 包括投多少钱、怎么投、预期效果
- 不用您先掏钱，方案做出来您觉得行咱们再聊

您看明天下午方便吗？我把方案发给您看看~"
```

### 电话约见面
```
"电话里聊得比较粗，要不咱们约个时间当面聊？
我去您店里看看实际情况，方案也能做得更精准。

您看明天还是后天方便？我过来也就半小时~"
```

## 六、常见异议处理

| 客户说 | 你怎么回 |
|---|---|
| "太贵了" | "我理解您的担心。不过抖音获客的成本其实比[美团/百度]低XX%，而且客户质量更高。要不我们先做小预算测试？" |
| "之前投过，亏了不少" | "亏的原因可能很多，不一定是平台的问题。您之前是怎么投的？我帮您分析一下问题在哪。" |
| "我没时间拍视频" | "不需要您天天拍。一周拍2-3条就行，每条15-30秒。或者我们可以帮您做素材。" |
| "我再考虑考虑" | "没问题，考虑是应该的。要不我先帮您免费做一个诊断报告？您看完再决定。" |
| "别人做的效果不好" | "效果好不好跟怎么做有很大关系。您说的哪家？我可以帮您对比一下他们的做法和我们的差异。" |
| "我自己也能投" | "当然可以！不过专业的事交给专业的人做，效率会高很多。您把精力放在服务客户上，我们来负责获客。" |
