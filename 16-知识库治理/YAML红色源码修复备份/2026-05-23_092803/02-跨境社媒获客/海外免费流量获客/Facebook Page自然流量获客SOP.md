# Facebook Page 自然流量获客 SOP

> **适用行业**：B2B 食品加工机械 / 通用机械设备出口
> **核心原则**：Page 是品牌信任基础，不是直接销售渠道。先建立专业形象，再引导询盘。
> **成本说明**：免费流量 ≠ 零成本。需要每日 30-60 分钟投入。

---

## 一、Page 主页包装

### 1.1 基础设置

| 项目 | 建议 |
|------|------|
| Page 名称 | 公司名 + 行业关键词（如 "ABC Machinery - Food Processing Equipment"） |
| 头像 | 工厂 Logo 或品牌标识（正方形，170x170px） |
| 封面图 | 工厂实拍 + 核心产品线（820x312px，展示规模感） |
| 简介 | 一行说明：核心产品 + 年限 + 出口范围（不超过 255 字符） |
| CTA 按钮 | "Send WhatsApp Message" 或 "Contact Us" |
| Bio 链接 | wa.me 短链接 或 Linktree（含 WhatsApp/邮箱/网站） |

### 1.2 关键信息页

| 项目 | 内容 |
|------|------|
| About | 公司简介、主营产品、出口国家、认证 |
| Contact | WhatsApp、邮箱、网站、工作时间 |
| Services | 列出 3-5 个核心服务（OEM/安装/售后/培训） |
| Photos | 工厂环境、生产线、发货现场、客户案例（至少 20 张） |
| Videos | 机器运转视频、工厂参观、产品演示（至少 5 个） |
| Reviews | 收集客户评价（自然积累，不刷） |

### 1.3 置顶帖设计

置顶帖是 Page 访客看到的第一条内容，必须包含：

1. **价值钩子**：工厂实力 / 核心产品 / 客户案例
2. **具体内容**：3-5 个核心产品亮点
3. **CTA**：WhatsApp 链接或询盘引导
4. **形式**：视频 + 文字（视频优先）

**置顶帖模板**：
```
🏭 [Company Name] - Food Processing Equipment Manufacturer
✅ 20+ years experience
✅ Export to 50+ countries
✅ CE / ISO certified

Top products:
• [Product 1] - [Key spec]
• [Product 2] - [Key spec]
• [Product 3] - [Key spec]

📩 Free quote → WhatsApp: wa.me/[number]
🌐 www.yoursite.com
```

## 二、内容栏目与混排

### 2.1 内容栏目（建议 5 个）

| 栏目 | 占比 | 内容 | 形式 |
|------|------|------|------|
| 产品展示 | 30% | 机器运转、功能演示、技术参数 | 视频（Reels） |
| 工厂背书 | 20% | 生产线、质检、发货、仓库 | 图片 + 文字 |
| 客户案例 | 20% | 安装现场、客户反馈、使用效果 | 视频 + 文字 |
| 行业知识 | 20% | 设备选型、维护技巧、趋势分析 | 文字 + 图片 |
| 互动/活动 | 10% | 问答、投票、限时优惠 | 文字 |

### 2.2 混排策略

| 形式 | 占比 | 最佳用途 |
|------|------|---------|
| Reels（短视频） | 40% | 机器运转、产品演示 → 最大覆盖 |
| 图片帖 | 25% | 工厂环境、发货现场 → 信任背书 |
| 文字帖 | 15% | 行业知识、FAQ → 专业形象 |
| 案例帖 | 20% | 客户成功故事 → 转化驱动 |

### 2.3 食品机械内容示例

| 栏目 | 示例 |
|------|------|
| 产品展示 | "Watch how this dumpling machine makes 1000pcs/hour" |
| 工厂背书 | "Our quality control process — every machine tested before shipping" |
| 客户案例 | "This UAE client increased production by 300% with our line" |
| 行业知识 | "How to choose the right capacity for your food business" |
| 互动 | "What's your biggest challenge in food processing? Comment below!" |

### 2.4 机械设备内容示例

| 栏目 | 示例 |
|------|------|
| 产品展示 | "Precision cutting — 0.1mm accuracy demonstrated" |
| 工厂背书 | "CNC workshop tour — 20+ machines, full production capability" |
| 客户案例 | "Factory in Mexico reduced labor cost by 60%" |
| 行业知识 | "5 factors to consider when buying packaging equipment" |
| 互动 | "Manual vs automatic — which do you prefer? Vote below." |

### 2.5 成人用品合规内容

| 可做 | 不可做 |
|------|--------|
| 产品科技角度（材料/工艺/设计） | 露骨画面或文字 |
| 行业趋势分析 | 性暗示内容 |
| OEM/制造能力展示 | 面向未成年人的内容 |
| 独立站/邮件承接 | 直接在 Page 推广敏感产品 |

## 三、发帖频率与时间

### 3.1 发帖频率

| 阶段 | 频率 | 说明 |
|------|------|------|
| 新 Page（0-30 天） | 每天 1 条 | 建立内容基础 |
| 成长期（30-90 天） | 每周 5-7 条 | 保持稳定活跃 |
| 成熟期（90 天+） | 每周 3-5 条 | 重质量轻数量 |

### 3.2 最佳发帖时间

| 地区 | 最佳时间（当地时间） |
|------|---------------------|
| 中东 | 周日-周四 10:00-14:00 |
| 东南亚 | 周一至周五 09:00-11:00 |
| 欧美 | 周二-周四 08:00-10:00 / 13:00-15:00 |
| 拉美 | 周一至周五 09:00-12:00 |

## 四、评论区承接

### 4.1 评论回复策略

| 用户评论 | 你的回复 |
|---------|---------|
| "Price?" | "Sent you a DM with pricing. Check your message requests! 📩" |
| "How does it work?" | "I'll send you a detailed video on WhatsApp for easier viewing: wa.me/[number]" |
| "Interested" | "Thanks! DM me your requirements and I'll prepare a custom quote." |
| 技术问题 | 在评论区专业回答，然后 "For full specs, DM or WhatsApp: wa.me/[number]" |

### 4.2 主动互动

- 每天检查评论（至少 2 次）
- 回复所有评论（1 小时内最佳）
- 在热门评论下引导 WhatsApp
- 不要删除负面评论，专业回应即可

## 五、私信承接

### 5.1 私信流程

```
用户看到 Page 内容
  ↓
用户评论或私信
  ↓
第一回合：专业回复 + 轻度 WhatsApp 引导
  ↓
用户添加 WhatsApp
  ↓
WhatsApp 承接 → 询盘跟进
```

### 5.2 私信话术

```
Hi [Name], thanks for reaching out!

Here's a quick overview of [product]:
• [Key spec 1]
• [Key spec 2]
• [Key spec 3]

For detailed specs, pricing, and a custom quote, 
you can reach me on WhatsApp: wa.me/[number]

Looking forward to helping you!
```

## 六、风险点

| 风险 | 触发条件 | 预防 |
|------|---------|------|
| Page 被限流 | 频繁发硬广告 | 80% 价值内容 + 20% 推广 |
| 评论被举报 | 重复复制同一条回复 | 个性化回复每条 |
| 成人用品违规 | 发布敏感内容 | 严格合规，独立站承接 |
| 账号被标记为 spam | 新号频繁加陌生人 | 先建立内容基础再互动 |

---

## 七、7 天启动计划

| 天数 | 任务 |
|------|------|
| Day 1 | 创建 Page，完善所有信息，上传 Logo 和封面 |
| Day 2 | 发布 3 条历史内容（产品展示/工厂/案例） |
| Day 3 | 设置 CTA 按钮 + Bio 链接 + 置顶帖 |
| Day 4 | 发布 1 条 Reels + 1 条行业知识 |
| Day 5 | 发布 1 条客户案例 + 互动帖 |
| Day 6 | 检查评论回复 + 私信处理 |
| Day 7 | 复盘：查看哪些内容互动最高，调整下周计划 |
