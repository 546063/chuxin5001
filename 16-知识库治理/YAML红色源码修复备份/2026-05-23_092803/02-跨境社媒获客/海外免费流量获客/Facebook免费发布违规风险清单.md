# Facebook 免费发布违规风险清单

> **适用场景**：Facebook Group / Page / 个人号 / Marketplace 免费发布
> **核心原则**：Facebook 的算法和政策比想象中严格。免费流量获客必须遵守规则，否则账号受限 = 所有努力归零。

---

## 一、垃圾发帖风险

### 触发条件

- 同一条内容发布到 5+ 个 Group
- 每天在多个 Group 发相同/相似内容
- 内容 80% 以上是硬广告
- 评论全是"Contact me"类引导

### 后果

| 严重程度 | 后果 |
|---------|------|
| 轻度 | 帖子被 Group 管理员删除 |
| 中度 | Group 禁止发帖（临时/永久） |
| 重度 | Facebook 账号限流/受限 |
| 严重 | 账号永久封禁 |

### 预防

1. 每个 Group 的内容做差异化（改标题/开头）
2. 80% 价值内容 + 20% 推广
3. 每个 Group 每天最多 1 条
4. 不要在帖子中堆砌联系方式

---

## 二、重复内容风险

### 触发条件

- 复制粘贴同一条帖子到多个 Group
- 使用相同的标题、描述、图片
- 短时间内发布多条高度相似内容

### 后果

- Facebook 算法检测为 spam
- 帖子被折叠/隐藏
- 账号发帖权限受限

### 预防

1. 每个 Group 的内容至少改 30%（标题/开头/CTA）
2. 使用不同的图片/视频
3. 间隔 2-4 小时再发下一条
4. 准备 5-10 条不同的内容轮换使用

---

## 三、群组规则风险

### 触发条件

- 不阅读 Group Rules 直接发帖
- 在禁止广告的 Group 发推广内容
- 违反特定 Group 的格式要求

### 后果

| 后果 | 说明 |
|------|------|
| 帖子被删 | Group 管理员手动删除 |
| 被踢出 Group | 无法再查看该 Group 内容 |
| 被禁止加入 | 永久无法加入该 Group |
| 被举报 | 可能触发 Facebook 账号审核 |

### 预防

1. 加入每个 Group 后先阅读 Rules
2. 前 2 周只回答问题，不发推广
3. 观察其他成员发什么内容
4. 如有疑问，先私信管理员确认

---

## 四、成人用品风险

### 触发条件

- 发布露骨产品图片或描述
- 在 Facebook 直接推广成人用品
- 发布性暗示内容
- 面向未成年人可见的受众

### 后果

| 平台 | 后果 |
|------|------|
| Facebook Page | 页面被删除 + 广告账户封禁 |
| Facebook Group | 群组被删除 |
| 个人号 | 账号永久封禁 |
| Marketplace | 帖子删除 + 权限永久禁止 |

### 预防

1. **不在 Facebook 直接推广成人用品**
2. 用科技角度描述（材料/工艺/设计）
3. 通过独立站承接
4. 使用 Reddit/X 等允许的平台
5. 确保所有内容标注 18+ 年龄限制
6. 绝不面向未成年人

---

## 五、夸大承诺风险

### 触发条件

- "World's Best"、"100% Effective"
- "Guaranteed ROI"、"No Risk"
- 虚假认证（假的 CE/ISO）
- 不实的客户案例

### 后果

- 帖子被举报
- Facebook 标记为误导性内容
- 商业信誉受损
- 可能面临法律风险

### 预防

1. 所有数据必须真实可验证
2. 认证必须真实有效
3. 客户案例需客户许可
4. 用具体数字代替模糊描述（"300 units/hour" 而非 "超快"）

---

## 六、侵权风险

### 触发条件

- 使用他人拍摄的图片/视频
- 使用有版权的音乐（Reels 背景音乐）
- 使用竞争对手的品牌 Logo
- 抄袭他人文案

### 后果

- DMCA 投诉
- 内容被删除
- 账号重复侵权 = 封禁
- 可能面临法律追责

### 预防

1. 只用自己拍摄的图片/视频
2. 使用无版权音乐（Facebook Sound Collection）
3. 不提及竞品品牌
4. 原创文案

---

## 七、私信骚扰风险

### 触发条件

- 给非好友发消息
- 群发相同消息
- 不回复后继续发消息
- 消息中包含大量链接

### 后果

| 严重程度 | 后果 |
|---------|------|
| 轻度 | 消息被折叠到 Message Requests |
| 中度 | 私信权限受限（临时） |
| 重度 | 账号被标记为 spam |
| 严重 | 账号封禁 |

### 预防

1. 只给主动联系你的人发消息
2. 不发群发消息
3. 每条消息个性化
4. 对方 2 次不回复后停止
5. 第一条消息不发链接

---

## 八、账号封禁风险

### 综合触发条件

- 多次违反以上任何规则
- 被多人举报
- 使用自动化工具（auto-posting/bot）
- 使用虚假身份

### 后果

| 类型 | 后果 |
|------|------|
| 临时封禁 | 24-72 小时无法登录 |
| 功能受限 | 无法发帖/评论/私信 |
| 永久封禁 | 账号永久不可恢复 |
| 关联封禁 | 同 IP/设备的其他账号也受影响 |

### 预防

1. 严格遵守 Facebook Community Standards
2. 不购买/使用自动化工具
3. 使用真实身份
4. 定期备份内容（以防账号恢复）
5. 保留所有合规证据

## 九、合规检查清单

发帖前逐项检查：

| # | 检查项 | 通过 |
|---|--------|------|
| 1 | 内容是否违反 Facebook Community Standards？ | ☐ |
| 2 | 是否阅读并遵守了 Group Rules？ | ☐ |
| 3 | 内容是否 80% 以上为价值型？ | ☐ |
| 4 | 是否没有夸大/虚假承诺？ | ☐ |
| 5 | 使用的图片/视频是否为自己原创？ | ☐ |
| 6 | 是否没有侵犯他人版权/商标？ | ☐ |
| 7 | 成人用品内容是否完全合规？ | ☐ |
| 8 | 是否没有群发相同内容？ | ☐ |
| 9 | 私信是否只发给主动联系的人？ | ☐ |
| 10 | 是否没有使用自动化工具？ | ☐ |

**全部通过 = 可以发布。有任何一项不通过 = 先修改再发布。**
