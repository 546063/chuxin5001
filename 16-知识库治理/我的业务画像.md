# 我的业务画像

## 我是谁

一人公司老板，主营新媒体运营、客户账号运营、投流规划、短视频文案策略。

## 我在做什么

- **跨境B2B社媒获客**（未来主营方向）
  - 平台：Facebook / LinkedIn / TikTok
  - 转化路径：社媒内容 → 私信 → WhatsApp → 成交
  - 行业：食品加工机械、VR成人科技、教育培训、医美旅游、装修家居、羽毛球馆

- **国内新媒体运营**（当前收入来源）
  - 客户账号运营、内容策略、投流规划
  - 拍摄交给摄像师，剪辑交给剪辑师
  - 我负责：运营策略、客户规划、文案、成交路径设计

## 我的核心能力

- 跨境B2B获客策略
- 新媒体账号定位和内容规划
- Facebook投流 / 巨量本地推投流
- 短视频文案和钩子设计
- WhatsApp私聊成交
- 客户分层和跟进SOP

## 我的客户类型

| 行业 | 特点 | 主要平台 |
|------|------|---------|
| 食品加工机械 | B2B、高客单、决策周期长 | Facebook、LinkedIn、TikTok |
| VR成人科技 | 新兴行业、灰色地带 | TikTok、TikTok海外 |
| 教育培训 | B2C为主、本地获客 | 巨量本地推、抖音 |
| 医美旅游 | 高利润、跨境服务 | Facebook、Instagram、小红书 |
| 装修家居 | 本地服务、低频高客单 | 巨量本地推、抖音 |
| 羽毛球馆 | 本地生活、会员制 | 巨量本地推、抖音 |

## 我的业务主线

```text
选品 / 客户项目
→ 市场调研
→ 账号定位
→ 内容策略
→ 广告投流
→ 私信成交
→ WhatsApp转化
→ 客户复盘
→ 知识库沉淀
```

## 我的原则

- 先测试，再放大
- 小预算、快验证、慢扩张
- 每个决策都记录，形成知识库
- 模型负责思考，SOP负责执行，案例负责验证
