# 投流与视频获客知识库每周深度补充机制

---

## 一、补充范围

### 每周必补模块
| 模块 | 补充内容 | 优先级 |
|------|---------|--------|
| 公开案例 | Meta Ad Library / TikTok Ad Library 新案例 | P1 |
| 平台规则 | 各平台政策/功能更新 | P0 |
| 竞品分析 | 竞品广告素材观察 | P1 |
| 素材方向 | 新素材方向/钩子/文案 | P1 |
| 数据更新 | 行业报告/成本数据更新 | P2 |

### 可选补充模块
| 模块 | 触发条件 |
|------|---------|
| 新平台 | 发现新的获客渠道 |
| 新行业 | 拓展到食品机械之外的行业 |
| 深度研究 | 用户提出特定深度问题 |
| 真实数据 | 有真实投流数据可导入 |

---

## 二、补充流程

```
1. 扫描新资料（平台文档/行业报告/竞品广告）
    ↓
2. 判断是否值得补充（与现有内容是否有差异化）
    ↓
3. 创建/更新文档（遵循 frontmatter + 结构规范）
    ↓
4. 更新路由文件（新增 routing_target）
    ↓
5. 更新总清单（如新增准备项）
    ↓
6. 生成补充报告（记录本周新增内容）
```
