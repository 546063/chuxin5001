# 食品机械B2B增长系统

## 一句话结论

食品机械增长不是单点投广告，而是“产品选择 -> 素材表达 -> 渠道测试 -> 私信承接 -> 报价跟进 -> 案例复盘”的闭环。

## 30天最小执行版

| 周期 | 主任务 | 验收标准 |
|---|---|---|
| 第1周 | 选定 1-3 个主推产品，整理参数、卖点、买家画像 | 每个产品至少 1 张产品卡、5 个痛点、5 个素材方向 |
| 第2周 | 产出短视频和图文素材，搭建 Facebook/LinkedIn/TikTok 基础入口 | 每个平台至少 5 条内容或素材草稿 |
| 第3周 | 跑小预算或自然发布测试，记录曝光、点击、私信、询盘 | 每日记录数据，保留截图或脱敏记录 |
| 第4周 | 复盘素材、平台、线索质量、WhatsApp 跟进结果 | 形成 1 份复盘报告和下一轮测试计划 |

## 平台组合建议

| 目标 | 优先组合 | 说明 |
|---|---|---|
| 快速验证海外询盘 | Facebook Ads + WhatsApp | 线索链路短，适合小预算测试 |
| 建立长期信任 | TikTok/YouTube Shorts/Instagram Reels | 适合设备演示、工厂实力、案例内容 |
| 找高客单决策者 | LinkedIn 内容 + 私信开发 | 慢，但更适合采购经理、老板、渠道商 |
| 补充国内业务 | 视频号/抖音 + 私域承接 | 适合国内 B2B 或代运营类业务 |
| 复用素材资产 | 一稿多发 + 平台适配 | 同一素材按平台重写标题、字幕、CTA |

## 本系统的核心检查清单

- [ ] 主推产品是否有明确目标市场？
- [ ] 产品素材是否能在 3 秒内让买家看懂用途？
- [ ] 每条内容是否有明确 CTA？
- [ ] Facebook / TikTok / LinkedIn 是否有不同定位？
- [ ] WhatsApp 承接话术是否能快速筛选预算、用途、采购时间？
- [ ] 每周是否形成至少 1 条可复用案例或失败归因？

## 维护规则

- 新增产品资料进入产品参数或设备知识总纲。
- 新增素材进入内容素材或对应平台素材库。
- 新增真实询盘只进入脱敏复盘模板，不写入隐私原文。
- 每轮测试结束后更新一次案例库和 QA 路由。
