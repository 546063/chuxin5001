# 食品机械设备知识总纲

**覆盖范围：** 清洗类/去皮脱水类/果蔬切割类/肉类切割类/成型类/面食输送整线类，共6大类，涵盖29个细分产品类别。

## 清洗设备

### 设备类型
- **鼓泡喷淋清洗机**：通过气泡和喷淋去除泥沙、杂质、农药残留
- **震动式涡流清洗机**：振动+水流冲刷，适合根茎类批量清洗

### 目标客户
净菜厂、中央厨房、农产品加工厂、团餐、预制菜工厂

### 关键参数
| 参数 | 说明 | 客户关注点 |
|---|---|---|
| 处理量 | kg/h 或 T/h | 必须满足客户产能需求 |
| 水循环 | 是否循环使用、耗水量 | 环保要求和成本控制 |
| 喷淋强度 | 多段喷淋压力可调 | 不同原料需要不同强度 |
| 臭氧/过滤 | 是否带杀菌去农残功能 | 高端客户在意 |
| 出料方式 | 斜坡出料、输送带出料 | 影响后端衔接 |

### 核心卖点
- 减少人工挑拣（每吨可省2-4人）
- 降低损伤（轻柔处理保持品相）
- 提升前处理效率（连续作业）
- 农残降解（臭氧/超声波辅助）

### 客户痛点
- 人工清洗慢、成本高、人员管理难
- 泥沙洗不干净导致投诉
- 伤果蔬外观影响卖价
- 水费太高

### 社媒适合拍
- 泥土冲洗前后对比（直观震撼）
- 连续出料镜头（展示效率）
- 杂质分离效果（展示专业性）
- 气泡翻滚+水流冲刷（视觉吸引）

## 去皮设备

### 设备类型
- **连续式削皮机**：土豆、胡萝卜等根茎类的连续去皮
- **毛刷去皮机**：柔和去皮，适合土豆、红薯、芋头
- **变频式离心甩干机**：去皮后快速脱水

### 目标客户
根茎类加工厂、净菜厂、中央厨房、食品厂前处理线

### 关键参数
| 参数 | 说明 | 客户关注点 |
|---|---|---|
| 去皮损耗率 | 去皮后原料损耗百分比 | 损耗太高客户不接受 |
| 适用原料尺寸 | 最小/最大原料直径 | 必须覆盖客户原料范围 |
| 毛刷材质 | 尼龙、钢丝、特殊材质 | 食品级认证 |
| 脱水时间 | 甩干转速和时间 | 脱水效果和产能 |
| 含水率控制 | 甩干后含水率 | 后端切割需要控制含水率 |

### 核心卖点
- 去皮损耗率可控（3-8%，可调节）
- 表面去皮均匀（品相好）
- 进入下道工序更稳定（标准化）
- 变频调速（适配不同原料）

### 客户痛点
- 手工削皮人工成本高
- 削皮不均匀导致成品率低
- 土豆去皮损耗太高（有的超15%）
- 后端切割含水率影响切面

### 社媒适合拍
- 去皮前后对比（萝卜、土豆、芋头）
- 批量出料一致性（展示标准化）
- 毛刷摩擦去皮特写（展示柔和处理）
- 变频调速演示（展示灵活性）

## 切割切丁设备

### 设备类型
- **分半机**：对半切分（胡萝卜、土豆等）
- **切丝机**：切制各种粗细的丝
- **切片机**：切制圆形/椭圆形片状
- **切丁机**：切制立方体规格（8mm/10mm/12mm/15mm/20mm等）
- **切块机**：切制不规则块状
- **切菜机**：一机多刀组切换，切丝切片切丁
- **滚刀切丝机**：大蒜、姜等特殊原料
- **剁块机**：大型切块（大块牛肉等）

### 目标客户
净菜厂、中央厨房、学校餐饮配送、商超切配中心、肉类加工厂

### 关键参数
| 参数 | 说明 | 客户关注点 |
|---|---|---|
| 切割尺寸 | 可调节范围（如2-25mm） | 必须覆盖客户规格需求 |
| 换刀效率 | 换刀组时间 | 频繁换规格的客户在意 |
| 产能 | kg/h | 满足客户产量需求 |
| 适用原料 | 鲜品、冷藏、微冻 | 客户加工什么原料 |
| 切面整齐度 | 是否平整、无毛边 | 影响卖相和二次加工 |

### 核心卖点
- 切割尺寸统一（规格稳定）
- 效率高（1台顶3-5人）
- 换型快（不同刀组快速切换）
- 一机多能（切丝切片切丁换刀即可）

### 客户痛点
- 人工切丝切片效率太低
- 规格不统一影响卖相
- 换刀太麻烦影响效率
- 鲜肉和冻肉切割效果差异大

### 社媒适合拍
- 不同刀组切不同形状（展示灵活性）
- 连续切割高速出料（展示效率）
- 切丝切片切丁一个镜头切换（展示一机多能）
- 冻肉切丁一致性展示（展示专业性）

## 肉类加工设备

### 设备类型
- **冻肉切片机**：-18°C冻肉切片，适合火锅、烤肉食材
- **鲜肉切片机**：0-4°C冷藏肉切片，适合中餐、日料
- **分半机/分层机**：大块肉分层切分
- **分条机**：肉条切制（肉干、肉脯加工）
- **鱼片机**：鱼柳切片
- **砍排机**：排骨砍段
- **锯骨机**：带骨肉类切割（排骨、鸡架等）
- **绞肉机**：肉馅、肉糜加工
- **肉饼成型机**：肉饼、汉堡肉饼加工
- **去皮机**：肉类去皮（如猪皮去毛）

### 目标客户
肉类加工厂、连锁餐饮供应链、火锅食材厂、预制菜工厂、肉制品厂

### 关键参数
| 参数 | 说明 | 客户关注点 |
|---|---|---|
| 温度适配 | 鲜肉模式(0-4°C)、冻肉模式(-18°C) | 客户加工什么温度的肉 |
| 切片厚度 | 可调节（如0.8-3mm） | 切出来的用途 |
| 切丁规格 | 立方体尺寸（8/10/12/15/20mm） | 客户最终产品规格 |
| 成品率 | 切割后成品重量/原料重量 | 损耗越低越好 |
| 刀组寿命 | 刀片更换周期 | 频繁切割的成本 |
| 冻肉兼容 | 是否能切-18°C冻肉 | 国内客户很多要冻肉 |

### 核心卖点
- 冻鲜两用（可切冻肉和鲜肉）
- 厚薄稳定（切割精度高）
- 减少碎料（成品率高）
- 效率替人（1台顶多人）

### 客户痛点
- 冻肉难切（硬度高、刀具损耗快）
- 切面不整齐影响卖相
- 碎肉太多损耗大
- 鲜肉和冻肉需要不同设备

### 社媒适合拍
- 冻肉切丁出库画面（展示能力）
- 鲜肉切片纹理对比（展示品质）
- 分条一致性展示（展示标准化）
- 砍排效率对比（展示速度）

## 面食设备

### 设备类型
- **包子机**：全自动包子成型
- **饺子机**：全自动饺子包合成型
- **推条机**：面条、刀削面胚推制
- **面团分割机**：面团定量分割
- **和面机**：面粉加水搅拌成面团
- **醒发箱**：面团醒发控制

### 目标客户
速冻食品厂、中央厨房、主食工厂、团餐工厂、包子店连锁、饺子厂

### 关键参数
| 参数 | 说明 | 客户关注点 |
|---|---|---|
| 产能 | 个/h或kg/h | 满足产量需求 |
| 成型精度 | 个重误差范围 | 连锁店需要标准化 |
| 成品一致性 | 每个包子/饺子重量差 | 客户品控要求 |
| 与后端衔接 | 能否连接蒸煮线/冷冻线 | 整线化需求 |
| 馅料兼容性 | 包子可包什么馅、饺子皮厚薄 | 客户产品线 |

### 核心卖点
- 全自动成型（减少人工）
- 成型标准化（每个一致）
- 连续化生产（接后端蒸煮/冷冻）
- 可定制馅料和皮厚

### 客户痛点
- 手工包饺子效率太低（人工短缺）
- 包子大小不一影响标准化
- 后端衔接需要人工二次搬运
- 馅料多样性需要快速切换

### 社媒适合拍
- 包子从成型到蒸制全流程
- 饺子机高速包合（展示效率）
- 面条从推制到煮制（展示一气呵成）
- 包子/饺子冷冻后品相展示

## 输送与整线方案

### 设备类型
- **提升输送机**：垂直提升、爬坡输送
- **挑选工作台**：人工质检、异物挑选
- **皮带输送机**：水平输送
- **滚筒输送机**：规则物体输送
- **整颗菜综合生产线**：洗-切-脱-输送全流程
- **根茎类加工生产线**：去皮-切片-切丁-脱水全流程

### 目标客户
净菜厂、中央厨房、预制菜工厂、食品厂整线改造

### 关键参数
| 参数 | 说明 | 客户关注点 |
|---|---|---|
| 占地 | 整线长度和宽度 | 工厂空间有限 |
| 前后端衔接 | 设备之间如何连接 | 整线运行顺畅 |
| 控制方式 | PLC/触摸屏/手动 | 客户操作习惯 |
| 定制能力 | 能否按客户厂房定制 | 很多需要非标 |
| 产能匹配 | 各工序产能是否匹配 | 避免瓶颈工序 |

### 核心卖点
- 整线化（减少人工转运）
- 非标定制（按厂房设计）
- PLC控制（自动化程度高）
- 模块化设计（分期投入）

### 客户痛点
- 各工序设备不匹配（产能瓶颈）
- 人工转运造成污染
- 整线太贵超预算
- 控制系统太复杂操作难

### 社媒适合拍
- 整线运行全景（展示规模感）
- 各工序无缝衔接特写
- 操作员触摸屏控制（展示简便）
- 非标定制案例展示

## 常见客户问题

### 问题1：你们设备能切我的原料吗？
需要确认：原料种类（蔬菜/肉类/面食）、原料状态（鲜/冷/冻）、成品规格（厚度/尺寸/形状）。

### 问题2：产能能达多少？
需要确认：原料处理量（kg/h）、成品产量（kg/h）、每天工作时间。

### 问题3：设备多少钱？
需要确认：哪类设备、规格配置、是否含税、是否含运费和安装。

### 问题4：能不能定制？
需要确认：定制内容（尺寸/电压/材质/功能）、起订量、交付周期。

### 问题5：售后怎么保障？
需要确认：质保期、备件供应、远程指导、现场服务。

### 问题6：设备认证有哪些？
需要确认：CE/UL认证需求、材质证明、食品接触材料说明。
