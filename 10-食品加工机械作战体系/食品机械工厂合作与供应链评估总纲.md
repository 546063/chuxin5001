# 食品机械工厂合作与供应链评估总纲

## 中文总结

本文档是食品机械工厂合作评估的核心标准文件，包含工厂探访问题清单、合作评估模型、河北产业带分析和合作风险控制。

**核心定位：** 当用户问"这家工厂能不能合作""探厂问什么""怎么筛工厂"时调用本文档。

**决策价值：** 避免与劣质工厂合作导致货期延误、质量问题和沟通成本损失。

## English Summary

This document covers factory cooperation assessment for food machinery: factory visit checklist (product/capacity/quality/certification/after-sales/customization/delivery/export experience/payment), factory cooperation evaluation model (product capability/delivery capability/foreign trade coordination/after-sales capability/documentation capability/price stability), Hebei industrial belt distribution and cooperation risk control.

## 一句话结论

好工厂的标志是"能交付、能沟通、能配合"——而不是规模最大或价格最低。

## 工厂探访问题清单

### 第一类：产品与产能

| 问题 | 目的 | 判断依据 |
|---|---|---|
| 你们的核心产品是什么？ | 识别主力产品线 | 和你的需求匹配度 |
| 月产能是多少台/套？ | 判断供货能力 | 能否满足你的订单量 |
| 最长单次能接多少订单？ | 判断产能上限 | 大单是否需要外协 |
| 生产周期是多长？ | 判断交期承诺 | 是否靠谱 |
| 设备主要零件哪里采购？ | 判断供应链稳定性 | 是否依赖单一供应商 |

### 第二类：质量与认证

| 问题 | 目的 | 判断依据 |
|---|---|---|
| 你们有哪些认证？ | 判断资质水平 | CE/ISO/其他 |
| 有检测设备吗？ | 判断质量控制 | 能否测试设备性能 |
| 来料检验怎么做？ | 判断质量控制 | 原料是否可控 |
| 出厂检验标准是什么？ | 判断品质保证 | 检验流程是否完整 |
| 有质量报告吗？ | 判断资料能力 | 能否提供检测数据 |

### 第三类：外贸经验

| 问题 | 目的 | 判断依据 |
|---|---|---|
| 做过哪些国家的出口？ | 判断外贸经验 | 和你的目标市场是否重叠 |
| 有出口美国/欧洲的经验吗？ | 判断高端市场能力 | 高端市场要求更严 |
| 有和外贸公司合作过吗？ | 判断外贸渠道 | 有没有外贸体系 |
| 能否提供出口案例？ | 判断真实能力 | 有无真实客户 |
| 包装和唛头怎么做？ | 判断出口细节 | 能否满足出口要求 |

### 第四类：定制能力

| 问题 | 目的 | 判断依据 |
|---|---|---|
| 能按客户需求定制吗？ | 判断定制灵活性 | 非标需求能否满足 |
| 最小起订量是多少？ | 判断定制门槛 | 单台能否定制 |
| 定制交期比标准品多长？ | 判断定制周期 | 能否接受 |
| 定制需要哪些资料？ | 判断配合度 | 能否提供图纸等 |

### 第五类：售后与备件

| 问题 | 目的 | 判断依据 |
|---|---|---|
| 质保期多长？ | 判断售后保障 | 是否合理 |
| 备件包怎么卖？ | 判断备件供应 | 是否需要单独购买 |
| 设备坏了怎么修？ | 判断售后响应 | 远程还是现场 |
| 刀组等耗材哪里买？ | 判断耗材供应 | 是否有稳定渠道 |

### 第六类：付款与合作

| 问题 | 目的 | 判断依据 |
|---|---|---|
| 接受什么付款方式？ | 判断合作灵活性 | TT/LC/DP |
| 预付款比例多少？ | 判断风险控制 | 是否合理 |
| 能签合同吗？ | 判断正规程度 | 有无法律效力 |
| 能开发票吗？ | 判断正规程度 | 发票类型 |

## 工厂合作评估模型

### 六维度评估表

| 维度 | 权重 | 评估项 | 高分标准 |
|---|---|---|---|
| 产品能力 | 25% | 设备质量、参数达标、认证齐全 | 参数达标、出口认证、案例丰富 |
| 交付能力 | 20% | 交期准时、产能充足、生产稳定 | 按期交付、无延期记录 |
| 外贸配合度 | 20% | 沟通顺畅、资料配合、响应及时 | 英文沟通、配合提供资料 |
| 售后能力 | 15% | 质保有效、备件供应、远程支持 | 有备件库存、可远程指导 |
| 资料能力 | 10% | 说明书、合格证、材质证明 | 资料齐全、可提供英文版 |
| 价格稳定性 | 10% | 报价稳定、涨价有理 | 价格透明、涨幅合理 |

### 评分判断

| 总分 | 判断 | 合作建议 |
|---|---|---|
| 80-100 | 优秀工厂 | 优先合作，建立长期关系 |
| 60-79 | 良好工厂 | 可合作，注意风险控制 |
| 40-59 | 一般工厂 | 谨慎合作，小单试用 |
| <40 | 差工厂 | 不合作，换供应商 |

## 河北食品机械产业带

### 核心区域判断

河北食品机械核心制造带以**石家庄为核心**，周边县区外延补充。

| 区域 | 高概率定位 | 代表能力 | 风险提示 |
|---|---|---|---|
| 石家庄高新区 | 核心装备制造与研发区 | 食品机械研发、专利、工程化整线 | 价格偏高 |
| 正定 | 面食设备、肉类设备、成套方案 | 品牌展示、展会、综合成套能力 | 交期可能偏长 |
| 石家庄经济技术开发区 | 切丁、切块、非标自动化 | 肉类、速冻调理、面食辅助设备 | 规模化程度一般 |
| 石家庄南郊工业区/窦妪 | 肉类加工设备集中 | 绞肉、肠类、切割、分割类 | 价格竞争激烈 |
| 行唐 | 面食设备、饺子机流水线 | 饺子机、蒸煮线、自动化面食线 | 出口经验可能不足 |
| 涿州 | 外资或专业化肉食加工设备 | 餐饮端、商超端、小型至中型肉机 | 规模偏小 |

### 河北 vs 其他产业带对比

| 对比地区 | 河北优势 | 河北短板 |
|---|---|---|
| 山东诸城 | 靠近京津，成本相对友好，肉类/面食叙事更自然 | 果蔬清洗、巴氏杀菌、整线行业心智不如诸城强 |
| 河南 | 北方面食和肉机工艺沟通顺，离原材料和华北客户近 | 价格内卷时未必比河南更有优势 |
| 广东 | 机械加工成本、华北供应链、北方食品工艺理解 | 自动化外观、国际化包装、英文营销不一定强于广东 |

### 适合外贸的方向

**优先推荐：**
- 肉类加工设备
- 面食设备
- 切割类设备
- 整线配套方案

**不建议先主推：**
- 没有案例支撑的大型综合生产线
- 参数模糊、竞品极多的通用切菜机

## 工厂资料核验

### 必须验证的资料

| 资料类型 | 验证要点 | 重要性 |
|---|---|---|
| 营业执照 | 公司名称、注册资本、成立时间 | 必须 |
| 产品目录 | 产品种类、参数范围、产能 | 必须 |
| 认证证书 | CE/ISO/其他出口认证 | 必须 |
| 工厂图片 | 车间、设备、库存实拍 | 必须 |
| 案例证明 | 出口案例、客户反馈 | 重要 |
| 联系方式 | 联系人、电话、邮箱、地址 | 必须 |
| 银行流水 | 正规结算账户 | 重要 |

### 核实方法

1. **视频看厂**：要求工厂提供实时视频或视频通话看厂
2. **第三方验证**：通过海关数据或客户评价验证
3. **实地考察**：有条件的情况下亲自或委托看厂
4. **行业打听**：通过同行了解工厂口碑

## 合作风险

### 高风险工厂特征

- 没有工厂（纯贸易商）
- 资料不全或提供假资料
- 沟通时闪烁其词
- 报价极低（低于市场30%+）
- 没有出口经验
- 交期承诺不切实际
- 付款要求极不合理（100%预付或货到付款）

### 风险控制方法

| 风险类型 | 控制方法 |
|---|---|
| 质量风险 | 出货前验货、预留质保金 |
| 交期风险 | 合同列明延期罚款条款 |
| 资料风险 | 合同列明资料交付标准 |
| 付款风险 | 采用TT+尾款或LC付款方式 |
