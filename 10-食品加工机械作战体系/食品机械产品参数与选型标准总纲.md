# 食品机械产品参数与选型标准总纲

## 中文总结

本文档是食品机械选型判断的核心标准文件，包含选型所需的所有参数体系、判断逻辑和报价前确认清单。

**核心定位：** 当用户问"这个客户适合哪台机器""报价前问什么""怎么判断型号"时调用本文档。

**覆盖范围：** 核心参数体系（产能/材质/电压/功率/尺寸/适用原料/成品规格等）、选型判断标准、采购决策模型、报价前必须确认的问题、贸易术语、错误选型风险。

**决策价值：** 避免客户需求和设备参数不匹配导致丢单（参数不够）或退货（买了用不上）。

## English Summary

This document covers food machinery selection standards: core parameter system (capacity/material/voltage/power/size/raw material/finished product specs), selection criteria by customer type, procurement decision model, pre-quotation checklist, common trade terms, and wrong selection risk warnings.

## 一句话结论

选型不对，努力全费——报价前必须确认原料、产能、成品规格、电压、包装六大类问题。

## 核心参数体系

### 1. 产能参数

| 参数 | 说明 | 客户关注点 |
|---|---|---|
| 处理量 | kg/h 或 T/h | 必须满足客户实际产能需求 |
| 成品率 | %（原料到成品转化率） | 影响客户采购成本 |
| 连续工作时间 | 小时/天 | 能否满足客户生产班次 |
| 启动时间 | 从静止到稳定运行 | 客户等待时间 |

### 2. 材质参数

| 参数 | 说明 | 客户关注点 |
|---|---|---|
| 主要材质 | 不锈钢型号（201/304/316） | 食品安全和耐用性 |
| 接触食品部分 | 食品接触材料说明 | 出口认证需要 |
| 刀具材质 | 不锈钢/碳钢/特殊合金 | 切割效果和寿命 |
| 毛刷材质 | 尼龙/钢丝/食品级 | 去皮效果和食品安全 |

### 3. 电气参数

| 参数 | 说明 | 客户关注点 |
|---|---|---|
| 电压 | 220V/380V/415V | 必须匹配客户当地电网 |
| 频率 | 50Hz/60Hz | 必须匹配客户当地电网 |
| 功率 | kW | 耗电量和电费成本 |
| 相数 | 单相/三相 | 大型设备通常三相 |
| 防护等级 | IP65/IP66等 | 潮湿环境需要 |
| 控制方式 | 触摸屏/按钮/PLC | 操作便捷性 |

### 4. 尺寸参数

| 参数 | 说明 | 客户关注点 |
|---|---|---|
| 设备尺寸 | 长×宽×高（m） | 能否放入客户厂房 |
| 重量 | kg | 地基承重要求 |
| 进出料高度 | 进料口/出料口高度 | 与前后端衔接 |
| 占地空间 | 操作空间需求 | 维护和操作需要 |

### 5. 适用原料参数

| 参数 | 说明 | 客户关注点 |
|---|---|---|
| 原料类型 | 蔬菜/肉类/面食/混合 | 设备是否支持 |
| 原料状态 | 鲜品/冷藏/微冻/-18°C冻品 | 温度适配 |
| 原料尺寸 | 最小/最大尺寸范围 | 能否加工客户原料 |
| 原料温度 | 适用的温度范围 | 冻肉和鲜肉需要不同设备 |

### 6. 成品规格参数

| 参数 | 说明 | 客户关注点 |
|---|---|---|
| 切割尺寸 | 厚度/宽度/长度（mm） | 必须满足产品规格 |
| 切丁规格 | 立方体尺寸（mm） | 客户最终产品要求 |
| 成型尺寸 | 重量/形状/一致性 | 标准化需求 |
| 含水率 | 脱水后含水率（%） | 后端加工需要 |

### 7. 自动化程度

| 参数 | 说明 | 客户关注点 |
|---|---|---|
| 自动化等级 | 半自动/全自动 | 减少人工还是提高效率 |
| 上料方式 | 人工/输送带/自动上料 | 减少人工需求 |
| 控制方式 | 手动/PLC/触摸屏+PLC | 操作便捷性 |
| 安全保护 | 急停/防护罩/过载保护 | 生产安全 |

### 8. 定制能力

| 参数 | 说明 | 客户关注点 |
|---|---|---|
| 电压定制 | 可定制多电压 | 全球市场通用 |
| 尺寸定制 | 按客户厂房设计 | 非标需求 |
| 功能定制 | 增加/减少功能模块 | 特定需求 |
| 材质定制 | 指定不锈钢型号 | 出口特定国家 |

## 选型判断标准

### 按客户行业判断

| 客户行业 | 推荐设备类型 | 重点参数 |
|---|---|---|
| 净菜厂 | 清洗+切割+脱水整线 | 产能匹配、规格统一 |
| 中央厨房 | 清洗+切割+输送设备 | 连续运行、卫生标准 |
| 预制菜工厂 | 切割+成型+整线方案 | 规格稳定、损耗低 |
| 肉类加工厂 | 切片/切丁/锯骨/绞肉 | 温度适配、冻鲜两用 |
| 火锅食材 | 冻肉切片机、锯骨机 | -18°C冻肉能力 |
| 餐饮连锁 | 小型切割/成型设备 | 操作简便、标准化 |
| 食品厂 | 整线方案、单机组合 | 前后端衔接、产能平衡 |

### 按原料类型判断

| 原料 | 推荐设备 | 关键判断点 |
|---|---|---|
| 叶菜类（菠菜、韭菜） | 气泡清洗机 | 损伤低、水循环 |
| 根茎类（土豆、胡萝卜） | 清洗+去皮+切割组合 | 去皮损耗率、切割规格 |
| 肉类（冻肉/鲜肉） | 切片/切丁机、绞肉机 | 温度适配、冻鲜两用 |
| 面食（面团） | 包子机/饺子机/推条机 | 成型精度、产能 |

### 按产能需求判断

| 产能需求 | 推荐方案 | 说明 |
|---|---|---|
| <100kg/h | 单机+人工 | 小批量多品种 |
| 100-500kg/h | 半自动单机组合 | 中等批量 |
| 500-2000kg/h | 全自动单机+输送 | 批量标准化 |
| >2000kg/h | 整线方案 | 大规模生产 |

### 按预算判断

| 预算范围 | 推荐方案 | 说明 |
|---|---|---|
| <5万 | 标准单机 | 基础功能 |
| 5-15万 | 高配单机/简单组合 | 自动化程度高 |
| 15-50万 | 整线方案 | 产能和自动化兼顾 |
| >50万 | 定制整线 | 按需求定制 |

## 采购决策模型

### 客户决策链

1. **需求确认**：客户确认要采购设备，识别问题（人工贵/效率低/质量不稳）
2. **方案调研**：客户了解市场可选方案，对比价格和功能
3. **供应商筛选**：客户筛选供应商（资质/案例/售后）
4. **技术确认**：客户确认技术参数（电压/产能/规格）
5. **商务谈判**：价格/交期/付款方式谈判
6. **合同签订**：签订采购合同
7. **交付执行**：设备生产/发货/安装/调试

### 不同决策角色的关注点

| 决策角色 | 关注重点 | 应对策略 |
|---|---|---|
| 老板/总经理 | 投资回报、价格、可靠性 | ROI分析、案例展示 |
| 采购经理 | 价格、付款方式、交期 | 灵活报价、快速响应 |
| 技术工程师 | 参数、规格、技术能力 | 详细技术资料、专业解答 |
| 生产负责人 | 操作简便、维保方便、产能 | 操作培训、易损件供应 |

## 报价前必须确认的问题

### 原料确认（必问）
- [ ] 加工什么原料？（蔬菜/肉类/面食/混合）
- [ ] 原料状态？（鲜品/冷藏/微冻/-18°C冻品）
- [ ] 原料尺寸范围？（最小/最大）
- [ ] 每天处理量？（kg或T）

### 产能确认（必问）
- [ ] 每小时需要多少产能？（kg/h）
- [ ] 每天工作多少小时？
- [ ] 需要连续运行还是间歇运行？
- [ ] 产能是否有季节性波动？

### 成品规格确认（必问）
- [ ] 切片厚度/切丁规格/成型尺寸？
- [ ] 规格是否需要可调？
- [ ] 成品用途是什么？（直接销售/再加工/包装）
- [ ] 对成品率有何要求？

### 电气确认（必问）
- [ ] 当地电压是多少？（220V/380V/415V）
- [ ] 当地频率是多少？（50Hz/60Hz）
- [ ] 是否需要认证（CE/UL）？
- [ ] 控制面板需要什么语言？

### 包装确认（必问）
- [ ] 包装要求？（木箱/托盘/纸箱）
- [ ] 是否需要防水防潮？
- [ ] 最小包装量是多少？
- [ ] 唛头/标签要求？

### 售后确认（必问）
- [ ] 质保期要求？（1年/2年）
- [ ] 是否需要现场安装？
- [ ] 是否需要培训？
- [ ] 备件包要求？（哪些配件）

### 合规确认（必问）
- [ ] 需要什么认证？（CE/SASO/其他）
- [ ] 材质证明是否需要？
- [ ] 食品接触材料说明？
- [ ] 说明书语言要求？

## 常见贸易术语解释

| 术语 | 全称 | 含义 | 适用场景 |
|---|---|---|---|
| FOB | Free On Board | 离岸价，卖方承担货到装港 | 客户指定货代 |
| CIF | Cost Insurance and Freight | 到岸价，含成本+保险+运费 | 卖方安排运输保险 |
| DDP | Delivered Duty Paid | 完税后交货 | 卖方承担所有费用和风险 |
| EXW | Ex Works | 工厂交货 | 客户自提 |
| MOQ | Minimum Order Quantity | 最小起订量 | 定制产品 |
| Lead Time | 交期 | 从下单到发货的时间 | 生产计划 |

## 错误选型风险

### 风险1：电压不匹配
- **问题**：设备380V/50Hz，发到客户那边无法使用
- **后果**：设备报废或需要大规模改造
- **防范**：报价前必须确认电压和频率

### 风险2：产能虚标
- [ ] **问题**：设备标称产能1000kg/h，实际只能到600kg/h
- **后果**：客户无法满足自己的产能需求，要求退货或索赔
- **防范**：提供真实测试数据，保守标注产能

### 风险3：原料不匹配
- **问题**：买了切片机，却发现无法切冻肉，只能切鲜肉
- **后果**：设备闲置，客户要求退货
- **防范**：明确客户原料状态，确认设备温度适配范围

### 风险4：规格无法达成
- **问题**：客户需要8mm切丁，设备实际最小只能切12mm
- **后果**：无法满足客户需求，产生纠纷
- **防范**：报价前确认所有规格参数，提供样品测试

### 风险5：交期延误
- **问题**：客户等设备2个月，错过生产季节
- **后果**：客户流失，损害口碑
- **防范**：如实评估交期，预留缓冲时间
