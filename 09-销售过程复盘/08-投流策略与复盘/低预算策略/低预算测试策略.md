## 最小可行预算计算

**案例依据: Dairy Machinery Meta Ads (印度) [确认]**
- 总预算: ₹8,500 (约USD 100) 测试6个创意
- 单创意预算: ~USD 16-17 per creative
- 日预算分配: USD 20-25/day total across 6 ad sets
- 最小可行: $20/day可保证基本数据积累

**案例依据: Food Processing Machinery B2B [确认]**
- 标准测试预算: $20-50/day per creative set
- EU市场注意: CPM高于亚洲，需要更高预算达到同样数据量
- 建议最低: $30/day for EU, $20/day for Asia

**推断补充**
- 最小预算公式: (目标转化数 × 平均CPC) / 转化率
- 通常需要至少50个转化事件才能得出统计显著性结论

## 测试优先级排序

**案例依据: Dairy Machinery Meta Ads [确认]**
优先级排序:
1. **创意格式**: Reels vs Static Image (5:1配比测试)
2. **内容方向**: 产品特写 vs 机器运转 vs 应用场景
3. **文案风格**: 短文案 vs 长描述
4. **受众细分**: 兴趣受众 vs 行为受众

**推断补充**
- 首轮测试聚焦创意格式和核心卖点
- 次轮测试优化受众定向

## 快速验证方法

**案例依据: Dairy Machinery Meta Ads [确认]**
- 测试窗口: 48小时快速判断
- 验证指标:
  - 视频完播率 >15% (Reels)
  - 点击率 >1% (所有格式)
  - 单次点击成本 <平均订单金额的10%
- 决策规则: 48小时后保留top 2创意，淘汰others

**案例依据: Food Processing Machinery B2B [确认]**
- CTR验证: >1.5% before scaling
- 低于1%CTR需要优化素材或定向
- 3-5天测试周期判断是否值得扩量

## 数据可信度判断

**综合推断**
- 统计显著性: 至少50-100转化事件
- 样本时间: 至少覆盖一个完整转化周期(通常7天)
- 异常值处理: 排除前24小时(冷启动波动)和最后24小时(疲劳期)
- 可信度信号:
  - 数据趋势一致: 连续3天表现稳定
  - 无明显异常: 没有单日CPM暴涨/跌

## 失败止损标准

**案例依据: Dairy Machinery Meta Ads [推断]**
止损标准:
- 48小时后CPM >$15且无任何转化 → 立即停止
- 48小时后点击率 <0.5% → 素材不达标，停止
- 任何时间单次获客成本 >目标LTV的30% → 停止

**Food Processing Machinery B2B [确认]**
- 3-5天测试后CTR仍 <1% → 判定失败
- CPM持续高于市场均值2倍 → 重新审视受众

## 数据复盘模型

**案例依据: Dairy Machinery Meta Ads [确认]**
复盘维度:
```
测试周期: 48小时
总花费: ₹8,500
创意数量: 6个
胜出创意: 1个Reel (机器运转，无声，最小文字)
胜出原因: 完播率高，受众互动积极
失败教训: Static Image CTR明显低于Reels
```

**推断补充**
- 每次测试记录: 日期/预算/创意/指标/结论
- 建立创意素材库，标注历史表现
- 按行业/地区积累基准指标
