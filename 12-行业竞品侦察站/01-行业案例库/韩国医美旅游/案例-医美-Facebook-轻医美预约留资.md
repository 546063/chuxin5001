# 医美 Facebook/Instagram 轻医美预约留资案例

## 一句话总结

美国中型医美诊所60天$15,000投放Facebook/Instagram Ads，480条线索、135单首次消费（$680/单），首单ROAS 6.1x。

## 核心结论

- 医美广告合规限制极严：不能有Before/After、不能有"效果承诺"
- 教程类素材不仅CPL最低（$22），线索质量也最高（48%预约率，35%成交率）
- 2小时内跟进是医美线索的"生死线"：30分钟内跟进预约率62%，超24小时仅8%
- 高LTV模式：首次消费$680，年均消费$2,500+，LTV ROAS可达10x+

## 适用场景

- 适合什么平台：Facebook + Instagram Ads
- 适合什么行业：轻医美（水光针、肉毒素、玻尿酸填充、光子嫩肤）
- 适合什么客户：有实体门店的医美诊所
- 适合什么投流目标：获取到店预约线索 + Instagram品牌认知

## 投流要素拆解

### 平台 / 行业 / 客户画像 / 投流目标 / 广告形式 / 素材类型

- **平台**: Facebook + Instagram Ads
- **行业**: 轻医美
- **客户画像**: 美国25-45岁关注护肤美容的女性，门店周边15英里
- **投流目标**: 获取到店预约线索
- **广告形式**: Lead Ads + Reels + Stories
- **素材类型**: Reels教程 / 客户体验vlog / 医生讲解 / 诊所实拍

### 核心钩子 / 转化路径 / 落地页/表单/私域承接

- **核心钩子**: "3 things I wish I knew before getting my first facial treatment"
- **转化路径**: 广告 → Lead Ads（4字段） → 30分钟内短信/电话跟进 → 预约到店 → 首次消费
- **承接方式**: 前台轮班 + 自动短信 + 电话跟进

### 关键数据指标 / 可复用策略 / 风险点 / 适合复用项目

- **关键指标**: CPL $31 / 预约率 40% / 首单ROAS 6.1x / LTV ROAS 10x+
- **可复用策略**: 软教育素材策略 + Lead Form 4字段设计 + 线索跟进SOP
- **风险点**: 广告账户被封（高风险类别）；Before/After被拒；季节性波动
- **适合复用项目**: 所有本地服务型高客单价业务（牙科、植发等）

## 可复用方法

### 方法1："软教育"素材策略

不直接卖项目 → 先教知识 → 建立信任 → 引导咨询。教程内容占比70%，项目推广占比30%。

### 方法2：Lead Form 设计

医美 Lead Form 最少4字段：姓名 / 电话（必选） / 感兴趣的项目（下拉） / 偏好时间（本周/下周/本月/先了解）。

### 方法3：线索跟进 SOP

- 第1接触（30分钟内）：短信 "Hi [Name], thanks for your interest! When would you like to visit?"
- 第2接触（2小时）：电话跟进
- 第3接触（24小时）：WhatsApp/短信 "We have 2 slots left this week"
- 第4接触（72小时）：电话 + "Limited-time first-visit discount"

## 对我的业务有什么价值

- 对跨境贸易：韩国医美旅游可复用此"软教育→咨询→预约"漏斗
