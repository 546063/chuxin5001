## B2B/机械类素材

### 1. 工厂实拍视频（通用模板）

**适用行业**：所有制造业/机械设备
**适用平台**：Facebook / LinkedIn / TikTok

```
结构:
0-3秒: 工厂大门/车间全景（视觉冲击：大、专业）
3-10秒: 生产线运转特写（机器噪音保留）
10-20秒: 质检环节（测量仪器/测试数据）
20-25秒: 成品展示（包装/出货）
25-30秒: CTA "Get Free Quote" + WhatsApp号码

关键: 不加滤镜，不加BGM（或极低音量），保留环境音
```

### 2. 客户案例图文（Before/After for B2B）

**适用行业**：所有B2B解决方案

```
结构:
标题: "How [Company] in [Country] [具体成果]"
内容:
  - 客户背景（行业/规模/问题）
  - 方案（设备/服务/产线）
  - 数据（产能+X% / 成本-Y% / 回本Z月）
  - 客户引言（直接引语，非编辑）
  - 现场照片（安装前/安装后）

关键: 必须有具体数字，不能只说"效果很好"
```

### 3. 价格对比图

**适用行业**：中国出口制造

```
结构:
标题: "Same Specs, [X]% Less Cost"
内容:
  中国产 vs 欧洲产（或印度产）
  维度: 价格 / 质量 / 保修 / 交期 / 认证
  数据来源: 公开市场报价（标注来源）

关键: 不能贬低竞品，用客观数据对比
```

## 美妆/DTC类素材

### 4. UGC GRWM 视频

**适用行业**：美妆/护肤/个护
**适用平台**：Facebook / Instagram / TikTok

```
结构:
0-3秒: "I tried [X] products before finding this one"（手持产品）
3-10秒: 质地展示（滴落/涂抹，微距镜头）
10-20秒: 上脸/使用过程（手机拍摄，无滤镜）
20-25秒: 使用感受（"I can already feel the difference..."）
25-30秒: "Link in bio / Shop now"

关键: 手机拍摄 > 专业拍摄，"不完美"增加真实感
```

### 5. Before/After 对比（合规版）

**适用行业**：美妆/护肤
**注意**：Facebook 允许，但需加 "Results may vary"

```
结构:
Day 1 vs Day [14/30/60]
同一光线 / 同一角度 / 无滤镜
只展示皮肤状态变化，不展示全脸（减少审核风险）
配文: "Results may vary. Individual experience depends on skin type and consistency."
```

### 6. 质地特写 ASMR

**适用行业**：美妆/护肤/食品

```
结构:
微距镜头 + ASMR音效
精华液滴落的声音 / 面霜涂抹的声音 / 食品切开的声音
无口播，纯视觉+听觉刺激
15-30秒

关键: 光线要好（自然光），背景干净（白色/浅色）
```

## 本地生活/餐饮类素材

### 7. 菜品特写视频

**适用行业**：餐饮/本地生活
**适用平台**：抖音 / TikTok / Instagram Reels

```
结构:
0-3秒: 沸腾的锅/滋滋作响的烤肉/冒热气的菜品（视觉+声音）
3-10秒: 夹菜/蘸料/吃的动作（让人流口水）
10-20秒: 环境/服务/其他顾客反应
20-30秒: "点击左下角，双人餐只要 ¥168"

关键: 声音比画面重要（咕嘟声/滋滋声）
```

### 8. 工地实拍（家装行业）

**适用行业**：家装/装修
**适用平台**：抖音 / 巨量本地推 / Facebook

```
结构:
0-3秒: 拆旧现场/水电管线（"乱"的反而真实）
3-15秒: 工艺展示（"我们的水电走线是这样做的"）
15-25秒: 完工效果（对比）
25-30秒: "免费上门量房，点击预约"

关键: 不需要滤镜，甚至有点乱增加可信度
```

## 医美类素材

### 9. 医生讲解视频

**适用行业**：医美/医疗
**适用平台**：Facebook / Instagram

```
结构:
医生白大褂出镜 + 简洁背景
"Today I'm going to explain [话题]..."
1-2分钟知识科普
结尾: "Book a free consultation"

关键: 医生 > 网红。"专业面孔"比"漂亮面孔"转化高25%
```

### 10. 客户体验vlog

**适用行业**：医美/本地生活

```
结构:
"跟我来做[项目]"第一视角
到店→登记→咨询→操作→离开
不展示操作细节（血腥/针头）
展示: 环境/服务/术后感受

关键: 真实顾客的"跟我来"比"广告"效果好
```

## 教育类素材

### 11. "3分钟学到X" 钩子

**适用行业**：在线教育
**适用平台**：Facebook / YouTube

```
结构:
0-3秒: "Learn [技能] in 3 minutes"
3-150秒: 直接教一个实用技能（让用户"做出来"）
结尾: "完整版在免费试听课里，点击领取"

关键: 试听课必须教到"用户能做出东西"，不是"讲概念"
```

### 12. 学员成功故事

```
结构:
Before: "之前是[职业]，月薪[金额]"
After: "现在是[职业]，月薪[金额]"
Process: "通过[课程]，用了[时间]"
CTA: "So can you. Free trial available."

关键: 真实照片 + 具体数据（非模糊描述）
```

## 敏感行业素材（情趣用品等）

### 13. "隐喻"素材

**适用行业**：成人用品/敏感品类
**适用平台**：TikTok

```
结构:
展示精美包装/礼盒
用户打开后的惊喜表情（不展示产品）
"Treat yourself to something special"
引导到店铺（店铺内才有产品详情）

关键: 广告负责"让人好奇"，店铺负责"让人购买"
```
