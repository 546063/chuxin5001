## B2B/机械类

### LinkedIn Ads 定向公式

```
食品机械/机械设备:
职位: "采购经理" OR "工厂负责人" OR "生产总监" OR "技术总监"
行业: "食品饮料制造" OR "农业机械" OR "工业自动化"
规模: 50-5000人
地域: 目标市场

关键词补充:
技能: "Manufacturing Engineering" / "Lean Manufacturing"
职能: 采购 / 运营 / 工程 / 管理层
```

### Facebook 定向公式

```
B2B设备:
兴趣: "食品制造" OR "农业机械" OR "工业设备" OR "小企业主"
行为: "Facebook Pages admins - Business"
地域: 目标市场（非洲/东南亚/中东）
年龄: 25-55

再营销:
- 网站访客（180天）
- 视频观看（50%+）
- 类似人群（1% LAL，基于已成交客户）
```

## 美妆/DTC类

### Meta ASC 模式

```
ASC 不设置定向，素材即人群筛选器:
- "油性皮肤适用" 自动吸引油性皮肤用户
- "抗老精华" 自动吸引 30+ 用户
- 重点: 15-20条多样化素材，让算法自主匹配

受众建议（非强制）:
- 排除: 已购买用户
- 建议: 现有客户（让算法参考）
```

## 人群定向通用原则

### 1. 冷启动期定向要"宽不要窄"

```
错误: 行业 + 职位 + 兴趣 + 行为 + 年龄（5层过滤后可能只剩几百人）
正确: 行业 + 地域（保留数万人规模）
```

### 2. 再营销期定向要"窄不要宽"

```
错误: 再营销 + 兴趣叠加（可能重叠人群太少）
正确: 单一再营销人群（网站访客 OR 视频观看者）
```

### 3. 决策人一致性原则

```
决策人 = 使用者: 直接定向使用者（如健身房→运动爱好者）
决策人 ≠ 使用者: 定向决策人（如教培→家长，医美→本人，家装→房主）
```
