## 2. 投放目标

**主目标**：获取发电机采购询盘
**次目标**：Google 品牌词保护 + Facebook 品类教育

**为什么 Google + Facebook 组合**：
- Google：捕获主动搜索（"diesel generator 50kVA"）的高意向用户
- Facebook：覆盖不知道具体型号、需要教育的用户（"power backup solutions"）
- 两者互补，总线索量比单平台高 60%

## 6. 投放结构

```
Google Campaign: Search Ads
├── 预算: $94/天
├── 广告组: 按功率分（5-10kVA / 20-50kVA / 100kVA+）
├── 匹配: 精准 + 词组
├── 出价: tCPA $30
└── 否定词: "used" "repair" "parts" "rental"

Facebook Campaign: Lead Ads
├── 预算: $62/天
├── 素材: 工厂视频 × 2 + 价格对比 × 2 + 场景 × 2
├── 表单: 姓名 / WhatsApp / 需要的功率 / 使用场景
└── 出价: Cost Cap $20
```

## 9. 可复用方法

### 方法1：Google + Facebook 互补

```
Google: 捕获搜索意图（高意向，高CPL）
Facebook: 扩大触达范围（中意向，低CPL）
预算比例: Google 60% + Facebook 40%
```

### 方法2：WhatsApp 优先（非洲市场）

```
表单中只收集 WhatsApp 号码（非邮箱）
2小时内自动发送产品目录PDF
销售团队轮班跟进（覆盖非洲时区）
```
