# 家居装修 巨量本地推 家装公司留资案例

## 一句话总结

成都中型家装公司60天¥60,000巨量本地推投放，380条线索、28单成交（客单价¥85,000），ROAS 39.7x。

## 核心结论

- 工地实拍是最优素材（CTR 2.2%，CPL ¥120，到店率32%），越"真实"转化越高
- 装修避坑科普建立信任效果最好，但到店率低于工地实拍
- 30分钟内电话跟进接通率75%，超24小时仅15%
- 装修旺季（3-5月/9-11月）线索量翻倍，淡季应降低预算

## 适用场景

- 适合什么平台：巨量本地推（抖音）
- 适合什么行业：家装公司、装修服务、旧房翻新、局部改造
- 适合什么客户：有实体门店/工地的本地家装公司
- 适合什么投流目标：获取装修咨询线索 → 邀约到店量房

## 投流要素拆解

### 平台 / 行业 / 客户画像 / 投流目标 / 广告形式 / 素材类型

- **平台**: 巨量本地推
- **行业**: 家装
- **客户画像**: 成都28-50岁有装修需求的业主（近期交房小区）
- **投流目标**: 获取装修咨询线索 → 邀约到店量房
- **广告形式**: 标准推广（表单目标）+ 短视频 + 直播
- **素材类型**: 工地实拍 / 业主见证 / 避坑科普 / 设计方案 / 直播

### 核心钩子 / 转化路径 / 落地页/表单/私域承接

- **核心钩子**: "成都这家90㎡老房子翻新，只花了8万5，效果惊艳。0增项，签合同价=最终价"
- **转化路径**: 巨量广告/直播/私信 → 表单留资 → 30分钟内电话 → 微信发送案例 → 邀约到店 → 量房报价 → 签约
- **承接方式**: 电话跟进 + 企业微信 + 到店送建材礼包

### 关键数据指标 / 可复用策略 / 风险点 / 适合复用项目

- **关键指标**: CPL ¥158 / 到店率 25% / 成交率 29.5% / ROAS 39.7x
- **可复用策略**: "真实工地"素材策略 + 线索跟进SOP + 直播获客
- **风险点**: 线索接通率低；到店率低（用户"随便问问"）；季节波动
- **适合复用项目**: 所有本地高客单价服务（装修/家具定制/全屋定制）

## 可复用方法

### 方法1："真实工地"素材策略

工地实拍 > 样板间 > 效果图。越"真实"（有灰尘有工人有工具），转化越高。

### 方法2：线索跟进 SOP

- 第1步（30分钟内）：电话 "您好，我们收到您的装修咨询..."
- 第2步（2小时）：微信添加，发送3个工地实拍案例
- 第3步（24小时）：电话邀约 "本周六我们有免费量房活动"
- 第4步（7天）：微信跟进 "这个月我们有老客户推荐的优惠"
