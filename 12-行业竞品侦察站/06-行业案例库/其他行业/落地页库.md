## B2B/机械类

### 产品详情页模板

```
1. Hero 区域
   - 产品高清图（至少3张，含不同角度）
   - 产品名称 + 一句话价值主张
   - "Get Quote" 按钮（醒目）
   - 信任徽章（CE/ISO/年数/客户数）

2. 技术规格
   - 详细参数表（功率/产能/尺寸/重量）
   - 适用场景
   - 可定制选项

3. 价格信息
   - 价格区间（或 "Starting from $X"）
   - 包含内容（出厂价 / FOB / CIF）
   - 付款方式（T/T, L/C）

4. 客户案例
   - 2-3个具体案例（行业/国家/成果数据）
   - 现场照片

5. 工厂实力
   - 车间照片
   - 年产能
   - 员工数/工程师数

6. 常见问题
   - 交期？保修？安装？售后？

7. 底部CTA
   - "Get Your Free Quote" 表单
   - WhatsApp 按钮
   - 邮箱/电话

关键: 移动端优先（B2B用户 60%+ 用手机浏览）
```

### 白皮书下载页模板

```
1. Hero
   - 白皮书封面（专业设计）
   - 标题 + 副标题
   - "下载人数" 社交证明

2. 内容预览
   - "Inside this guide:" 列出5-7个要点
   - 2-3页预览截图

3. 表单（最少字段）
   - 姓名 / 公司 / 工作邮箱 / 职位
   - 不要加太多字段（降低提交率）

4. 信任要素
   - "We respect your privacy. No spam."
   - 公司信息 / 认证
```

## 美妆/DTC类

### 产品页模板（Shopify）

```
1. Hero
   - 产品图（5-7张，含质地特写/使用场景/包装）
   - 产品名称
   - 价格（含折扣对比 "~~$58~~ $38"）
   - 评分（⭐⭐⭐⭐⭐ 4.8 / 2,340 reviews）

2. 价值主张
   - 3-5个 bullet points（成分/功效/适用肤质）
   - "Before/After"（合规版，加 "Results may vary"）

3. 社交证明
   - 用户评价（含照片的优先）
   - 媒体提及（"As seen in Vogue..."）

4. 成分透明
   - 完整成分表
   - 关键成分说明
   - "Free from"（不含 Parabens/Sulfates 等）

5. 购买保障
   - "30-day money-back guarantee"
   - Free shipping threshold

6. 加购/购买
   - 数量选择
   - "Subscribe & Save" 选项
   - "Add to Cart"（醒目按钮）
```

## 医美类

### 预约咨询页模板

```
1. Hero
   - 诊所外观/接待区高清照片
   - "Book Your Free Consultation"
   - 医生照片 + 资质（MD/Board Certified/Years）

2. 服务项目
   - 卡片式展示（水光针/肉毒素/玻尿酸/光子嫩肤）
   - 每项含: 简介 / 适合人群 / 恢复时间 / 价格区间

3. 信任要素
   - 医生团队介绍（照片+简历）
   - 认证/资质展示
   - 客户评价（真实姓名+照片）

4. 预约表单
   - 姓名 / 电话 / 邮箱
   - 感兴趣的项目（多选）
   - 偏好时间（下拉：本周/下周/本月/先了解）
   - "预约" 按钮

5. FAQ
   - 安全吗？恢复时间？价格？
```

## 家装类

### 免费量房预约页模板

```
1. Hero
   - 完工实景大图（非效果图）
   - "免费上门量房 + 出设计方案"
   - 0增项承诺（醒目）

2. 公司实力
   - 成立年限 / 服务家庭数 / 好评率
   - 工地实拍（水电/泥工/木工/油漆）

3. 案例展示
   - Before/After 对比（3-5个案例）
   - 含: 面积 / 预算 / 工期

4. 服务流程
   1. 预约 → 2. 上门量房 → 3. 出方案 → 4. 报价 → 5. 签约 → 6. 施工

5. 预约表单
   - 姓名 / 电话 / 小区名称
   - 房屋面积 / 装修类型（整装/局装/翻新）
   - 预算范围（下拉）

6. 信任保障
   - 0增项承诺（写进合同）
   - 质保期（水电5年/其他2年）
   - 第三方监理可选
```

```

6. 讲师介绍
   - 照片 / 背景 / 教学经验
```

## 落地页优化 checklist

### 通用（所有行业）

```
□ 首屏 3 秒内传达清楚"我是谁 / 我能帮你什么 / 你该怎么做"
□ CTA 按钮在首屏可见
□ 移动端加载 < 3 秒
□ 表单字段 ≤ 4 个
□ 有至少 2 个信任要素（认证/评价/数据/案例）
□ 页面有明确的单一CTA（不要多个CTA分散注意力）
□ 有转化追踪代码（Pixel / Insight Tag / GA4）
```

### B2B 专属

```
□ 有技术参数/规格表
□ 有客户案例/参考项目
□ 有工厂/公司实力展示
□ 有 WhatsApp/邮箱/电话多种联系方式
```

### DTC 电商专属

```
□ 有用户评价（带照片）
□ 有退换货政策
□ 有运费信息
□ 有安全支付标识
```
