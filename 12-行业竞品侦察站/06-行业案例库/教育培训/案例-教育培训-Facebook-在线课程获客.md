# 教育培训 Facebook 在线课程获客案例

## 一句话总结

中国出海在线教育平台90天$18,000 Facebook投放，3,600条线索、720单付费（$49/单），通过"免费试听→付费转化"漏斗实现首单ROAS 1.96x。

## 核心结论

- 不直接在广告中卖正价课（转化极低），先用"3分钟钩子"引流到免费试听
- 学员故事素材的付费转化率最高（28%），真实照片+薪资变化数据最有效
- 课程大纲展示带来的线索量最少但完课率最高（55%），被内容本身吸引的用户质量最高
- 东南亚定价为欧美的1/3-1/4（$29-$69 vs $99-$199）

## 适用场景

- 适合什么平台：Facebook + Instagram Ads

## 投流要素拆解

### 平台 / 行业 / 客户画像 / 投流目标 / 广告形式 / 素材类型

- **平台**: Facebook + Instagram Ads
- **行业**: 在线教育
- **客户画像**: 东南亚18-30岁有自我提升需求的年轻人
- **投流目标**: 获取免费试听用户 → 转化为付费用户
- **广告形式**: Video Ads + Lead Ads + Messenger
- **素材类型**: 讲师授课片段 / 学员成功故事 / "3分钟学到X"钩子 / 课程大纲

### 核心钩子 / 转化路径 / 落地页/表单/私域承接

- **核心钩子**: "From $300/month to $800/month — Maria's Python journey in 12 weeks"
- **转化路径**: 广告 → Lead Ads（免费领取试听） → 邮件序列+Messenger跟进 → 试听 → 完课 → 限时优惠 → 付费
- **承接方式**: 邮件自动化序列（5封）+ Messenger答疑

### 关键数据指标 / 可复用策略 / 风险点 / 适合复用项目

- **关键指标**: CPL $5 / CPA $25 / 完课率 40% / 付费率 20%
- **可复用策略**: 免费试听课漏斗 + 东南亚定价策略 + Messenger互动促转化
- **风险点**: 完课率低（用户领了课不看）；付费率不足；退款率15%+
- **适合复用项目**: 所有有引流课/体验课的在线教育

## 可复用方法

### 方法1：免费试听课漏斗

广告钩子（3分钟教学片段）→ 免费试听课（30分钟）→ 完课邮件提醒 → 限时优惠。关键：试听课必须"教到用户能做出东西"。

### 方法2：东南亚定价策略

- 欧美正价课：$99-$199
- 东南亚正价课：$29-$69（欧美价格的1/3-1/4）
- 引流课：免费
