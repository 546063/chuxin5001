# Facebook B2B 广告结构

## 一句话总结
> 食品机械Facebook B2B广告分漏斗四层：Awareness→Consideration→Conversion→Retargeting；素材分产品视频、客户案例、技术参数三类。

---

## 广告漏斗结构

### 第一层: Awareness (认知)
**目标**: 曝光品牌，建立认知
**KPI**: CPM, Reach, Brand Awareness

| 广告类型 | 素材 | 文案方向 |
|----------|------|----------|
| 品牌视频 | 工厂全景/产品展示 | "中国专业食品机械厂家" |
| 产品介绍 | 4-6产品展示 | "出口60+国家，ISO认证" |
| 行业内容 | 食品加工趋势 | "如何提升产能30%" |

---

### 第二层: Consideration (考虑)
**目标**: 培育兴趣，引导询盘
**KPI**: Engagement, Video Views, Website Traffic

| 广告类型 | 素材 | 文案方向 |
|----------|------|----------|
| 产品视频 | 机器运行+切割效果 | "观看实际切割效果" |
| 客户案例 | 工厂实拍+客户评价 | "XX食品厂产能提升50%" |
| 技术参数 | 参数图表 | "产能/功率/电压对比" |
| 展会邀请 | Gulfood展会 | "邀请参观工厂" |

---

### 第三层: Conversion (转化)
**目标**: 获取询盘和线索
**KPI**: Lead Form, Message, Website Purchase Intent

| 广告类型 | 素材 | 文案方向 |
|----------|------|----------|
| 促销视频 | 限时折扣 | "本月FOB报价享9折" |
| 样品Offer | 免费样品 | "免费寄送样品" |
| 报价CTA | 报价单下载 | "获取最新报价" |
| WhatsApp CTA | 直接咨询 | "WhatsApp立即询价" |

---

### 第四层: Retargeting (再营销)
**目标**: 找回流失访客，促进成交
**KPI**: ROAS, Conversion Rate

| 广告类型 | 素材 | 文案方向 |
|----------|------|----------|
| 产品提醒 | 浏览过的产品 | "您浏览的锯骨机还有库存" |
| 折扣提醒 | 限时优惠 | "优惠截止本周五" |
| 新品推送 | 新产品 | "新品多功能切菜机上市" |

---

## 受众定向

### 核心受众
| 定向 | 设置 |
|------|------|
| 国家 | 马来西亚/印尼/沙特/阿联酋 |
| 年龄 | 25-55 |
| 兴趣 | Food Processing, Commercial Kitchen, Catering Equipment, Meat Processing |
| 行为 | B2B Purchase Decision Makers, Business Owners |
| 语言 | 英语/阿拉伯语/马来语 |

### 相似受众 (Lookalike)
| 来源 | 规模 |
|------|------|
| 网站访客 | 1-3% |
| 询盘客户 | 1-5% |
| 成交客户 | 1-3% |
| 邮件列表 | 1-5% |

## 素材类型

### 1. 产品视频 (15-30秒)
**内容结构**:
```
0-3秒: 开头吸引 - 切割效果特写
3-10秒: 产品全貌 - 整机运行展示
10-20秒: 参数说明 - 产能/功率字幕
20-30秒: CTA - 询盘呼吁
```

### 2. 客户案例视频 (45-60秒)
**内容结构**:
```
0-5秒: 客户背景 - 公司介绍
5-15秒: 问题痛点 - 产能不足问题
15-35秒: 解决方案 - 设备引入+效果
35-50秒: 客户证言 - 客户评价
50-60秒: CTA - 联系我们
```

### 3. 静态图 (轮播)
```
第1张: 产品图+型号
第2张: 产能参数
第3张: 认证证书
第4张: 客户案例
第5张: CTA+联系方式
```

## 广告文案模板

### 标题模板
```
[产品名] - [产能]kg/h | [功率]kW | [认证]
中国专业[产品名]厂家 | 出口60+国家
[产品名] for [市场] - FOB/CIF报价
```

### 正文模板
```
✈️ 专业[产品名]厂家，出口[国家]

📊 产品参数：
- 产能: [X]-[Y] kg/h
- 功率: [Z] kW
- 电压: 220/380V
- 认证: CE/SASO/SIRIM

🏭 10年专业生产，ISO认证工厂
📦 20GP/40GP可装[X]台

💬 立即询价，获取FOB报价
```
