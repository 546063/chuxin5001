# 食品加工机械 LinkedIn Ads B2B询盘获取案例

## 一句话总结

中国食品机械出口商在LinkedIn投放B2B广告，90天$12,000预算获86条线索，成交7单（客单价$35,000），ROAS 20.4x。

## 核心结论

- LinkedIn适合高客单价B2B机械获客（客单价>$5,000），CPL虽高（$139）但SQL率优秀（36%）
- 缩小定向范围后线索数量减少62%，但SQL率提升80%，最终成交数反而增加
- 工厂实拍视频是CTR最高的素材（1.8%），真实感胜过精致感
- 客户案例必须包含具体数据（"日产从5吨提升到7吨"），不能只说"效果很好"

## 适用场景

- 适合什么平台：LinkedIn Ads
- 适合什么行业：B2B机械制造、食品加工机械、工业设备出口
- 适合什么客户：年出口额$500万+的食品机械制造商
- 适合什么投流目标：B2B询盘获取、高价值线索收集、行业品牌认知

### 转化链路

```
LinkedIn 广告 → (CTR 1.5%) → Lead Gen Form → (提交率 12%) → 86条线索
→ (48h内邮件+电话跟进) → 31条SQL（36%）
→ (寄送样品测试视频 + 技术方案) → 18条深度洽谈
→ (报价 + 在线工厂参观) → 7单成交 → GMV $245,000
```

### 投放结构

```
Campaign 1: 品牌认知（Awareness）- $40/天 → 视频观看，工厂实拍视频 × 3
Campaign 2: 线索获取（Lead Generation）- $70/天 → Lead Gen Form，客户案例 × 3 + 白皮书 × 3
Campaign 3: 线索培育（Retargeting）- $23/天 → 已互动但未留资人群
```

### 线索质量分析

| 阶段 | 线索数/月 | SQL率 |
|------|---------|------|
| 第1-30天（宽定向） | 42条 | 25% |
| 第31-60天（加职位+规模过滤） | 28条 | 38% |
| 第61-90天（加再营销+精准职位） | 16条 | 45% |

**高质量线索特征**：公司邮箱、50-500人公司、表单写明具体需求、主动要求样品测试
**低质量线索特征**：个人邮箱（Gmail/Yahoo）、小公司（<10人）、无明确需求描述、不接电话

## 投流要素拆解

### 平台 / 行业 / 客户画像 / 投流目标 / 广告形式 / 素材类型

- **平台**: LinkedIn Ads

- **客户画像**: 东南亚+中东中型食品加工厂采购决策者
- **投流目标**: B2B询盘（Lead Gen Form）

- **素材类型**: 工厂实拍视频 / 客户案例图文 / 产品对比轮播 / 白皮书

### 核心钩子 / 转化路径 / 落地页/表单/私域承接

- **核心钩子**: "食品包装效率低？越南客户用这台设备日产从5吨提升到7吨"
- **转化路径**: LinkedIn广告 → Lead Gen Form（3字段：姓名+公司+邮箱） → CRM → 销售跟进
- **承接方式**: 邮件+电话48小时内跟进，寄送样品测试视频 + 技术方案

### 关键数据指标 / 可复用策略 / 风险点 / 适合复用项目

- **关键指标**: CPL $139 / CPSQL $387 / SQL率 36% / 成交率 22.6% / ROAS 20.4x
- **可复用策略**: 缩小定向提高SQL率；Lead Gen Form三字段原则；客户案例"3数据1故事"结构
- **风险点**: LinkedIn CPC年增15%；客单价<$5,000时ROI为负；成交周期60-120天
- **适合复用项目**: 高客单价B2B机械出口、工业设备、整厂方案销售

## 可复用方法

### 方法1：Lead Gen Form 三字段原则

最少必要信息 = 姓名 + 公司 + 邮箱/电话。不要加"预算范围""采购时间"，增加填表负担降低提交率。后续信息在电话跟进时问。

### 方法2：客户案例"3数据1故事"结构

- 3个具体数据：产能提升%、成本降低%、回本周期
- 1个故事：客户是谁、遇到了什么问题、怎么解决的

### 方法3：LinkedIn B2B预算分配

品牌认知30% / 线索获取55% / 线索培育15%

### 方法4：线索评分模型

| 信号 | 分值 |
|------|------|
| 使用公司邮箱 | +30 |
| 公司规模50-500人 | +20 |
| 明确写出具体需求 | +25 |
| 目标市场匹配 | +15 |
| 访问过官网产品页 | +10 |

>=60分进入销售跟进。
