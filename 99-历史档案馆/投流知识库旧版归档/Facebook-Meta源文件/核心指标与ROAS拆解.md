# 核心指标与ROAS拆解

## 一句话总结

ROAS 由三个核心变量决定：CPM（竞争环境）、CTR（素材吸引力）、CVR（落地页转化力），优化任一个都能提升 ROAS。

## 核心结论

- 改善 ROAS 的优先级：CVR（最易短期提升）> CTR（素材驱动）> CPM（长期降低）> AOV（产品策略）
- MER 比 ROAS 更可靠（不需要归因），MER 与 ROAS 差异 > 20% 需排查
- 交叉诊断矩阵比单一指标更能发现深层问题

## 适用场景

- ROAS 下滑快速定位
- 数据异常判断真伪
- 多维度交叉诊断
- MER 与 ROAS 差异分析

## 具体内容

### 1. ROAS 拆解公式

```
ROAS = (客单价 × 转化率) ÷ CPC
     = 客单价 ÷ CPA

其中：
CPA = CPC ÷ 转化率
CPC = CPM ÷ (1000 × CTR)
CTR = 点击数 ÷ 展示数
```

**核心推论**：ROAS 受三个变量影响 — CPM（市场竞争）、CTR（素材吸引力）、CVR（落地页转化力）。

**优化优先级**：
1. **CVR**（最容易短期提升）：优化落地页
2. **CTR**（素材驱动）：优化素材
3. **CPM**（长期降低）：扩大受众、优化广告质量分
4. **AOV**（产品策略）：捆绑销售、Upsell

### 2. 全链路指标

**曝光层**：CPM（$8-15 美国）、Impressions、Reach、Frequency（冷受众 1-2，再营销 3-5）

**互动层**：CTR(Link) 1-2%、CPC(Link) $0.50-2.00、Thumb-Stop Rate > 25%、ThruPlay

**转化层**：Landing Page View 80-90%、ATC Rate 5-15%、IC Rate 40-60%、Purchase CVR 1-5%、CPA

**业务层**：ROAS（> 2.0）、AOV、MER（> 3.0）、LTV/CAC（> 3.0）

### 3. ROAS 下滑排查顺序

```
ROAS 下滑
  ├── CPM 上涨 > 30% → 竞争问题（等/换市场/调出价）
  ├── CTR 下降 > 30% → 素材疲劳（换新素材）
  ├── CVR 下降 > 30% → 落地页/产品问题（查网站/价格/库存）
  └── 三个都正常但 ROAS 降 → 客单价降或归因问题
```

### 4. 核心指标联动关系

```
CPM ↑ → CPC ↑（如果 CTR 不变）
CTR ↓ → CPC ↑（如果 CPM 不变）
CVR ↓ → CPA ↑（如果 CPC 不变）
频次 ↑ → CTR ↓ → CPC ↑ → CPA ↑（疲劳链条）
```

**诊断口诀**：
- CPM 异常 → 竞争环境/竞价问题
- CTR 异常 → 素材/受众匹配问题
- CVR 异常 → 落地页/价格/产品问题
- CPA 异常 → 以上三者之一或组合

### 5. CPM × CTR 交叉诊断矩阵

| | CPM 正常 | CPM 偏高 | CPM 极高 |
|--|---------|---------|---------|
| **CTR 高** | 理想状态 | 素材好但竞争贵 | 素材够硬，但成本吃紧 |
| **CTR 正常** | 健康 | 需关注 | 考虑扩受众降 CPM |
| **CTR 低** | 素材问题 | 双重问题 | 立即换素材 + 扩受众 |

### 6. CVR × CPA 交叉诊断矩阵

| | CPA 正常 | CPA 偏高 | CPA 极高 |
|--|---------|---------|---------|
| **CVR 高** | 完美 | 可能 CPM 太高 | 查 CPM |
| **CVR 正常** | 健康 | 轻微优化空间 | 需优化落地页 |
| **CVR 低** | CPA 靠低 CPM 撑着 | 落地页 + 成本问题 | 双重问题 |

### 7. MER 框架深度应用

```
ROAS = Meta 归因收入 ÷ Meta 花费
MER  = 总收入（所有渠道）÷ 总广告花费（所有渠道）
```

| 情况 | MER vs ROAS | 原因 |
|------|------------|------|
| MER > ROAS | 正常 | 有自然流量、SEO、邮件等渠道贡献 |
| MER < ROAS | 异常 | Meta 过度归因 |
| MER ≈ ROAS | 少见 | 几乎 100% 依赖 Facebook 流量 |

**MER 看板**：MER > 4.0 非常高效，3.0-4.0 健康，2.0-3.0 勉强，< 2.0 效率低。

### 8. 数据异常排查速查表

**"变好了"——是不是真的？**
- ROAS 突然翻倍 → 对比 Shopify 实际订单
- CPA 突然降 50% → 检查是否有延迟归因
- CTR 突然飙升 → 检查是否新增 Reels 版位

**"变差了"——是不是真的？**
- ROAS 突然掉 50% → 对比 Shopify + 归因窗口
- 转化数骤降 → 检查 Pixel 是否正常回传
- CPM 暴涨 3 倍 → 查是否进入 Q4/黑五
- 不花费了 → 查审核状态 + 支付方式

### 9. 指标联动模拟计算器

**示例：CTR 从 1% 提升到 1.5%**
- CPM = $15，CVR = 2%，AOV = $50
- CTR = 1%: CPC = $1.50，CPA = $75，ROAS = 0.67x
- CTR = 1.5%: CPC = $1.00，CPA = $50，ROAS = 1.0x
- 结论：CTR 提升 50%，ROAS 从 0.67x → 1.0x（+49%）

**示例：CVR 从 2% 提升到 3%**
- CPA 从 $50 → $33.33，ROAS 从 1.0x → 1.5x（+50%）

### 10. 行业 ROAS 基准

| 行业 | 盈亏平衡 ROAS | 健康 ROAS |
|------|------------|----------|
| 电商（标品）| 2.0-3.0x | 3.5-5.0x |
| 电商（非标品）| 1.5-2.5x | 3.0-4.0x |
| B2B | 1.0-2.0x | 3.0-5.0x |
| App | 1.0-1.5x | 2.0-3.0x |

**盈利线公式**：CPA < 客单价 × 毛利率

## 可复用方法

### 素材效率评分公式

```
素材效率分 = (CTR × 10) + (3s留存率 × 0.3) + (CVR × 20) - (Frequency × 3)
```

- > 50 分 = 优质素材，持续投放
- 30-50 分 = 中等，可以优化
- < 30 分 = 差，关掉或重做
