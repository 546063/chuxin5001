## 核心结论

Meta 广告政策宽泛但执行严格。**90% 的拒登可以通过在提交前自查图片、文案、落地页三个维度避免**。账户封禁的最好解法是预防：养号期从小预算开始，不触发平台风控阈值。

## 1. 广告审核机制

### 审核流程
- 广告提交后，Meta 通常在 **24 小时内** 完成审核
- 热门类别（医疗、金融、政治等）可能需要 **3-5 天**
- 审核状态：`Pending` → `Approved` / `Disapproved` / `Request Re-review`

### 审核范围
Meta 审核广告的三个组成部分：
1. **广告内容**：图片/视频/文案/CTA
2. **目标落地页**：网站内容/功能/购买流程
3. **广告账户历史**：历史违规记录/账户信誉

### 需后台复核
广告审核标准可能随平台政策更新，以下内容**需对照 Meta 官方文档复核**：
- 具体受限产品类别清单（不断变化）
- 当前有效的绝对化用语黑名单
- 视频内容的裸露标准（当前阈值）
- 竞品对比广告的允许边界

## 2. 常见拒登原因与修复

### 2.1 图片违规

| 违规类型 | 触发词/示例 | 修复方法 |
|---------|------------|---------|
| 前后对比图（Before/After） | 减肥/美容/金融类产品 | 删除对比图，用真实用户证言替代 |
| 过度修图/虚假效果 | 瘦身前后的身材对比 | 用真实产品实物图，不用效果图 |
| 裸露或暗示性内容 | 模特姿势过于暴露 | 检查 Facebook Community Standards 当前标准 |
| 误导性数字 | "3天减20斤" | 删除具体效果承诺，改为"产品效果因人而异" |
| 恐慌营销图片 | 疾病/死亡的恐吓图片 | 移除恐慌元素，改用问题+解决方案框架 |

**修复后复审**：通常 24 小时内完成，最长 48 小时。

### 2.2 文案违规

| 违规类型 | 触发词/示例 | 修复方法 |
|---------|------------|---------|
| 绝对化用语 | "100% 有效"/"保证治愈"/"最佳" | 删除绝对化表述，改用"很多人"/"大多数" |
| 误导性声明 | "治愈"/"彻底解决"/"保证" | 改为"有助于"/"可能有助于" |
| 个人信息攻击 | "你太胖了吗"/"你单身吗" | 改为间接表述，如"想要改变身形的人" |
| 不支持的健康声明 | 未经证实的医疗效果 | 对照各国法规，必要时加免责声明 |
| 金融承诺 | "保证赚钱"/"稳赚不赔" | 删除承诺，改用"本产品不保证投资回报" |

**文案自查清单**（提交前必查）：
- [ ] 没有"保证"/"100%"/"绝对"等绝对化用语
- [ ] 效果声明有依据或加免责声明
- [ ] 没有直接攻击竞争对手（用"市场上某些产品"替代）
- [ ] CTA 按钮文字与落地页一致
- [ ] 价格/促销信息与落地页完全一致

### 2.3 落地页违规

| 违规类型 | 问题 | 修复方法 |
|---------|------|---------|
| 落地页与广告不符 | 广告说低价，落地页价格更高 | 确保完全一致 |
| 虚假促销 | 广告说"限免"，落地页无此促销 | 删除或真实呈现促销 |
| 缺失必要资质 | 医疗/金融产品无对应资质 | 补充资质或申请白名单 |
| 落地页无法正常加载 | 死链/页面报错 | 修复网站问题 |
| 隐形费用 | 结账时有隐藏费用 | 透明呈现所有费用 |
| 隐私政策缺失 | 无隐私政策页面 | 添加符合 GDPR/CCPA 的隐私政策 |

### 2.4 受众定向违规

| 违规类型 | 示例 | 修复方法 |
|---------|------|---------|
| 歧视性定向 | 基于种族/宗教/健康状况的定向 | 使用行为/兴趣替代人口属性 |
| 敏感类别定向 | "疾病"相关的精准定向 | 改用宽泛兴趣标签 |

## 3. 受限行业与特殊要求

> ⚠️ 以下行业分类可能随 Meta 政策更新而变化，**需后台/官方文档复核**最新要求。

### 3.1 金融类广告

| 要求 | 说明 |
|------|------|
| 需预审 | 金融产品和服务广告需提前向 Meta 申请 |
| 必要披露 | 必须包含利率、费用、风险提示等法定披露信息 |
| 目标受众限制 | 不能定向 18-24 岁用户（某些产品） |
| 禁止内容 | 不能承诺投资回报/稳赚不赔 |

**申请白名单流程**：Business Settings → Brand Safety → Ad Restrictions → Financial Products

### 3.2 医疗/健康类广告

| 要求 | 说明 |
|------|------|
| 需预审 | 药品/医疗器械/健康服务需申请 |
| 功效声明限制 | 不能声明未经证实的医疗效果 |
| 竞品对比限制 | 不能对比其他药品/医疗器械 |
| 地域限制 | 部分药品有地域投放限制 |

### 3.3 酒精类广告

- 不能定向 18 岁以下用户
- 部分国家禁止酒精广告，需按国家设置地域限制
- 不能暗示饮酒可以提升社会地位/浪漫成功

### 3.4 博彩/彩票类广告

- 需要事先书面授权
- 目标受众不能包含未成年人
- 受限于特定国家/地区

### 3.5 成人内容广告

- 只能投放非艳情性质的成人用品（如性健康/计生用品）
- 禁止露骨的性内容
- 禁止约会/找伴类广告（可投但限制严格）

## 4. 申诉流程

### 4.1 广告被拒后的申诉

**步骤**：
1. 进入 Ads Manager → 广告管理工具 → 查看广告状态
2. 点击被拒广告 → 查看具体违规原因（Meta 会给出违规条款编号）
3. 修改违规内容
4. 点击"申请复审"（Request Re-review）
5. 等待 24-48 小时

**申诉通过率提升技巧**：
- 在申诉说明中清晰解释为何修改（"已删除绝对化用语，已修改为..."）
- 提供支撑材料（如产品认证/功效测试报告）
- 如果是误判，提供证据说明

### 4.2 账户被封后的申诉

**步骤**：
1. 查看邮件/通知中的封禁原因
2. 进入 Business Settings → 账户内容品质 → 查看详细违规记录
3. 整改所有违规内容
4. 提交申诉（Submit Appeal）
5. 附上业务证明文件（营业执照/产品资质/品牌授权等）

**账户解封所需材料**：
- 营业执照或公司注册证明
- 产品合规认证（CE/FDA 等）
- 品牌授权书（如是分销商）
- 网站域名验证
- 详细整改说明信

**预防账户封禁的操作规范**：
- 新账户：Day 1 预算不超过 $20-50，逐步加到 $100
- 账户内不要突然大幅提预算（触发风控）
- 同一产品/落地页不要短时间创建大量广告
- 避免使用被大量拒登过的素材/文案
- 保持网站和广告内容 100% 一致

### 4.3 BM（商务管理平台）被封

BM 被封会影响所有广告账户和数据：
- 提前申请多个 BM 备用
- 主 BM 验证域名 + 添加所有相关人员
- 不要在主 BM 上测试风险广告

## 5. 品牌安全与版位控制

### 5.1 版位排除

部分版位可能损害品牌：
- **Instagram Explore**（可能展示在不适当内容旁）
- **Audience Network**（质量参差，已逐步停用）
- **Facebook Feed 以外的某些自动版位**

排除方式：Ads Manager → 广告组设置 → 版位 → 手动选择或排除特定版位

### 5.2 内容安全工具

- **Brand Safety 规则**：在 Business Settings 中设置版位黑名单
- **分类定向**：可排除某些内容类别
- **Placement 报告**：查看广告实际展示位置，确保符合品牌调性

## 6. 政策变化监控

### 需定期复核的内容
- Meta 官方广告政策页面（facebook.com/business/help）
- Meta 商业帮助中心更新公告
- 受限行业清单（每年更新 2-4 次）
- 新广告格式的功能限制

### 监控频率建议
- 每月检查一次官方政策更新
- 每次上新品/新市场前对照政策清单自检
- 每次被拒登后记录违规原因，防止重复

---

## 7. 快速自查清单

**发布广告前逐项确认**：

- [ ] 图片无前后对比/过度修图/裸露/误导性数字
- [ ] 文案无绝对化用语（100%/保证/彻底）
- [ ] 落地页可正常访问，内容与广告一致
- [ ] 促销信息（价格/折扣）与落地页一致
- [ ] 落地页有隐私政策（符合 GDPR/CCPA）
- [ ] 受众定向无歧视性内容
- [ ] 目标落地页 URL 与验证域名一致
- [ ] 产品符合广告可推广类别（无受限品类违规）
- [ ] 金融/医疗等产品有对应资质文件

---

## 8. 常见错误

| 错误 | 后果 | 避免方法 |
|------|------|---------|
| 使用 Before/After 对比图 | 拒登 | 用真实证言替代 |
| 文案含"保证"/"100%" | 拒登 | 改用"很多人"/"可能" |
| 落地页价格与广告不符 | 拒登+账户警告 | 上线前交叉核对 |
| 新账户第一天大额投放 | 触发风控封号 | 预算从小到大，逐步增加 |
| 测试敏感品类未申请白名单 | 广告拒登+账户受限 | 先申请白名单再投 |
| 多个广告同时违规 | 账户被封 | 每次只上 1-2 条广告测试 |
