# 生物过滤原理

> 水族箱生态核心

---

## 概述

生物过滤是水族箱生态系统的核心，通过硝化细菌将有毒的氨转化为低毒的硝酸盐。

## 硝化作用

```
氨 (NH3) → 亚硝酸盐 (NO2) → 硝酸盐 (NO3)
   ↑           ↑              ↑
有毒       有毒           低毒（通过换水去除）
```

### 第一阶段：氨氧化

```
NH3 + O2 → NO2 + H+ + H2O
```
- 菌种：亚硝化单胞菌
- 产物：亚硝酸盐（有毒）

### 第二阶段：亚硝酸盐氧化

```
NO2 + O2 → NO3
```
- 菌种：硝化杆菌
- 产物：硝酸盐（低毒）

## 硝化细菌特性

| 特性 | 说明 |
|------|------|
| 生长速度 | 慢（倍增周期 8-24 小时） |
| 附着方式 | 需要固体表面 |
| 氧气需求 | 高（好氧菌） |
| 温度范围 | 20-30°C |
| pH 范围 | 6.5-8.5 |

## 培养方法

1. 提供足够附着面（滤材）
2. 持续供氧（打氧/水流）
3. 提供氨源（[[闯缸鱼与硝化建立]]）
4. 保持水温 25-28°C
5. 避免使用抗生素

## 维护要点

- **不要一次性清洗所有滤材**（破坏硝化系统）
- **清洗用原缸水**（保留硝化细菌）
- **定期检测 NO2/NO3**（监测硝化效率）
- **避免过度清洁**

## 硝化崩溃

### 症状
- NO2/氨氮骤升
- 鱼只浮头
- 水体浑浊

### 原因
- 过度清洗滤材
- 使用抗生素
- 停电导致供氧中断
- 大量换水

### 处理
1. 立即大量换水
2. 停止喂食
3. 加强打氧
4. 添加硝化细菌
