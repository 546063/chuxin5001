# 底滤系统

> 大型缸专业过滤方案

---

## 概述

底滤系统利用缸底空间作为过滤仓，优点：
- 过滤容量极大
- 隐藏式设计
- 维护方便

## 适用场景

- 120 cm 以上大型缸
- 大型鱼混养缸
- 高密度养殖

## 过滤仓配置

| 仓 | 功能 | 滤材 |
|----|------|------|
| 第1仓 | 物理过滤 | 过滤棉 |
| 第2仓 | 生物过滤 | 生化环/细菌屋 |
| 第3仓 | 生物+化学 | 陶瓷环/活性炭 |
| 第4仓 | 出水/加热 | 水泵/加热棒 |

## 水流路径

```
主缸 → 下水管 → 物理仓 → 生物仓 → 化学仓 → 水泵 → 主缸
```

## 维护

- 物理仓：每周清洗
- 生物仓：每月清洗 1/3
- 化学仓：每月更换
