# 鱼缸项目路由

## 中文总结
本路由管理鱼缸相关的水质、设备、换水、喂食、鱼病、日志、自动化和维护SOP。AI遇到鱼缸问题时，应优先进入本路由，再进入具体记录或SOP。

## English Summary
This note routes aquarium-related questions to maintenance logs, water quality notes, equipment records, feeding plans, disease handling notes, and automation workflows. AI should first route to this document when encountering aquarium-related questions.

## 一句话结论
鱼缸问题先分成水质、设备、生物、维护、异常和自动化六类，再进入对应文稿。

---

## 鱼缸问题类型路由

### 1. 水质问题
**场景**：
- 水浑 / 水发白
- 氨氮 / 亚硝酸盐 / 硝酸盐
- PH值异常
- 温度异常
- TDS过高
- 硝化系统崩溃

**路由到**：
- [[鱼缸水质记录]]
- [[鱼缸换水记录]]
- [[鱼缸异常处理SOP]]

### 2. 设备问题
**场景**：
- 过滤器堵塞/效率下降
- 水泵流量异常
- 加热棒失灵
- 灯光故障
- 氧气泵停转
- 定时器设置错误
- 自动喂食器故障

**路由到**：
- [[鱼缸设备清单]]
- [[鱼缸设备维护记录]]
- [[鱼缸自动化方案]]

### 3. 生物问题
**场景**：
- 鱼病 / 细菌感染
- 死鱼 / 异常死亡
- 打架 / 领地冲突
- 拒食 / 食欲下降
- 新鱼入缸暂养
- 虾 / 螺 / 水草状态异常

**路由到**：
- [[鱼缸生物记录]]
- [[鱼病观察记录]]
- [[鱼缸异常处理SOP]]

### 4. 日常维护
**场景**：
- 换水 / 部分换水
- 洗滤棉 / 换滤材
- 喂食计划调整
- 清藻 / 除藻
- 补水 / 水位调整
- 添加硝化菌

**路由到**：
- [[鱼缸维护SOP]]
- [[鱼缸换水记录]]
- [[鱼缸喂食记录]]

### 5. 异常处理
**场景**：
- 水突然变浑
- 鱼浮头 / 缺氧
- 鱼批量死亡
- 设备突然停电
- 水温异常升高/降低
- 过滤器完全停转

**路由到**：
- [[鱼缸异常处理SOP]]
- [[鱼缸日志]]
- [[鱼缸复盘记录]]

### 6. 自动化与AI管家
**场景**：
- 自动喂食提醒设置
- 数据记录自动化
- 传感器数据采集
- 摄像头监控
- 水质表格自动生成
- 自动化脚本维护

**路由到**：
- [[鱼缸AI管家方案]]
- [[鱼缸自动化工作流]]
- [[鱼缸数据记录模板]]
