## 开缸步骤

### Day 1：设备安装

1. 放置鱼缸
2. 安装过滤系统
3. 安装加热棒
4. 安装灯具

### Day 1-2：造景

1. 铺设底床（红沙，3-5 cm）
2. 摆放沉木
3. 种植水草
4. 缓慢注水（避免冲散底床）

### Day 3：启动设备

1. 启动过滤
2. 启动加热（目标 26°C）
3. 启动灯具（6-8 小时/天）
4. 添加除氯剂

### Day 3-21：养水

1. 添加硝化细菌
2. 加入氨源（少量鱼食或[[闯缸鱼与硝化建立]]）
3. 每日检测水质
4. 每周换水 20%

### Day 14-21：硝化完成

检测标准：
- [ ] 氨氮 = 0
- [ ] NO2 = 0
- [ ] NO3 > 0

### Day 21+：入鱼

1. 先入闯缸鱼测试
2. 观察 1 周
3. 逐步入目标鱼

## 水质目标

| 参数 | 目标值 |
|------|--------|
| pH | 6.0-7.0 |
| 硬度 | 软水 |
| 温度 | 24-28°C |
| NO2 | 0 |
| NO3 | <20 |
