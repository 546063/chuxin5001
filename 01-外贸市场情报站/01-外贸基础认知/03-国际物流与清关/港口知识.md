# 港口知识

## 一句话定义

> **港口 = 货物装船/卸船的地方 = 影响运费和交货**

---

## 常用起运港（中国）

| 港口 | 城市 | 特点 |
|------|------|------|
| **上海港** | 上海 | 最大港，航线最多 |
| **深圳港** | 深圳 | 华南主要港 |
| **宁波港** | 宁波 | 长江三角洲大港 |
| **青岛港** | 青岛 | 北方重要港 |
| **天津港** | 天津 | 北方大港 |
| **厦门港** | 厦门 | 东南重要港 |
| **广州港** | 广州 | 华南综合港 |

> [!tip] 食品机械最常用：上海港（华东）、深圳港/广州港（华南）、青岛港（华北）

## 港口费用构成

### 起运港费用（FOB包含）

| 费用 | 说明 |
|------|------|
| 港口费 | 码头装卸费 |
| 拖车费 | 工厂到港口 |
| 报关费 | 出口清关 |
| 仓储费 | 临时存储（如果有） |

### 目的港费用（买家承担）

| 费用 | 说明 |
|------|------|
| 港口费 | 卸货费 |
| 堆存费 | 超期存放费 |
| 清关费 | 清关代理费 |
| 关税 | 海关收取 |
| 送货费 | 从港口到客户工厂 |

## 什么影响港口选择

| 因素 | 说明 |
|------|------|
| 客户位置 | 尽量选靠近客户的港口 |
| 运费 | 不同港口运费不同 |
| 船期 | 大港口船期多 |
| 航线 | 不是每个港口都有直达船 |
| 费用 | 不同港口费用不同 |

---

## ⚠️ 内陆国家怎么交货

> [!warning] 内陆国家注意
> 如果客户在 内陆国家（没有海港）：
> - 走海运到邻国港口
> - 再用卡车运到内陆
> - 例如：沙特利雅得 → 海运到吉达 → 陆运到利雅得

## 实战话术

### 客户问目的港

> "您的工厂在哪个城市？如果在墨西哥城附近，建议走曼萨尼约港，靠近墨西哥中南部。如果在美墨边境，可以考虑走洛杉矶转陆运。"

### 客户指定港口

> "好的，我们按您指定的港口报价。但请确认您的清关代理在那个港口可以操作。有些小港口可能没有直达船，需要中转。"

### 客户问运费为什么差这么多

> "运费差异主要取决于：1）港口远近 2）船公司 3）淡旺季 4）是否有直达船。上海到汉堡有直达船，费用相对稳定；到曼萨尼约可能需要中转，费用会高一些。"
