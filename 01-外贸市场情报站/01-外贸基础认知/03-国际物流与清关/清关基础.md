# 清关基础

## 一句话定义

> **清关 = 把货物合法弄进一个国家 = 海关批准后才能提货**

---

## 清关基本流程

```
货物到港 → 提交单据 → 海关审核 → 缴关税 → 海关放行 → 提货
```

---

## 食品机械场景解释

### 🇨🇳 中国工厂 → 🇸🇦 沙特阿拉伯客户

场景：货物到吉达港，客户要清关

**清关步骤：**
1. 货到吉达港，等待清关
2. 清关行提交文件（发票、装箱单、BL、证书）
3. 海关审核HS Code，计算关税
4. 客户缴关税（大约10-15%）
5. 海关放行
6. 客户提货，送工厂

---

## 清关需要的文件

| 文件 | 谁准备 | 说明 |
|------|--------|------|
| Commercial Invoice（商业发票） | 供应商 | 形式发票，清关用 |
| Packing List（装箱清单） | 供应商 | 货物明细 |
| BL（提单） | 船公司 | 提货凭证 |
| Certificate of Origin（原产地证） | 供应商/商会 | 证明中国原产 |
| HS Code | 供应商提供 | 海关归类参考 |
| 认证证书 | 供应商 | CE/FDA/SASO等 |
| 进口许可证 | 客户 | 部分产品需要 |

## 各地区清关特点

| 地区 | 清关难度 | 特点 |
|------|----------|------|
| 美国 | 中等 | 文件要求严格，查验率高 |
| 欧盟 | 中等 | CE认证是门槛 |
| 中东（沙特） | 较高 | SASO认证必须，清关慢 |
| 东南亚 | 较易 | 速度较快 |
| 拉美 | 较高 | 文件多，流程慢，关税高 |
| 非洲 | 高 | 腐败风险，清关慢 |
| 澳洲 | 高 | 世界上最严 |

## ⚠️ 食品机械清关的特殊问题

### 1. 认证问题

> 食品机械如果没有对应认证，清关会被卡：
> - 欧洲：无CE不行
> - 美国：无FDA不行
> - 沙特：无SASO不行

### 2. 木箱问题

> 木箱没有熏蒸/无IPPC标识 → 被扣或退运

### 3. HS Code归类错误

> 海关认为归类不对 → 重新审核 → 延误

### 4. 关税争议

> 海关认为价格偏低 → 要求重估 → 补税

---

## 关税怎么计算

```
关税 = 货值 × 关税率

例如：
- 货值：$10,000
- 关税：10%
- 关税金额：$1,000

最终客户要付：$10,000 + $1,000 = $11,000（加清关费）
```

## 实战话术

### 客户问清关要多久

> "正常清关3-5个工作日，但沙特比较慢，大约需要5-10天。如果遇到查验，可能要2-3周。建议您提前准备好所有文件，配合清关代理。"

### 客户问关税多少

> "关税取决于HS Code归类。食品机械的关税因国家而异：沙特约5-15%，欧盟约2-5%，美国约2-7%。具体金额需要您的清关代理确认。"

### 客户清关被扣怎么办

> "如果是文件问题，我们配合补件；如果是认证问题，我们可以提供认证文件；如果是关税问题，需要您承担。建议提前准备好所有文件，选择有经验的清关代理。"
