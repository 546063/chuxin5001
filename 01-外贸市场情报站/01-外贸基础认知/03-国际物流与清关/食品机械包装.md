# 食品机械包装

## 一句话定义

> **食品机械包装 = 让设备安全漂洋过海到达客户手里 = 必须结实+防潮+防锈**

---

## 食品机械为什么需要特殊包装

| 特点 | 影响 |
|------|------|
| 重 | 可能几吨重，需要加固 |
| 精密 | 怕震动，怕碰撞 |
| 金属 | 容易生锈 |
| 食品相关 | 要防潮、防尘 |
| 外贸时间长 | 海运30-45天，要防海运潮湿 |

---

## 包装类型

### 1. 木箱（最常用）

| 类型 | 适用 | 特点 |
|------|------|------|
| 熏蒸木箱 | 几乎所有机械 | 结实，通用，但需要熏蒸 |
| 免熏蒸胶合板箱 | 避免熏蒸麻烦 | 不需要熏蒸，但贵 |
| 框架箱 | 超大型设备 | 四周框架，露空部分 |

### 2. 纸箱

| 类型 | 适用 | 特点 |
|------|------|------|
| 普通纸箱 | 配件、耗材、小件 | 轻便，但强度低 |
| 加重纸箱 | 较重小件 | 强度高，但有限 |

### 3. 托盘

| 类型 | 适用 | 特点 |
|------|------|------|
| 熏蒸托盘 | 整柜发货，底部固定 | 方便装卸 |
| 免熏蒸托盘 | 需要免熏蒸时 | 成本高 |

---

## 食品机械出口包装标准

### 标准木箱结构

```
┌─────────────────────┐
│                     │
│   木箱顶盖          │
│                     │
├─────────────────────┤
│                     │
│   设备              │
│   (防潮布包裹)       │
│                     │
├─────────────────────┤
│   木箱底座          │
│   (加厚，加强)       │
└─────────────────────┘
```

### 包装步骤

1. **清洁设备** - 除去油污、水分
2. **防潮处理** - 缠绕防潮膜
3. **防锈处理** - 涂防锈油，贴防锈纸
4. **固定** - 用绑带、木块固定在木箱内
5. **装箱** - 封箱，打唛头

## 包装材料清单

| 材料 | 用途 |
|------|------|
| 防潮膜 | 防海运潮湿 |
| 防锈纸/防锈油 | 防金属生锈 |
| 气泡膜 | 防震 |
| 泡沫板 | 填充，保护 |
| 木板 | 箱体 |
| 绑带 | 固定 |
| 唛头 | 标记 |

## 不同设备的包装要求

| 设备类型 | 包装要求 | 说明 |
|---------|---------|------|
| 小型切片机 | 木箱或纸箱 | 轻量，便携 |
| 中型设备（灌装机） | 标准木箱 | 要加固防震 |
| 大型整线 | 框架箱+木箱 | 超大，需要特别处理 |
| 精密部件 | 纸箱+防震填充 | 怕震动 |
| 电子控制面板 | 单独纸箱+防潮 | 防潮防震 |
| 配件/耗材 | 纸箱 | 普通包装 |

---

## ⚠️ 注意事项

### 1. 重量问题

> [!warning] 重要
> 木箱重量会影响运费计算。
> - 木箱本身重量约50-200kg
> - 要在Packing List里标注清楚
> - 防止超重（集装箱限重）

### 2. 重心问题

> 大型食品机械重心高，包装时要确保稳固，防止运输中倾倒。

### 3. 潮湿问题

> 海运经过热带地区，潮湿严重。务必做好防潮措施。

## 实战话术

### 客户问包装方式

> "我们的标准包装是熏蒸木箱，内部有防潮膜、防震填充。设备会用绑带固定，防止运输中移动。您可以放心，包装可以经受长途海运。"

### 客户要求换包装

> "我们可以根据您的要求调整包装，但要注意：纸箱强度有限，不适合大型设备。如果您有特殊包装要求，请提前告诉我们，我们可以提供方案和报价。"

### 客户问包装尺寸

> "您订货的设备型号是XXX，包装后尺寸约为 1.5m x 1.2m x 1.8m，木箱重量约300kg。具体以实际为准，我们会在发货前给您详细的包装尺寸和重量。"

