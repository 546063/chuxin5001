# 海运流程

## 一句话定义

> **海运流程 = 从工厂把货送到目的港客户手里的完整步骤**

---

## 完整海运流程

```
Step 1: 确认订单
    ↓
Step 2: 订舱 (提前5-7天)
    ↓
Step 3: 工厂装货
    ↓
Step 4: 拖车到港口
    ↓
Step 5: 出口报关
    ↓
Step 6: 装船
    ↓
Step 7: 海运
    ↓
Step 8: 目的港清关
    ↓
Step 9: 提货
    ↓
Step 10: 送货到客户
```

---

## Step 1: 确认订单

| 内容 | 说明 |
|------|------|
| 客户付款确认 | 收到定金/全款 |
| PI签字 | 正式订单 |
| 产品生产 | 按交期生产 |
| 验货 | 确保质量 |

---

## Step 2: 订舱 (提前5-7天)

| 内容 | 说明 |
|------|------|
| 联系货代 | 提供货量、目的地 |
| 订舱 | 确认船公司、舱位 |
| 拿到订舱单 | S/O (Shipping Order) |
| 确认截关时间 | 截关前要送货到港口 |

---

## Step 3: 工厂装货

| 内容 | 说明 |
|------|------|
| 包装 | 符合出口标准（木箱熏蒸） |
| 装货 | 工厂到货代仓库或直接到港口 |
| 唛头 | 箱号、目的地标记 |

---

## Step 4: 拖车到港口

| 内容 | 说明 |
|------|------|
| 安排拖车 | 货代安排 |
| 工厂装柜 | 如果是整柜 |
| 送货到港口 | 截关前送到 |

---

## Step 5: 出口报关

| 内容 | 说明 |
|------|------|
| 准备单据 | 发票、装箱单、合同 |
| 报关 | 委托报关行 |
| 查验（可能） | 海关随机抽查 |
| 放行 | 货物可以装船 |

---

## Step 6: 装船

| 内容 | 说明 |
|------|------|
| 装船 | 货物装上船 |
| 拿到BL | 提单 |
| 核对信息 | 确认货物信息正确 |

---

## Step 7: 海运

| 内容 | 说明 |
|------|------|
| 起运 | 船离开中国港口 |
| 中转（可能） | 如果没有直达船 |
| 到港 | 船到目的港 |

**海运时效参考：**

| 航线 | 时效 |
|------|------|
| 中国 → 东南亚 | 5-10天 |
| 中国 → 中东（迪拜） | 12-18天 |
| 中国 → 印度 | 10-15天 |
| 中国 → 欧洲 | 25-35天 |
| 中国 → 美国西海岸 | 15-20天 |
| 中国 → 美国东海岸 | 28-35天 |
| 中国 → 南美 | 30-45天 |
| 中国 → 非洲 | 25-40天 |

---

## Step 8: 目的港清关

| 内容 | 说明 |
|------|------|
| 提交单据 | 发票、装箱单、BL、证书 |
| 缴关税 | 海关核算关税 |
| 放行 | 海关放行 |

---

## Step 9: 提货

| 内容 | 说明 |
|------|------|
| 拿BL | 正本BL或电放 |
| 去港口提货 | 或让货代提货 |
| 检查货物 | 确认数量 |

---

## Step 10: 送货到客户

| 内容 | 说明 |
|------|------|
| 安排送货 | 拖车/配送 |
| 客户签收 | 验收货物 |
| 完成 | 交易完成 |

---

## ⚠️ 新人注意

> [!warning] 重要提示
> 1. **截关时间** - 要在截关前把货送到港口
> 2. **报关时间** - 至少提前2天报关
> 3. **BL核对** - 拿到BL后立刻核对信息
> 4. **清关时间** - 目的港清关需要时间，提前通知客户准备

---

## 实战话术

### 客户问发货流程

> "我们会按以下流程发货：确认订单 → 订舱 → 工厂装货 → 拖车到港口 → 出口报关 → 装船 → 海运 → 目的港清关 → 提货送货。整个流程约需要30-45天，具体看目的地。"

### 客户问怎么追踪货物

> "货物装船后，我们会提供BL号码和船名。您可以通过船公司网站输入BL号追踪货物。我们也会定期更新货物状态给您。"

### 客户问到了没

> "根据船期，您的货预计XX日到达目的港。到港后清关约3-5个工作日。我们会配合您的清关代理尽快完成清关。"

## 时间线示例

> [!tip] 中国 → 墨西哥曼萨尼约港 时间线
> 
> - Day 1: 客户付款确认，订舱
> - Day 3: 货物备齐，送工厂
> - Day 5: 拖车到上海港
> - Day 6: 出口报关
> - Day 7: 装船
> - Day 8: 起航
> - Day 25: 到达墨西哥曼萨尼约港
> - Day 28: 清关完成
> - Day 29: 提货，送客户
> 
> **总共约29天**
