# 整柜 vs 拼柜

## 一句话定义

> **整柜 (FCL) = 整集装箱，你一个人用**
> **拼柜 (LCL) = 集装箱里塞多家货物，大家分摊**

---

## 食品机械场景解释

### 🇨🇳 中国工厂 → 🇲🇽 墨西哥客户

**场景1：整柜**
- 客户订了5台食品机械，货量足够装满一个20GP集装箱
- 单独用一个集装箱，货权清晰，运输快

**场景2：拼柜**
- 客户只订了1台切片机，货量很少
- 和其他供应商的货拼在一个集装箱里
- 费用分摊，但需要等货凑齐

---

## 集装箱类型

| 类型 | 内部尺寸 | 容量 | 适合 |
|------|----------|------|------|
| 20GP (20尺普柜) | 5.9m x 2.35m x 2.4m | 约28立方米 | 小批量机械 |
| 40GP (40尺普柜) | 12m x 2.35m x 2.4m | 约58立方米 | 中批量机械 |
| 40HQ (40尺高柜) | 12m x 2.35m x 2.7m | 约68立方米 | 批量较大机械 |
| 45HQ (45尺高柜) | 13.5m x 2.35m x 2.7m | 约75立方米 | 大批量整线设备 |

---

## 食品机械用柜参考

| 设备类型 | 数量 | 推荐柜型 |
|---------|------|----------|
| 小型切片机/切菜机 | 1-3台 | 20GP或拼柜 |
| 中型设备（真空包装机） | 2-4台 | 20GP |
| 大型设备（整条生产线） | 1套 | 40HQ或多个 |
| 配件/工具箱 | - | 拼柜 |

---

## 整柜 (FCL) 优缺点

| 优点 | 缺点 |
|------|------|
| 货权清晰（自己负责） | 费用高（整柜费用） |
| 运输快（直达目的港） | 需要足够货量 |
| 减少转运风险 | 可能浪费空间 |
| 清关简单 | |
| 减少货物损坏 | |

---

## 拼柜 (LCL) 优缺点

| 优点 | 缺点 |
|------|------|
| 费用分摊（便宜） | 费用计算复杂 |
| 适合小批量 | 需要等货凑齐（约1周） |
| 灵活 | 多个货主，货权不清晰 |
| 小单也能发 | 可能有转运，增加风险 |
| | 清关更复杂（要分开报关） |
| | 货物损坏责任难分 |

---

## ⚠️ 什么情况用拼柜

> [!tip] 实战建议
> - 货量少于5立方米 → 建议拼柜
> - 货值低的产品 → 可以拼柜
> - 配件/耗材 → 可以拼柜
> - 整机/大型设备 → 尽量整柜

---

## 实战话术

### 客户问是整柜还是拼柜

> "您的订单是2台真空包装机，体积约6立方米，用20GP整柜更划算。如果您只有1-2台小型设备，我们会建议拼柜，费用会更低。"

### 客户嫌拼柜慢

> "拼柜需要等货凑齐才能发船，一般需要3-5天。但我们会在货物备齐后第一时间安排发货。如果您急的话，可以考虑空运，虽然贵但更快。"

### 计算用柜

> "您的货物体积约12立方米，建议用20GP整柜。如果是多个小件货物，我们可以安排拼柜，费用约比整柜便宜30-40%。"
