# 木箱与熏蒸

## 一句话定义

> **木箱熏蒸 = 用化学方法处理木质包装，杀虫防病 = 国际贸易强制要求**

没有熏蒸的木箱会让货物在目的港被扣或销毁。

---

## 为什么需要熏蒸

> [!warning] 核心原因
> 各国海关要求：木质包装必须经过处理，防止害虫、病菌跨境传播。
>
> 《ISPM15标准》要求：所有木质包装材料（实木木板、托盘等）必须经过批准的处理方法。

---

## 食品机械场景解释

### 🇨🇳 中国工厂 → 🇧🇷 巴西客户

场景：一台大型食品加工设备用木箱包装

**如果木箱没有熏蒸：**
- 中国出境可能受阻
- 巴西入境会被扣
- 可能被要求退运或销毁
- 客户清不了关

**正确做法：**
- 找专业熏蒸机构处理
- 木箱打上IPPC标识（国际植物保护公约）
- 随货附带熏蒸证明

---

## 什么需要熏蒸

| 需要熏蒸 | 不需要熏蒸 |
|----------|------------|
| 实木木板箱 | 胶合板箱（plywood） |
| 实木托盘 | 密度板（MDF） |
| 实木支架 | 塑料箱 |
| 天然木材包装 | 纸箱 |
| | 压缩木（pressed wood） |

> [!tip] 简单判断：能看出木头纹理的通常是实木，需要熏蒸。胶合板/密度板是复合材料，不需要。

---

## IPPC标识

**什么是IPPC标识：**

```
IPPC
│
└── 国际植物保护公约标志

XX
│
└── 国家代码（如CN=中国）

000
│
└── 熏蒸处理机构代码

YY
│
└── 处理方式（HT=热处理，MB=溴甲烷熏蒸）
```

**示例：**

```
IPPC: CN-999: HT
```

---

## 熏蒸处理方式

| 方式 | 说明 | 优缺点 |
|------|------|--------|
| 热处理 (HT) | 木材加热到56°C以上 | 效果好，推荐 |
| 溴甲烷熏蒸 (MB) | 化学药剂熏蒸 | 效果好，但有污染 |
| 窑干 (KD) |  kiln dry，烘干处理 | 效果好 |
| 防腐处理 | 药水浸泡 | 效果一般 |

## 木箱包装要求

### 食品机械木箱标准

| 要求 | 说明 |
|------|------|
| 木板厚度 | 至少15mm |
| 木板间距 | 不超过25mm |
| 箱体牢固 | 能承受运输压力 |
| 箱底加厚 | 至少25mm |
| 固定 | 用木钉或螺栓固定 |

## ⚠️ 注意事项

### 1. 免熏蒸木箱

> [!tip] 选择
> 不想用熏蒸木箱？
> - 用胶合板箱（plywood）代替
> - 胶合板是加工过的，不含原木，不需要熏蒸
> - 但价格比实木箱贵20-30%

### 2. 熏蒸证明

> 熏蒸后会有证明文件，随货走，清关时出示。

### 3. 熏蒸时间

> 约3-5天完成，需要提前规划。

## 各国对木质包装的要求

| 国家/地区 | 要求 | 说明 |
|-----------|------|------|
| 美国 | 必须ISPM15 | 实木包装要熏蒸 |
| 欧盟 | 必须ISPM15 | 实木包装要熏蒸 |
| 中国 | 必须ISPM15 | 出口要熏蒸 |
| 日本 | 必须ISPM15 | 实木包装要熏蒸 |
| 韩国 | 必须ISPM15 | 实木包装要熏蒸 |
| 澳大利亚 | 非常严格 | 必须熏蒸，证书要随货 |
| 新西兰 | 必须ISPM15 | 实木包装要熏蒸 |
| 中东 | 必须ISPM15 | 越来越严格 |
| 非洲 | 严格 | 部分国家要求 |

## 实战话术

### 客户问包装

> "我们的包装是熏蒸木箱，符合ISPM15国际标准。木箱上会有IPPC标识，随货有熏蒸证明。到了贵国海关清关不会有问题。"

### 客户问能不能换纸箱

> "食品机械比较重，纸箱强度不够，无法保护设备。建议您还是用木箱。如果您希望避免熏蒸的麻烦，可以用胶合板箱，这种材料不需要熏蒸，但价格会高一些。"

### 客户要求免熏蒸

> "我们可以提供免熏蒸的胶合板箱，但价格比实木箱贵约20%。或者如果您自己能处理目的国的清关，实木熏蒸箱也可以。"
