# Packing List 是什么

## 一句话定义

> **Packing List = 装箱清单 = 列出每个箱子里面有什么的文件**

随货走，清关时核对货物数量用。

---

## 食品机械场景解释

### 🇨🇳 中国工厂 → 🇵🇭 菲律宾客户

场景：福建工厂发一批食品机械给马尼拉进口商

**Packing List包含：**
- 箱号（1/5, 2/5...）
- 每箱内产品名称、型号
- 数量
- 净重、毛重
- 箱子的尺寸
- 总净重、总毛重

海关开箱核对时用Packing List对应

## Packing List vs 发票区别

| 项目 | Packing List | 发票(Invoice) |
|------|-------------|---------------|
| 重点 | 货物包装细节 | 货物金额 |
| 是否需要 | 必须 | 必须 |
| 金额 | 不写价格 | 必须写金额 |
| 用途 | 核对货物 | 银行结汇 |
| 海关 | 核对货物数量 | 核查价格 |

---

## 客户为什么需要Packing List

1. **清关** - 海关核对货物用
2. **提货** - 知道自己要提多少箱
3. **核对** - 收到货后核对是否齐全
4. **保险** - 保险公司需要看货物的重量和体积

---

## ⚠️ 注意

> [!warning] 重要提示
> 1. Packing List不写价格（价格只在Invoice里）
> 2. 箱数和实际必须一致
> 3. 毛重/净重要准确（影响海运费计算）
> 4. 木箱要单独标注（熏蒸证明）

## 木箱包装在Packing List里的标注

```
| 4/5 | 肉类切片机 | JRL-200 | 1 | 150 | 170 | 120x100x90 |
|    | ^ Wooden Case, Fumigated | | | | |
```

> [!tip] 实战提示
> 清关时海关会检查木箱是否有熏蒸标签。Packing List里要标注清楚木箱，让客户知道是熏蒸过的合规木箱。
