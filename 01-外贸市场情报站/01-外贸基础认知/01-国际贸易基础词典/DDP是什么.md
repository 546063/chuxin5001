# DDP 是什么

## 一句话定义

> **DDP = Delivered Duty Paid = 完税后交货（含关税）**

供应商负责把货送到客户工厂门口，包含所有费用。

---

## 食品机械场景解释

### 🇨🇳 中国工厂 → 🇸🇦 沙特阿拉伯客户

场景：浙江工厂卖一台肉类加工设备给利雅得进口商

**DDP Riyadh 含义：**
- 工厂负责从中国工厂到利雅得客户工厂的所有费用
- 包含：中国运输 → 出口清关 → 海运 → 目的港清关 → 关税 → 送货上门
- 客户什么都不用做，等着收货

**为什么沙特客户喜欢DDP？**
- 简单：报总价就行
- 省心：不用跟报关行、海关打交道
- 沙特进口流程复杂，找本地清关公司很难

## DDP责任划分

| 环节 | DDP责任 |
|------|--------|
| 国内运输 | ✅ 供应商 |
| 出口清关 | ✅ 供应商 |
| 海运费 | ✅ 供应商 |
| 海运保险 | ✅ 供应商 |
| 目的港卸货 | ✅ 供应商 |
| 进口清关 | ✅ 供应商 |
| 关税/税费 | ✅ 供应商 |
| 送货到客户工厂 | ✅ 供应商 |

---

## ⚠️ 供应商注意

> [!danger] 重大风险提示
> DDP对供应商风险最高：
> 1. **关税不确定** - 不同国家关税不同，可能差很多
> 2. **清关延误** - 目的港清关出问题，供应商要负责
> 3. **沙特/中东特殊** - 需要SASO认证，否则清不了关
> 4. **价格波动大** - 海运费涨、汇率变，供应商承担

---

## 什么客户喜欢DDP

| 客户类型 | 原因 |
|--------|------|
| 中东客户 | 进口流程复杂，希望供应商全包 |
| 新手进口商 | 不懂清关，希望简单 |
| 大客户 | 要总预算，不喜欢意外费用 |
| 沙特/阿联酋/卡塔尔 | 常用DDP |

---

## 食品机械行业DDP注意事项

1. **沙特**：必须要有SASO认证，否则无法清关
2. **食品设备**：需要目的地国的食品卫生认证
3. **大型设备**：需要确认目的港吊装设备是否具备
4. **包装**：木箱必须熏蒸，否则清关会被扣

## 实战话术

### 客户问能否做DDP

> "我们可以做DDP到利雅得，包含海运费、保险、沙特进口清关和关税。请确认您的收货地址在利雅得市区内我们会报一个含税到门价。但需要您提供公司商业执照用于清关。"

### 客户觉得DDP太贵

> "DDP价格包含了所有费用，您不用操心清关、关税、找货代。对您来说是把复杂的事情交给我们处理，省去很多麻烦。如果是FOB价格，您还需要自己找货代、买保险、办理清关，这些都有隐性成本。"
