# HS Code 是什么

## 一句话定义

> **HS Code = Harmonized System Code = 国际统一的商品编码（海关税号）**

每个商品有一个国际通用的"身份证号"，用于确定关税税率。

---

## 食品机械场景解释

### 🇨🇳 中国工厂 → 🇧🇷 巴西客户

场景：广东工厂卖一台肉类切片机给圣保罗进口商

**产品HS Code：**
- 肉类切片机 → **8438.60.00**（食品加工机械）
- 关税：12%（巴西对中国机械）

进口商清关时：
- 报关要用HS Code
- 海关根据Code查税率
- 错了要补税或被扣

## 常见食品机械HS Code

| 设备类型 | HS Code | 说明 |
|---------|---------|------|
| 切片机 | 8438.60.00 | 食品加工机械 |
| 搅拌机 | 8438.10.00 | 糕点加工机械 |
| 包装机 | 8422.30.10 | 包装机械 |
| 灌装机 | 8422.30.20 | 液体灌装机械 |
| 冷藏柜 | 8418.50.00 | 冷藏设备 |
| 食品烤箱 | 8417.80.00 | 烘焙设备 |

---

## HS Code结构（6+4+2位）

```
8438.60.00
│    │   │
│    │   └── 子目（2位）
│    └── 副目（4位）
└── 章节（4位）
```

- **前4位**：国际通用
- **5-6位**：各国略有差异
- **7-8位**：中国海关特定
- **9-10位**：最细化分类

---

## 客户为什么关心HS Code

1. **算关税** - 不同Code关税不同
2. **清关** - 报关必需
3. **限制** - 某些产品需要进口许可
4. **补贴** - 某些农用机械有关税优惠

---

## ⚠️ 常见问题

### 1. HS Code错误

> [!warning] 风险提示
> 用错HS Code可能导致：
> - 被海关查
> - 补缴关税+罚款
> - 货物扣押
> - 延误清关

### 2. 不同国家HS Code不同

> 同一个产品，不同国家可能用不同细分Code。
> 中国用的是**中国海关HS Code**（10位）
> 美国用的是**HTS Code**（10位，但前6位不同）

## 实战话术

### 客户问HS Code

> "这款真空包装机的HS Code是8422.30.10。巴西对这类机械的关税是10%左右。请让您的清关代理确认一下，因为最终税率由当地海关决定。"

### 客户要我们查HS Code

> "我们可以提供产品的HS Code参考，但建议您让清关代理最终确认，因为HS Code的最终归类由目的国海关决定。特别是如果您要进口同类不同型号的产品，可能归类不同。"

### 食品机械HS Code特殊注意

> [!tip] 重要
> 食品机械涉及**食品安全**，有些国家要求：
> - 额外的卫生许可
> - 食品接触材料认证
> - 兽医健康证明
>
> 这些会跟HS Code一起要求，不只是关税问题。
