# T/T 是什么

## 一句话定义

> **T/T = Telegraphic Transfer = 电汇 = 银行转账**

客户通过银行把款汇到我们账户，最常见的外贸付款方式。

---

## 食品机械场景解释

### 🇨🇳 中国工厂 → 🇵🇭 菲律宾客户

场景：卖一台食品切片机，货值$10,000

**T/T付款流程：**
1. 客户下订单，签PI
2. 客户付30%定金 = $3,000（开始生产）
3. 货物做好，等待发货
4. 客户付70%尾款 = $7,000（我们发货）
5. 我们收到全款，发货

---

## 常见T/T付款结构

| 结构 | 说明 | 适用 |
|------|------|------|
| 30%定金 + 70%见BL副本 | 最常见，最安全 | 大多数情况 |
| 30%定金 + 70%发货前 | 常见，客户要先发货 | 信任度高 |
| 100% T/T before shipment | 全款先付 | 新客户首批货 |
| 100% T/T against BL copy | 先付款，后发货 | 客户急货 |
| 50%定金 + 50%到港前 | 大型设备，高总价 | 整线/大机械 |

---

## T/T的优点

| 优点 | 说明 |
|------|------|
| 简单 | 银行转账，容易理解 |
| 快速 | 通常1-5个工作日到账 |
| 费用低 | 手续费大约0.1%，双方分担 |
| 安全 | 有银行作为中间方 |
| 透明 | 双方都能追踪 |

---

## T/T的缺点

| 缺点 | 说明 |
|------|------|
| 汇率风险 | 汇率波动影响实际收入 |
| 手续费 | 国际转账有手续费 |
| 不可撤销 | 付款后难以追回 |
| 信任问题 | 新客户不愿先付 |

---

## ⚠️ 新人注意

### 1. 什么时候让客户付全款

> [!tip] 实战建议
> - 新客户首批货：100% T/T before shipment
> - 试单金额小：可以100%先付
> - 老客户/信任度高：30%定金 + 70%发货前

### 2. 定金收了必须生产吗

> 不一定。但如果客户付了定金，取消订单要退定金或承担损失。收了定金就意味着承诺了货期。

### 3. 尾款没付能发货吗

> **绝对不能。** 货发出去了但没收到全款，风险极大。货物在海上，你无法控制。

---

## 实战话术

### 解释付款方式

> "我们的付款方式是30%定金，70%见BL副本。您签PI后付30%定金，我们开始生产。货物上船后我们把BL副本发您，您付清70%尾款，我们给您正本BL。"

### 客户要求先发货再付款

> "我理解您希望收到货再付款。但对供应商来说，货物上船后我们就失去了控制权，需要收到款项才放心发货。您可以在国内找个信得过的验货代理帮您验货，或者您考虑用L/C付款方式来保护双方。"

### 客户说资金周转问题

> "我们可以理解资金周转的压力。有几个方案：1. 您可以先付30%定金，我们生产，完成后您再付70%发货。2. 如果金额较大，可以考虑分期付款。3. 您也可以申请信用额度或L/C。"

## T/T银行手续费

| 项目 | 说明 |
|------|------|
| 汇款手续费 | 约$20-50/笔 |
| 中转行手续费 | 约$10-30/笔 |
| 收款入账费 | 约$10-20/笔 |
| 总计 | 约$50-100/笔 |

> [!tip] 提示
> 一般T/T手续费由客户承担。报价时可以说明"价格是FOB价，汇款手续费由买方承担"。
