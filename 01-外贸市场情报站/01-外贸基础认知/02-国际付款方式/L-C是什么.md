# L/C 是什么

## 一句话定义

> **L/C = Letter of Credit = 信用证 = 银行担保的付款方式**

银行作为中间人，保证买家付款、卖家发货。适用于大额交易或新客户之间的信任保障。

---

## 食品机械场景解释

### 🇨🇳 中国工厂 → 🇩🇪 德国客户

场景：卖一条食品加工整线，货值$150,000

**L/C付款流程：**
1. 买家（德国）找银行申请开证
2. 德国银行把L/C发给我们的银行（通知行）
3. 我们审核L/C条款，确认可以接受
4. 生产、发货物
5. 准备单据（发票、装箱单、BL等）
6. 把单据交给银行（议付行）
7. 银行审核单据，符合则付款
8. 德国买家去银行赎单，提货

---

## L/C专业术语

| 术语 | 解释 |
|------|------|
| 开证申请人 (Applicant) | 买家，德国进口商 |
| 受益人 (Beneficiary) | 我们，中国供应商 |
| 开证行 (Issuing Bank) | 德国银行 |
| 通知行 (Advising Bank) | 我们的银行（中国） |
| 议付行 (Negotiating Bank) | 议付单据的银行 |
| 交单期 (Presentation Period) | 发货后多少天内交单 |
| 有效期 (Expiry Date) | L/C最后期限 |

---

## 常见L/C类型

| 类型 | 说明 | 风险 |
|------|------|------|
| 即期L/C (Sight L/C) | 单据到了就付款 | 低 |
| 远期L/C (Usance L/C) | 发货后XX天付款 | 中 |
| 可转让L/C (Transferable L/C) | 可转让给供应商 | 中 |
| 背对背L/C (Back-to-Back) | 用L/C抵押再开L/C | 高 |

---

## L/C的优点

| 优点 | 说明 |
|------|------|
| 银行担保 | 有银行信用，风险低 |
| 双方保障 | 买家款到账，卖家发货 |
| 金额大 | 适合大额交易 |
| 进出口商都接受 | 国际通用 |

---

## L/C的缺点

| 缺点 | 说明 |
|------|------|
| 手续复杂 | 需要很多单据 |
| 费用高 | 开证费、议付费约0.5-1%
| 银行审核严格 | 单据不符会被拒 |
| 周期长 | 开证、审核、付款需要时间 |
| 新人难操作 | 需要经验 |

---

## ⚠️ 风险提示

### 单据不符 (Discrepancy)

> [!danger] 最常见风险
> L/C要求单据完全符合。如果：
> - 发票金额不对
> - 晚了1天交单
> - BL上的日期不对
> - 少了某个证书
>
> **银行会拒绝付款。**

### 怎么避免

> [!warning] 防范措施
> 1. 收到L/C后仔细审核每一个条款
> 2. 不明白的地方立即问客户修改L/C
> 3. 严格按L/C要求制作单据
> 4. 发货前再次核对单据
> 5. 找有经验的货代/银行帮忙

---

## 什么情况用L/C

| 情况 | 建议 |
|------|------|
| 大额交易（$50,000+） | 建议用 |
| 新客户大订单 | 用L/C保护双方 |
| 信任度不够 | L/C比T/T安全 |
| 买家要求 | 欧洲/中东常见 |
| 高风险国家 | 非洲/战乱地区建议用 |

## 实战话术

### 客户问能不能做L/C

> "我们可以接受L/C，但需要您承担开证费用。通常L/C会加0.5-1%的开证费。请确认您的银行可以开证给我们。另外，L/C需要严格按条款操作，建议您提前了解L/C的要求。"

### 收到L/C后审核

> "我们收到了L/C，正在审核条款。如果有任何不符的地方，我们会立即通知您修改。请您也确认L/C的金额、有效期、装运期是否正确。"

### 单据被拒怎么办

> "非常抱歉，单据有不符点。请您联系开证行，看看能否接受不符点并付款。如果不行，我们需要修改单据重新提交，这会产生额外费用。"

## L/C vs T/T 对比

| 项目 | T/T | L/C |
|------|-----|-----|
| 手续费 | 低 | 高 |
| 风险 | 中（先发货风险高） | 低（银行担保） |
| 复杂度 | 简单 | 复杂 |
| 适用金额 | 小/中 | 中/大 |
| 付款时间 | 灵活 | 固定 |
| 新人友好度 | 高 | 低 |
