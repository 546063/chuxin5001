# 马来西亚市场分析

## 一句话总结
> 马来西亚是穆斯林市场，SIRIM和Halal认证，清真食品需求大，电压220V/50Hz，连锁快餐发达，餐饮连锁化率高，适合首进东南亚。

## 进口特点

### 核心特点
1. **Halal认证需求** - 清真食品厂必须Halal认证
2. **连锁快餐发达** - McDonald's, KFC, Burger King本地化
3. **SIRIM认证** - 通信和电子产品的强制认证
4. **性价比导向** - 价格敏感但更重品质

### 进口税率

|------|------|
| 普通产品 | 0-10% |
| 食品机械 | 0-5% |
| ASEAN国家 | 0% (中国享受优惠) |
