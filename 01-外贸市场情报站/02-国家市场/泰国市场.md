# 泰国市场分析

## 一句话总结
> 泰国是东南亚旅游餐饮中心，TISI认证，旅游酒店业发达，日系连锁多，价格竞争激烈，电压220V/50Hz，佛学为主不强制Halal。

## 进口特点

### 核心特点
1. **旅游餐饮中心** - 酒店和餐饮业发达
2. **日系连锁多** - 日本企业在泰国制造和餐饮
3. **价格竞争激烈** - 本地厂家和中国厂家竞争
4. **TISI认证** - 非强制但有优势
5. **非穆斯林市场** - Halal认证不是必需

### 进口税率
| 类型 | 税率 |
|------|------|
| 普通产品 | 0-10% |
| 食品机械 | 0-5% |
| ASEAN国家 | 0% (中国享受优惠) |
