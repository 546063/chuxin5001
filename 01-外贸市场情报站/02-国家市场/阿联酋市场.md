## 进口特点

### 核心特点
1. **自由港优势** - 迪拜为自由港，转口关税低
2. **市场开放** - CE/SASO/UL认证均可接受
3. **展会活跃** - Gulfood是全球最大食品展之一
4. **贸易中心** - 覆盖中东、非洲、南亚市场

### 转口贸易
```
中国工厂 → 迪拜 → 沙特/卡塔尔/科威特/非洲
```

### 进口税率
| 类型 | 税率 |
|------|------|
| 普通产品 | 5% |
| 食品机械 | 0-5% |
| 自由港 | 0% |
