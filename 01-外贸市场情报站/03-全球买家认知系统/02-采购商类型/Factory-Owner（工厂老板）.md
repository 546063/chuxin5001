# Factory Owner（工厂老板）

## 一句话定义

> **Factory Owner = 自己开食品厂、自己用设备、自己做决策的人 = 真正的终端买家**

工厂老板是最终掏钱的人，他关心的是设备能不能帮他赚钱。

---

## 工厂老板分类

| 类型 | 说明 | 特点 |
|------|------|------|
| 小型食品厂 | 几台设备，几十个人 | 决策快，老板直接说了算 |
| 中型食品厂 | 几十台设备，100-500人 | 有采购流程，但老板最终决定 |
| 大型食品厂 | 整条生产线，500+人 | 有专业采购团队，老板看投资回报 |
| 中央厨房 | 连锁餐厅自用 | 关注效率、卫生、标准化 |
| 肉制品厂 | 专门做肉 | 设备要求高，认证要全 |

---

## 他真正关心什么

### 1. 设备能不能帮他赚钱

> [!tip] 核心关注
> 工厂老板最关心：**这个设备能不能提高我的效率/降低我的成本？**
>
> - 产量能不能提高
> - 人工能不能省
> - 质量能不能稳定
> - 能用多少年

### 2. 投资回报

> - 设备多少钱
> - 能省多少人工
> - 几年回本
> - 维护成本多少

### 3. 可靠性

> - 会不会经常坏
> - 坏了能不能修
> - 供应商靠不靠谱

---

## 他最怕什么

> [!warning] 工厂老板最怕的 5 件事

| 恐惧 | 说明 |
|------|------|
| 买了设备成摆设 | 不好用，用不起来 |
| 设备天天坏 | 影响生产，影响交货 |
| 找不到人修 | 供应商不理人 |
| 质量不稳定 | 产品出来不一致 |
| 算不过账 | 回本时间太长 |

---

## 他会问什么问题

> - "这个设备一天能处理多少？"
> - "能省多少人工？"
> - "保修多久？"
> - "坏了多久能修好？"
> - "操作复不复杂？"
> - "能不能看设备运行视频？"
> - "有参考客户吗？"

---

## 他最容易被什么打动

| 打动点 | 说明 |
|--------|------|
| **ROI数据** | 具体能省多少钱、几年回本 |
| **现场演示** | 看到设备真的能用 |
| **参考客户** | 同行在用，说好 |
| **售后承诺** | 坏了能快速修 |
| **试用期** | 先试再买 |

---

## 不同规模工厂老板的特点

### 小型工厂老板

| 特点 | 应对 |
|------|------|
| 价格敏感 | 强调性价比 |
| 决策快 | 看对眼就买 |
| 喜欢WhatsApp | 直接沟通 |
| 怕被骗 | 需要信任建立 |

### 中型工厂老板

| 特点 | 应对 |
|------|------|
| 看ROI | 计算给他看 |
| 要看数据 | 技术参数、测试报告 |
| 考虑售后 | 配件、响应时间 |
| 决策稍慢 | 耐心跟进 |

### 大型工厂老板

| 特点 | 应对 |
|------|------|
| 投资回报 | 做完整方案 |
| 合规要求 | CE/FDA/NSF |
| 技术审查 | 工程师参与 |
| 决策周期长 | 6-12个月 |

---

## Facebook/LinkedIn怎么打

### Facebook（小型工厂）

**文案风格：**
- 视频优先（设备运行）
- 价格明确
- WhatsApp直接联系
- 西班牙语/当地语言

**示例：**
> "¡Productor de jamon necesito cortadora automatica! Precio bajo, economia de mano de obra,video de prueba disponible. Contact via WhatsApp."

### LinkedIn（大型工厂）

**文案风格：**
- 专业、ROI导向
- 英文
- 白皮书/案例研究

**示例：**
> "[Factory Name] replacing manual slicers with automated systems. 40% labor cost reduction, 2-year payback. CE certified. Contact for ROI calculation."

---

## 成交关键

> [!success] 成交工厂老板的关键
> 1. **ROI要清楚** - 帮他算能赚多少
> 2. **视频要真实** - 让他看到设备在跑
> 3. **售后要承诺** - 坏了能找到人
> 4. **参考客户** - 同行在用最有说服力
> 5. **试用机会** - 先试再买
