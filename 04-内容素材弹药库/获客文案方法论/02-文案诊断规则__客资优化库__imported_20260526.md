## 一、线索获取阶段优化

### 1. 表单设计优化

**低质量线索的常见原因：表单太简单**

```
最低有效表单（B2B）:
✅ 姓名 / 公司 / 邮箱 / 职位 → 4字段
❌ 只留邮箱 → 无法判断决策者身份

最低有效表单（本地生活）:
✅ 姓名 / 电话 / 感兴趣项目 → 3字段
❌ 只留电话 → 无法匹配需求
```

### 2. 筛选问题设计

在表单中加入 1 个"筛选问题"，自动标记低优先级线索：

```
B2B: "您计划采购的时间？"
  □ 1个月内  □ 1-3个月  □ 3-6个月  □ 先了解

家装: "您的装修预算范围？"
  □ <¥5万  □ ¥5-10万  □ ¥10-20万  □ ¥20万+

医美: "您之前做过类似项目吗？"
  □ 是（做过） □ 否（首次） → 首次需要更多教育
```

### 3. 线索评分模型

| 信号 | B2B分值 | 本地生活分值 | 说明 |
|------|---------|------------|------|
| 公司邮箱（非个人邮箱） | +30 | N/A | B2B专用 |
| 公司规模匹配 | +20 | N/A | |
| 明确需求描述 | +25 | +20 | |
| 预算匹配 | +15 | +15 | |
| 访问过产品页 | +10 | +10 | |
| 下载过资料 | +5 | +5 | |
| 采购时间 < 3个月 | +20 | +15 | |

**阈值**: ≥60分 → 销售立即跟进 / <60分 → 邮件培育

## 二、线索筛选阶段优化

### 4. 线索分级标准

```
A级（高意向）: 评分≥60 + 30分钟内响应
  → 销售经理直接跟进
  → 48小时内安排面谈/量房/咨询

B级（中意向）: 评分40-59
  → 普通销售跟进
  → 72小时内电话/微信联系

C级（低意向）: 评分<40
  → 自动化邮件/短信培育
  → 每月推送 2-3 次有价值内容
```

### 5. 低质量线索识别

**自动标记低优先级的信号**:

| 信号 | 含义 | 动作 |
|------|------|------|
| 个人邮箱（Gmail/Yahoo） | 可能不是决策者 | 降低评分，不优先跟 |
| 电话空号/关机 | 无效线索 | 标记，优化表单验证 |
| 表单乱填（姓名填"test"） | 恶意/测试提交 | 直接排除 |
| 3次未接听 | 跟进困难 | 转短信/微信 |
| 需求模糊（只填"Interested"） | 意向不明确 | 邮件培育，不浪费销售时间 |

## 三、线索跟进阶段优化

### 6. 跟进时效（最重要！）

| 行业 | 最佳跟进时间 | 超时后果 |
|------|------------|---------|
| B2B机械 | 48小时内 | 线索冷却50% |
| 家装 | 2小时内 | 到店率从40%→15% |
| 医美 | 30分钟内 | 预约率从62%→25% |
| 教育 | 24小时内 | 试听率从50%→20% |
| 本地生活 | 即时（自动确认） | 核销率从70%→40% |

### 7. 跟进话术模板

#### B2B 首次电话

```
"Hi [Name], I saw you downloaded our [白皮书/目录].
I'm [Your Name] from [Company]. I'd love to understand your needs better —
are you looking to [specific problem the guide addresses]?
It'll take just 5 minutes."
```

#### 家装 首次电话

```
"您好[姓名]，我们收到您的装修咨询。
想了解下您的房子情况——是哪个小区？面积多大？
我们这周正好在您那附近有工地，可以顺便免费上门量一下。"
```

#### 医美 首次短信

```
"Hi [Name]! Thanks for your interest in [项目].
We have 2 free consultation slots available this week.
Reply with your preferred time and I'll book it for you! 📅"
```

### 8. 多触点跟进 SOP

```
第1触点（即时）: 自动短信/WhatsApp "收到您的咨询，稍后联系您"
第2触点（2小时内）: 电话跟进
第3触点（24小时）: 微信/短信 + 有价值内容（案例/报价）
第4触点（72小时）: 电话 + 限时优惠
第5触点（7天）: 邮件/微信 + 客户见证
第6触点（14天）: "Last chance" + 最终优惠
第7触点（30天）: 转入长期培育（月度内容推送）
```

## 四、转化阶段优化

### 9. 转化率提升方法

#### B2B

```
- 寄送样品/测试视频
- 在线工厂参观
- 提供 ROI 计算器
- 免费技术方案
- 客户参考项目推荐
```

#### 家装

```
- 免费上门量房
- 3套设计方案可选
- 到店送建材礼包
- 老客户优惠推荐
- 限时工艺升级
```

#### 医美

```
- 免费面诊（15分钟）
- 首次体验折扣
- 闺蜜同行优惠
- 术后关怀（提升复购）
```

### 10. 流失线索挽回

```
30天未转化线索:
→ "我们最近有[新项目/新优惠]，想跟您分享"
→ 附上最新客户案例

60天未转化线索:
→ "距离上次联系已经2个月了，您的[需求]有变化吗？"
→ 提供新的价值内容

90天未转化线索:
→ "这是我们本季度的最后一个活动..."
→ 制造紧迫感
```
