# LinkedIn私信开发SOP

> **适用行业**：B2B 食品加工机械出口
> **定位**：轻量框架

---

## 一、搜索客户

### 1.1 搜索关键词

| 关键词 | 说明 |
|--------|------|
| "Food Processing Manager" | 食品工厂生产经理 |
| "Procurement Manager Food" | 食品采购经理 |
| "Factory Owner Food" | 食品工厂老板 |
| "Food Manufacturing Director" | 食品制造总监 |
| "Food Processing Equipment Buyer" | 食品设备采购 |

### 1.2 筛选条件

| 条件 | 建议 |
|------|------|
| 地理位置 | 目标国家/地区 |
| 行业 | Food & Beverages |
| 公司规模 | 50+ 员工 |
| 职位层级 | Manager+ |

---

## 二、加好友策略

### 2.1 连接请求模板

**通用模板**：
```
Hi [Name], I noticed you're in the food processing industry. I work with food factories on production equipment. Would love to connect and share industry insights. Best, [Your Name]
```

**个性化模板**：
```
Hi [Name], saw your post about [topic]. I've been helping food factories optimize production with [equipment]. Would love to connect and discuss. Best, [Your Name]
```

### 2.2 每日限额

| 账户状态 | 每日连接请求 |
|---------|------------|
| 新账户 | 10-15/天 |
| 成熟账户 | 20-30/天 |

---

## 三、私信开发流程

### 3.1 通过后的首条消息

```
Thanks for connecting, [Name]! 

I help food factories improve production efficiency with automated processing equipment. 

If you're ever looking to upgrade your production line or reduce labor costs, I'd be happy to share some insights.

Best, [Your Name]
```

### 3.2 跟进节奏

| 天数 | 动作 |
|------|------|
| Day 0 | 通过好友 + 首条消息 |
| Day 7 | 发一篇相关文章/帖子 |
| Day 14 | 分享客户案例 |
| Day 21 | 询问是否有设备需求 |
| Day 30 | 标记沉默 |

### 3.3 避免的做法

- ❌ 通过后立即推销
- ❌ 群发模板消息
- ❌ 过于频繁的跟进
- ❌ 长篇大论
