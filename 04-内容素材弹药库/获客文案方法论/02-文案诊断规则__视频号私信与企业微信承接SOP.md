# 视频号私信与企业微信承接SOP

> 适用对象：食品加工机械、通用工业机械设备出口企业的销售/运营人员
> 目标：将视频号私信咨询高效转化为企业微信客户，并通过标准化跟进流程促成成交

---

## 一、私信转企业微信路径

### 1.1 路径汇总

```
路径1（主页引导）：
视频号主页 → "添加企业微信"组件 → 客户扫码/点击 → 添加企微

路径2（私信引导）：
客户发私信 → 自动回复引导 → 发送企微二维码 → 客户添加 → 开始跟进

路径3（评论区引导）：
视频评论区置顶 → "添加企业微信获取详细资料" → 客户主动联系 → 添加企微

路径4（直播引导）：
直播间挂载企微名片 → 观众扫码 → 添加企微

路径5（公众号引导）：
公众号文章 → 嵌入企微二维码 → 扫码添加 → 自动欢迎语
```

### 1.2 路径选择建议

| 场景 | 推荐路径 | 原因 |
|------|----------|------|
| 日常运营 | 主页引导+私信引导 | 覆盖最全面 |
| 直播期间 | 直播引导 | 实时转化效率最高 |
| 公众号文章 | 公众号引导 | 深度阅读后转化意愿强 |
| 评论区活跃 | 评论区引导 | 利用热度引流 |

## 三、客户标签管理

### 3.1 标签体系

| 标签类别 | 标签值 | 说明 |
|----------|--------|------|
| 客户来源 | 视频号/朋友圈/公众号/社群/直播/搜索 | 标记线索来源 |
| 行业 | 肉制品/烘焙/饮料/乳品/休闲食品/日化/电子/物流/... | 客户所属行业 |
| 感兴趣产品 | 灌肠机/滚揉机/包装机/灌装机/码垛机/... | 关注的产品线 |
| 地区 | 华南/华东/华北/西南/东南亚/中东/非洲/拉美/欧美 | 客户所在地区 |
| 意向等级 | A(高)/B(中)/C(低) | 购买意向评估 |
| 跟进阶段 | 初次接触/需求确认/报价中/方案中/谈判中/已成交/已流失 | 销售阶段 |
| 预算范围 | <5万/5-20万/20-50万/50-100万/>100万/不确定 | 预算评估 |
| 紧急度 | 紧急(1个月内)/一般(1-3个月)/不急(3个月+)/仅了解 | 采购紧急程度 |
| 决策角色 | 老板/采购经理/技术负责人/其他 | 客户在决策链中的角色 |

### 3.2 标签使用规则

```
添加客户时：
  → 必打标签：来源+行业+意向等级
  → 选打标签：地区+感兴趣产品

沟通后补充：
  → 根据对话内容补充：预算+紧急度+决策角色
  → 根据跟进进度更新：跟进阶段

定期清理：
  → 每月检查一次标签准确性
  → 清理重复/过时标签
  → 更新意向等级和跟进阶段
```

### 3.3 标签筛选应用

| 筛选场景 | 标签组合 | 动作 |
|----------|----------|------|
| 新品推送 | 感兴趣产品=灌肠机 + 跟进阶段≠已成交 | 群发新品信息 |
| 催单 | 意向等级=A + 跟进阶段=报价中 + 紧急度=紧急 | 电话跟进 |
| 激活沉睡客户 | 跟进阶段=初次接触 + 最后联系>30天 | 发送最新案例 |
| 复购提醒 | 已成交 + 行业=肉制品 | 推送配套设备 |
| 展会邀请 | 地区=华南 + 意向等级=A/B | 发送展会邀请函 |

## 四、跟进节奏

### 4.1 标准跟进节奏

| 时间 | 动作 | 内容 | 负责人 |
|------|------|------|--------|
| 0分钟 | 自动欢迎语 | 系统自动发送 | 系统 |
| 30分钟内 | 首次人工回复 | 了解需求+确认信息 | 销售 |
| 当天 | 发送资料包 | 产品目录/案例/选型指南 | 销售 |
| Day 1 | 需求确认 | 确认需求细节 | 销售 |
| Day 3 | 第一次跟进 | "您看了资料后有什么问题吗？" | 销售 |
| Day 7 | 价值输出 | 分享行业知识/新案例 | 销售 |
| Day 14 | 报价/方案 | 根据需求发送报价/方案 | 销售/工程师 |
| Day 21 | 第二次跟进 | "方案/报价看了有什么需要调整的？" | 销售 |
| Day 30 | 持续影响 | 发送最新活动/案例/资讯 | 销售 |
| Day 45+ | 长期培育 | 定期发送有价值内容 | 销售/运营 |

### 4.2 不同意向等级的跟进节奏

| 意向等级 | 跟进频率 | 跟进方式 | 目标 |
|----------|---------|---------|------|
| A(高) | 每2-3天 | 电话+微信 | 快速成交 |
| B(中) | 每周1次 | 微信为主 | 推进决策 |
| C(低) | 每2周1次 | 朋友圈+群发 | 培育信任 |

## 八、适用/不适用场景

### ✅ 适用

- 有企业微信账号
- 有专人/兼职负责私信承接
- 产品有一定客单价（值得人工跟进）
- 目标客户用微信沟通

### ❌ 不适用

- 无企业微信（应先注册配置）
- 无人力及时回复私信
- 产品客单价极低（ROI不划算）
- 纯C端消费品（策略不同）

## 九、风险点

| 风险 | 说明 | 应对措施 |
|------|------|----------|
| 响应不及时 | 客户流失给竞品 | 设置自动回复+SLA |

| 跟进不足 | 客户被竞品抢走 | 设置跟进提醒 |
| CRM不同步 | 信息丢失 | 定期同步+备份 |
| 人员流动 | 客户流失 | 离职继承+客户分配 |
| 数据泄露 | 客户信息外泄 | 权限管理+数据加密 |
