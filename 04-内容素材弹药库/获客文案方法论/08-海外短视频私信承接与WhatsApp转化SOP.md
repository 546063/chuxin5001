# 海外短视频私信承接与WhatsApp转化SOP

> **当前状态**：参考级 SOP，基于行业经验和平台规则，待真实运营后验证。
> **适用场景**：B2B 食品加工机械出海，从短视频自然流量到 WhatsApp 对话的转化。

---

## 一、各平台私信规则

| 平台 | 私信规则 | 限制 |
|------|---------|------|
| TikTok | 互关可无限发消息；非互关每日 1 条 | 新号私信限制更严 |
| YouTube | 无限制（但无原生私信，通过评论/邮件） | 评论公开可见 |
| Instagram | 互关可无限发；非粉丝进请求列表 | 请求列表不通知 |
| Facebook | 互关可无限发；非粉丝进请求列表 | 群组私信需对方同意 |

---

## 二、通用承接流程

```
用户看视频 → 产生兴趣 → 发私信/评论
    ↓
自动/手动回复（5 分钟内）
    ↓
需求确认（产品/产能/地区）
    ↓
发送资料（产品目录/报价单/视频）
    ↓
引导到 WhatsApp（快速沟通）
    ↓
WhatsApp 深度沟通 → 报价 → 跟进
```

---

## 三、各平台话术模板

### 3.1 TikTok 私信话术

**首次回复：**
```
Hi! 👋 Thanks for your interest in our food machines.

To give you the best recommendation:
1️⃣ What product do you want to process?
2️⃣ What capacity do you need (kg/hour)?
3️⃣ Which country are you in?

For faster reply, add us on WhatsApp: [number] 📱
```

**发送资料：**
```
Here's our product catalog: [link]
The [machine name] is our best seller for [application].

Would you like a detailed quote? We can chat on WhatsApp 
for faster communication: [number]
```

**跟进（24 小时未回复）：**
```
Hi! Here's a short video of the machine in action:
[video link]

Still interested? We offer free shipping to [country] 
this month! 📦
```

## 四、WhatsApp 承接流程

### 4.1 WhatsApp Business 设置
| 设置 | 说明 |
|------|------|
| 商业资料 | 公司名称、地址、网站 |
| 自动问候语 | 新对话自动发送 |
| 快速回复 | /catalog, /quote, /shipping |
| 标签 | 新线索/报价中/已成交/流失 |
| 离线消息 | 非工作时间自动回复 |

### 4.2 自动问候语
```
Welcome to [Brand]! 🏭

We manufacture food processing machines. 
Please tell us:
1. What product you want to process
2. Your desired capacity (kg/hour)
3. Your country

We'll send a tailored quote within 2 hours! ⚡
```

### 4.3 快速回复设置
| 快捷词 | 内容 |
|--------|------|
| /catalog | 产品目录链接 |
| /quote | 报价流程说明 |
| /shipping | 运输方式和时间 |
| /cert | 认证证书链接 |
| /warranty | 质保条款 |
| /video | 产品运转视频链接 |

### 4.4 线索标签体系
| 标签 | 含义 | 颜色 |
|------|------|------|
| 🔴 新线索 | 首次联系，未报价 | 红 |
| 🟡 报价中 | 已发报价，等回复 | 黄 |
| 🟢 意向强 | 多次沟通，有采购意向 | 绿 |
| 🔵 已成交 | 下单 | 蓝 |
| ⚪ 流失 | 无回复/价格不合适 | 灰 |

---

## 五、转化指标追踪

| 指标 | 目标 |
|------|------|
| 私信回复率 | > 90%（5 分钟内） |
| 私信 → WhatsApp 转化率 | > 30% |
| WhatsApp 首次回复时间 | < 5 分钟（工作时间内） |
| 询盘 → 报价转化率 | > 70% |
| 报价 → 成交转化率 | > 10% |

---

## 六、注意事项

| 注意 | 说明 |
|------|------|
| 时差 | 标注工作时间和时区 |
| 语言 | 用简单英语，避免复杂句式 |
| 响应速度 | 5 分钟内回复是关键 |
| 不要催单 | B2B 决策周期长，耐心跟进 |
| 记录 | 每个线索记录来源平台/视频/时间 |
| 跟进节奏 | 第 1 天 → 第 3 天 → 第 7 天 → 第 14 天 |
