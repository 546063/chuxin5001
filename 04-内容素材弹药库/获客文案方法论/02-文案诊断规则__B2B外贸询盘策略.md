## 平台选择与组合

**案例依据: Aerogel Insulation LinkedIn (UAE) [确认]**
- 主平台: LinkedIn (B2B工业品首选)
- 平台组合: LinkedIn + Google Ads (据LinkedIn B2B Consultancy UK案例)
- 原因: LinkedIn触达决策者，Google捕捉主动搜索意图

**Food Processing Machinery B2B [推断]**
- 工业机械出口: LinkedIn + Google + Meta组合
- LinkedIn: 品牌建设 + 决策者触达
- Google: 工程采购搜索意图
- Meta: 行业社群 + 再营销

**Dairy Machinery Meta Ads [确认]**
- 平台: Meta (Facebook/Instagram)
- 适合: 中小B2B，预算有限场景
- 印度市场Meta渗透率高

## 询盘导向的素材设计

**案例依据: Aerogel Insulation LinkedIn [确认]**
- Document Ads (PDF datasheet) 是最有效格式
- 带来42%的验证B2B线索
- 内容: 产品技术规格 + 应用案例 + 竞争优势
- 设计要点: 专业排版，数据清晰，联系方式突出

**Dairy Machinery Meta Ads [确认]**
- 最佳创意: Reel展示机器运转，无声音，最小文字
- 视频特点: 真实展示设备运行过程
- 避免: 过度包装的营销内容

**推断补充**
- B2B询盘素材要素:
  1. 明确价值主张 (30字内)
  2. 产品核心参数/规格
  3. 应用行业/场景
  4. 社会证明 (客户Logo/案例)
  5. CTA: 获取报价/下载资料/联系销售

## 多语言与本地化

**案例依据: Aerogel Insulation MENA [确认]**
- 涉及市场: UAE, Oman, Qatar
- 语言: 英文为主 (GCC地区商务英语通用)
- 本地化: 货币用USD/AED，联系方式本地化
- 注意事项: Oman/Qatar需要单独优化Procurement函数

**Food Processing Machinery B2B [推断]**
- EU市场: 英德法西等语言
- 本地化优先级: 德国 > 法国 > 意大利 > 西班牙
- 细节: 公制单位，符合EU标准认证标识

**推断补充**
- 本地化清单:
  1. 语言 (不是翻译，是本地化表达)
  2. 货币/价格格式
  3. 联系方式 (本地电话/办公时间)
  4. 证书/合规标识
  5. 支付方式偏好

## 长周期跟进策略

**案例依据: Aerogel Insulation [确认]**
- 询盘后: 自动触发销售邮件序列
- Day 1: 确认收到询盘，发送产品资料
- Day 3: 技术参数对比表 + 常见问题
- Day 7: 行业应用案例 + 客户评价
- Day 14: 定制方案初稿 + demo邀请
- Day 30: 案例更新 + 优惠截止提醒

**LinkedIn B2B Consultancy UK [确认]**
- 8个月思想领导力 → 培养信任 → 自然转化
- 高客单价需要更长培育周期
- 不追求即时询盘转化

**推断补充**
- 询盘分级跟进:
  - A级询盘 (有具体项目需求): 24小时电话跟进
  - B级询盘 (了解阶段): 邮件序列 + 定期内容推送
  - C级询盘 (收集信息): 自动培育序列

## 询盘质量优化

**案例依据: Aerogel Insulation LinkedIn [确认]**
- 验证机制: 42% of leads是verified B2B leads
- 验证方法: 职位/公司规模/采购需求确认
- 质量优化: Document Ads自然过滤无效询盘

**推断补充**
- 询盘质量指标:
  1. 职位级别: Manager以上占比
  2. 公司信息完整度: 是否有公司名/网站/电话
  3. 需求描述清晰度: 是否明确产品/数量/时间
  4. 主动程度: 是否主动要报价/联系

**优化询盘质量方法:**
- 定向收窄: 职位级别设高
- 素材筛选: 专业内容过滤非目标客户
- 表单设计: 必填项包含公司/职位/具体需求
- 初步筛选: AI客服先过滤无效询盘

```
Campaign: Aerogel Insulation MENA
总预算: USD 200,000
周期: 6个月
平台: LinkedIn
国家: UAE, Oman, Qatar

关键指标:
- 总询盘数: X条
- Verified B2B leads: 42%
- Document Ads转化占比: 42%
- 平均询盘成本: USD X

国家差异:
- UAE: Engineering函数转化好
- Oman/Qatar: Procurement函数转化好

结论:
- Document Ads是最具成本效益的B2B格式
- 按国家+函数分层优化有效
- 27个广告变体帮助精准测试
```

**Food Processing Machinery B2B [推断]**
- 复盘维度:
  - 询盘成本 (CPA)
  - 询盘有效率 (有效/总询盘)
  - 询盘转成交率
  - 客户LTV
- 优化方向: 提高有效询盘比例比增加询盘数量更重要
