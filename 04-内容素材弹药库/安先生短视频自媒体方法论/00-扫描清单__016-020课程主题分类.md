# 016-020 课程扫描清单与主题分类

## 存放位置
`04-内容素材弹药库/安先生短视频自媒体方法论/`

## 课程清单
### 016｜基本功篇：内容数据复盘以及运营规划
- 主题分类：内容数据复盘 / 运营规划
- 精读笔记：[[07-原课逐篇精读__原课016-内容数据复盘以及运营规划]]
- 一句话：内容复盘要先明确目标，再围绕播放量、互动、粉丝模型、数据流失和内容动作做诊断。

### 017｜进阶篇：流量漏斗思维
- 主题分类：流量漏斗 / 精准流量
- 精读笔记：[[07-原课逐篇精读__原课017-流量漏斗思维]]
- 一句话：流量漏斗不是模板，而是从精准流量到扩散流量的注意力路径设计。

### 018｜进阶篇：信息效率思维
- 主题分类：信息效率 / 信息密度
- 精读笔记：[[07-原课逐篇精读__原课018-信息效率思维]]
- 一句话：短视频的信息效率取决于单位时间内给目标用户提供的新信息、路标和可感知推进。

### 019｜进阶篇：政治思维
- 主题分类：政治思维 / 归属感
- 精读笔记：[[07-原课逐篇精读__原课019-政治思维]]
- 一句话：IP 的本质是影响力，内容要让用户产生归属感、自我认同和“我们是一类人”的关系。

### 020｜进阶篇：结构的精细化处理(钩子)
- 主题分类：钩子结构 / 动力结构
- 精读笔记：[[07-原课逐篇精读__原课020-结构精细化处理钩子]]
- 一句话：钩子不是脚本模板，而是抓住注意力的动力成分，可以与骨架、情绪刺激叠加使用。
