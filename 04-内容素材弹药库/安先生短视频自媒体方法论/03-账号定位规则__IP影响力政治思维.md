# IP影响力政治思维

## 一句话总结
IP 的价值来自影响用户决策，而不只是让用户看见你。

## 适用场景
- 个人IP定位
- 专家型账号
- 高客单服务转化

## 判断规则
- 如果内容只能获得注意力，商业价值有限。
- 如果内容能影响用户判断、选择和购买，IP含金量更高。

## 操作步骤
1. 定义你能影响的决策。
2. 展示判断标准。
3. 建立可信身份。
4. 引导用户行动。

## 文案诊断用法
判断文案是否只在表达观点，还是能改变用户决策。
