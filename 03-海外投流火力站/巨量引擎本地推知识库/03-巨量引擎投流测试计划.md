# 巨量引擎投流测试计划

> **当前状态**：参考级测试计划，待真实投放后验证。

---

## 一、测试目标

验证巨量引擎是否适合 B2B 食品机械线索收集。

### 核心指标
| 指标 | 目标 |
|------|------|
| CPL（单条线索成本） | < ¥100 |
| 线索有效率 | > 20% |
| 日均线索数 | > 3 |
| 有效询盘率 | > 10% |

---

## 二、14 天测试计划

### 第 1-3 天：冷启动
| 项目 | 设置 |
|------|------|
| 广告目标 | 线索收集（表单） |
| 日预算 | ¥200 |
| 素材 | 3 条产品运转视频 |
| 定向 | 行业兴趣（食品加工/餐饮设备） |
| 地域 | 全国 |
| 出价 | 系统推荐 |

**观察指标**：展示量、点击率、表单提交数

### 第 4-7 天：素材测试
| 项目 | 设置 |
|------|------|
| 日预算 | ¥300 |
| 素材 | 5 条（产品 2 + 工厂 2 + 案例 1） |
| 定向 | 同上 |
| 新增 | 加入图文广告测试 |

**观察指标**：各素材 CPL 对比

### 第 8-10 天：受众测试
| 项目 | 设置 |
|------|------|
| 日预算 | ¥300 |
| 素材 | 保留 CPL 最低的 2 条 |
| 定向 | 测试 2 组不同定向 |
| A 组 | 行业兴趣 |
| B 组 | 行为标签（搜索过相关设备） |

**观察指标**：A/B 组 CPL 和线索质量

### 第 11-14 天：优化
| 项目 | 设置 |
|------|------|
| 日预算 | ¥400 |
| 素材 | 最优 1-2 条 |
| 定向 | 最优受众 |
| 表单 | 根据前 10 天数据优化字段 |

**观察指标**：稳定 CPL、线索质量、询盘转化

---

## 三、测试结论判定

| 结果 | 判定 | 建议 |
|------|------|------|
| CPL < ¥50，有效率 > 30% | 优秀 | 加大投入 |
| CPL ¥50-100，有效率 20-30% | 可行 | 继续优化 |
| CPL > ¥100 或有效率 < 20% | 不推荐 | 转向其他渠道 |
