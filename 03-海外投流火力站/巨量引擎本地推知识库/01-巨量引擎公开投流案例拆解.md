## 三、食品机械投放建议

### 如果投巨量引擎
| 建议 | 说明 |
|------|------|
| 首选 AD 广告 | 本地推更适合 B2C，AD 适合线索收集 |
| 精准定向 | 行业兴趣 + 职业标签 |
| 表单过滤 | 加入行业/公司规模等过滤问题 |
| 素材方向 | 工厂实力 + 产品运转 + 客户案例 |
| 预算 | ¥200-500/天 测试 |

### 风险提示
- B2B 受众在抖音上较少
- 线索质量可能偏低
- 建议先小预算测试
- 与海外投流并行，不替代
