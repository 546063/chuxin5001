# 巨量引擎投流前准备清单

> **当前状态**：参考级准备方案，待真实投放后验证。

---

## 一、账户准备

| 检查项 | 操作说明 | 状态 |
|--------|---------|------|
| 注册巨量引擎 | oceanengine.com | ☐ |
| 企业验证 | 营业执照 + 法人身份证 | ☐ |
| 行业资质 | 食品机械相关资质 | ☐ |
| 支付方式 | 绑定对公账户 | ☐ |
| 账户充值 | 首次充值 ¥1000+ | ☐ |

---

## 二、抖音号准备

| 检查项 | 操作说明 | 状态 |
|--------|---------|------|
| 企业号认证 | 抖音企业号 | ☐ |
| 主页完善 | 头像、简介、联系方式 | ☐ |
| 首发内容 | 5-10 条视频 | ☐ |
| 主页链接 | 网站/表单链接 | ☐ |
