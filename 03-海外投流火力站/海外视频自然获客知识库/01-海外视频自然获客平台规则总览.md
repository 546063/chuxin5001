# 海外视频自然获客平台规则总览

> **当前状态**：参考级研究，基于各平台公开规则和行业观察，非真实投放数据。
> **适用场景**：B2B 食品加工机械出海，通过短视频自然流量获取线索。

## 二、TikTok 自然流量获客规则

### 核心机制
- **For You Page (FYP)**：算法推荐，不依赖粉丝数
- **初始流量池**：新视频先推 200-500 播放
- **晋升条件**：完播率 > 30%、互动率 > 5%
- **二次爆发**：发布后 3-7 天可能二次推荐

### 获客路径
1. 视频 → 主页 → Bio 链接 → WhatsApp/落地页
2. 视频 → 评论区引导 → 私信 → WhatsApp
3. 视频 → TikTok Shop（如开通）→ 咨询

### 限制
- 新号前 5 条视频流量较低
- 带外部链接可能被限流
- 私信未互关用户有每日上限
- 商业内容需标注 #ad 或 #sponsored

---

## 三、YouTube Shorts 自然流量获客规则

### 核心机制
- **Shorts Feed**：垂直滑动推荐
- **初始流量**：比 TikTok 更依赖频道历史
- **搜索流量**：可被 YouTube 搜索索引（最大优势）
- **长尾效应**：发布后数周仍可获播放

### 获客路径
1. Shorts → 描述区链接 → 网站/WhatsApp
2. Shorts → 频道主页 → About 页面 → 联系方式
3. Shorts → 相关长视频 → 深度了解 → 询盘

### 限制
- 描述区链接点击率较低
- 新频道需要时间积累权重
- Shorts 和长视频的推荐逻辑不同

---

## 四、Instagram Reels 自然流量获客规则

### 核心机制
- **Explore + Reels Feed**：双入口推荐
- **标签依赖**：3-5 个精准标签比泛标签更有效
- **社交放大**：分享和收藏权重高于点赞
- **跨平台复用**：可与 Facebook Reels 同步

### 获客路径
1. Reels → 主页 → Bio 链接 → WhatsApp/网站
2. Reels → DM → 自动回复 → WhatsApp
3. Reels → Story → 链接 Sticker → 落地页

### 限制
- 新号前 30 天流量受限
- 带水印内容会被降权
- 私信非粉丝有隐藏请求限制

---

## 五、Facebook Reels 自然流量获客规则

### 核心机制
- **Reels Feed + Groups**：推荐 + 社群双入口
- **主页权重**：粉丝数影响初始推荐
- **群组扩散**：分享到行业群组可获精准流量
- **WhatsApp 整合**：主页可直接添加 WhatsApp 按钮

### 获客路径
1. Reels → 主页 → WhatsApp 按钮 → 直接对话
2. Reels → 群组分享 → 精准受众 → 私信
3. Reels → 转发 → 二次传播 → 主页

### 限制
- 纯商业内容 reach 较低
- 群组分享需谨慎（避免 spam）
- Reels 算法仍在发展中

## 六、B2B 食品机械适配建议

### 首选平台
| 优先级 | 平台 | 理由 |
|--------|------|------|
| P0 | TikTok | 算法公平、东南亚/中东覆盖广 |
| P1 | YouTube Shorts | 搜索流量、长尾效应强 |
| P1 | Instagram Reels | 与 Facebook 生态打通 |
| P2 | Facebook Reels | WhatsApp 整合好，但 reach 有限 |

### 内容策略
- **TikTok**：原生感、机器运转 ASMR、工厂日常
- **YouTube Shorts**：产品科普、FAQ、操作演示
- **Instagram Reels**：成品效果、客户案例、对比展示
- **Facebook Reels**：与 TikTok/IG 复用，加 WhatsApp 引导

### 跨平台发布节奏
1. 先发布 TikTok（原创平台）
2. 24 小时后去水印发 IG Reels
3. 同步发 Facebook Reels
4. 48 小时后发 YouTube Shorts

## 七、通用规则

| 规则 | 说明 |
|------|------|
| 比例 | 9:16 竖屏（1080x1920） |
| 时长 | 15-30 秒最佳，最长不超过 60 秒 |
| 钩子 | 前 1-3 秒必须出钩子 |
| 音乐 | 使用平台商业音乐库 |
| 标签 | 3-5 个精准标签 |
| 频率 | 日更或每周 5 条 |
| 互动 | 发布后 1 小时内回复所有评论 |
| 引导 | 评论区置顶引导语 |
