# YouTube Shorts自然流量获客深度SOP

> **当前状态**：参考级 SOP，基于平台规则和行业案例，待真实运营后验证。
> **适用场景**：B2B 食品加工机械出海，通过 YouTube Shorts 搜索 + 推荐获取询盘。

---

## 一、频道设置与包装

### 1.1 频道基础设置
| 项目 | 设置建议 |
|------|---------|
| 频道名称 | 品牌名 + 关键词（如：FoodTech Machines） |
| 频道描述 | 200 字，包含行业关键词、联系方式、网站 |
| 频道头像 | 品牌 Logo |
| Banner | 产品展示 + 联系方式 + 网站链接 |
| 自定义链接 | 网站、WhatsApp、社交媒体 |

### 1.2 频道 About 页面模板
```
Welcome to [Brand] - Food Processing Machine Manufacturer

🏭 15+ years in food machinery manufacturing
🌍 Exported to 50+ countries
✅ CE, ISO, SGS certified

Products: Dumpling machines, vegetable processing lines, 
meat processing equipment, packaging machines...

📩 Contact us:
Website: [URL]
WhatsApp: [+XX XXX XXXX]
Email: [email]
```

## 二、内容发布策略

### 2.1 Shorts 规格
| 项目 | 要求 |
|------|------|
| 时长 | 15-60 秒（不超过 60 秒） |
| 比例 | 9:16 竖屏（1080x1920） |
| 格式 | MP4 |
| 标题 | 100 字符以内 |
| 标签 | #Shorts 必加 + 2-3 个行业标签 |

### 2.2 发布频率
| 阶段 | 频率 |
|------|------|
| 冷启动（1-30 天） | 每周 3-5 条 |
| 成长期（31-90 天） | 每周 5-7 条 |
| 成熟期（90+ 天） | 每周 5-7 条 + 长视频 1 条 |

### 2.3 标题模板
```
✅ 产品演示：
"This machine cuts 200kg vegetables per hour! 🥕 #foodmachine #Shorts"

✅ 科普类：
"How to choose the right dumpling machine for your factory 🥟 #foodprocessing #Shorts"

✅ 工厂展示：
"Inside our food machinery factory - 5000sqm of production 🏭 #manufacturing #Shorts"
```

## 三、SEO 优化（YouTube 最大优势）

### 3.1 标题 SEO
- 包含搜索关键词（food processing machine, dumpling maker 等）
- 用数字和动词提升点击率
- 格式："[动作] + [产品] + [结果] #Shorts"

### 3.2 描述区优化

- [Feature 2]
- [Feature 3]

## 四、转化链路

### 4.1 主要转化路径
1. **Shorts → 描述区链接 → 网站 → 询盘**
2. **Shorts → 频道主页 → About → 联系方式**
3. **Shorts → 相关长视频 → 深度了解 → 询盘**
4. **Shorts → YouTube 搜索 → 重复曝光 → 询盘**

### 4.2 描述区链接技巧
- 第一行放 WhatsApp 链接（最高优先级）
- 第二行放网站链接
- 用 emoji 引导点击（📩 →、🏭 Visit）
- 链接用短链接（bit.ly 或自建短链）

## 五、长短视频联动

### 5.1 联动策略
| 短视频内容 | 对应长视频 |
|-----------|-----------|
| 产品运转 15s | 完整操作演示 5-10 分钟 |
| 工厂片段 30s | 工厂全景 tour 10-15 分钟 |
| 客户案例 30s | 客户访谈 5-8 分钟 |
| FAQ 15s | 完整教程 8-12 分钟 |

### 5.2 联动执行
1. 先发 Shorts 测试话题热度
2. 播放量好的话题制作长视频
3. 长视频评论区关联 Shorts
4. Shorts 描述区引导观看长视频

## 六、数据分析与优化

### 6.1 关键指标
| 指标 | 目标 |
|------|------|
| 播放量 | > 1000/条（搜索流量加持） |
| 平均观看时长 | > 50% |
| 订阅转化 | > 1% |
| 描述区点击 | > 0.5% |

### 6.2 YouTube Studio 数据分析
- **流量来源**：Shorts Feed vs 搜索 vs 推荐
- **观众留存**：哪一秒流失最多
- **搜索关键词**：用户通过什么词找到你
- **地理分布**：哪些国家观众最多

## 七、与 TikTok 的差异

| 维度 | TikTok | YouTube Shorts |
|------|--------|---------------|
| 内容风格 | 原生/娱乐 | 信息/教育 |
| 标题 | 短 + 标签 | 含搜索关键词 |
| 描述 | 不重要 | 重要（SEO + 链接） |
| 内容寿命 | 数天 | 数周-数月 |
| 获客方式 | 私信 + 主页 | 描述链接 + 搜索 |
