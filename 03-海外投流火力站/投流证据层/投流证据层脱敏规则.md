# 投流证据层脱敏规则

> **适用范围**：所有涉及客户个人信息的证据层文档。
> **原则**：最小化暴露、可逆性不保证、内部可追溯。

---

## 一、脱敏方法总览

| 字段类型 | 脱敏方法 | 示例 |
|----------|---------|------|
| 姓名 | 替换为 Customer-X | Ahmed Al-Rashid → Customer-A |
| 手机号 | 保留国家码 + 前3位 + **** + 末4位 | +966 50 123 4567 → +966****4567 |
| 邮箱 | 保留首字母 + *** + @域名 | ahmed@example.com → a***@example.com |
| 公司名 | 缩写首字母 + Corp | Riyadh Food Equipment → RFE-Corp |
| 地址 | 保留城市 + 国家 | 123 King Fahd Rd, Riyadh → Riyadh, SA |
| WhatsApp ID | 同手机号脱敏 | |
| LinkedIn URL | 保留名字 + /in/*** | linkedin.com/in/ahmed-rashid → ahmed-rashid/linkedin |
| 网站 | 保留域名 | www.riyadhfood.com → riyadhfood.com |

---

## 二、客户代号规则

### 分配规则

| 规则 | 说明 |
|------|------|
| 格式 | Customer-A, Customer-B, Customer-C... |
| 分配时机 | 首次录入线索时分配 |
| 递增方式 | 按录入顺序递增字母 |
| 持久性 | 同一客户所有记录使用同一代号 |

### 映射表（内部保密）

> 此映射表不应与脱敏后的数据存放在同一文件中。
> 建议在独立的、受密码保护的文件或系统中维护。

| customer_id | 真实姓名 | 真实公司 | 分配日期 |
|-------------|---------|---------|---------|
| Customer-A | *(独立存储)* | *(独立存储)* | |
| Customer-B | *(独立存储)* | *(独立存储)* | |

---

## 三、各场景脱敏规范

### 3.1 客户转化记录

| 字段 | 脱敏后格式 |
|------|-----------|
| customer_id | Customer-A |
| contact_email | a***@example.com |
| contact_phone | +966****4567 |
| company | RFE-Corp |
| 成交金额 | 保留（内部可见，不公开分享时） |

### 3.2 询盘跟进记录

| 字段 | 脱敏后格式 |
|------|-----------|
| customer_id | Customer-A |
| inquiry_content | 去除所有可识别个人身份的信息 |
| follow_up_log | 所有提及客户名字的地方替换为 Customer-X |

### 3.3 案例报告

| 字段 | 脱敏后格式 |
|------|-----------|
| 客户名称 | Customer-A |
| 客户公司 | RFE-Corp |
| 客户国家 | 保留（SA） |
| 客户行业 | 保留（Food Machinery） |
| 客户反馈 | 去除可识别信息 |

### 3.4 截图

| 内容 | 处理方式 |
|------|---------|
| 截图中的客户名字 | 截图后用图片编辑工具涂抹 |
| 截图中的手机号 | 涂抹 |
| 截图中的邮箱 | 涂抹 |
| 截图中的公司 Logo | 保留（非个人隐私） |
| 截图中的广告数据 | 保留（非个人隐私） |

---

## 四、不可脱敏的字段

以下字段可以保留原始值，因为不涉及个人隐私：

| 字段 | 说明 |
|------|------|
| platform | 平台名称 |
| country | 国家代码 |
| industry | 行业 |
| product | 产品 |
| campaign_name | 广告系列名称（不包含客户信息时） |
| spend / impressions / clicks | 广告指标数据 |
| cpm / ctr / cpc / cpl | 广告效率数据 |
| date | 日期 |

---

## 五、脱敏检查清单

录入后检查以下项目：

- [ ] 所有客户姓名已替换为 Customer-X
- [ ] 所有手机号已脱敏（保留国家码和末4位）
- [ ] 所有邮箱已脱敏（首字母 + *** @ 域名）
- [ ] 所有公司名已缩写（如 RFE-Corp）
- [ ] 询盘内容中无可识别个人信息
- [ ] 跟进日志中无可识别个人信息
- [ ] 截图中的敏感信息已涂抹
- [ ] 映射表单独存放，不与脱敏数据混放
