# 投流证据层字段规范

> **定位**：定义所有证据层文档的字段类型、格式、取值范围。
> **适用范围**：投流证据层全部 10 个文档。

---

## 一、ID 编号规范

| 类型 | 格式 | 示例 | 说明 |
|------|------|------|------|
| 案例编号 | CASE-YYYYMMDD-001 | CASE-20260522-001 | 每日从 001 递增 |
| 数据记录 | DATA-YYYYMMDD-001 | DATA-20260522-001 | 每日从 001 递增 |
| 线索编号 | LEAD-YYYYMMDD-001 | LEAD-20260522-001 | 每日从 001 递增 |
| 客户转化 | CONV-YYYYMMDD-001 | CONV-20260522-001 | 每日从 001 递增 |
| 客户代号 | Customer-A | Customer-A, Customer-B | 按字母递增 |

---

## 二、平台字段取值

| 字段 | 取值 | 说明 |
|------|------|------|
| platform | facebook / tiktok / linkedin / google_ads | 小写英文，下划线分隔复合词 |
| country | ISO 3166-1 alpha-2 | 小写，如 sa / us / ae / ph |
| objective | lead_generation / conversion / traffic / brand_awareness / engagement | 下划线分隔 |
| contact_method | whatsapp / email / phone / linkedin_dm / wechat | 小写 |

---

## 三、线索分级标准

### 分级维度

| 等级 | 需求明确度 | 预算明确度 | 时间线明确度 | 响应时效 | 跟进频率 |
|------|-----------|-----------|-------------|---------|---------|
| A | 明确 | 明确 | 明确（30天内） | 5 分钟内 | 每日 |
| B | 明确 | 不确定 | 不确定 | 30 分钟内 | 每 2-3 天 |
| C | 有兴趣 | 未提及 | 未提及 | 2 小时内 | 每周 |
| D | 无效/其他 | - | - | 24 小时内 | 不主动跟进 |

### 升级/降级规则

| 触发条件 | 操作 |
|----------|------|
| C 级明确需求 + 预算 → | 升级 B |
| B 级确认时间线 → | 升级 A |
| A 级超过 7 天无回应 → | 降级 B |
| B 级明确无预算 → | 降级 C |
| 确认不需要/竞品调研 → | 降级 D |

## 四、转化状态流转

```
new → contacted → qualified → quoted → won
                            ↓
                           lost
  任何阶段 → dormant（超过 30 天无互动）
```

| 状态 | 说明 |
|------|------|
| new | 新获取的线索，尚未联系 |
| contacted | 已首次响应 |
| qualified | 确认为有效询盘（B 级以上） |
| quoted | 已发送报价 |
| won | 已成交 |
| lost | 已丢单 |
| dormant | 休眠状态，超过 30 天无互动 |

---

## 五、丢单原因分类

| 分类 | 说明 |
|------|------|
| price | 价格不匹配（客户觉得太贵） |
| competitor | 选择了竞品 |
| no_budget | 客户无预算/资金不足 |
| no_need | 客户实际无需求 |
| timing | 时间不合适（暂缓采购） |
| specs | 技术规格不匹配 |
| trust | 信任度不足（不愿从海外采购） |
| other | 其他原因 |

## 六、数据来源字段

| 取值 | 说明 |
|------|------|
| ads_manager | 从广告管理后台手动录入 |
| api | 通过 API 自动同步 |
| manual_entry | 手动录入（如客户沟通记录） |
| crm_export | 从 CRM 系统导出 |
| form_export | 从表单平台导出 |
| screenshot_ocr | 从截图 OCR 提取 |

---

## 七、验证状态字段

| 取值 | 说明 |
|------|------|
| pending | 待验证，数据已录入但尚未确认 |
| verified | 已验证，数据经确认准确 |
| flagged | 已标记，数据可能存在异常需复核 |

---

## 八、金额字段规范

| 字段 | 货币 | 格式 | 说明 |
|------|------|------|------|
| budget_daily | USD | number | 日预算 |
| budget_total | USD | number | 总预算 |
| spend | USD | number | 实际花费 |
| cpm | USD | number, 2 decimals | 千次展示成本 |
| cpc | USD | number, 2 decimals | 单次点击成本 |
| cpl | USD | number, 2 decimals | 单条线索成本 |
| cpa | USD | number, 2 decimals | 单次转化成本 |
| revenue | USD | number | 产生收入 |
| quote_amount | USD | number | 报价金额 |
| closed_amount | USD | number | 成交金额 |

---

## 九、百分比字段规范

| 字段 | 格式 | 说明 |
|------|------|------|
| ctr | number, 2 decimals | 点击率，如 1.50 表示 1.50% |
| cvr | number, 2 decimals | 转化率，如 3.50 表示 3.50% |
| roas | number, 2 decimals | 广告回报率，如 2.50 表示 2.5x |
