## 数据记录模板

### 记录格式

每条记录代表一个 Campaign 或 Ad Set 在某个时间周期的数据汇总。

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| record_id | string | ✅ | 格式：DATA-YYYYMMDD-001 |
| date | date | ✅ | 数据日期或周期起止 |
| platform | string | ✅ | Facebook / TikTok / LinkedIn / Google Ads |
| country | string | ✅ | 投放国家/地区 |
| industry | string | ⬜ | 行业 |
| product | string | ⬜ | 产品 |
| campaign_name | string | ✅ | 广告系列名称 |
| adset_name | string | ⬜ | 广告组名称 |
| objective | string | ✅ | Lead Gen / Conversion / Traffic / Awareness |
| budget_daily | number | ⬜ | 日预算 (USD) |
| budget_total | number | ⬜ | 总预算 (USD) |
| impressions | number | ✅ | 展示次数 |
| clicks | number | ✅ | 点击次数 |
| spend | number | ✅ | 花费 (USD) |
| cpm | number | ✅ | 千次展示成本 (USD) |
| ctr | number | ✅ | 点击率 (%) |
| cpc | number | ✅ | 单次点击成本 (USD) |
| leads | number | ⬜ | 线索数 |
| cpl | number | ⬜ | 单条线索成本 (USD) |
| conversions | number | ⬜ | 转化数 |
| cpa | number | ⬜ | 单次转化成本 (USD) |
| revenue | number | ⬜ | 产生收入 (USD) |
| roas | number | ⬜ | 广告回报率 |
| cvr | number | ⬜ | 转化率 (%) |
| data_source | string | ✅ | ads_manager / api / manual_entry |
| verified_status | string | ✅ | pending / verified / flagged |
| notes | string | ⬜ | 备注 |
