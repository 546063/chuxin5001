## 通用投放原则（适用于所有品类）

### 广告目标层级选择

| 层级 | 推荐选择 | 说明 |
|------|----------|------|
| Campaign Objective | **Leads**（线索） | B2B 机械首选，直接收集客户信息 |
| Campaign Objective（备选） | **Traffic**（流量） | 用于冷启动测试素材 + 积累像素数据 |
| Conversion Location | **Instant Form**（原生表单） | 降低跳转流失，适合移动端 |
| Conversion Location（备选） | **Website**（落地页） | 适合有成熟落地页 + 像素的网站 |

### 无效线索通用过滤策略

| 过滤手段 | 配置方法 | 效果 |
|----------|----------|------|
| 表单自定义问题 | 添加"您的公司类型？""预计采购时间？"等 | 过滤 C 端个人 |
| 预算门槛问题 | "您的预算范围？" 选项不含低预算 | 过滤小买家 |
| 手机号格式 | 要求含国家代码 | 减少假号码 |
| 公司邮箱验证 | 排除 Gmail/Yahoo/Hotmail | 提高 B 端比例 |
| 否定关键词 | 排除 job, salary, training, course, free | 过滤求职者/学生 |

### 复盘指标通用框架

| 指标 | 基准线（待实测） | 预警线 |
|------|-----------------|--------|
| CPL (Cost per Lead) | $8-$30 | >$50 需排查 |
| 有效线索率 | 20%-40% | <15% 需优化表单 |
| 询盘转化率 | 10%-25% | <8% 需优化承接话术 |
| 样品/视频请求率 | 5%-15% | <3% 素材吸引力不足 |
| ROI | 待实测 | <1 需全链路排查 |

## 附录：阿拉伯语文案注意事项

参考 [[03-海外投流火力站/Facebook投流知识库/24-阿拉伯语Facebook广告文案模板]]，投放中东市场时需注意：

1. **文案方向**：从右到左（RTL），确保排版正确
2. **语气**：正式商务语气，避免过于口语化
3. **卖点强调**：质保（ضمان）、发货速度（شحن سريع）、认证（شهادات）
4. **禁忌**：避免使用猪/猪肉相关内容在纯中东定向中
5. **时间**：投放时段选择当地 9:00-22:00（避开祈祷时间）
6. **本地化**：展示当地客户案例、使用当地货币报价
