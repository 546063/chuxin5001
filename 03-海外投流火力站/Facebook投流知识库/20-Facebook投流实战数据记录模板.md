## 数据判断规则（逻辑，非真实数据）

### CTR（点击率）判断

| CTR 范围 | 判断 | 可能原因 | 建议动作 |
|---------|------|---------|---------|
| > 2% | 优秀 | 素材吸引力强，受众精准 | 保持，考虑加预算 |
| 1-2% | 正常 | 素材和受众基本匹配 | 继续观察 3 天 |
| 0.5-1% | 偏低 | Hook 不够吸引 / 受众不匹配 | 换 Hook 或换受众 |
| < 0.5% | 危险 | 素材严重不匹配 / 受众错误 | 暂停，检查受众和素材 |

### CPC（单次点击成本）判断

| CPC 范围（B2B 食品机械参考） | 判断 | 可能原因 | 建议动作 |
|---------------------------|------|---------|---------|
| < $1 | 优秀 | CPM 低 + CTR 高 | 保持 |
| $1-2 | 正常 | 市场平均水平 | 继续观察 |
| $2-4 | 偏高 | CPM 高或 CTR 低 | 优化素材 |
| > $4 | 危险 | 竞争激 / 素材差 / 受众窄 | 检查 CPM 和 CTR 哪个出问题 |

> 注意：以上 CPC 范围为参考，不同国家/季节/竞争环境差异巨大，以实际数据为准。

### CPL（单次线索成本）判断

| CPL 范围（B2B 食品机械参考） | 判断 | 可能原因 | 建议动作 |
|---------------------------|------|---------|---------|
| < $10 | 优秀 | 素材好 + 受众精准 + 表单门槛合适 | 保持，考虑加预算 |
| $10-20 | 正常 | B2B 机械行业常见范围 | 关注有效线索率 |
| $20-40 | 偏高 | 受众太广 / 素材不相关 / 竞争强 | 优化受众或素材 |
| > $40 | 危险 | 方向可能不对 | 全面复盘 |

> 注意：B2B 线索成本远高于 B2C，关键看线索质量而非数量。CPL $15 的线索如果 50% 有效，实际成本优于 CPL $8 但只有 10% 有效。

### CPA（单次行动成本）判断

| 情况 | 判断 | 建议动作 |
|------|------|---------|
| CPA < 目标值 | 健康 | 考虑加预算扩量 |
| CPA = 目标值 ±20% | 可接受 | 继续优化 |
| CPA > 目标值 30%+ | 需优化 | 走决策树排查 |
| CPA > 目标值 100%+ 持续 5 天 | 危险 | 建议暂停重做 |

### ROAS（投入产出比）判断

| ROAS 范围 | 判断 | 建议动作 |
|----------|------|---------|
| > 5x | 优秀 | 大幅加预算 |
| 3-5x | 健康 | 稳步加预算 |
| 1.5-3x | 及格 | 找优化空间 |
| 1-1.5x | 微亏 | 优化后观察 |
| < 1x | 亏损 | 暂停或大改 |

> 注意：B2B 食品机械的 ROAS 计算需要考虑决策周期（30-90天），短期 ROAS 可能偏低。建议用"询盘价值"替代"即时 ROAS"做判断。

### 询盘有效率（Valid Lead Rate）判断

| 有效率 | 判断 | 建议动作 |
|--------|------|---------|
| > 50% | 优秀 | 受众和表单设计合理 |
| 30-50% | 正常 | 有优化空间 |
| 10-30% | 偏低 | 需要加强过滤 |
| < 10% | 危险 | 大量无效线索，需要全面调整 |

### WhatsApp/Messenger 回复率判断

| 回复率 | 判断 |
|--------|------|
| > 60% | 优秀 |
| 40-60% | 正常 |
| 20-40% | 偏低 |
| < 20% | 需优化首句消息和响应速度 |

### 报价率（Leads → Quotes）判断

| 报价率 | 判断 |
|--------|------|
| > 40% | 优秀（线索质量高，客服跟进及时） |
| 20-40% | 正常 |
| 10-20% | 偏低（需优化线索质量或客服话术） |
| < 10% | 危险（线索太差或客服跟进有问题） |

### 成交率（Quotes → Orders）判断

| 成交率 | 判断 |
|--------|------|
| > 20% | 优秀（B2B 机械行业） |
| 10-20% | 正常 |
| 5-10% | 偏低 |
| < 5% | 需要优化报价和谈判流程 |

## 数据判断决策树

### 有线索但没回复

```
有 Leads 但 0 Messenger/WhatsApp 回复
├── 1. 查响应时间 → 超过 1 小时 → 加快响应速度
├── 2. 查首句消息 → 太正式/太销售 → 改轻松开场
├── 3. 查线索来源 → Lead Ads 无意图用户 → 加表单门槛
├── 4. 查国家 → 低质量国家 → 排除该国
└── 5. 查号码有效性 → 空号/假号 → 表单加验证
```

### 询盘多但都无效

```
Leads 多但 Valid Lead Rate < 10%
├── 1. 查受众 → 太广泛 → 收窄兴趣/行为定向
├── 2. 查表单 → 字段太少 → 加"公司名""月需求量"
├── 3. 查素材 → 吸引好奇用户而非买家 → 加价格锚点
├── 4. 查国家 → 低意图国家占比高 → 排除或降预算
└── 5. 查 CTA → "了解更多" → 改为"获取报价"
```

### ROAS 不稳定

```
ROAS 忽高忽低
├── 1. 查归因窗口 → B2B 决策周期长 → 用 7D Click + 1D View
├── 2. 查样本量 → 数据太少 → 等更多转化再判断
├── 3. 查大单影响 → 1-2 个大单决定 ROAS → 看长期均值
├── 4. 查线索周期 → 30-90 天决策期 → 用询盘价值替代 ROAS
└── 5. 查归因设置 → Pixel/CAPI 数据丢失 → 检查配置
```

## 人工补数说明

### 如何记录手动数据

以下数据无法通过 Facebook Ads Manager 自动获取，需要人工记录：

| 手动数据字段 | 记录方式 |
|------------|---------|
| Valid Leads | 客服人工判断后在表格中标记 #valid 或 #invalid |
| Quotes Sent | CRM 或表格中记录已发送报价的线索数 |
| Orders | 从销售系统/ERP 导出 |
| Revenue | 从订单金额汇总 |
| WhatsApp Conversations | 从 WhatsApp Business 后台导出 |
| Messenger Conversations | 从 Messenger 后台或聊天机器人平台导出 |

### 推荐工具

- **Google Sheets**：免费，适合起步阶段
- **Airtable**：更好的数据管理和筛选
- **CRM（HubSpot/Zoho）**：自动化线索管理
- **Google Data Studio / Looker**：数据可视化

## 使用方式

1. 复制上方表格到 Google Sheets
2. 每天从 Facebook Ads Manager 导出当日数据填入
3. 客服团队每天更新 Valid Leads 和回复数据
4. 每周末汇总一周数据
5. 每月底做月度复盘

> 积累 4-8 周数据后，你就能建立自己的食品机械行业基准，比任何公开 benchmark 都准确。
