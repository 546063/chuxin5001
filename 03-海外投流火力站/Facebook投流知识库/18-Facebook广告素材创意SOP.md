# Facebook广告素材创意 SOP（B2B食品加工机械出口专用）

> [!summary] 本文档适用对象
> 面向 B2B 食品加工机械出口企业，指导 Facebook/Instagram 广告素材的策划、设计、制作、测试与迭代全链路。整单金额通常 ≥ ¥20,000，目标受众为海外工厂老板、采购经理、经销商和中央厨房负责人。

---

## 1. 图片广告设计规范

### 1.1 尺寸与比例

| 广告位置 | 推荐比例 | 像素尺寸 | 文件限制 |
|---------|---------|---------|---------|
| Feed（信息流） | 1:1 正方形 | 1080 × 1080 | JPG/PNG ≤ 4 MB |
| Feed 竖版 | 4:5 | 1080 × 1350 | JPG/PNG ≤ 4 MB |
| Right Column | 1:1 | 1080 × 1080 | JPG/PNG ≤ 4 MB |
| Stories / Reels | 9:16 | 1080 × 1920 | JPG/PNG ≤ 4 MB |
| Marketplace | 1:1 | 1080 × 1080 | JPG/PNG ≤ 4 MB |

### 1.2 文字比例与排版

| 规则 | 说明 |
|-----|------|
| 图片上文字覆盖 | ≤ 20%（虽然 Facebook 已取消硬性限制，但超过 20% 仍会影响触达） |
| 主标题字号 | 图片高度的 8%-12% |
| 辅助文字 | 不超过 2 行，字号 ≤ 主标题的 70% |
| CTA 按钮 | 使用高对比色（橙/红/绿），文字 2-4 个单词 |

### 1.3 配色方案（食品机械 B2B 风格）

| 用途 | 色值 | 说明 |
|-----|------|------|
| 主色（品牌/信任） | `#1A5276` 深蓝 | 工业感、专业感 |
| 强调色（CTA/钩子） | `#E67E22` 橙色 | 醒目、促点击 |
| 背景色 | `#F8F9FA` 浅灰白 | 干净、不抢主体 |
| 文字色 | `#2C3E50` 深灰 | 易读、不刺眼 |
| 成功/绿色 | `#27AE60` | 认证/合格/节能标签 |

### 1.4 排版模板

```
┌──────────────────────────┐
│  [LOGO] 左上角，高度 8%  │
│                          │
│   机器设备高清图（居中）  │
│   占比 60%-70%           │
│                          │
│  主文案：Capacity 500kg/h│
│  副文案：CE Certified    │
│                          │
```

## 4. Reels 素材规范（竖版 9:16）

### 4.1 技术规格

| 项目 | 要求 |
|-----|------|
| 分辨率 | 1080 × 1920 |
| 比例 | 9:16 竖版 |
| 时长 | 15-90秒（推荐 15-30秒） |
| 文字安全区 | 上下各留 14% 不被 UI 遮挡 |
| 文件格式 | MP4（H.264） |
| 文件大小 | ≤ 4 GB |

### 4.2 安全区域布局

```
┌──────────────────┐
│   ← 14% 安全区   │ ← 被 Reels 顶部 UI 遮挡
│                  │
│   核心内容区     │
│   （占 72%）     │
│                  │
│   ← 14% 安全区   │ ← 被底部 CTA 和用户名遮挡
└──────────────────┘
```

### 4.3 Reels 内容策略

| 内容类型 | 占比 | 示例 |
|---------|------|------|
| 机器运行实拍 | 40% | 设备高速运转，配节奏感音乐 |
| Before/After | 20% | 人工操作 vs 机器自动 |
| 工厂实力展示 | 15% | 车间全景、发货现场 |
| 客户证言 | 15% | 外国客户竖大拇指/评价 |
| 趣味/幕后 | 10% | 工程师调试、搞笑反差 |

## 6. 产品演示素材

### 6.1 设备运行效果展示模板

| 步骤 | 画面 | 时长 | 标注 |
|-----|------|------|------|
| 1. 原料展示 | 原始食材（整块肉/蔬菜） | 2s | "Before processing" |
| 2. 投料过程 | 食材进入机器入口 | 3s | 箭头指引 |
| 3. 加工中 | 机器内部运转（可用动画示意） | 4s | 速度/温度等参数 |
| 4. 成品输出 | 成品连续产出（慢动作） | 4s | "Uniform slices, 2mm" |
| 5. 成品特写 | 成品排列整齐 | 2s | 标尺/对比物 |
| 6. CTA | Logo + 联系方式 | 2s | "Get specs + price" |

### 6.2 拍摄要点

| 要点 | 说明 |
|-----|------|
| 光线 | 车间光线不足时补 LED 灯，确保机器和成品清晰可见 |
| 速度 | 关键过程用 60fps 拍摄，后期可慢放 |
| 声音 | 收录真实机器运转声，BGM 音量 ≤ 30% |
| 干净 | 设备表面不能有油污/锈迹，拍摄前清洁 |
| 安全 | 操作员穿戴合规（安全帽、手套），避免不合规画面被拒审 |

## 8. 客户痛点素材

### 8.1 常见痛点与视觉转化

| 客户痛点 | 视觉表达方式 | 广告文案钩子 |
|---------|-------------|-------------|
| "人工成本越来越高" | 工资单堆叠动画 + 机器取代人 | "Labor cost eating your margin?" |
| "产品规格不统一" | 歪歪扭扭的切片 vs 整齐的机器切片 | "Tired of inconsistent cuts?" |
| "卫生检查不过关" | 红色 X 标记手工作业 → 绿色 ✓ 机器封闭式 | "Fail hygiene audits again?" |
| "产能跟不上订单" | 堆积如山的订单 vs 机器高速产出 | "Can't keep up with orders?" |
| "设备频繁故障" | 维修工满头大汗 vs 我们的设备安静运转 | "Machine breakdown costing you?" |
| "找不到可靠供应商" | 地图标记 60+ 国家 | "Need a supplier you can trust?" |

### 8.2 痛点素材制作流程

1. **收集痛点**：从销售对话、客户邮件、展会交流中提取 Top 5 痛点
2. **场景还原**：用图片/短视频还原客户痛点场景（可用演员或真实客户）
3. **对比解决**：紧接着展示我们的方案如何解决问题
4. **数据支撑**：用具体数字（产能、人数、成本）增强说服力
5. **CTA 引导**：引导到落地页或 WhatsApp

## 10. 3秒钩子设计（10种 Hook 类型 + 示例）

### 10.1 10种钩子类型

| # | 钩子类型 | 原理 | B2B 食品机械示例（英文） |
|---|---------|------|------------------------|
| 1 | **数字冲击** | 用具体数字抓眼球 | "5,000kg per day. 1 machine." |

| 3 | **对比冲突** | Before/After 制造落差 | "This: 200kg/day. That: 5,000kg/day." |
| 4 | **恐惧损失** | FOMO / 怕落后 | "Your competitors already automated. Have you?" |
| 5 | **利益承诺** | 直接给好处 | "Cut your processing cost by 60%. Here's how." |
| 6 | **好奇悬念** | 引发好奇心 | "The machine that changed meat processing in Africa" |
| 7 | **权威背书** | 借助第三方信任 | "CE certified. Exported to 60 countries." |
| 8 | **场景代入** | 让观众想象自己 | "Imagine running your entire line from a touchscreen" |
| 9 | **反常识** | 打破认知 | "You don't need 10 workers. You need 1 machine." |
| 10 | **紧迫感** | 限时/限量 | "Factory direct price — only this month" |

### 10.2 钩子使用原则

| 原则 | 说明 |
|-----|------|
| 前 3 秒必须有运动画面 | 静态图在 Reels/Stories 中完播率极低 |
| 文字叠加在画面上 | 40% 用户静音观看，字幕是刚需 |
| 一钩一主题 | 不要试图在 3 秒内传达多个信息 |
| 配合音效 | 重低音/机械声/提示音都能提升注意力 |

## 12. 素材测试节奏

### 12.1 测试时间线

| 阶段 | 时间 | 动作 | 判断标准 |
|-----|------|------|---------|
| 冷启动 | 第 1-3 天 | 上 6-8 条不同创意（3图+3视频+2轮播） | 不干预，让算法学习 |
| 初步筛选 | 第 4-5 天 | 暂停 CPA 最高的 50% | CPA ≤ 目标值 × 1.5 |
| 赢家识别 | 第 6-7 天 | 保留 Top 2-3 条，加注预算 | CTR ≥ 1.5%, CPC ≤ $1.50 |
| 扩量 | 第 8-14 天 | 赢家预算 × 1.5-2x | ROAS ≥ 3 或 CPL ≤ $30 |
| 迭代 | 第 15-21 天 | 制作赢家变体（换钩子/换文案/换配色） | 变体 CPA 低于原版 |
| 全面替换 | 第 22-30 天 | 旧素材逐步下线，新素材上线 | 广告疲劳（Frequency > 3） |

### 12.2 关键指标参考（B2B 食品机械）

| 指标 | 优秀 | 良好 | 需优化 |
|-----|------|------|-------|
| CTR（点击率） | ≥ 2.0% | 1.0%-2.0% | < 1.0% |
| CPC（单次点击） | ≤ $1.00 | $1.00-$2.00 | > $2.00 |
| CPM（千次展示） | ≤ $15 | $15-$25 | > $25 |
| CPL（单次询盘） | ≤ $20 | $20-$50 | > $50 |
| 视频完播率（15s） | ≥ 25% | 15%-25% | < 15% |
| 视频 ThruPlay | ≤ $0.10 | $0.10-$0.20 | > $0.20 |

### 12.3 测试矩阵模板

| 变量 | A 版本 | B 版本 | 结论 |
|-----|-------|-------|------|
| 钩子 | "5,000kg/day" | "Stop losing money on labor" | 保留胜出者 |
| 画面 | 机器特写 | 分屏对比 | 保留胜出者 |

## 15. 常见坑

### 15.1 审核拒审常见原因

| 问题 | 原因 | 解决方法 |
|-----|------|---------|
| 文字超过 20% | 图片上文字太多 | 用文字检查工具，控制在 20% 以内 |
| Before/After 太夸张 | Facebook 认为误导 | 用真实对比，标注"实际效果因工厂而异" |
| 使用他人商标 | 展示竞品 Logo | 只展示自己的设备和品牌 |
| 不安全操作画面 | 操作员未戴安全帽 | 拍摄时确保穿戴合规 PPE |
| 虚假承诺 | "100% profit" 类描述 | 用具体数据和客户案例说话 |
| 低质量图片 | 模糊/过暗/过曝 | 拍摄后调色、补光 |

### 15.2 投放常见错误

| 错误 | 后果 | 正确做法 |
|-----|------|---------|
| 1 条素材跑所有广告组 | 疲劳快、数据无法对比 | 至少 3 条素材并行测试 |
| 每天改预算 | 打断算法学习 | 每 3-5 天调整一次 |
| 只看 CPC 不看转化 | 便宜但没询盘 | 以 CPL/CPA 为核心指标 |
| 素材长期不更新 | 广告疲劳、CPM 飙升 | 每 2-3 周换新素材 |
| 忽略再营销素材 | 只投冷启动浪费流量 | 准备专门的再营销素材 |
| 不考虑移动端 | 桌面设计、手机看变形 | 100% 按移动端预览设计 |

### 15.3 B2B 特有大坑

| 坑 | 说明 | 避免方法 |
|---|------|---------|
| 素材太 C 端 | 用卖消费品的方式卖设备 | 强调产能、参数、投资回报 |
| 没有明确 CTA | 用户看完不知道做什么 | 每条素材必须有明确行动指引 |
| 落地页不匹配 | 广告说 A，落地页是 B | 广告与落地页信息一致性 ≥ 80% |
| 忽略询盘质量 | 只追求数量不筛选 | 在文案中写明 MOQ 和目标客户类型 |

## 16. 可复制 SOP（Step-by-Step Checklist）

### Phase 1: 策划准备（每月初第 1-2 天）

- [ ] **Step 1**: 回顾上月数据，确定 Top 3 表现最好的素材类型
- [ ] **Step 2**: 与销售团队沟通，收集本月 Top 5 客户痛点
- [ ] **Step 3**: 确定本月素材主题（如：肉类加工线、蔬菜清洗设备、包装机械）
- [ ] **Step 4**: 编写 10 个钩子文案（参考第 10 节）
- [ ] **Step 5**: 编写 5 套广告文案（覆盖 PAS/AIDA/FAB/4U/Social Proof 框架）
- [ ] **Step 6**: 制定本月拍摄计划（设备、场地、人员、时间）

### Phase 2: 素材制作（第 3-10 天）

- [ ] **Step 7**: 拍摄工厂实景（全景、细节、发货、质检）
- [ ] **Step 8**: 拍摄产品演示（原料→加工→成品全流程）
- [ ] **Step 9**: 拍摄 Before/After 对比素材
- [ ] **Step 10**: 设计师完成 15-20 张图广告（1080×1080 / 1080×1350）
- [ ] **Step 11**: 剪辑师完成 8-10 条 15s 短视频
- [ ] **Step 12**: 剪辑师完成 4-6 条 30s 中视频
- [ ] **Step 13**: 剪辑师完成 2-3 条 60s 长视频
- [ ] **Step 14**: 制作 3-4 套轮播素材（每套 5 张）
- [ ] **Step 15**: 制作 8-10 条竖版 Reels 素材
- [ ] **Step 16**: 所有素材按「类型_日期_主题_版本」命名归档

### Phase 3: 上传测试（第 11-12 天）

- [ ] **Step 17**: 在 Ads Manager 创建测试 Campaign
- [ ] **Step 18**: 设置 3-4 个 Ad Set（不同受众/地区/兴趣）
- [ ] **Step 19**: 每个 Ad Set 上 6-8 条素材（图+视频+轮播混合）
- [ ] **Step 20**: 设置预算（建议 $30-$50/Ad Set/天）
- [ ] **Step 21**: 确认 Pixel/CAPI 已正确安装
- [ ] **Step 22**: 设置自定义事件追踪（ViewContent, Lead, Contact）

### Phase 4: 数据监控与优化（第 13-28 天）

- [ ] **Step 23**: 第 3 天：初步查看数据，不调整预算
- [ ] **Step 24**: 第 5 天：暂停 CPA 最高的 50% 素材
- [ ] **Step 25**: 第 7 天：识别 Top 2-3 赢家，记录特征
- [ ] **Step 26**: 第 10 天：制作赢家变体（换钩子/配色/文案）
- [ ] **Step 27**: 第 14 天：赢家预算 × 1.5-2x
- [ ] **Step 28**: 第 21 天：全面检查广告疲劳（Frequency > 3 则替换）
- [ ] **Step 29**: 第 25 天：准备下月素材，开始新一轮拍摄
- [ ] **Step 30**: 第 28 天：撰写本月素材效果报告

### Phase 5: 复盘迭代（月末最后 2 天）

- [ ] **Step 31**: 统计本月素材总消耗、总询盘、CPL
- [ ] **Step 32**: 分析 Top 5 和 Bottom 5 素材特征
- [ ] **Step 33**: 提炼可复用的创意模式
- [ ] **Step 34**: 更新素材库（将表现好的素材标记为"Winner"）
- [ ] **Step 35**: 与销售团队同步询盘质量和反馈
