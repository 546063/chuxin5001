# 账户结构与出价策略

## 一句话总结

精简结构优于分散结构：3-6 个 Campaign、每个 1-4 个 Ad Set、每个 Ad Set 至少 2-3 条素材。

## 核心结论

- 使用 CBO 的广告主 CPA 平均降低 12%
- 每日手动调整超过 1 次会导致 ROAS 下降 15%
- 预算调整幅度不要超过 20%（超过会重置学习期）
- 勘探 vs 再营销预算比例 = 60-70% : 30-40%

## 适用场景

- 新账户从零搭建
- 现有账户结构重组
- 选择 CBO vs ABO vs ASC
- 不同预算等级的投放方案

## 具体内容

### 1. Meta 广告三层架构

```
广告账户 (Ad Account)
├── Campaign（广告系列）← 选目标 + 预算模式
│   ├── Ad Set（广告组）← 定向 + 出价 + 版位
│   │   ├── Ad（广告）← 素材 + 文案 + 链接
│   │   └── Ad（广告）← A/B 测试变体
│   └── Ad Set（广告组）
└── Campaign（广告系列）
```

### 黄金法则

| 法则 | 说明 |
|------|------|
| 精简结构 | 每个账户 3-6 个 Campaign，每个 Campaign 1-4 个 Ad Set |
| 预算集中 | 宁可用 $100/天 × 2 个 Ad Set，不用 $20/天 × 10 个 Ad Set |
| 避免自竞争 | 同一受众不要在多个 Ad Set 中重叠 |
| 命名规范 | `[目标]_[市场]_[受众]_[日期]` |
| 隔离学习 | 每个 Ad Set 需要独立的转化数据来完成学习期 |

### 2. Campaign Objective 选择

| 目标 | 适用场景 | 备注 |
|------|---------|------|
| Sales | 电商转化、独立站购买 | 需要 Pixel + CAPI 数据 |
| Leads | B2B 留资、表单收集、WhatsApp | 即时表单 + 后续跟进 |
| Engagement | 视频观看、帖文互动 | 品牌曝光 + 暖受众 |
| Traffic | 引流到网站 | 仅做点击优化，不保证转化 |
| Awareness | 品牌认知、大范围曝光 | 最低 CPM，不做转化 |

### 3. 预算模式选择

| 模式 | 适用场景 | 优点 | 缺点 |
|------|---------|------|------|
| CBO | 稳定投放、成熟账户 | AI 自动分配预算 | 单个 Ad Set 可能被边缘化 |
| ABO | 测试阶段、需要精确控制 | 精确控制每组预算 | 需要手动调优 |
| ASC | 电商放量 | 全自动，效果通常优于手动 | 黑盒、无法精确控制 |

### 4. 出价策略对比

| 策略 | 控制级别 | 放量能力 | 适合阶段 |
|------|---------|---------|---------|
| Lowest Cost | 无控制 | 最大 | 冷启动/测试期 |
| Cost Cap | 平均成本控制 | 中等 | 稳定期/扩展期 |
| Bid Cap | 每次拍卖上限 | 受限 | 严格单位经济模型 |
| ROAS Target | 价值优化 | 中等 | 电商多 SKU |

### 出价策略迁移路径

```
新账户（0 转化）
  ↓ Lowest Cost（2-4 周，100+ 转化）
  ↓ Cost Cap = 实际 CPA × 1.2（2-4 周，验证稳定性）
  ↓ Cost Cap = 实际 CPA × 0.9（收紧成本）
  ↓ 如果 ROAS > 目标且稳定
  ↓ Target ROAS = 实际 ROAS × 0.8（最大化利润）
```

### 5. 按预算等级的模板化结构

**$20-50/天（极简测试）**
```
Campaign: [PURCHASE]_Testing_$30/day (ABO)
├── Ad Set 1: Broad_US_18-65 ($10/day)
├── Ad Set 2: Interest_Core ($10/day)
└── Ad Set 3: Retargeting_30d ($10/day)
```

**$50-200/天（双轨测试+扩量）**
```
Campaign A: Testing (ABO, $150/day) — 3 Ad Set 测受众
Campaign B: Scaling (CBO, $50/day) — 已验证受众
```

**$200-1000/天（成熟扩量）**
```
Campaign A: TOF (CBO, $700/day) — Broad + LAL + Interest
Campaign B: ASC (ASC, $200/day) — 10-15 条已验证素材
Campaign C: Retargeting (CBO, $100/day) — 30d Visitors
```

### 6. B2B 线索广告结构

```
Campaign: [LEADS]_B2B_$100/day (CBO)
├── Ad Set 1: Interest_Industry ($40/day)
├── Ad Set 2: LAL-1%_Leads ($30/day)
└── Ad Set 3: Retargeting_60d ($30/day)

优化目标：Leads（即时表单）或 WhatsApp 消息
素材类型：案例研究/产品演示/白皮书下载
关键指标：CPL < $30, 有效线索率 > 20%
```

### 7. 上线前检查清单

- [ ] Pixel 已安装且正常回传
- [ ] CAPI 已启用且 Event Match Quality ≥ 6/10
- [ ] 域名已验证
- [ ] Aggregated Event Measurement 已配置
- [ ] 支付方式有效
- [ ] 目标选对（Sales 不是 Traffic）
- [ ] 预算 ≥ 目标 CPA × 3
- [ ] 受众大小 ≥ 10 万人
- [ ] 没有受众重叠 > 30%
- [ ] 每个 Ad Set 至少 2-3 条素材
- [ ] 素材尺寸适配版位
- [ ] 文案含明确 CTA
- [ ] 落地页加载 < 3 秒

## 可复用方法

### ASC 设置清单

1. 选目标：Sales → Conversions → Purchase
2. 选 Advantage+ Shopping Campaign
3. 设预算：≥ 历史最佳 Ad Set 预算的 2-3 倍
4. 设置建议受众（非强制）
5. 上传素材：至少 5-10 个创意变体
6. 准备 3-5 个文案变体
7. 初期用 Lowest Cost，稳定后切 Cost Cap
8. 发布后 7-14 天学习期不干预

### ASC 效果评估标准

| 指标 | 达标线 | 优秀线 |
|------|--------|--------|
| ROAS | ≥ 2.5x | ≥ 4x |
| CPA | ≤ 历史手动 80% | ≤ 历史手动 60% |
| 新客占比 | ≥ 50% | ≥ 65% |
| 预算消耗率 | ≥ 95% | 100% |

## 对我的业务有什么价值

- 有现成的模板可以直接套用，不需要从零设计
- 知道 CBO/ABO/ASC 的选择依据，不做拍脑袋决策
- 上线前检查清单避免漏掉关键设置
- ASC 效果评估标准帮助判断是否值得用
