# B2 核心指标与ROAS拆解

## 一句话总结
ROAS拆解公式、核心指标联动关系、行业基准值、频率监控标准、广告质量诊断三指标

## 核心结论
- **ROAS 核心公式**：ROAS = 客单价 ÷ CPA = (客单价 × 转化率) ÷ CPC，受 CPM（市场竞争）、CTR（素材吸引力）、CVR（落地页转化力）三个变量影响
- **ROAS 下滑排查顺序**：先查 CPM（竞争加剧？）→ 再查 CTR（素材疲劳？）→ 再查 CVR（落地页/产品问题？）→ 最后查 CPA（以上组合问题）
- **优化优先级**：CVR（落地页，最容易短期提升）> CTR（素材驱动）> CPM（长期降低）> AOV（产品策略：捆绑销售、Upsell）
- **MER 是最终校准器**：MER = 总收入 ÷ 总广告花费，当 MER 稳定时即使广告后台数据有偏差，整体盈利能力可信；MER 与 ROAS 差异 > 20% 需排查数据源
- **盈利线公式**：CPA < 客单价 × 毛利率。例：客单价 $50，毛利率 60% → CPA < $30 才盈利

## 适用场景
- 适合平台：Facebook/Meta Ads Manager（跨工具数据对比涉及 GA4、Shopify、Triple Whale）
- 适合行业：DTC 电商（ROAS 导向）、B2B（CPL 导向）、SaaS（CAC/LTV 导向）、本地服务（线索成本导向）
- 适合场景：日常数据监控、ROAS 下滑诊断、跨工具数据差异解读、指标联动分析、行业基准对比

## 投流要素拆解
- 平台：Facebook/Meta
- 核心指标体系：曝光层（CPM/Frequency）→ 互动层（CTR/CPC/Thumb-Stop）→ 转化层（CVR/CPA）→ 业务层（ROAS/MER/LTV）
- 关键公式：CPA = CPC ÷ CVR、CPC = CPM ÷ (1000 × CTR)、ROAS = AOV ÷ CPA
- 诊断方法：多维度交叉（CPM×CTR、CVR×CPA、Frequency×生命周期）
- 关键指标：CTR(Link) > 1%、LPV/Link Click > 80%、Purchase CVR > 1%、Frequency < 2.5
- 可复用策略：ROAS 下滑排查树、指标联动模拟计算器
- 风险点：只看单一指标忽略联动、归因窗口变化导致误判、季节性 CPM 波动误认为广告变差
- 适合复用：所有投流场景的数据诊断

## 可复用方法
1. **ROAS 快速排查树**：CPM 涨 → 竞争问题；CTR 降 → 素材疲劳；CVR 降 → 落地页问题
2. **指标联动计算器**：改变 CTR/CVR/AOV 任一指标，推算 ROAS 变化幅度
3. **交叉诊断矩阵**：CPM×CTR 看素材-竞争组合、CVR×CPA 看落地页-成本组合、Frequency×周期看疲劳
4. **MER 校准法**：MER 与 ROAS 差异 > 20% 时排查数据源，MER 稳定则整体盈利可信
5. **数据异常排查**："变好了"对比 Shopify 验证、"变差了"先查 Pixel/归因窗口/审核状态

## 对我的业务有什么价值
- 对跨境贸易的价值：B2B 食品机械用 CPL（单条线索成本）和线索→成交转化率衡量效果，而非 ROAS
- 对 Facebook 投流的价值：有了明确的排查顺序和交叉诊断矩阵，快速定位问题根因
- 对巨量本地推的价值：巨量的核心指标体系与 Meta 基本一致，诊断框架可直接迁移
- 对客户开发的价值：理解 MER 和 ROAS 差异，正确解读投放效果，避免因数据不一致做错误判断
- 对知识库沉淀的价值：核心指标是数据优化的度量基础，所有优化策略都依赖正确的指标解读

## 1. ROAS 拆解公式

### 核心公式
```
ROAS = (客单价 × 转化率) ÷ CPC
     = 客单价 ÷ CPA

其中：
CPA = CPC ÷ 转化率
CPC = CPM ÷ (1000 × CTR)
CTR = 点击数 ÷ 展示数
```

### 核心推论
- ROAS 受三个变量影响：**CPM（市场竞争）**、**CTR（素材吸引力）**、**CVR（落地页转化力）**
- 改善 ROAS = 降低 CPM 或 提高 CTR 或 提高 CVR

### 优化优先级
1. **CVR**（最容易短期提升）：优化落地页
2. **CTR**（素材驱动）：优化素材
3. **CPM**（长期降低）：扩大受众、优化广告质量分
4. **AOV**（产品策略）：捆绑销售、Upsell

## 2. 全链路指标

### 曝光层指标
| 指标 | 公式 | 基准参考 | 说明 |
|------|------|---------|------|
| CPM | 花费 ÷ 展示 × 1000 | $8-15 美国, $3-8 东南亚 | 市场竞争度指标 |
| Impressions | 展示次数 | - | 广告被展示的次数 |
| Reach | 独立用户数 | - | 触达了多少不同的人 |
| Frequency | Impressions ÷ Reach | 冷受众 1-2, 再营销 3-5 | 每人平均看到几次 |

### 互动层指标
| 指标 | 公式 | 基准参考 | 说明 |
|------|------|---------|------|
| CTR (All) | 所有点击 ÷ 展示 | 1-3% | 含点赞/评论/分享 |
| CTR (Link) | 链接点击 ÷ 展示 | 1-2% | 核心指标 |
| CPC (Link) | 花费 ÷ 链接点击 | $0.50-2.00 | 点击成本 |
| Thumb-Stop Rate | 3秒播放 ÷ 展示 | > 25% 视频广告 | 素材Hook能力 |
| ThruPlay | 15秒播放数 | - | 视频完播能力 |

### 转化层指标
| 指标 | 公式 | 基准参考 | 说明 |
|------|------|---------|------|
| Landing Page View | LPV ÷ Link Click | 80-90% | 加载速度指标 |
| ATC Rate | ATC ÷ LPV | 5-15% | 商品页吸引力 |
| IC Rate | IC ÷ ATC | 40-60% | 结账意愿 |
| Purchase CVR | Purchase ÷ LPV | 1-5% | 整体转化能力 |
| CPA / Cost Per Result | 花费 ÷ 转化数 | 因行业而异 | 核心优化指标 |

### 业务层指标
| 指标 | 公式 | 基准参考 | 说明 |
|------|------|---------|------|
| ROAS | 收入 ÷ 花费 | > 2.0（取决于利润率） | 最终业务指标 |
| 客单价 (AOV) | 收入 ÷ 订单数 | 因行业而异 | 提升 ROAS 的杠杆 |
| MER | 总收入 ÷ 总营销花费 | > 3.0 | 全域营销效率 |
| LTV/CAC | 用户终身价值 ÷ 获客成本 | > 3.0 | 长期健康度 |

## 3. ROAS 下滑排查顺序

### 排查顺序
1. **先看 CPM** → 是否竞争加剧？（季节性/行业/竞品）
2. **再看 CTR** → 素材是否疲劳？
3. **再看 CVR** → 落地页/产品价格/库存是否出问题？
4. **最后看 CPA** → 如果 CVR 没问题但 CPA 高，可能是 CPM 问题

### 快速判断框架
```
ROAS 下滑
  ├── CPM 上涨 > 30% → 竞争问题（等/换市场/调出价）
  ├── CTR 下降 > 30% → 素材疲劳（换新素材）
  ├── CVR 下降 > 30% → 落地页/产品问题（查网站/价格/库存）
  └── 三个都正常但 ROAS 降 → 客单价降或归因问题
```

## 4. 核心指标联动关系

```
CPM ↑ → CPC ↑（如果 CTR 不变）
CTR ↓ → CPC ↑（如果 CPM 不变）
CVR ↓ → CPA ↑（如果 CPC 不变）
频次 ↑ → CTR ↓ → CPC ↑ → CPA ↑（疲劳链条）
```

**诊断口诀**：
- CPM 异常 → 竞争环境/竞价问题
- CTR 异常 → 素材/受众匹配问题
- CVR 异常 → 落地页/价格/产品问题
- CPA 异常 → 以上三者之一或组合

## 5. 行业基准值

### ROAS 基准

| 行业 | 盈亏平衡 ROAS | 健康 ROAS |
|---|---|---|
| 电商（标品）| 2.0-3.0x | 3.5-5.0x |
| 电商（非标品）| 1.5-2.5x | 3.0-4.0x |
| B2B | 1.0-2.0x | 3.0-5.0x |
| App | 1.0-1.5x | 2.0-3.0x |

### 盈利线公式
- `CPA < 客单价 × 毛利率`
- 例：客单价 $50，毛利率 60% → CPA < $30 才盈利

## 6. MER 框架深度应用

### MER 与 ROAS 的关系

```
ROAS = Meta 归因收入 ÷ Meta 花费
MER  = 总收入（所有渠道）÷ 总广告花费（所有渠道）
```

### 什么时候 MER > ROAS？什么时候 MER < ROAS？

| 情况 | MER vs ROAS | 原因 |
|------|------------|------|
| MER > ROAS | 正常 | 有自然流量、SEO、邮件等渠道贡献 |
| MER < ROAS | 异常 | Meta 过度归因（如品牌词搜索被误归因） |
| MER ≈ ROAS | 少见 | 几乎 100% 依赖 Facebook 流量 |

### MER 看板搭建

| 指标 | 公式 | 健康值 |
|------|------|--------|
| MER | 总收入 ÷ 总广告花费 | > 3.0 |
| 广告贡献占比 | 广告归因收入 ÷ 总收入 | 40-70%（太低=广告不发力，太高=过度依赖） |
| 自然流量占比 | 自然流量收入 ÷ 总收入 | 30-60% |
| 广告 ROI | (总收入 - 总广告花费) ÷ 总广告花费 | > 200% |

## 7. 多维度交叉诊断矩阵

> 单一指标正常不代表健康。用二维交叉看深层问题。

### CPM × CTR 交叉

| | CPM 正常 | CPM 偏高 | CPM 极高 |
|--|---------|---------|---------|
| **CTR 高** | 🟢 理想状态 | 🟡 素材好但竞争贵 | 🟡 素材够硬，但成本吃紧 |
| **CTR 正常** | 🟢 健康 | 🟡 需关注 | 🟠 考虑扩受众降 CPM |
| **CTR 低** | 🟠 素材问题 | 🔴 双重问题 | 🔴 立即换素材 + 扩受众 |

### CVR × CPA 交叉

| | CPA 正常 | CPA 偏高 | CPA 极高 |
|--|---------|---------|---------|
| **CVR 高** | 🟢 完美 | 🟡 可能 CPM 太高 | 🟠 查 CPM |
| **CVR 正常** | 🟢 健康 | 🟡 轻微优化空间 | 🟠 需优化落地页 |
| **CVR 低** | 🟠 CPA 靠低 CPM 撑着 | 🔴 落地页 + 成本问题 | 🔴 双重问题 |

### 频率 × 素材生命周期交叉

| | 跑了 <7 天 | 跑了 7-21 天 | 跑了 >21 天 |
|--|-----------|-------------|-------------|
| **Frequency < 1.5** | 🟢 刚开始，正常 | 🟢 受众够大 | 🟢 受众极大或预算极低 |
| **Frequency 1.5-2.5** | 🟡 偏快，检查受众大小 | 🟢 正常 | 🟡 关注 CTR 趋势 |
| **Frequency > 2.5** | 🔴 受众太小或预算太高 | 🟠 疲劳初期 | 🔴 立即换素材 |

## 8. 数据异常排查速查表

> 数据突然"变好了"或"变差了"，怎么判断是真的还是数据问题？

### "变好了"——是不是真的？

| 症状 | 排查方法 | 判断 |
|------|---------|------|
| ROAS 突然翻倍 | 对比 Shopify 实际订单 | 如果 Shopify 也涨了 → 真的 |
| CPA 突然降 50% | 检查是否有延迟归因 | 可能前几天转化延迟到现在归因 |
| CTR 突然飙升 | 检查是否新增 Reels 版位 | Reels 的 CTR(All) 含互动会拉高 |
| 花费突然暴增 | 检查是否有人改了预算 | 也可能是 CBO 重新分配了 |

### "变差了"——是不是真的？

| 症状 | 排查方法 | 判断 |
|------|---------|------|
| ROAS 突然掉 50% | 对比 Shopify + 7D+1D 归因窗口 | 可能归因窗口变化导致 |
| 转化数骤降 | 检查 Pixel 是否正常回传 | 用 Test Events 验证 |
| CPM 暴涨 3 倍 | 查是否进入 Q4/黑五 | 季节性正常 |
| 不花费了 | 查审核状态 + 支付方式 | 最常见是广告被拒审 |

## 9. 指标联动模拟计算器

> 改变一个指标，对其他指标的连锁影响。

### 示例 1：CTR 从 1% 提升到 1.5%

```
假设：CPM = $15，CTR = 1% → 2%，CVR = 2%，AOV = $50

CTR = 1%:
CPC = $15 / (1000 × 0.01) = $1.50
CPA = $1.50 / 0.02 = $75
ROAS = $50 / $75 = 0.67x

CTR = 1.5%:
CPC = $15 / (1000 × 0.015) = $1.00
CPA = $1.00 / 0.02 = $50
ROAS = $50 / $50 = 1.0x

结论：CTR 提升 50%（1%→1.5%），ROAS 从 0.67x → 1.0x（+49%）
```

### 示例 2：CVR 从 2% 提升到 3%

```
假设：CPM = $15，CTR = 1.5%，CVR = 2% → 3%，AOV = $50

CVR = 2%:
CPA = $1.00 / 0.02 = $50
ROAS = $50 / $50 = 1.0x

CVR = 3%:
CPA = $1.00 / 0.03 = $33.33
ROAS = $50 / $33.33 = 1.5x

结论：CVR 提升 50%（2%→3%），ROAS 从 1.0x → 1.5x（+50%）
```

### 优化投入回报排序（同等努力下）

| 优化目标 | 从 X 到 Y | ROAS 提升 | 实施难度 |
|---------|----------|----------|---------|
| CTR 提升 | 1% → 1.5% | +49% | 中（换素材） |
| CVR 提升 | 2% → 3% | +50% | 中（优化落地页） |
| AOV 提升 | $50 → $65 | +30% | 低（捆绑销售） |
| CPM 降低 | $15 → $10 | +50% | 高（扩受众） |
