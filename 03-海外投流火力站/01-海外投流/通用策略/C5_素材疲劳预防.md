## 1. 素材疲劳 5 要素诊断

### 疲劳信号
| 要素 | 诊断方法 | 阈值 |
|------|---------|------|
| Creative Fatigue | 运行时长 | > 2-4 周 |
| Emotional Fatigue | 同样情绪类型重复出现 | 连续 3+ 条 |
| Frequency | 人均看到次数 | > 2.5 |
| CTR | 点击率趋势 | 下降 > 20% |
| CPA | 获客成本趋势 | 上升 > 30% |

**满足任意 2 项 → 进入迭代流程**

---

## 2. 轮换节奏基准

### 素材轮换周期

| 预算+受众组合 | 轮换周期 |
|---|---|
| 高预算+小受众 | 3-4天 |
| 中等预算+中等受众 | 7-10天 |
| 低预算+大受众 | 14-21天 |
| Facebook整体 | 2-4周 |
| TikTok | 每周/每日 |

### 触发换素材的信号
- Frequency > 2.5
- CTR 下降 > 20%
- CPA 上升 30%+
- 素材跑了 2-4 周以上

---

## 3. 迭代三大策略

### 3.1 单素材迭代
- 保留表现最好的素材框架
- 只改变量：Hook 开头 / CTA / 文案
- 每周迭代 1-2 条

### 3.2 多素材迭代
- 保留 Top 3 素材
- 每周新增 2-3 条不同角度素材
- 淘汰表现最差的 20%

### 3.3 跨行迭代
- 借鉴其他行业/竞品的成功素材
- 适配到自己的产品/受众
- 注意：不要直接抄袭，要创新

---

## 4. 素材疲劳预防 SOP

### 日常预防
1. **每周产出**：5-8 条新素材
2. **每周测试**：2-3 条
3. **每周替换**：淘汰 1-2 条疲劳素材
4. **监控 Frequency**：> 2.5 时准备新素材

### 紧急处理
- **CTR 暴跌**：立即换 Hook
- **CPA 飙升**：检查是否素材疲劳或竞争加剧
- **Frequency > 3.5**：立即扩展受众或换素材

---

## 5. 诊断流程

```
素材表现下降
  ├── 查 Frequency → > 2.5？→ 素材疲劳 → 换素材
  ├── 查 CTR → 下降 > 20%？→ Hook 疲劳 → 改 Hook
  ├── 查 CVR → 下降？→ 落地页问题 → 查网站
  └── 都正常？→ 可能是竞争加剧 → 等/换市场
```

## 6. 素材生命周期管理

> 每条素材从出生到死亡有明确的阶段。了解阶段才能精准行动。

### 素材生命周期五阶段

```
诞生 → 测试 → 放量 → 稳定 → 衰退
  │       │       │       │       │
  1-3天   3-7天   7-14天  14-28天  28天+
  制作    A/B/C   加预算  维持    替换
          测试    扩量    轮换    关停
```

### 各阶段操作清单

| 阶段 | 特征 | 核心动作 | 禁忌 |
|------|------|---------|------|
| **诞生** | 刚制作完成 | 立即进入测试 Campaign | 不要直接上生产 |
| **测试** | 3天数据 | 看 CTR + Thumb-Stop | 不要大改，只看数据 |
| **放量** | CTR > 1.5%, CPA < 目标 | 导入生产 Campaign，加预算 | 不要一次加 > 20% |
| **稳定** | 数据平稳 7天+ | 维持现状，准备裂变版本 | 不要频繁调预算 |
| **衰退** | CTR 下降, CPA 上升 | 换素材/换 Hook | 不要继续硬撑 |

## 7. 疲劳预警日历

> 不用每天查疲劳。用日历系统化管理。

### 周一检查（全量）

- [ ] 列出所有在跑素材（> 7 天的）
- [ ] 标出 Frequency > 2.0 的
- [ ] 标出 CTR 比上周下降 > 15% 的
- [ ] 标出 CPA 比上周上涨 > 20% 的
- [ ] 标记"预警"和"需替换"

### 周三检查（预警素材）

- [ ] 复查周一标记的"预警"素材
- [ ] 如果指标继续恶化 → 升级到"需替换"
- [ ] 如果指标稳定 → 降级为"观察"

### 周五检查（行动日）

- [ ] 关停所有"需替换"素材
- [ ] 上线本周制作的新素材
- [ ] 更新素材状态标签

### 素材疲劳预测公式

```
预计疲劳天数 = (受众大小 ÷ 日均触达) × 2.5

示例：
受众 = 500万人
日均触达 = 10万人
预计疲劳天数 = (5,000,000 ÷ 100,000) × 2.5 = 125 天

受众 = 10万人
日均触达 = 5万人
预计疲劳天数 = (100,000 ÷ 50,000) × 2.5 = 5 天
```

**结论**：小受众 + 高预算 = 素材疲劳极快（3-5天），需要高频换素材。

## 8. 疲劳后的紧急应对手册

### 场景 A：唯一跑量素材突然疲劳

```
症状：之前 CPA $15 的素材，今天 CPA 涨到 $35
动作：
1. 立即复制该素材，改 3 个不同 Hook（V1-Hook-A/B/C）
2. 同时上新 2 条全新素材
3. 紧急情况下，可临时降低预算 30% 延缓疲劳
4. 24 小时内必须上线新素材
```

### 场景 B：多条素材同时疲劳

```
症状：3+ 条素材在同周出现疲劳信号
原因：通常是季节性竞争加剧或受众耗尽
动作：
1. 不要慌，先确认是素材疲劳还是市场竞争
2. 如果 CPM 同步上涨 → 竞争问题，接受短期成本上涨
3. 如果 CPM 正常但 CTR 降 → 素材疲劳，需要全新方向
4. 紧急从竞品广告库找灵感（Facebook Ad Library）
5. 一周内上线 5-8 条全新素材
```

### 场景 C：新素材上线就疲劳

```
症状：新素材跑了 3 天，CTR 只有 0.5%
原因：可能是素材类型/方向根本不对
动作：
1. 不要继续用同一个素材类型
2. 换完全不同的素材类型（如从品牌片换 UGC）
3. 重新审视产品卖点和目标受众的匹配度
4. 测试 3 种完全不同的 Hook 方向
```
