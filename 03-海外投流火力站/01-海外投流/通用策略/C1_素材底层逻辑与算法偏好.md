## 1. 注意力经济

### 核心结论
- Meta 广告的本质是**注意力竞价**：谁能在前 3 秒抓住用户，谁就赢了一半
- **47% 的广告成果在前 3 秒决定**
- 算法偏好"让用户停留更久"的素材 → 停留时间 = 信号 = 更便宜的 CPM

### Thumb-Stop Rate（拇指停止率）
- **定义**：3秒播放 ÷ 展示
- **基准**：视频广告 > 25% 为合格，> 35% 为优秀
- **提升方法**：强视觉 Hook、对比色、动态元素、人脸/情绪

## 2. Hook 5 大类型

| 类型 | 示例 | 适用场景 |
|------|------|---------|
| **情绪触发型** | "你还在用X方法？小心Y后果" | 恐惧/焦虑驱动 |
| **意外场景型** | Pattern Interrupt（打破预期） | 吸引注意力 |
| **数字钩子型** | "月薪3万的人在用什么理财方式？" | 好奇心驱动 |
| **反常识陈述型** | "停止健身，否则你会..." | 悬念驱动 |
| **问题钩子型** | "眼睛累？试试这个" | 痛点共鸣 |

## 3. 三段式脚本框架

### 结构
```
Hook（前3秒）→ Body（核心价值）→ CTA（行动号召）
```

### Hook（前3秒）
- **目标**：让用户停下来
- **方法**：强视觉/情绪/数字/反常识
- **禁忌**：不要放 Logo 或品牌名

### Body（核心价值）
- **时长**：15-30 秒为黄金时长（Facebook Feed）
- **内容**：产品价值/使用场景/社会证明
- **节奏**：每 3-5 秒一个视觉变化

### CTA（行动号召）
- **方法**：明确指令 + 紧迫感
- **测试**：不同 CTA 文案有 20-40% CTR 差异
  - "Shop Now" vs "Get Offer"
  - "Learn More" vs "Send Message"

## 5. DCO 动态创意优化

### 原理
- Meta 自动组合不同的素材/文案/CTA，找到最佳组合
- 适合：多素材/多文案测试

### 设置
1. 上传 3-5 张图片/视频
2. 写 3-5 条文案
3. 选 2-3 个 CTA
4. 开启 DCO → Meta 自动测试所有组合

## 6. 静音优先设计

### 核心结论
- **85% 的 Facebook 视频在静音状态下观看**
- 素材必须在静音状态下也能传达核心信息
- 使用字幕、文字叠加、视觉叙事

### 静音优化清单
- [ ] 添加字幕（硬编码）
- [ ] 前 3 秒用视觉 Hook，不依赖声音
- [ ] 关键信息用文字叠加
- [ ] 测试静音播放效果

## 8. 视频脚本模板（3 套直接用）

### 模板 A：15 秒带货视频（电商标准）

```
0-3s  [HOOK] 视觉冲击 + 问题
      "你是不是也在为这个问题烦恼？"
      （画面：痛点场景展示）

3-8s  [VALUE] 产品展示 + 核心卖点
      "这个产品帮你解决了……"
      （画面：产品使用过程）

8-12s [PROOF] 社会证明
      "已有 10,000+ 用户选择"
      （画面：用户评价/数据）

12-15s [CTA] 明确指令
      "点击链接，今天下单享 XX 折"
      （画面：产品 + 折扣信息 + 箭头指向按钮）
```

### 模板 B：30 秒 UGC 证言视频

```
0-3s  [HOOK] 真实场景开场
      "我用这个产品已经 3 个月了"
      （画面：真实用户在家中）

3-10s [PROBLEM] 使用前痛点
      "之前我一直在用 X 方法，但是……"
      （画面：痛点展示）

10-20s [SOLUTION] 使用过程和效果
      "用了这个之后，最大的变化是……"
      （画面：产品使用 + 效果展示）

20-25s [RESULT] 量化结果
      "具体来说，节省了 XX 时间/钱"
      （画面：数据/对比）

25-30s [CTA] 行动号召
      "链接在下方，推荐你们试试"
```

### 模板 C：6 秒 Reels 短视频

```
0-2s  [HOOK] 结果前置
      （画面：最终效果/震撼视觉）

2-4s  [VALUE] 一句话卖点
      （画面：产品特写 + 文字叠加）

4-6s  [CTA] 快速引导
      （画面：箭头 + "Shop Now" 按钮指向）
```

## 9. 素材制作成本矩阵

> 知道花多少钱做素材，才能算出"素材 ROI"。

### 制作方式 vs 成本 vs 质量

| 制作方式 | 单条成本 | 周期 | 质量 | 适合 |
|---------|---------|------|------|------|
| **手机自拍** | $0-50 | 1 天 | 中（但真实感强） | UGC/工厂实拍 |
| **Fiverr 外包** | $50-200 | 3-5 天 | 中 | 图文/简单视频 |
| **Upwork 专业** | $200-800 | 1-2 周 | 高 | 品牌片/教程 |
| **UGC Creator** | $100-500/条 | 1 周 | 高（真实感最强） | UGC/证言 |
| **本地 Agency** | $500-3000 | 2-4 周 | 最高 | 品牌形象片 |
| **AI 生成** | $10-50 | 几小时 | 低-中 | 图文/简单动画 |

### 素材 ROI 计算

```
素材 ROI = 该素材带来的总收入 - 制作成本 - 广告花费

示例：
UGC 视频制作成本 $200
广告花费 $3,000
带来收入 $12,000
素材 ROI = $12,000 - $200 - $3,000 = $8,800
```

### 各阶段素材预算占比

| 阶段 | 月广告预算 | 建议素材月预算 | 占比 |
|------|-----------|--------------|------|
| 测试期 | $1,000-3,000 | $200-500 | 15-20% |
| 成长期 | $3,000-10,000 | $500-1,500 | 10-15% |
| 成熟期 | $10,000-50,000 | $1,500-5,000 | 8-12% |
| 企业级 | $50,000+ | $5,000-15,000 | 5-10% |

## 10. 图片广告 vs 视频广告选择

### 何时用图片？何时用视频？

| 场景 | 推荐 | 原因 |
|------|------|------|
| 新产品上市 | 视频 | 需要展示功能和使用场景 |
| 促销/折扣 | 图片 | 信息简单直接 |
| B2B 询盘 | 图片 + 视频混合 | 图片做品牌，视频做信任 |
| Reels 版位 | 视频（必须） | 版位限制 |
| Feed 版位 | 混合测试 | 先测再决定 |
| Stories 版位 | 视频（9:16） | 竖屏视频效果最佳 |

### 图片广告的最佳实践

- **文字占比 < 20%**（Facebook 已不限制，但过多文字 CTR 仍低）
- **产品占画面 60%+**（不要放太小）
- **用高对比色**（在灰色/白色 Feed 中突出）
- **人脸 > 产品 > 文字**（有脸的图片 CTR 高 30%+）
- **用箭头/指向引导视线**（指向 CTA 方向）
