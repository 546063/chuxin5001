## 1. CPM 暴涨排查

### 症状
CPM 从正常水平上涨 > 30%，且持续 3 天以上

### 排查路径
```
CPM 暴涨
  ├── 检查时间 → 是否 Q4 / 黑五 / 大促期？
  │     ├── 是 → 正常季节性上涨，等竞争消退或调高预算应对
  │     └── 否 → 继续排查
  ├── 检查受众 → 是否受众太小？
  │     ├── 是 → 切 Broad 或 Advantage+ Audience
  │     └── 否 → 继续排查
  ├── 检查版位 → 是否手动选了贵版位？
  │     ├── 是 → 切 Advantage+ Placements
  │     └── 否 → 继续排查
  ├── 检查广告质量排名 → 是否 Below Average？
  │     ├── 是 → 素材质量问题，换素材
  │     └── 否 → 继续排查
  └── 竞品分析 → 是否竞品加大了投放？
        ├── 是 → 考虑错峰或差异化受众
        └── 否 → 可能是平台级 CPM 上涨，监控行业基准
```

### 常见原因与解法

| 原因 | 判断方法 | 解法 |
|---|---|---|
| 季节性竞争 | 查看是否进入 Q4/黑五/圣诞/双11 | 提前 2 周加预算，避开高峰时段 |
| 受众太窄 | 受众规模 < 10 万 | 扩大受众：LAL 1%→3%，或加兴趣标签 |
| 素材质量分低 | Quality Ranking = Below Average | 换素材，提高参与度 |
| 版位选择不当 | 只选了 Feed（最贵版位） | 开启 Advantage+ Placements，让系统自动分配 |
| 账户评分下降 | 账户质量评分降低 | 减少拒审广告，提升落地页质量 |

### 解法总结
- **季节性**：接受或上调 Cost Cap 20-30%
- **受众窄**：切 Broad，受众越大 CPM 越低
- **版位**：用自动版位，手动版位通常更贵
- **素材质量**：低质量素材 CPM 更高（Meta 惩罚）

## 2. CTR 低排查

### 症状：CTR < 0.8% 或 Link CTR 下降 > 30%

**排查顺序：素材 → 受众 → 版位**

### 排查路径
```
CTR 下降
  ├── 检查运行时长 → 是否跑了 2 周+？
  │     ├── 是 → 素材疲劳，换新素材
  │     └── 否 → 继续排查
  ├── 检查频率 → 是否 > 3？
  │     ├── 是 → 同一用户看太多次，扩展受众或换素材
  │     └── 否 → 继续排查
  ├── 检查素材与受众匹配度 → 素材是否对的人看？
  │     ├── 不匹配 → 调整定向或用 Broad 让算法找
  │     └── 匹配 → 继续排查
  └── 检查展示环境 → 是否主要在 Reels 展示？
        └── Reels CTR 通常低于 Feed → 正常，关注 ThruPlay 替代
```

### 常见原因与解法

| 原因 | 判断方法 | 解法 |
|---|---|---|
| 素材不吸引 | Hook 3s 留存率 < 40% | 重做 Hook（参考 C1 的 5 种 Hook 类型） |
| 受众不匹配 | 同一素材在不同受众 CTR 差异大 | 换受众测试 |
| 版位竞争 | Stories 版位 CTR 通常低于 Feed | 调整素材尺寸适配版位（9:16 vs 1:1） |
| 广告文案差 | 素材点击率正常但链接点击率低 | 优化 CTA 和链接描述 |

### 解法总结
- **素材疲劳**（最常见）：每周上 2-3 条新素材
- **频率过高**：扩展受众或轮换素材
- **受众不匹配**：Broad 定向 + 好素材
- **版位差异**：Reels CTR 天然低于 Feed，用 ThruPlay 评估

## 3. CPA 高排查（漏斗分析）

### 症状：CPA > 毛利的 50% 或 CPA 上涨 > 30%

```
排查路径：
CPA = 总花费 ÷ 转化数
     = (CPM × 1000) ÷ (CTR × CVR)
     
CPA 高 = CPM高 或 CTR低 或 CVR低
```

### 排查路径
```
CPA 升高
  ├── 检查 CPM → 涨了？→ 见上方 CPM 排查
  ├── 检查 CTR → 降了？→ 见上方 CTR 排查
  ├── 检查 CVR → 降了？→ 落地页/产品问题
  │     ├── 网站加载速度？→ 优化速度
  │     ├── 价格竞争力？→ 对比竞品
  │     ├── 库存/断货？→ 补充库存
  │     ├── 支付问题？→ 测试结账流程
  │     └── 社会证明不足？→ 加评价/案例
  ├── 检查学习期 → 是否刚重置？
  │     └── 是 → 等 3-5 天再评估
  ├── 检查竞争 → 是否有新竞品进入？
  │     └── 是 → 差异化或错峰
  └── 检查归因 → 是否 iOS 归因变化？
        └── 对比第三方数据（Shopify/GA4）
```

### 漏斗分层排查

| 环节 | 症状 | 原因 | 解法 |
|---|---|---|---|
| 展示层 | CPM > 行业均值×2 | 受众窄/竞争大 | 扩受众 / 换版位 |
| 点击层 | CTR < 1% | 素材差/受众不匹配 | 换素材（优先）/ 换受众 |
| 转化层 | CVR < 1.5% | 落地页问题 | 优化加载速度/产品图/信任元素 |
| 定价层 | AOV 太低导致 ROAS 差 | 产品定价策略 | 捆绑销售 / Upsell / 提价测试 |

## 4. 学习期过不了排查

### 症状：Ad Set 状态一直是 "Learning Limited"

| 原因 | 判断标准 | 解法 |
|---|---|---|
| 事件太少 | 优化事件 < 50 次/周 | 换优化目标（Purchase→AddToCart）或加预算 |
| 预算太低 | 日预算 < $5/Ad Set | 每组至少 $10-15/天 |
| 受众太窄 | 受众 < 1000 人 | 扩大到至少 10 万 |
| 素材差 | CTR < 0.5% | 换素材 |
| 落地页差 | 跳出率 > 80% | 优化落地页 |
| 出价太低 | 竞拍排名很低 | 提高出价或换最低成本 |

---

## 5. ROAS 下滑排查

### 症状：ROAS 从 3.5 掉到 2.0

**排查顺序：先素材 → 再受众 → 最后出价**

1. **查素材**：Frequency > 2.5？CTR 连续下降？→ 素材疲劳，换素材
2. **查受众**：受众是否已经洗过？→ 扩 LAL 百分比 / 加 Broad Interest
3. **查出价**：出价是否被竞品超越？→ 适当提高出价上限
4. **查季节**：是否进入淡季？→ 调整预算分配，测试新市场
5. **查归因**：归因窗口是否被改？→ 检查 Ads Manager 设置

### 快速判断框架
```
ROAS 下滑
  ├── CPM 上涨 > 30% → 竞争问题（等/换市场/调出价）
  ├── CTR 下降 > 30% → 素材疲劳（换新素材）
  ├── CVR 下降 > 30% → 落地页/产品问题（查网站/价格/库存）
  └── 三个都正常但 ROAS 降 → 客单价降或归因问题
```

## 6. 不花费 / 花费极慢排查

### 排查路径
```
不花费
  ├── 检查广告审核 → 是否 Disapproved / Under Review？
  │     └── 是 → 修改违规内容或申诉
  ├── 检查出价 → Bid Cap / Cost Cap 是否太低？
  │     └── 是 → 上调或切 Lowest Cost
  ├── 检查受众 → 是否太小？
  │     └── 是 → 扩展受众
  ├── 检查预算 → 日预算是否太低？
  │     └── 是 → 提高预算到目标 CPA × 3 以上
  ├── 检查优化事件 → 是否选了太深的事件但数据不够？
  │     └── 是 → 先优化更浅层事件积累数据
  └── 检查支付 → 付款方式是否过期/被拒？
        └── 是 → 更新付款信息
```

## 7. 广告被拒审排查

### 常见拒审原因

| 原因 | 典型触发 | 解法 |
|---|---|---|
| 误导性声明 | "100%治愈"/" Guaranteed结果" | 删除绝对化承诺 |
| 不合规图片 | 前后对比图/过度修图 | 使用真实产品图 |
| 落地页问题 | 虚假促销/隐藏费用 | 确保落地页与广告一致 |
| 受限品类 | 医疗/金融/成人内容 | 查白名单政策 |
| 个人属性 | "你胖吗"/"单身吗" | 改为间接表述 |

**申诉流程**：Ads Manager → 帐户内容品质 → 申请复审 → 附说明信

## 8. 频率过高排查

### 症状
Frequency > 5（冷受众）或 > 8（再营销）

### 解法
- **冷受众**：扩展受众 / 加新素材 / 降低预算
- **再营销**：缩短再营销窗口（180天 → 30天）/ 轮换再营销素材
- **通用**：检查是否有多个 Campaign 重叠投放同一受众

## 9. 诊断速查表

| 症状 | 首要排查指标 | 最可能原因 | 第一行动 |
|------|------------|-----------|---------|
| CPA 高 | CPM / CTR / CVR | 三选一问题 | 看哪个指标异常 |
| 不花费 | 审核状态 / 出价 | 审核未通过 / 出价太低 | 查审核，调出价 |
| ROAS 降 | AOV / CVR | 客单价降 / 转化降 | 查业务数据 |
| 花费太快 | CPM 低 / CTR 高 | 竞争激烈但素材好 | 正常，检查 CPA |
| 频率高 | Reach / 受众大小 | 受众太小 | 扩展受众 |

## 10. 广告被拒/账户被封深度排查

### 账户质量评分自查

| 评分 | 含义 | 行动 |
|------|------|------|
| 4-5/5 | 优秀 | 保持现状 |
| 3/5 | 警告 | 检查被拒广告，清理违规内容 |
| 1-2/5 | 危险 | 立即停投，全面整改 |

### 封号分级应对

| 级别 | 症状 | 应对 | 恢复周期 |
|------|------|------|---------|
| **广告被拒** | 单条广告不通过 | 修改后申诉 | 2-24h |
| **广告账户受限** | 无法创建新广告 | 提交业务证明 | 3-7 天 |
| **BM 受限** | 整个 BM 无法操作 | 准备完整业务材料 | 7-30 天 |
| **BM 永久封禁** | 无法申诉 | 准备新 BM 重新开始 | 1-3 天 |

### 新 BM 安全启动清单

- [ ] 全新电脑/IP（不要用被封的设备的浏览器指纹）
- [ ] 全新企业信息（名称、地址、网站）
- [ ] 全新支付方式
- [ ] 前 3 天只投品牌宣传类广告
- [ ] 预算从 $10-20/天开始
- [ ] 落地页 100% 合规（隐私政策、联系方式、退换货）
- [ ] 广告文案不含敏感词（赚钱、减肥、美白、治疗）

## 11. 落地页诊断 SOP

### 广告 OK 但转化差的落地页排查

```
落地页诊断
  ├── 加载速度（PageSpeed Insights 测试）
  │     ├── > 3s → 优化图片/启用 CDN/减少 JS
  │     └── < 3s → 继续排查
  ├── 移动端适配
  │     ├── 按钮太小/文字太小/图片变形 → 修复
  │     └── 移动端 OK → 继续排查
  ├── 首屏与广告一致性
  │     ├── 首屏承诺 ≠ 广告 Hook → 统一
  │     └── 一致 → 继续排查
  ├── 社会证明
  │     ├── 无评价/无销量/无信任标志 → 添加
  │     └── 有 → 继续排查
  ├── CTA 明确度
  │     ├── 没有明确按钮 → 加大/加颜色/改文案
  │     └── 有 → 继续排查
  ├── 支付/表单复杂度
  │     ├── 表单 > 5 个字段 / 结账 > 3 步 → 精简
  │     └── 简洁 → 继续排查
  └── 价格竞争力
        ├── 比竞品贵 > 20% → 调整或增加价值包装
        └── 合理 → 可能是流量本身质量问题
```

## 12. 跨账户交叉诊断

### 多 Campaign 同时异常的排查

| 症状 | 排查方向 | 解法 |
|------|---------|------|
| 所有 Campaign CPA 同时升高 | 平台级 CPM 上涨 / 季节性 | 接受短期波动或错峰 |
| 所有 Campaign 花费突然下降 | 付款方式问题 / 账户级别限流 | 查付款状态、联系 Meta 支持 |
| 所有 Campaign CTR 同时下降 | 素材类型过时 / 市场变化 | 换全新素材方向 |
| 只有某个 Campaign 异常 | 该 Campaign 内部问题 | 按漏斗排查 |
| Pixel 事件突然停止回传 | 网站更新/代码变动/插件冲突 | 用 Pixel Helper 排查 |

## 13. 诊断优先级矩阵

### 遇到问题时按此顺序排查

| 优先级 | 排查项 | 耗时 | 解决概率 |
|--------|--------|------|---------|
| **P0** | 广告审核状态 | 1 分钟 | 20% |
| **P0** | Pixel/CAPI 是否正常回传 | 3 分钟 | 15% |
| **P1** | 素材疲劳（CTR + Frequency） | 2 分钟 | 35% |
| **P1** | CPM 异常 | 2 分钟 | 15% |
| **P2** | 落地页问题 | 5 分钟 | 25% |
| **P2** | 受众重叠/饱和 | 3 分钟 | 10% |
| **P3** | 竞争加剧 | 5 分钟 | 5% |
| **P3** | 归因窗口变化 | 1 分钟 | 5% |

**经验法则**：80% 的问题出在素材疲劳或 CPM 变化，先查这两个。
