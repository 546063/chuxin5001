# 学习期与归因机制

## 一句话总结

Meta 默认归因窗口已从 28 天缩短至 7 天点击，学习期需 7 天内积累 50 次优化事件才能退出。

## 核心结论

- BM 和 GA 数据差异 20-40% 属正常，不要用 GA 作为 FB 广告的"真实数据源"
- 每日手动调整超过 1 次会导致 ROAS 下降 15%
- 使用 CBO 的广告主 CPA 平均降低 12%
- 勘探 vs 再营销预算比例 = 60-70% : 30-40%

## 适用场景

- 新广告上线后学习期管理
- 理解为什么 BM 数据和独立站数据对不上
- 选择合适的归因窗口
- 预算分配策略制定

## 具体内容

### 1. 归因模型演变

| 时期 | 归因窗口 | 影响 |
|------|---------|------|
| 2021 前 | 28天点击 + 28天浏览 | 数据完整 |
| 2021-2023 | 7天点击 + 1天浏览 | 数据缩减 |
| 2024+ | 7天点击为默认 | 进一步缩减 |

同一广告，归因窗口缩短后 ROAS 可能"看起来"降了 20-40%，但实际转化没变。

### 2. 归因窗口选择

| 归因窗口 | 说明 | 何时使用 |
|----------|------|---------|
| 7-Day Click | 点击广告后 7 天内转化 | 默认推荐 |
| 1-Day View | 观看广告后 1 天内转化 | 叠加使用 |
| 7D Click + 1D View | 两者都算 | 品牌类广告 |
| 28-Day Click（旧制）| 点击后 28 天 | 长决策周期产品 |

### 3. 学习期核心规则

- **退出条件**：7 天内 50 次优化事件
- **公式**：最低日预算 = (50 × 目标 CPA) ÷ 7

| 目标 CPA | 最低日预算 | 建议 |
|----------|-----------|------|
| $10 | $72/天 | 可行 |
| $20 | $143/天 | 可行 |
| $30 | $214/天 | 需要一定预算 |
| $50 | $357/天 | 预算压力大 |
| $100 | $714/天 | 考虑优化更浅层事件 |

### 4. 预算不够 50 转化的解法

1. 优化更浅层事件：Purchase → AddToCart → Landing Page View
2. 减少 Ad Set 数量：集中数据到 1-2 个 Ad Set
3. 提高预算：集中预算
4. 合并 Ad Set：如果多个 Ad Set 都卡在有限学习

### 5. 跨工具数据差异

| 工具 | 数据差异 | 原因 |
|------|---------|------|
| Meta Ads Manager | 基准 | Meta 归因模型 |
| Google Analytics 4 | 通常低 15-30% | 最后点击归因，丢失 fbclid |
| Shopify 后台 | 通常低 10-20% | 自有归因逻辑 |
| Triple Whale | 通常高 10-20% | 第三方像素，归因更宽松 |

### 6. 增量测量方法

| 方法 | 原理 | 适用场景 | 成本 |
|------|------|---------|------|
| MTA（多触点归因）| 追踪每个触点贡献 | 中小规模 | 低 |
| MMM（营销组合模型）| 统计建模分析渠道贡献 | 大预算、多渠道 | 中 |
| Incrementality（增量测试）| A/B 实验：看 vs 不看广告的差异 | 验证广告真实 ROI | 高 |

### 7. 加速学习期的 6 种方法

| 方法 | 效果 | 风险 |
|------|------|------|
| 提高日预算 | 最直接有效 | 素材差会多花钱 |
| 降优化事件层级 | 事件量可增 3-5 倍 | 目标变浅 |
| 合并 Ad Set | 数据集中 | 丧失受众细分 |
| 用 CBO 替代 ABO | 预算自动倾斜 | 可能忽视慢热 Ad Set |
| Broad 替代精准定向 | 展示量可增 5-10 倍 | 初期转化率可能下降 |

## 可复用方法

### 数据信任优先级

```
当不同数据源冲突时：
1. CAPI + Pixel 去重后数据（最可信）
2. 广告管理器内显示数据（含建模转化）
3. GA4 数据（注意归因窗口差异）
4. 第三方归因工具（参考用）
5. 独立站后台订单数据（最准确但不包含广告归因）
```

### 每日监控优先级

1. Meta Ads Manager → CPA、ROAS、消耗率
2. Shopify 后台 → 实际订单数、收入
3. GA4 → 趋势验证（不纠结绝对值）

### 归因窗口与产品决策周期匹配

| 产品类型 | 决策周期 | 推荐归因窗口 |
|---------|---------|-------------|
| 冲动购买（<$20） | 0-24 小时 | 7-Day Click |
| 普通电商（$20-100） | 1-7 天 | 7-Day Click |
| 高客单价（$100-500） | 3-14 天 | 7D Click + 1D View |
| B2B/SaaS（$500+） | 7-30 天 | 需要增量测试验证 |

## 对我的业务有什么价值

- 不再为 BM 和 GA 数据差异而焦虑
- 知道学习期的正确管理方式，避免频繁调整
- 能用 MER 做最终校准，不依赖单一归因
- 预算分配有据可依（60-70% 勘探 + 30-40% 再营销）

## 相关案例

- 某电商发现 BM 显示 10 个转化，Shopify 显示 15 个，差异 50% 正常
- 学习期退出后 3-7 天蜜月期，CPA 比稳定期低 10-20%
