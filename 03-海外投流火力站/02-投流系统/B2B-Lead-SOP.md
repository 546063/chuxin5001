# B2B Lead SOP

## 一句话总结
> B2B Lead SOP分五个阶段：Lead获取→初次响应→需求诊断→方案报价→成交维护，每阶段有标准动作和时效要求，目标是提高转化率。

---

## Lead 五个阶段

### Stage 1: Lead 获取 (Lead Capture)
**目标**: 全渠道获取潜在客户
**KPI**: Lead数量, 来源分布

**Lead来源**:
| 来源 | 占比目标 | 质量 |
|------|----------|------|
| Facebook广告 | 30% | 中 |
| TikTok询盘 | 20% | 中 |
| LinkedIn开发 | 20% | 高 |
| WhatsApp咨询 | 20% | 高 |
| 官网表单 | 10% | 中 |

**捕获标准**:
```
必填信息：
□ 姓名/公司名
□ 国家
□ 产品需求
□ 联系方式

可选信息：
□ 公司规模
□ 采购预算
□ 采购时间
□ 现有供应商
```

---

### Stage 2: 初次响应 (Initial Response)
**目标**: 4小时内回复，赢得沟通机会
**时效**: <4小时 (黄金时间是1小时内)
**KPI**: 响应率, 首次回复时长

**初次响应流程**:
```
1. 收到Lead通知
2. 识别来源和需求
3. 选择话术模板
4. 发送首次回复
5. 记录到CRM
6. 设置跟进提醒
```

**首次回复模板**:
```
[来自广告/询盘]

您好[客户姓名]！👋

感谢您关注我们的[产品名]！

我是[姓名]，负责[市场]市场的销售。

您咨询的[产品名]参数如下：
- 产能: [X]-[Y] kg/h
- 功率: [Z] kW
- 材质: SUS304不锈钢
- 认证: CE/SASO/SIRIM

想进一步了解：
1. 您主要用于什么产品/场景？
2. 需要的产能大概是多少？
3. 您的目标市场是哪个国家？

我可以根据您的需求给您推荐合适的型号和报价 😊

期待您的回复！
```

---

### Stage 3: 需求诊断 (Needs Discovery)
**目标**: 全面了解客户需求
**时长**: 1-2周
**KPI**: 信息完整度, 需求明确度

**诊断清单**:
```
【必问问题】
□ 具体产品需求 (哪种设备)
□ 应用场景 (食品类型)
□ 产能要求 (kg/h)
□ 目标市场 (哪个国家)
□ 认证要求 (SASO/SIRIM等)
□ 采购时间 (何时需要)
□ 预算范围 (大致的)
□ 现有设备 (是否替换)

【可选问题】
□ 公司规模人数
□ 决策链 (谁决定)
□ 现有供应商
□ 期望的合作方式
```

**诊断记录模板**:
```
客户信息：
- 姓名: [XXX]
- 公司: [XXX]
- 国家: [XXX]
- 规模: [XXX]

需求分析：
- 产品: [XXX]
- 产能: [XXX]
- 场景: [XXX]
- 市场: [XXX]

采购信息：
- 预算: [XXX]
- 时间: [XXX]
- 决策链: [XXX]

痛点：
1. [XXX]
2. [XXX]

销售机会：⭐⭐⭐⭐⭐
推荐方案：[XXX]
下一步：[XXX]
```

---

### Stage 4: 方案报价 (Proposal & Quote)
**目标**: 提供专业方案，促成询价
**时长**: 1-3天
**KPI**: 报价发出率, 报价转化率

**报价流程**:
```
1. 根据需求选配产品
2. 确定价格条款 (FOB/CIF/DDP)
3. 计算总价和交期
4. 生成报价单 (PDF/Excel)
5. 发送并跟进
6. 记录报价版本
```

**报价单结构**:
```
[公司Logo]

报价单 QUOTATION

客户: [客户名]
日期: [日期]
有效期: 30天

产品清单：
┌─────────────────────────────────────────┐
│ # │ 产品 │ 型号 │ 数量 │ 单价 │ 总价 │
├─────────────────────────────────────────┤
│ 1 │ [产品A] │ [型号] │ [数量] │ $XXX │ $XXX │
│ 2 │ [产品B] │ [型号] │ [数量] │ $XXX │ $XXX │
└─────────────────────────────────────────┘

总价: $XXX (FOB Qingdao)

付款条款: T/T 30%预付 + 70%发货前
交货期: 收到预付后[X]-[Y]天
有效期: 30天

备注:
- [认证信息]
- [包装信息]
- [服务承诺]
```

**报价后跟进**:
| 时间 | 内容 |
|------|------|
| 第1天 | 确认收到报价 |
| 第3天 | 询问是否有问题 |
| 第7天 | 发送客户案例 |
| 第14天 | 发送优惠信息 |
| 第21天 | 最终确认 |

---

### Stage 5: 成交维护 (Closing & After-sales)
**目标**: 完成订单，建立长期关系
**KPI**: 成交率, 复购率, 客户满意度

**成交流程**:
```
1. 确认订单细节
2. 发送PI (Proforma Invoice)
3. 收到预付
4. 进入生产
5. 生产完成验货
6. 收到尾款
7. 发货
8. 客户收货
9. 安装调试 (如有)
10. 售后服务跟进
11. 复购和口碑推荐
```

**成交维护话术**:
```
【确认订单】
您好！感谢您的信任决定合作 🙏

订单确认：
- 产品: [产品名] x [数量]台
- 总价: $XXX (FOB Qingdao)
- 付款: 30%预付 + 70%发货前

生产计划我会每周给您更新进度 📊

【发货通知】
您好！您的订单已发货 ✈️

预计到达: [日期]
追踪号: [XXX]

清关文件已发送到您邮箱 📄

【使用关怀】
您好！设备使用得怎么样？
有问题随时联系我 😊

另外，如果您的朋友或同行需要设备，推荐成功有奖励哦 🎁
```

## Lead 状态管理

### 状态定义
| 状态 | 定义 | 后续动作 |
|------|------|----------|
| New | 新获取的Lead | 4小时内首次响应 |
| Contacted | 已首次联系 | 进入需求诊断 |
| Qualified | 需求已明确 | 进入方案报价 |
| Quoted | 已报价 | 持续跟进 |
| Negotiating | 谈判中 | 价格/交期谈判 |
| Won | 已成交 | 进入维护 |
| Lost | 流失 | 记录原因 |
| Nurturing | 培育中 | 持续培育 |

### 超时提醒
| 状态 | 超时时间 | 提醒 |
|------|----------|------|
| New | 4小时未响应 | 优先级提醒 |
| Contacted | 7天无跟进 | 跟进提醒 |
| Qualified | 14天无进展 | 激活提醒 |
| Quoted | 21天未回复 | 最终确认 |
| Won | 30天未复购 | 复购激活 |
