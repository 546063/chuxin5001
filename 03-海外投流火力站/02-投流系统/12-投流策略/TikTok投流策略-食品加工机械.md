# TikTok投流策略-食品加工机械

## 平台定位

TikTok在食品机械行业的角色：**低成本引流+品牌曝光**
- CPM仅为Facebook的1/5-1/3
- 适合获取中小B端客户（餐厅老板、小食品厂）
- 直播是差异化的获客方式

---

## 一、受众定向

### Spark Ads（自然内容加热）

| 维度 | 设置 |
|------|------|
| **地域** | 目标市场（建议聚焦1-3个国家） |
| **年龄** | 22-45岁 |
| **兴趣** | 系统自动匹配（基于视频标签） |
| **性别** | 不限 |

### In-Feed Ads（信息流广告）

| 维度 | 设置 |
|------|------|
| **地域** | 东南亚/中东/非洲 |
| **年龄** | 22-50岁 |
| **兴趣标签** | Food, Business, Manufacturing, Restaurant |
| **行为** | 观看过商业/机械类视频的用户 |

---

## 二、素材策略

### 高转化视频特征

| 特征 | 说明 |
|------|------|
| **前3秒** | 必须展示核心卖点（设备运行/成品效果） |
| **时长** | 15-30秒最佳 |
| **节奏** | 快剪，每3-5秒切换镜头 |
| **字幕** | 英文字幕必备，本地语言加分 |
| **BGM** | TikTok热门音乐（非版权限制） |
| **CTA** | 评论区置顶链接+主页Bio |

### 内容类型（按效果排序）

1. **设备运行实拍**：一台机器工作的特写，配"ASMR"风格音效
2. **对比型**：左边手工慢、右边机器快
3. **工厂Tour**：第一人称视角逛工厂
4. **成品展示**：设备做出来的食品成品
5. **知识科普**：食品行业小知识+设备植入

---

## 三、预算设置

| 阶段 | 日预算 | 目标 |
|------|--------|------|
| **测试期** | $5-10/天 | 测试3-5条视频 |
| **优化期** | $10-20/天 | 给表现好的视频加热 |
| **放量期** | $20-50/天 | 多素材并行 |

---

## 四、直播策略

### 直播前

- [ ] 提前3天发布3条预热短视频
- [ ] 准备好设备演示区
- [ ] 测试网络（4G/5G备份）
- [ ] 准备英文/阿拉伯语关键词卡片

### 直播中（60分钟结构）

| 时间段 | 内容 |
|--------|------|
| 0-5 min | 开场+自我介绍+今日内容预告 |
| 5-20 min | 工厂参观（走动拍摄） |
| 20-40 min | 设备演示（核心产品运行） |
| 40-50 min | QA问答（评论提问） |
| 50-60 min | CTA+下期预告 |

### 直播后

- [ ] 24小时内跟进所有评论中的询盘
- [ ] 剪辑直播精华片段，发布2-3条短视频
- [ ] WhatsApp跟进留资客户

---

## 五、免费流量方法

1. **热门话题标签**：#FoodMachinery #FoodProcessing #FactoryTour #BusinessTips
2. **发布频率**：每天1-2条，保持稳定
3. **互动**：回复每条评论，提高互动率
4. **合拍/Duet**：与行业KOL合拍，借力引流
5. **SEO优化**：标题中包含搜索关键词（如 "meat processing machine price"）
