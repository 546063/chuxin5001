# 案例-FB-中央厨房整线解决方案

## 案例概述

| 字段 | 内容 |
|------|------|
| **平台** | Facebook |
| **广告目标** | 留资 + WhatsApp咨询 |
| **推广产品** | 中央厨房整线解决方案 |
| **目标市场** | 中东（沙特、阿联酋、卡塔尔） |
| **广告周期** | 2个月 |

## 受众定向

| 维度 | 设置 |
|------|------|
| **地域** | 沙特+阿联酋+卡塔尔 |
| **年龄** | 28-60岁 |
| **语言** | 阿拉伯语+英语 |
| **兴趣定向** | Central kitchen, Food service, Catering, Restaurant management, Food manufacturing |
| **行为定向** | Business decision makers |
| **排除** | B2C食品消费者兴趣标签 |

## 广告素材

| 要素 | 内容 |
|------|------|
| **素材类型** | 长视频（90秒）+ 案例图片 |
| **内容** | 中央厨房整线运行流程+产能数据+ROI计算 |
| **文案结构** | "一个中央厨房能服务50家餐厅" + 产能数据 + 免费方案咨询CTA |
| **CTA按钮** | "Send WhatsApp Message" |

## 关键数据

| 指标 | 数值 |
|------|------|
| **日均预算** | $40-60 |
| **CPC** | $0.8-1.5 |
| **CTR** | 1.2-2.0% |
| **CPL** | $12-20 |
| **月均留资** | 30-50条 |
| **WhatsApp咨询** | 15-25条/月 |
| **转化率** | 约6% |
| **ROAS** | 约4:1 |

## 成功因素

1. **WhatsApp直连CTA**：中东客户偏好WhatsApp沟通，跳过表单直接对话
2. **产能数据驱动**：用"服务50家餐厅"等具体数字吸引决策者
3. **阿拉伯语+英语双语**：覆盖不同语言偏好的中东客户

## 风险与教训

- **客单价高决策慢**：中央厨房整线客单价$50K-$200K+，决策周期3-6个月
- **竞争对比**：需要与土耳其/欧洲品牌做价格和配置对比
