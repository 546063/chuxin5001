# 案例-LinkedIn-B2B设备经销商招募

## 案例概述

| 字段 | 内容 |
|------|------|
| **平台** | LinkedIn |
| **广告目标** | 经销商合作意向收集 |
| **推广产品** | 经销代理机会 |
| **目标市场** | 全球（重点东南亚+中东+非洲） |
| **广告周期** | 持续投放 |

## 受众定向

| 维度 | 设置 |
|------|------|
| **地域** | 全球（重点发展中国家） |
| **职位** | Business Owner, Managing Director, Sales Director, Distributor |
| **行业** | Industrial Machinery, Food Equipment, Manufacturing |
| **关键词** | Food machinery distributor, Industrial equipment dealer |

## 广告素材

| 要素 | 内容 |
|------|------|
| **素材类型** | Sponsored InMail + Sponsored Content |
| **内容** | 经销商政策介绍 + 利润空间分析 + 合作案例 |
| **文案结构** | "成为XX品牌在贵国的独家经销商" + 利润数据 + 申请CTA |

## 关键数据

| 指标 | 数值 |
|------|------|
| **CPL** | $25-45 |
| **月均意向** | 5-15条 |
| **有效经销商** | 2-5家/月 |
| **最终签约** | 0-1家/月 |

## 成功因素

1. **精准触达**：LinkedIn是唯一能精准找到"设备经销商老板"的平台
2. **InMail直投**：直接发送到目标人群收件箱，打开率高于信息流广告

## 风险与教训

- **成交率极低**：经销商合作是重大决策，从意向到签约可能需6-12个月
- **需要线下跟进**：经销商必须实地考察工厂或参加展会
