# 案例-TikTok-工厂实拍短视频获客

## 案例概述

| 字段 | 内容 |
|------|------|
| **平台** | TikTok |
| **广告目标** | 主页访问 + WhatsApp咨询 |
| **推广产品** | 食品加工机械（通用） |
| **目标市场** | 东南亚（印尼、泰国、越南、菲律宾） |
| **广告周期** | 持续投放 |

## 受众定向

| 维度 | 设置 |
|------|------|
| **地域** | 印尼+泰国+越南+菲律宾 |
| **年龄** | 22-45岁 |
| **兴趣标签** | Food business, Factory, Manufacturing, Business owner |
| **行为** | 观看过机械类视频的用户 |

## 广告素材

| 要素 | 内容 |
|------|------|
| **素材类型** | 短视频（15-30秒） |
| **内容** | 工厂车间实拍+设备运行+工人操作 |
| **风格** | 第一人称视角，"沉浸式参观工厂" |
| **音乐** | TikTok热门BGM（非版权音乐） |
| **文字叠加** | 英文字幕+关键参数 |

## 关键数据

| 指标 | 数值 |
|------|------|
| **日均预算** | $10-20 |
| **CPM** | $2-4 |
| **CPC** | $0.1-0.3 |
| **CTR** | 3-5% |
| **视频完播率** | 25-35% |
| **主页访问量** | 200-500/周 |
| **WhatsApp咨询** | 20-40/周 |

## 成功因素

1. **低成本高曝光**：TikTok CPM仅为Facebook的1/5-1/3，预算利用率高
2. **工厂实拍真实感**：第一人称视角让用户"身临其境"，建立初步信任
3. **热门BGM引流**：借助平台热门音乐提高推荐权重

## 风险与教训

- **线索质量参差**：TikTok用户偏年轻，部分为好奇浏览而非真实采购 → 需要在主页设置筛选问题
- **内容节奏快**：3秒内必须展示核心卖点，否则用户划走
