# 案例-FB-肉制品加工设备留资广告

## 案例概述

| 字段 | 内容 |
|------|------|
| **平台** | Facebook |
| **广告目标** | 留资（Lead Generation） |
| **推广产品** | 肉制品加工设备（绞肉机/斩拌机/灌肠机） |
| **目标市场** | 东南亚（泰国、越南、马来西亚、印尼） |
| **广告周期** | 3个月 |

## 受众定向策略

| 维度 | 设置 |
|------|------|
| **地域** | 泰国+越南+马来西亚+印尼 |
| **年龄** | 25-55岁 |
| **语言** | 英语+泰语+越南语 |
| **兴趣定向** | Food processing, Food industry, Meat processing, Restaurant equipment |
| **行为定向** | Facebook Page admins, Small business owners |
| **自定义受众** | 网站访问者（过去180天） |
| **类似受众** | 已有客户邮箱的1% Lookalike |

## 广告素材

| 要素 | 内容 |
|------|------|
| **素材类型** | 短视频（30-60秒）+ 轮播图 |
| **视频内容** | 工厂实拍+设备运行展示+成品效果 |
| **文案结构** | 痛点钩子（"还在人工切肉？"）+ 价值主张 + CTA |
| **CTA按钮** | "Get Quote" / "Learn More" |
| **广告形式** | Facebook Lead Ad（站内留资表单） |

## 关键数据

| 指标 | 数值 |
|------|------|
| **日均预算** | $30-50 |
| **总花费** | $2,700-4,500/月 |
| **CPC** | $0.5-1.2 |
| **CTR** | 1.5-2.5% |
| **CPL（每条线索成本）** | $8-15 |
| **月均留资数** | 50-80条 |
| **有效询盘率** | 约40-50% |
| **最终转化率** | 约5-8% |

## 成功因素

1. **精准兴趣定向**：锁定"食品加工"相关兴趣标签，减少无效曝光
2. **工厂实拍素材**：真实工厂视频建立信任感，降低B2B客户疑虑
3. **站内留资表单**：减少跳出率，用户无需离开Facebook即可提交信息
4. **多语言覆盖**：英语+本地语言双语文案，提高东南亚市场接受度

## 风险与教训

- **低质量线索**：站内留资门槛低，约50%线索为无效信息 → 需要快速WhatsApp跟进筛选
- **广告疲劳**：素材2-3周后CTR下降 → 需每月更新2-3套新素材
- **地区差异**：不同东南亚国家对设备的偏好不同 → 需要分国家单独优化广告组

## 可复用方法

- 工厂实拍短视频模板可复用到其他产品线
- Lead Ad表单模板可复用到其他平台的留资广告
- 兴趣关键词库可用于TikTok和LinkedIn的定向参考
