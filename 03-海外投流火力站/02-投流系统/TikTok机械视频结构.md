# TikTok 机械类视频结构

## 一句话总结
> 食品机械TikTok视频分效果展示型、产能对比型、工厂探秘型、客户证言型四类；时长15-60秒，字幕必备，前3秒决定留存。

---

## 四大视频类型

### 类型1: 效果展示型 (最高播放)
**目标**: 展示产品核心功能
**时长**: 15-30秒
**爆款率**: 高

**脚本结构**:
```
0-3秒: [开头] 切割特写/清洗效果/去皮效果
      - "Watch this potato peeler in action"
      - 画面: 土豆投入→去皮完成

3-8秒: [中段] 成品展示
      - 展示清洗/切割/去皮后的成品
      - 特写镜头展示效果

8-15秒: [参数] 快速参数说明
      - 字幕: "500kg/h | 1.5kW | SUS304"
      - 展示机器型号

15-30秒: [CTA] 询盘呼吁
      - "Tap to inquire / Link in bio"
      - 展示联系方式
```

**拍摄要点**:
- 近景特写展示切割面光滑度
- 充足光线，背景干净
- 快节奏剪辑 (1-2秒切换)

---

### 类型2: 产能对比型 (高转化)
**目标**: 展示效率优势
**时长**: 30-45秒
**爆款率**: 中

**脚本结构**:
```
0-5秒: [痛点] 人工效率低
      - "Manual cutting vs machine cutting"
      - 对比画面: 人工切 vs 机器切

5-15秒: [方案] 机器运行
      - 机器运行全景
      - 原料投入→成品输出

15-25秒: [数据] 效率对比
      - 字幕: "Manual: 50kg/h | Machine: 500kg/h"
      - 数据可视化

25-35秒: [背书] 客户证言
      - 客户评价片段
      - "We increased capacity by 300%"

35-45秒: [CTA] 询盘呼吁
      - "Get your quote now"
```

**拍摄要点**:
- 对比镜头要有明显差异
- 数据用大字字幕突出
- 客户证言要真实可信

---

### 类型3: 工厂探秘型 (建立信任)
**目标**: 展示生产实力
**时长**: 45-60秒
**爆款率**: 中低

**脚本结构**:
```
0-10秒: [工厂全景] 规模化生产
      - 航拍工厂外景
      - 车间生产线展示

10-20秒: [工艺展示] 精密制造
      - 数控加工过程
      - 焊接/组装特写

20-35秒: [质检展示] 品质把控
      - 质检过程
      - 设备测试运行

35-45秒: [认证展示] 资质齐全
      - CE/SASO/ISO证书
      - 出口包装展示

45-60秒: [CTA] 合作邀请
      - "Factory direct | OEM/ODM welcome"
      - 联系方式
```

**拍摄要点**:
- 工厂要干净整洁
- 工艺要展示专业性
- 员工穿戴整齐

---

### 类型4: 客户证言型 (高信任)
**目标**: 真实案例背书
**时长**: 30-45秒
**爆款率**: 中

**脚本结构**:
```
0-5秒: [客户背景] 身份介绍
      - "Factory in Malaysia | 10 years in business"
      - 工厂外观

5-15秒: [痛点] 之前的问题
      - "We needed to increase capacity"

15-30秒: [解决方案] 设备引入
      - 设备安装过程
      - 调试运行展示

30-40秒: [效果] 改变后的效果
      - 产能数据提升
      - 客户评价

40-45秒: [CTA] 模仿邀请
      - "Get the same result"
      - 联系方式
```

## 爆款公式

### 前3秒留存法则
| 元素 | 示例 |
|------|------|
| 数字冲击 | "500kg/h in just 3 minutes" |
| 效果对比 | Before/After 展示 |
| 悬念提问 | "What happens when..." |
| 动态特写 | 切割特写/清洗涡流 |
| 客户证言 | "This machine changed our business" |

## TikTok广告 vs 自然流量

### 自然流量内容
- 更真实、更随意
- 工厂员工出镜
- 真实客户评价
- 现场生产画面

### 广告投放内容
- 更精致、更专业
- 专业拍摄剪辑
- 明确CTA
- 优化过的转化路径
