## 脚本创作模板

### Hook公式（3秒法则）
```
[惊人数字] + [痛点场景] + [视觉冲击]
例："10人的活现在1个人干" + 分屏对比
```

### Body公式（数据支撑）
```
[客户故事/场景] + [具体数字] + [解决方案展示]
例：迪拜中央厨房 + $28,000/月人工 + 自动化产线演示
```

### CTA公式（降低门槛）
```
[免费/限时] + [具体行动] + [获得价值]
例：Comment 'AUTO' → 获得免费自动化方案
```
