# WhatsApp 转化结构

## 一句话总结
> WhatsApp是中东和东南亚B2B客户主要沟通工具，转化分首次询盘、培育跟进、报价谈判、成交维护四阶段；核心是及时回复和专业话术。

## 转化漏斗四阶段

### 阶段1: 首次询盘 (First Inquiry)
**目标**: 快速响应，获取需求
**响应时间**: <1小时 (最佳), <4小时

**常见询盘类型**:
```
1. "Hi, is this available?"
2. "Price for [product]?"
3. "Do you ship to [country]?"
4. "MOQ and delivery time?"
```

**回复模板**:
```
你好！👋
感谢您的询价！

【[产品名]】报价如下：

📋 基本参数：
- 产能: [X]-[Y] kg/h
- 功率: [Z] kW
- 材质: SUS304不锈钢
- 认证: CE/SASO/SIRIM

💰 价格条款:
- FOB Qingdao: $XXX/台
- CIF [目的港]: $XXX/台

📦 交期: [X]-[Y]天
📦 MOQ: 1台

请告诉我：
1. 目的港是哪个国家？
2. 需要什么认证？
3. 数量多少？

这样我可以给您更准确的报价 😊
```

---

### 阶段2: 培育跟进 (Nurturing)
**目标**: 建立信任，了解需求
**跟进频率**: 第1/3/7/14天

**跟进话题**:
| 时间 | 内容 |
|------|------|
| 第1天 | 感谢询价，补充产品资料 |
| 第3天 | 分享相关客户案例 |
| 第7天 | 询问项目进展 |
| 第14天 | 发送优惠信息 |

**培育内容**:
```
[Day 3 跟进 - 客户案例分享]

您好！
之前分享了我们的土豆去皮机资料，今天给您分享一个马来西亚客户的案例 👇

客户: 马来西亚XX薯条厂
问题: 人工去皮效率低，损耗高
方案: PP-500土豆去皮机
效果: 产能提升300%，损耗从15%降到5%

客户反馈: "机器用了2年，性能稳定，售后响应快"
[客户视频评价] 📹

您有兴趣了解更多吗？可以视频看厂或者寄送样品 😊
```

---

### 阶段3: 报价谈判 (Negotiation)
**目标**: 完成报价，促成订单
**关键节点**: 参数确认→价格谈判→付款条件→交期确认

**报价谈判话术**:
```
【讨价还价】
客户: "Price too high"
回应: "We understand budget is important. For orders of 5+ units, we can offer 5% discount. Also, our machine has 30% lower maintenance cost compared to competitors."

【交期谈判】
客户: "Need faster delivery"
回应: "We have some units in stock. For stock items, we can ship within 7 days. For production order, the standard lead time is 20-30 days."

【付款谈判】
客户: "Can you accept L/C?"
回应: "Yes, for orders over $20,000, we can accept L/C at sight. For smaller orders, we typically use T/T 30% deposit + 70% before shipment."
```

---

### 阶段4: 成交维护 (After Sales)
**目标**: 订单执行，客户复购
**关键节点**: 生产→验货→发货→收货→使用→复购

**成交后维护**:
```
【订单确认】
您好！您的订单已确认 🙏

订单详情：
- 产品: [产品名] x [数量]台
- 总价: $XXX
- 付款: 30%预付 + 70%发货前

生产计划：
- 第1周: 材料准备
- 第2-3周: 生产组装
- 第4周: 测试验货
- 第5周: 包装发货

我会每周给您更新进度 📊

【发货通知】
您好！您的订单已发货 ✈️

发货信息：
- 订单号: [XXX]
- 货代: [XXX]
- 预计到港: [日期]
- 追踪号: [XXX]

清关文件我会同步发送给您 📄

【使用关怀】
您好！设备应该已经到货了吧？
安装过程有问题吗？需要技术支持可以随时联系我 😊

另外，我们有备件套餐优惠，首单订购备件享8折。
```

## 常见问题话术

### 产品相关
| 问题 | 回答 |
|------|------|
| "能用多久？" | "正常使用下寿命10+，主要部件5年" |

### 基础技巧
