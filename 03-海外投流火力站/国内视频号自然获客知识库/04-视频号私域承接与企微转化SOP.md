# 视频号私域承接与企微转化SOP

> **当前状态**：参考级 SOP，基于行业经验，待真实运营后验证。
> **适用场景**：B2B 食品加工机械，通过视频号导入企业微信私域。

---

## 一、承接链路设计

```
视频号内容 → 用户感兴趣 → 访问主页
                          ↓
              ┌───────────┼───────────┐
              ↓           ↓           ↓
         企微名片     公众号关注     私信咨询
              ↓           ↓           ↓
         企微对话     产品目录      人工回复
              ↓           ↓           ↓
         报价跟进     持续关注      导流企微
              ↓
            成交
```

---

## 二、主页包装

### 2.1 主页设置
| 项目 | 设置建议 |
|------|---------|
| 名称 | 品牌名 + 行业关键词（如：XX食品机械） |
| 头像 | 品牌 Logo |
| 简介 | 行业 + 优势 + CTA（50 字以内） |
| 企微名片 | 主页添加企业微信名片 |
| 公众号 | 关联公众号 |

### 2.2 简介模板
```
XX食品机械 | 15年专注食品加工设备
CE/ISO认证 | 出口50+国家
📩 免费获取产品手册 → 点击下方名片添加客服
🏭 工厂地址：XX省XX市
```

---

## 三、企微承接流程

### 3.1 企业微信设置
| 设置 | 说明 |
|------|------|
| 企业资料 | 公司名称、地址、网站、Logo |
| 自动欢迎语 | 新好友自动发送 |
| 快捷回复 | 预设常用话术 |
| 客户标签 | 按行业/地区/意向度分类 |
| 朋友圈 | 定期发布产品和工厂内容 |

### 3.2 自动欢迎语
```
您好！欢迎咨询 XX食品机械 🏭

我们专注食品加工设备 15 年，主要产品：
- 饺子/包子/馄饨机
- 蔬菜/肉类加工设备
- 包装/清洗/烘干设备

请告诉我您的需求：
1. 加工什么产品？
2. 产能要求（公斤/小时）？
3. 所在地区？

我将为您发送详细产品资料！📩
```

### 3.3 快捷回复设置
| 快捷词 | 内容 |
|--------|------|
| /产品目录 | 产品手册 PDF / 链接 |
| /报价 | 报价流程说明 |
| /视频 | 产品运转视频 |
| /案例 | 客户案例链接 |
| /发货 | 发货流程和时间 |
| /售后 | 质保和售后服务说明 |

## 四、公众号联动

### 4.1 公众号设置
| 设置 | 说明 |
|------|------|
| 菜单栏 | 产品中心 / 客户案例 / 联系我们 |
| 自动回复 | 关键词触发自动回复 |
| 内容 | 每周发布 1-2 篇行业文章 |

### 4.2 关键词自动回复
| 关键词 | 回复 |
|--------|------|
| 产品 | 产品目录链接 |
| 价格 | 引导添加企微获取报价 |
| 案例 | 客户案例文章 |
| 联系 | 联系方式 + 企微名片 |

### 4.3 内容方向
- 产品深度介绍
- 客户案例分享
- 行业知识科普
- 展会/活动报道
- 公司动态

---

## 五、社群运营

### 5.1 社群类型
| 类型 | 用途 |
|------|------|
| 行业交流群 | 食品加工行业交流 |
| 产品咨询群 | 已咨询客户的专属群 |
| VIP 客户群 | 已成交客户的售后群 |

### 5.2 群运营规则
- 每周 1-2 次有价值内容分享
- 允许成员提问，及时回复
- 不发硬广，以价值输出为主
- 定期邀请行业专家分享
