# 视频号自然获客平台规则总览

> **当前状态**：参考级研究，基于平台规则和行业观察，待真实运营后验证。
> **适用场景**：B2B 食品加工机械，通过视频号自然流量获客，导入私域。

---

## 一、平台定位

| 维度 | 说明 |
|------|------|
| 用户意图 | 社交 + 信息获取（非纯娱乐） |
| 流量来源 | 社交推荐（朋友点赞）+ 算法推荐 |
| B2B 适配度 | ★★★ 私域整合是最大优势 |
| 内容寿命 | 数天-数周（长于 TikTok） |
| 核心优势 | 微信生态内无缝转化到私域 |

---

## 二、推荐机制

### 流量来源分布
| 来源 | 占比 | 说明 |
|------|------|------|
| 社交推荐 | 60-70% | 朋友点赞/转发后出现在你的信息流 |
| 算法推荐 | 20-30% | 基于兴趣和行为推荐 |
| 关注 | 10-20% | 已关注账号的新内容 |

### 推荐逻辑
1. **社交裂变**：朋友点赞 → 你看到 → 你也点赞 → 你的朋友看到
2. **初始流量池**：先推给关注者和社交圈
3. **晋升条件**：点赞率、完播率、转发率
4. **冷启动友好**：新号可以通过社交推荐获得曝光

---

## 三、内容规则

### 视频规格
| 项目 | 要求 |
|------|------|
| 比例 | 9:16 竖屏（推荐）或 16:9 横屏 |
| 时长 | 最长 30 分钟（推荐 30-60 秒） |
| 格式 | MP4 |
| 封面 | 支持自定义 |
| 定位 | 可添加位置标签 |

### 发布规则
- 每天可发布多条，但建议 1 条/天
- 可关联公众号文章
- 可添加位置、话题标签
- 可同步到朋友圈

---

## 四、获客链路

### 主要转化路径
```
视频 → 主页 → 关注公众号/添加企微
视频 → 评论区 → 私信 → 企微
视频 → 直播 → 互动 → 企微
视频 → 转发到朋友圈 → 熟人推荐 → 关注
```

### 私域整合能力
| 功能 | 说明 |
|------|------|
| 公众号关联 | 主页可关联公众号 |
| 企微名片 | 主页可添加企业微信名片 |
| 客服消息 | 用户可通过公众号发消息 |
| 社群引流 | 视频可分享到微信群 |
| 朋友圈 | 视频号内容可转发到朋友圈 |
| 小程序 | 可关联小程序（如商城） |

---

## 五、B2B 适配度分析

### 适合的理由
| 理由 | 说明 |
|------|------|
| 私域整合 | 无缝导入企微/公众号 |
| 社交信任 | 朋友点赞 = 社交证明 |
| 决策者覆盖 | 微信覆盖各年龄层决策者 |
| 长期运营 | 内容寿命较长，可持续获客 |
| 客户留存 | 导入私域后可长期跟进 |

### 不适合的理由
| 理由 | 说明 |
|------|------|
| 流量规模 | 不如抖音（但更精准） |
| 内容调性 | 偏生活化，工业内容较少 |
| 年龄偏大 | 年轻用户较少 |
| 传播速度 | 社交推荐比算法推荐慢 |

---

## 六、与抖音对比

| 维度 | 视频号 | 抖音 |
|------|--------|------|
| 流量来源 | 社交推荐为主 | 算法推荐为主 |
| 用户意图 | 社交 + 信息 | 娱乐为主 |
| 获客方式 | 导入私域（企微） | 私信/主页链接 |
| 内容调性 | 相对正式 | 更原生/娱乐 |
| B2B 适配 | 私域整合强 | 流量规模大 |
| 适合行业 | 所有行业 | 偏消费品 |

## 七、运营建议

| 建议 | 说明 |
|------|------|
| 与公众号联动 | 视频号内容引导关注公众号 |
| 与企微联动 | 主页添加企微名片 |
| 社群扩散 | 发布后分享到相关微信群 |
| 朋友圈配合 | 转发视频号到朋友圈 |
| 直播配合 | 定期做直播深度沟通 |
| 与海外内容差异化 | 国内用中文，侧重工厂实力+客户案例 |
