# TikTok广告素材创意 SOP

> **适用行业**：B2B 食品加工机械出口
> **官方来源**：https://www.tiktok.com/creative | https://ads.tiktok.com/help/article/ad-requirements

---

## 一、TikTok 视频广告格式规范

### 1.1 基础规格

| 参数 | 推荐值 | 最低要求 |
|------|--------|---------|
| 比例 | 9:16 竖屏 | 9:16, 1:1, 16:9 |
| 分辨率 | 1080x1920 | 540x960 |
| 时长 | 15-30秒 | 5-60秒 |
| 大小 | < 500MB | < 500MB |
| 格式 | MP4 / MOV | MP4 / MOV / AVI |
| 帧率 | 30fps | 24-60fps |

### 1.2 关键原则

- **前 3 秒定生死**：TikTok 用户注意力极短
- **竖屏优先**：90%+ TikTok 用户使用竖屏
- **声音重要**：60%+ TikTok 用户开声音
- **文字叠加**：关键信息加字幕/文字覆盖

## 二、TikTok 广告钩子设计（Hook）

### 2.1 前 3 秒钩子类型

| 钩子类型 | 示例 | 适合场景 |
|---------|------|---------|
| 问题型 | "Can this machine cut 500kg/hour?" | 产品展示 |
| 数字型 | "Output: 1000 dumplings per minute" | 产能展示 |
| 对比型 | "Manual vs Machine: Watch the difference" | 对比展示 |
| 悬念型 | "The secret behind fast food production..." | 引起好奇 |
| 痛点型 | "Tired of slow production speed?" | 问题解决 |
| 展示型 | 直接展示机器运转画面 | 视觉冲击 |

### 2.2 B2B 食品机械钩子示例

| 品类 | 钩子（英文） | 时长 |
|------|-------------|------|
| 切菜机 | "This machine cuts 200kg vegetables in 5 minutes" | 3s |
| 饺子机 | "Watch 1000 dumplings roll out in 1 minute" | 3s |
| 生产线 | "From raw material to packaged product: Full line demo" | 3s |
| 包装机 | "This machine wraps 60 items per minute. Here's how." | 3s |
| 绞肉机 | "500kg/hour meat grinding. See it in action." | 3s |

## 三、UGC vs 专业广告

### 3.1 UGC 风格广告

**特点**：
- 看起来像普通用户视频
- 手机拍摄，非专业制作
- 真实感强
- 适合 TikTok 原生生态

**适合**：
- 工厂日常拍摄
- 工人操作展示
- 客户反馈视频
- "Behind the scenes" 内容

### 3.2 专业广告

**特点**：
- 专业拍摄和剪辑
- 高质量画面和音效
- 品牌标识清晰
- 信息结构完整

**适合**：
- 品牌展示
- 产品发布
- 工厂实力展示
- 国际展会素材

### 3.3 B2B 推荐比例

| 类型 | 占比 | 说明 |
|------|------|------|
| UGC 风格 | 60% | TikTok 原生感，转化更好 |
| 专业广告 | 40% | 品牌信任背书 |

## 四、Spark Ads

### 4.1 什么是 Spark Ads

Spark Ads 允许将已有的有机 TikTok 视频（包括自己或创作者的）作为广告投放。

**官方来源**：https://ads.tiktok.com/help/article/spark-ads

### 4.2 Spark Ads 优势

- 保留原有视频的赞/评/分享数据
- 看起来更像原生内容
- 可以放大已有的热门视频
- 适合 B2B 工厂的"日常运营"视频

### 4.3 Spark Ads 使用流程

1. 发布有机 TikTok 视频
2. 观察自然流量表现（3-7 天）
3. 选择表现好的视频（高完播率/高互动）
4. 在 Ads Manager 中选择 Spark Ads
5. 设置目标受众和预算
6. 投放并监控

## 五、食品机械 TikTok 素材方向

### 5.1 10 个素材方向

| # | 素材方向 | 时长 | 说明 |
|---|---------|------|------|
| 1 | 机器运转实拍 | 15-30s | 直接展示生产能力 |
| 2 | 加工前后对比 | 15-20s | 人工 vs 机器效率对比 |
| 3 | 工厂参观 | 30-60s | 展示工厂规模和实力 |
| 4 | 客户案例 | 30-60s | 海外客户使用场景 |
| 5 | 产品细节 | 15-30s | 材质/工艺/认证展示 |
| 6 | 操作教学 | 30-60s | 如何操作/维护设备 |
| 7 | 打包发货 | 15-30s | 展示出口实力 |
| 8 | 常见问题解答 | 30-60s | 回答客户关心的问题 |
| 9 | 行业应用场景 | 15-30s | 设备在不同场景的使用 |
| 10 | 价格解释 | 30-60s | 解释设备价值构成 |

### 5.2 视频脚本模板

**模板 A：产品演示（15秒）**
```
0-3s: 钩子（产能/效果数字）
3-10s: 机器运转画面（多角度切换）
10-15s: CTA（Contact Us for Quote + WhatsApp 按钮）
```

**模板 B：对比型（20秒）**
```
0-3s: 钩子（人工 vs 机器）
3-8s: 人工操作画面（慢/费力）
8-15s: 机器操作画面（快/高效）
15-20s: CTA + 价格锚点
```

**模板 C：工厂实力（30秒）**
```
0-3s: 钩子（工厂规模/年产量数字）
3-12s: 工厂全景（车间/设备/工人）
12-22s: 生产线流程（原料→成品）
22-30s: CTA + 认证展示
```

## 七、TikTok vs Facebook 素材差异

| 维度 | TikTok | Facebook |
|------|--------|----------|
| 主导格式 | 竖屏视频 | 视频+图片+轮播 |
| 最佳时长 | 15-30秒 | 30-60秒 |
| 风格偏好 | 原生/UGC 感 | 可以更精致 |
| 声音 | 必须考虑 | 可选 |
| 文字覆盖 | 推荐 | 推荐 |
| 音乐 | 重要（TikTok 音乐库） | 可选 |
| 钩子位置 | 前 1-3 秒 | 前 3-5 秒 |

## 八、素材测试方法论

### 8.1 A/B 测试维度

| 维度 | 测试方法 |
|------|---------|
| 钩子 | 同一素材，换前 3 秒 |
| 时长 | 同一内容，15s vs 30s vs 60s |
| 风格 | UGC vs 专业 |
| CTA | Learn More vs Contact Us vs Get Quote |
| 文案 | 数字型 vs 问题型 vs 痛点型 |

### 8.2 测试节奏

| 阶段 | 动作 |
|------|------|
| Week 1 | 测试 5-8 个不同钩子 |
| Week 2 | 保留 Top 3 钩子，测试不同时长 |
| Week 3 | 保留 Top 2 组合，测试不同文案 |
| Week 4 | 确定最佳组合，开始制作变体 |

## 九、常见失败原因

| 问题 | 原因 | 解决方案 |
|------|------|---------|
| 完播率低 | 钩子不够吸引 | 换更直接的数字/对比型钩子 |
| 互动率低 | 内容太像广告 | 改用 UGC 风格 |
| 点击率低 | CTA 不明确 | 改为直接利益驱动型 CTA |
| 转化率低 | 落地页/表单体验差 | 优化移动端表单 |
| 素材衰减快 | TikTok 素材寿命短 | 每周更新 2-3 个新素材 |
