# TikTok Ads平台规则与广告产品总览

> **当前状态**：参考级研究，基于 TikTok Business 官方文档（https://www.tiktok.com/business/en/learn）。
> **最后更新**：2026-05-22，建议定期检查官方文档获取最新规则。

---

## 一、广告格式

| 格式 | 说明 | 适合场景 |
|------|------|---------|
| In-Feed Ads | 信息流原生广告，9:16 竖屏 | 品牌曝光、流量引导 |
| Spark Ads | 推广已有 organic 帖子 | 利用已有内容效果 |
| TopView | 开屏广告 | 品牌大曝光 |
| Brand Takeover | 开屏 + 信息流 | 品牌活动 |
| Branded Effects | 品牌定制滤镜/贴纸 | 品牌互动 |
| Collection Ads | 产品展示卡 | 电商产品目录 |
| Video Shopping Ads | 视频 + 产品链接 | 电商转化 |
| Lead Generation Ads | 内置表单收集线索 | B2B 线索收集 |

---

## 二、投放目标（Campaign Objectives）

| 目标 | 说明 | B2B 适配度 |
|------|------|-----------|
| Reach | 最大化覆盖人数 | ★★ 品牌认知 |
| Traffic | 引导到网站/落地页 | ★★★ 适合有落地页 |
| Video Views | 最大化视频播放 | ★★★ 内容传播 |
| Lead Generation | 内置表单收集线索 | ★★★★ B2B 首选 |
| Conversions | 网站转化事件 | ★★★★ 有 Pixel 时 |
| Community Interaction | 主页互动/关注 | ★★ 辅助 |

---

## 三、定向方式

| 定向 | 说明 | B2B 适配 |
|------|------|---------|
| Interest | 兴趣标签 | ★★★ 标签偏消费者 |
| Behavior | 行为标签 | ★★ B2B 行为标签少 |
| Demographics | 年龄/性别/地域 | ★★★ 基础定向 |
| Hashtag | 话题定向 | ★★ 偏消费者内容 |
| Audience | Custom/Lookalike | ★★★★ 有数据时很有效 |
| Device | 设备类型/网络 | ★★ 辅助 |

> B2B 适配度评估：TikTok 的定向系统偏向消费者，B2B 精准度不如 Facebook/LinkedIn。

---

## 四、素材要求

| 项目 | 要求 |
|------|------|
| 视频比例 | 9:16 竖屏（推荐） |
| 视频时长 | 9-60 秒（推荐 15-30s） |
| 视频格式 | MP4 / MOV |
| 文件大小 | ≤ 500MB |
| 文字限制 | 建议不超过 100 字符 |
| 标签建议 | 3-5 个 |
| 音乐 | 使用 TikTok Commercial Music Library |

## 五、Spark Ads 基础

### 什么是 Spark Ads
推广已有的 organic 帖子，保留点赞、评论、分享数据。

### 使用场景
1. 某条 organic 视频自然播放量不错
2. 想用已有内容做广告，节省制作成本
3. 想借已有数据的社交证明

### 操作步骤
1. 发布 organic 视频
2. 等待 24-48 小时观察自然表现
3. 在 TikTok Ads Manager 中选择该帖子
4. 创建 Spark Ads Campaign
5. 设置预算、受众、目标

### B2B 适配度
★★★★ 适合 B2B，因为 organic 视频可以先测试哪些内容吸引目标受众。

## 六、Lead Generation 基础

### 什么是 Lead Generation Ads
内置表单，用户在 TikTok 内填写信息，无需跳转。

### 表单字段
| 字段 | 是否自动填充 |
|------|-------------|
| 姓名 | 是（从 TikTok Profile） |
| 邮箱 | 是 |
| 电话 | 是 |
| 公司名 | 否（自定义字段） |
| 需求描述 | 否（自定义字段） |

### B2B 适配度
★★★★ 适合 B2B，降低了转化门槛。但需注意：
- TikTok 用户可能填写随意
- 需要配合过滤问题提高线索质量
- 建议加"您的年产量"等过滤问题

---

## 七、Website Conversion 基础

### 什么是 Website Conversion
引导用户到网站，完成指定转化事件（如表单提交）。

### 前置条件
- TikTok Pixel 安装在网站上
- Pixel 配置了转化事件

### B2B 适配度
★★★★ 适合有独立站/落地页的 B2B 企业。

---

## 八、素材测试逻辑

| 阶段 | 操作 |
|------|------|
| 第 1 周 | 发布 3-5 条 organic 视频，观察自然表现 |
| 第 2 周 | 选表现最好的 2 条做 Spark Ads |
| 第 3 周 | 根据 Spark Ads 数据优化素材方向 |
| 第 4 周 | 固定最优素材，测试受众 |

> TikTok 的素材测试比 Facebook 更依赖 organic 数据，因为 organic 表现好的内容通常 Ads 表现也好。

---

## 九、学习期

| 规则 | 说明 |
|------|------|
| 学习期 | Ad Group 需要积累 50 个优化事件（如 Lead）才能完成学习 |
| 学习期时长 | 通常 3-7 天 |
| 学习期注意事项 | 不要频繁调整预算/定向/素材 |
| 学习失败 | 如果 7 天内没达到 50 个事件，标记为"学习失败"，需调整策略 |

---

## 十、合规要点

| 项目 | 要求 |
|------|------|
| 广告标识 | 必须标注"Sponsored" |
| 音乐 | 使用 Commercial Music Library，不能用个人音乐 |
| 内容 | 不能虚假宣传、不能贬低竞品 |
| 行业限制 | 某些行业（赌博、武器等）不允许投放 |
| 数据合规 | 遵守各国数据保护法 |

> 详见 TikTok Business 官方政策：https://www.tiktok.com/business/en-US/policy

## 十一、B2B 设备是否适合 TikTok Ads？

### 适合的理由
| 理由 | 说明 |
|------|------|
| 覆盖面广 | 东南亚/中东 TikTok 用户量大 |
| 素材成本低 | 手机拍摄即可 |
| 品牌认知 | 适合建立品牌存在感 |
| Spark Ads | 可以先 organic 测试再付费 |

### 不适合的理由
| 理由 | 说明 |
|------|------|
| 用户意图 | 娱乐为主，不是来找供应商的 |
| 定向精度 | B2B 兴趣标签少 |
| 线索质量 | 可能较低（个人用户多） |
| 决策场景 | B2B 决策需要理性判断，TikTok 是感性平台 |

### 建议
1. TikTok 适合做品牌认知和线索补充
2. 不作为唯一投放渠道，配合 Facebook/LinkedIn
3. 先 organic 测试，再用 Spark Ads 放大
4. 重点关注东南亚/中东市场（TikTok 渗透率高）
