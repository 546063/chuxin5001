# TikTok投流复盘指标体系

> **适用行业**：B2B 食品加工机械出口
> **注意**：以下基准值为参考级，需实际投放后验证。

---

## 一、核心指标定义

### 1.1 漏斗指标

| 指标 | 全称 | 说明 | B2B重要性 |
|------|------|------|-----------|
| 展示 | Impressions | 广告被看到的次数 | ★★ |
| CPM | Cost Per Mille | 千次展示成本 | ★★ |
| CTR | Click-Through Rate | 点击率 | ★★★★ |
| CPC | Cost Per Click | 单次点击成本 | ★★★ |
| CVR | Conversion Rate | 转化率 | ★★★★★ |
| CPL | Cost Per Lead | 单条线索成本 | ★★★★★ |
| CPA | Cost Per Action | 单次行为成本 | ★★★★ |
| ROAS | Return on Ad Spend | 广告支出回报 | ★★★ |

### 1.2 视频专属指标

| 指标 | 说明 | B2B重要性 |
|------|------|-----------|
| 播放率 | 视频开始播放/展示 | ★★★ |
| 3秒播放率 | 播放超过3秒的比例 | ★★★★ |
| 完播率 | 完整播放的比例 | ★★★★★ |
| 平均播放时长 | 平均观看秒数 | ★★★★ |
| 互动率 | 赞/评/分享/展示 | ★★★ |

---

## 二、参考级基准值（待验证）

> ⚠️ 以下数据为行业参考值，非实际投放数据。需真实账户回填。

### 2.1 TikTok Ads 参考基准

| 指标 | 参考范围 | 说明 |
|------|---------|------|
| CPM | $3-10 | 因市场而异，美国更贵 |
| CTR | 0.8-2.0% | 素材质量影响大 |
| CPC | $0.50-2.00 | 因竞争度而异 |
| CVR | 2-8% | B2B 偏低 |
| CPL | $10-50 | B2B 设备类可能更高 |
| 完播率 | 15-30% | 15秒视频参考值 |
| 3秒播放率 | 50-70% | 取决于钩子 |

### 2.2 TikTok vs Facebook 参考对比

| 指标 | TikTok（参考） | Facebook（参考） |
|------|--------------|-----------------|
| CPM | $3-10 | $5-25 |
| CTR | 0.8-2.0% | 1.0-3.0% |
| CPL（B2B） | $10-50 | $5-30 |

> **注意**：TikTok 的 CPL 在 B2B 场景可能高于 Facebook，因为 TikTok 用户更偏 C 端。

## 三、三步诊断框架

### Step 1: 检查 CPM

| CPM 状态 | 判断 | 动作 |
|---------|------|------|
| 高于参考值 | 竞争大/受众窄 | 扩展受众或换市场 |
| 低于参考值 | 正常/竞争小 | 进入下一步 |

### Step 2: 检查 CTR

| CTR 状态 | 判断 | 动作 |
|---------|------|------|
| 低于 0.8% | 素材/钩子问题 | 换钩子、换素材风格 |
| 0.8-1.5% | 一般 | 优化 CTA |
| 高于 1.5% | 好 | 进入下一步 |

### Step 3: 检查 CVR

| CVR 状态 | 判断 | 动作 |
|---------|------|------|
| 低于 2% | 落地页/表单问题 | 优化表单体验 |
| 2-5% | 一般 | 优化受众精准度 |
| 高于 5% | 好 | 考虑放量 |

---

## 四、素材衰减判断

### 4.1 衰减信号

| 信号 | 阈值 | 说明 |
|------|------|------|
| CTR 下降 | 连续 3 天下降 > 15% | 素材疲劳 |
| 完播率下降 | 下降 > 10% | 用户失去兴趣 |
| 频率升高 | > 2.5 | 同一用户看到太多次 |
| CPL 上升 | 连续 3 天上升 > 20% | 素材效率降低 |
| 互动率下降 | 赞/评/分享持续下降 | 内容不再吸引 |

### 4.2 应对策略

| 阶段 | 动作 |
|------|------|
| 早期衰减（1-3天） | 换钩子，保留素材主体 |
| 中期衰减（4-7天） | 换素材，保留受众 |
| 晚期衰减（7天+） | 换素材 + 换受众 |

---

## 五、预算放量逻辑

### 5.1 放量条件

满足以下 3 个条件方可放量：

1. CPL 低于目标值（或低于参考值 20%）
2. 广告运行 ≥ 7 天，积累 ≥ 50 个转化
3. CTR 稳定在 1.0% 以上

### 5.2 放量方法

| 方法 | 幅度 | 频率 |
|------|------|------|
| 增加日预算 | +20-30% | 每 3-4 天一次 |
| 拓展新市场 | 新增 Ad Group | 每周 1-2 个 |
| 拓展新受众 | 新增兴趣定向 | 每周 1-2 个 |
| 复制优秀广告 | 完全复制 | 根据需要 |

### 5.3 放量风险控制

- 每次只调整一个变量
- 调整后观察 48-72 小时
- 如果 CPL 上升超过 30%，回退调整
- 保持素材更新节奏

## 六、B2B 特殊考量

### 6.1 B2B 不能只看 ROAS

B2B 食品机械的特点：

- 决策周期长（2周-3个月）
- 单笔金额大（¥20,000+）
- 需要多轮沟通
- 无法即时追踪成交

### 6.2 B2B 替代指标

| 替代指标 | 说明 | 追踪方式 |
|---------|------|---------|
| 询盘价值评分 | A/B/C/D 分级 | 客服人工判断 |
| 有效线索率 | Valid Lead / Total Lead | CRM 记录 |
| 询盘→报价率 | 报价数 / 询盘数 | 销售记录 |
| 询盘→成交率 | 成交数 / 询盘数 | 销售系统 |
| 平均成交金额 | 总成交额 / 成交数 | 财务数据 |

## 八、月度复盘 Checklist

- [ ] 导出本月所有广告数据
- [ ] 计算各 Campaign 的平均 CPM/CTR/CPL
- [ ] 对比上月数据，标记变化趋势
- [ ] 识别 Top 3 最佳素材
- [ ] 识别 Bottom 3 最差素材
- [ ] 分析素材衰减周期
- [ ] 评估受众定向效果
- [ ] 计算总线索量和有效线索率
- [ ] 追踪线索→报价→成交转化
- [ ] 制定下月优化计划
