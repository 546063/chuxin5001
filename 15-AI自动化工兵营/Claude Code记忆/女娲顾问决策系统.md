# 女娲顾问决策系统

## 核心原则

**资料可以多，但进入决策系统的内容必须少。**

## 目录结构

`11-智囊团与知识蒸馏/女娲顾问/`

## 四层架构

| 层级 | 位置 | 内容 |
|------|------|------|
| 原始资料 | `00_原始资料/` | 只存索引，不存全文 |
| 决策层 | `10_顾问心智模型/` | 核心董事 |
| 业务工具层 | `20_业务工具顾问/` | 业务工具卡 |
| 观察层 | `30_观察型顾问/` | 观察型顾问卡 |
| 委员会 | `50_决策委员会/` | 5个委员会，调用3-6个顾问 |
| 个人原则 | `40_我的业务系统/` | 个人原则沉淀 |

## 核心文件

- 8 张核心董事卡
- 5 张业务工具卡
- 3 张观察型卡
- 5 个委员会文件

## 决策时如何调用

通过 `50_决策委员会/` 中的委员会文件调用对应顾问，每个委员会配置 3-6 个顾问。
