# 银羽招生短视频剪辑 SOP V3 — B-roll 混剪强化版

## 一句话原则

> 每 1-2 秒换画面，关键词匹配内容，女孩口播不超过 2 句，其余全是 B-roll。

---

## 1. 核心问题

- 画面一直是女孩坐着说话，观众容易划走
- B-roll 太少或随机插入，和文案不匹配
- 节奏不够快，不像抖音/视频号风格

## 2. B-roll 混剪规则

### 2.1 口播画面限制

- 女孩坐着说话：最多 1-2 句
- 女孩站着说话：最多 1 句（如果有素材）
- 口播画面总占比 ≤ 25%
- 其余时间全部用 B-roll 覆盖

### 2.2 快切节奏

| 文案长度 | 画面切换频率 |
|---|---|
| 短句（<5字） | 整句一个画面 |
| 中句（5-12字） | 1-2 秒一切 |
| 长句（>12字） | 拆为 2-3 个画面，每个 1.5-2.0 秒 |

**每句话尽量换一个场景或角度。**

### 2.3 文案关键词 → B-roll 语义映射

| 文案关键词 | 优先画面 | 备选画面 |
|---|---|---|
| 握拍 | 手部/球拍/握拍特写 | 教练纠正握拍 |
| 打了一年 | 孩子打球/训练 | 多人训练 |
| 动作错 | 错误动作/教练纠正 | 挥拍动作 |
| 难改 | 重复训练/动作修正 | 孩子练习 |
| 发力不对 | 挥拍/击球/手臂动作 | 球拍特写 |
| 伤肩 | 肩部/挥拍/动作变形感 | 教练提醒 |
| 动作评估 | 教练指导/观察动作 | 教练示范 |
| 步伐 | 脚步移动/跑动 | 场地移动 |
| 小班教学 | 多人训练/教练带练 | 分组练习 |
| 免费试听 | 教练/球馆/报名氛围 | 孩子体验 |
| 动作测评 | 教练纠正/动作反馈 | 教练指导 |
| 暑假 | 球馆环境/集体训练 | 训练氛围 |

**如果没有完全对应的画面：**
1. 优先羽毛球训练素材
2. 其次教练指导
3. 再选球馆环境
4. 不用无关画面凑时长

### 2.4 B-roll 选择优先级

1. **语义匹配度** — 画面和文案关键词最接近
2. **画面质量** — 清晰、不糊、不抖
3. **可用时长** — 至少 2 秒可用内容
4. **视觉多样性** — 避免连续使用同一角度

### 2.5 B-roll 静音

所有 B-roll：
```bash
-an  # 完全静音
```

最终音轨 = master_audio.wav（唯一）

## 3. 画面时间线搭建

### 3.1 每句话的画面计划

```
句子1: 口播画面（开头 Hook）
句子2: B-roll A → B-roll B（1.5s 一切）
句子3: B-roll C → B-roll D → B-roll E（2.0s 一切）
句子4: B-roll F → B-roll G
句子5: 口播画面（中间信任）
句子6: B-roll H → B-roll I
句子7: B-roll J → B-roll K
句子8: B-roll L → B-roll M
句子9: 口播画面（结尾 CTA）
```

### 3.2 B-roll 时长控制

- 最短：1.0 秒
- 常规：1.5-2.0 秒
- 最长：2.5 秒（仅完整动作片段）
- 不允许一个 B-roll 停留超过 2.5 秒

## 4. 字幕规则

同 V1/V2，ASS 格式，PlayResX=1080, PlayResY=1920, Fontsize=40, PingFang SC。

字幕分行：
- 每行 10-14 个中文
- 最多两行
- 长句自动拆分

## 5. 验收标准

1. ✅ 女孩口播 ≤ 2 句
2. ✅ B-roll 覆盖率 ≥ 75%
3. ✅ 每 1-2 秒切换画面
4. ✅ B-roll 与文案关键词语义匹配
5. ✅ B-roll 全部静音
6. ✅ 1080×1920 正向竖屏
7. ✅ 无黑屏/卡顿/空白
