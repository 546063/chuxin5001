# 15-AI自动化工兵营 目录总览

## 中文总结

本目录是AI工具与自动化体系的指挥中心，涵盖Claude Code配置、AI知识库路由系统、提示词工程、SOP执行库、模型路由决策、Agent记忆管理、用量统计等核心AI工作流。为跨境B2B、工业品获客、社媒运营提供智能化能力支撑。

## English Summary

This directory is the command center for AI tools and automation systems. Covers Claude Code configuration, knowledge base routing, prompt engineering, SOP execution library, model routing decisions, Agent memory management, and usage statistics. Provides intelligent capability support for cross-border B2B, industrial product acquisition, and social media operations.

## 一句话结论

AI自动化工兵营是全库智能化运营的核心引擎，通过标准化AI工作流驱动所有业务目录效率提升。

## 本目录适用场景

- AI工具配置与工作流设计
- 提示词工程与优化
- 知识库路由与读取规则
- Claude Code配置与记忆管理
- 模型选择与切换决策

## 本目录不适用场景

- 具体业务内容生产（见对应业务目录）
- 财务分析（见17-个人财务与账单分析）
