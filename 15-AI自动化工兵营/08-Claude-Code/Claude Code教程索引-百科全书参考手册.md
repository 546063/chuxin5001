# Claude Code 从入门到精通 — 百科全书与参考手册

> **来源**：[0xfnzero/AI-Code-Tutorials](https://github.com/0xfnzero/AI-Code-Tutorials)

## 概述

从零基础到高级应用，系统学习 Claude Code，掌握 AI 辅助编程技能。教程采用循序渐进的方式，分为**入门、进阶、高级**三个阶段。

## 学习路线

### 零基础用户
第一课 → 第二课 → 第三课 → 第四课 → 第六课（实战）
**预计时间**：2-3 周

### 有编程基础
第一课（快速浏览）→ 第二课 → 第四课 → 第五课 → 第六课 → 第七课 → 第八课 → 第九课 → 第十课
**预计时间**：1-2 周

### 专业开发者
第二课（安装）→ 第五课 → 第七课 → 第八课 → 第九课（必学）→ 第十课（必学）
**预计时间**：3-5 天

## 学习建议

1. **按顺序学习** — 后面的课程会用到前面的知识
2. **动手实践** — 每课都有练习任务，一定要自己做
3. **不要跳过** — 即使觉得简单，也要过一遍
4. **多提问** — 遇到不懂的，直接问 Claude Code
5. **做笔记** — 记录常用命令和技巧
6. **创建项目** — 学完每个阶段，做一个小项目巩固

## 学习目标

完成本教程后，你将能够：

- 熟练使用 Claude Code 进行日常开发
- 独立完成前端和后端项目
- 运用最佳实践编写高质量代码
- 设计和实现复杂的应用架构
- 建立完整的开发工作流
- 使用 AI 提升 10 倍开发效率
