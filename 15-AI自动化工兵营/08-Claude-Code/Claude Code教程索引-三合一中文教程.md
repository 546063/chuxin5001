# Claude Code + OpenClaw + Codex 三合一中文教程

> **来源**：[KimYx0207/AI-Coding-Guide-Zh](https://github.com/KimYx0207/AI-Coding-Guide-Zh)
> **协议**：MIT
> **规模**：39篇完整教程 | 18万字 | 90+代码示例 | 230+FAQ

## 概述

一套系统化、小白友好的三顶流合璧 AI 工具中文教程，涵盖三大产品：

| 工具 | 是什么 | 干什么 | 教程数 |
|------|--------|--------|--------|
| **Claude Code** | Anthropic 官方 AI 编程 CLI | 终端写代码、调Bug、做架构 | 13篇 + 1速查卡 |
| **OpenClaw** | 开源 AI 私人助手框架 | 管邮件、排日程、自动化一切 | 12篇完整教程 |
| **Codex** | OpenAI 编程 Agent 平台 | App/CLI/Web/IDE/SDK 分层协作 | 14篇完整教程 |

## 为什么放在一起？

1. **血脉相连** — OpenClaw 原名 Clawdbot，DNA 里流着 Claude 的血
2. **互补工具链** — Claude Code 管编程，OpenClaw 管日常，Codex 管快速 Agent 任务
3. **双强协作** — Codex 和 Claude Code 通过 MCP Server 共享工具链
4. **中国用户刚需** — 一站式中文教程
