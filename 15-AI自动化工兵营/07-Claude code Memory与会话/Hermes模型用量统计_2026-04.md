## ⚠️ 注意事项

1. **OpenClaw 原生不记录 Input/Output 拆分**，大部分 `totalTokens = 0`（响应截断）
2. **每日新闻 cron 推送用 MiniMax M2.7 API，不经过 OpenClaw**，那部分 token 不在此统计中
3. **精确费用需查看各平台后台**：
