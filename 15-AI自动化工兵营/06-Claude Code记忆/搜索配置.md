# 搜索配置

## 搜索工具状态

| 搜索服务 | 状态 | 优先级 |
|---------|------|--------|
| Tavily | ✅ 已配置 | 1 - 优先使用 |
| Serper | ✅ 已配置 | 2 - 备用 |
| Firecrawl | ✅ 已配置 | 3 - 网页抓取兜底 |

## 搜索优先级工作流

1. **Tavily** — 优先使用，适合通用搜索、新闻、技术文档
2. **Serper** — Tavily 不可用时使用，Google Search API，适合精确搜索、本地搜索、购物搜索
3. **Firecrawl** — 最后备用，适合网页抓取和深度内容提取

## Claude Code 内置工具

- `WebSearch` — 内置搜索（调用 Tavily）
- `WebFetch` — 抓取指定 URL 内容

## 终端调用

```bash
/Users/mac/.openclaw/scripts/web-search.sh "搜索关键词"
ws "搜索关键词"  # (已添加 alias)
```

## 配置文件位置

- `~/.claude/settings.json` — 所有 API Keys (TAVILY_API_KEY, SERPER_API_KEY, FIRECRAWL_API_KEY)
- `~/.openclaw/workspace/skills/firecrawl-search/.env` — Firecrawl API Key
- `~/.openclaw/scripts/web-search.sh` — 统一搜索脚本
