# 字段字典

| 字段 | 说明 |
| --- | --- |
| clean_transaction_id | 清洗后的唯一流水 ID |
| source_platform | 来源平台：微信、支付宝、工商银行 |
| source_file | 来源文件 |
| transaction_time | 交易时间 |
| quarter | 年度季度，例如 2025-Q2 |
| raw_type | 原始账单里的交易类型 |
| counterparty | 交易对方/商户 |
| description | 商品说明或银行摘要 |
| direction | 收入/支出/退款/转账方向 |
| amount | 正数金额 |
| signed_amount | 带方向金额，收入为正，支出为负 |
| payment_method | 支付方式或银行渠道 |
| include_in_analysis | 是否纳入财务分析 |
| exclude_reason | 不纳入分析的原因 |
| primary_category | 一级分类 |
| secondary_category | 二级分类 |
| cash_layer | 现金流层级：收入、真实消费、业务支出等 |
| classification_confidence | 分类置信度 |
| classification_rule | 分类依据/人工确认规则 |
| need_review | 是否仍需人工复核 |
| duplicate_match_id | 如果为重复通道，对应匹配ID |
| duplicate_reason | 重复或通道排除原因 |
