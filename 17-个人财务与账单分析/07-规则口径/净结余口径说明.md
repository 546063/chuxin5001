# 净结余口径说明

## 严格口径

```text
严格口径净结余 = 确认收入 - 真实生活/家庭消费 - 业务支出
```

这个口径适合判断“钱最终有没有留在个人手里”，但会被保姆/阿姨代收款放大支出。

## 阿姨代收抵消口径

```text
阿姨代收抵消净结余 = 确认收入 - (真实生活/家庭消费 - 保姆/阿姨代收款) - 业务支出
```

这个口径适合判断“排除代收代付后，你自己的真实压力”。

## 不进入净结余的项目

- 微信/支付宝/银行卡之间的充值提现。
- 退款、还款、投资理财。
- 开票/代收代付流转。
- 银行/支付平台重复通道。
