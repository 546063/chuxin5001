# 人工确认规则库 最终版

## 客户收入 / 个人进账

- 张峰 = 客户收款。
- 刘彦威 = 客户收款。
- 刘兰 = 个人收款。
- 陶之初 / 河北陶之初文化发展有限公司 = 你的公司；公司转给你个人，记为个人进账收入。
- 赵总 / 银羽客户转账 = 客户收入。
- 曾学勤 / 高老师 / 米高教育 / 韩教练 / 范范儿保定执行局 / 沈清华青岛娱乐 / 小猫43直聘 = 客户收入或业务收入。

## 家庭 / 老婆 / 保姆

- 都爽 / Jane / 朵朵 / 老婆 = 家庭支出或家庭内部资金流转。
- 吴楠 / 自动售卖纪念币中每月约 6000 = 保姆/阿姨代收款。
- 郝金龙、爸 6000 等按你确认 = 保姆/阿姨代收款。
- 阿姨费用通常不是你直接转给阿姨，而是经老婆/家庭账户转出，因此分析时保留“阿姨代收抵消口径”。

## 业务支出

- 故意走红 = 给客户转的广告费，业务支出。
- 书宁 = 给客户的返点/提成，业务支出。
- 张良 = 工资给员工，业务支出。
- 河传兼职不错 = 给兼职发的工资 / 剪辑费用 / 项目支出。
- 刘欢增摄像、剪辑、拍摄、素材、运营 = 业务支出。
- AI工作室 Coze、ChatGPT、Claude、API、剪映、巨量、DOU+ = 业务工具或广告投放支出。

## 开票 / 代收代付

- 田明明 = 开票/代办票款，转入转出单独列为代收代付/开票流转。
- 桥西区馥醇广告设计中心 = 收款再转出，开票/代收代付。
- 郑丝羽妈妈 / 办九中幼儿园 = 客户项目，收 15000、转 10000，净盈利约 5000。

## 真实生活 / 家庭消费

- 美团 = 消费。
- 拼多多 = 消费。
- 泰享曼 / 曼悦 / 按摩 = 消费。
- 闪动 / 银羽 / 银羽运动中心 = 羽毛球运动费用。
- 喜德盛 = 运动/公路车消费。
- 摄影器材、索尼镜头、办公家具：如用于业务，归业务设备；如不能明确，保留规则说明。
