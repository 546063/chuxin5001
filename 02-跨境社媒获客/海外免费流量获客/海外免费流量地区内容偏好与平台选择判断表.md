# 海外免费流量地区内容偏好与平台选择判断表

> 覆盖 7 大目标地区，每个地区给出平台组合、内容偏好、食品机械/机械设备策略、成人用品合规边界。

---

## 一、中东（沙特 / 阿联酋 / 科威特 / 卡塔尔）

### 推荐平台组合

| 平台 | 优先级 | 说明 |
|------|--------|------|
| TikTok | P0 | 沙特用户超 2500 万，食品机械内容效果好 |
| WhatsApp | P0 | 商业沟通首选，承接必备 |
| YouTube Shorts | P1 | 搜索长尾 + 推荐双引擎 |
| Facebook | P1 | 年长决策者使用，Group 活跃 |
| Instagram | P2 | 品牌展示 |
| X | P2 | 商业讨论 |

### 内容偏好

| 维度 | 偏好 | 食品机械适配 | 机械设备适配 |
|------|------|------------|------------|
| 语言 | 阿拉伯语为主，英语为辅 | 阿拉伯语字幕必加 | 英语可接受 |
| 内容风格 | 奢华感、科技感、干净现代 | 全自动产线实拍 + 产能数字 | CNC 精度 + 质检特写 |
| 信息偏好 | ROI、投资回报周期、分期付款 | "日产 X 吨""回本 X 个月" | "能耗降低 X%""人工节省 X 人" |
| 信任要素 | 认证、案例、本地化 | CE/ISO + 中东客户案例 | 售后支持 + 阿拉伯语技术 |
| 承接方式 | WhatsApp 直聊 | wa.me 短链接 + 阿语欢迎语 | 同上 |

### 成人用品合规

| 项目 | 说明 |
|------|------|
| 平台限制 | 沙特/阿联酋审查极严，TikTok 中东区严禁 |
| 文化风险 | 宗教禁忌，绝对不能出现暴露/性暗示 |
| 建议 | **不建议在中东地区拓展成人用品市场** |

### 内容禁忌

- 宗教敏感（伊斯兰教必须尊重）
- 酒精/猪肉相关机械需提及清真认证
- 以色列国旗/元素/地图
- 女性穿着暴露的模特
- 夸大承诺（"世界第一""100%"）

## 二、东南亚（印尼 / 泰国 / 越南 / 菲律宾 / 马来西亚）

### 推荐平台组合

| 平台 | 优先级 | 说明 |
|------|--------|------|
| TikTok | P0 | 用户量大，食品机械运转视频效果好 |
| Facebook | P0 | 社群活跃，Group 获客有效 |
| WhatsApp/LINE/Zalo | P0 | 不同国家偏好不同通讯工具 |
| YouTube Shorts | P1 | 搜索属性强 |
| Instagram | P2 | 品牌展示 |

### 内容偏好

| 维度 | 偏好 | 食品机械适配 | 机械设备适配 |
|------|------|------------|------------|
| 语言 | 本地语言字幕优先 | 印尼语/泰语/越南语字幕 | 英语可接受 |
| 内容风格 | 真实感、性价比 | 工厂实拍（不需精致剪辑） | 效率对比 + ROI |
| 信息偏好 | 价格敏感、投资回报 | 强调"性价比""回本快" | 强调"替代人工" |
| 信任要素 | 同行在用、本地案例 | "东南亚 500+ 工厂选择" | 本地安装案例 |
| 承接方式 | WhatsApp/LINE/Zalo | 按国家选工具 | 同上 |

### 成人用品合规

| 项目 | 说明 |
|------|------|
| 平台限制 | 印尼/马来西亚伊斯兰文化，审查严格 |
| 文化风险 | 宗教禁忌，马来西亚/印尼严禁公开推广 |
| 建议 | 越南/泰国/菲律宾相对宽松，用独立站承接 |

### 内容禁忌

- 印尼/马来西亚的宗教敏感内容
- 过度暴露画面
- 与当地文化冲突的产品展示

## 三、欧美（美国 / 英国 / 德国 / 法国 / 澳洲）

### 推荐平台组合

| 平台 | 优先级 | 说明 |
|------|--------|------|
| YouTube Shorts | P0 | 搜索驱动，B2B 决策者使用 |
| LinkedIn | P0 | B2B 决策者聚集 |
| Reddit | P1 | 精准行业社区 |
| Instagram | P1 | 品牌展示 |
| Facebook | P2 | Group 可用 |
| X | P2 | 行业讨论 |

### 内容偏好

| 维度 | 偏好 | 食品机械适配 | 机械设备适配 |
|------|------|------------|------------|
| 语言 | 英语为主 | 英语字幕清晰 | 英语专业术语 |
| 内容风格 | 专业、数据驱动 | 产能数据 + 认证 + 合规 | 技术参数 + 精度 + 效率 |
| 信息偏好 | 合规、认证、售后 | FDA/CE/ISO + 保修条款 | 技术参数 + ROI 分析 |
| 信任要素 | 真实案例、第三方评测 | 客户见证 + 视频工厂参观 | 精度测试报告 + 用户评测 |
| 承接方式 | 邮件 + WhatsApp | 邮件优先，WhatsApp 辅助 | 邮件为主 |

### 成人用品合规

| 项目 | 说明 |
|------|------|
| 平台限制 | 欧美相对宽松，X 允许广告，YouTube 允许教育内容 |
| 文化风险 | 低，但需注意平台年龄限制 |
| 建议 | 优先 Reddit + X + 独立站，YouTube 做科技角度 |

### 内容禁忌

- 虚假认证（CE/FDA 必须真实）
- 夸大性能数据
- 侵犯知识产权的内容
