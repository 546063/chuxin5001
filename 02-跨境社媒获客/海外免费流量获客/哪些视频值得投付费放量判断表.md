# 哪些视频值得投付费放量判断表

> **声明：** 本文档中涉及的具体数值阈值标注为 `[待补充真实数据]` 的，需基于实际投放数据校准。不得编造数据，以下为行业通用逻辑框架。

---

## 一、核心判断标准

决定一条有机视频是否值得投付费放量，需同时满足以下三个维度：

| 维度 | 核心问题 | 判断方式 |
|------|----------|----------|
| **视频质量** | 内容本身能否留住观众？ | 完播率、平均观看时长、点赞/评论/分享比 |
| **受众精准度** | 吸引的是不是目标客户？ | 观众画像（地区/行业）、评论内容质量、自然询盘来源 |
| **询盘潜力** | 有无转化为商业机会的信号？ | 自然询盘数、评论区商业意图词（price/MOQ/delivery 等） |

**一条视频必须同时通过三个维度的最低门槛，才进入放量候选池。**

## 三、评分卡模型

每条视频按以下三个维度打分，加权计算总分。

### 3.1 评分维度与权重

| 维度 | 权重 | 评分标准（1-5分） |
|------|------|-------------------|
| **视频质量** | 40% | 5=完播率优秀+画面清晰+节奏紧凑<br>4=完播率良好+基本要素齐全<br>3=完播率一般+无明显硬伤<br>2=完播率低+有改进空间<br>1=完播率差+内容粗糙 |
| **受众精准度** | 35% | 5=目标市场观众占比高+评论专业<br>4=目标市场为主+偶有非目标观众<br>3=混合观众+部分相关评论<br>2=非目标观众为主<br>1=完全不对口 |
| **询盘潜力** | 25% | 5=自然询盘多且意向明确<br>4=有询盘但需进一步跟进<br>3=有询问但未达询盘标准<br>2=仅点赞无评论<br>1=零互动 |

### 3.2 总分计算

```
总分 = (视频质量得分 × 0.4) + (受众精准度得分 × 0.35) + (询盘潜力得分 × 0.25)
```

| 总分 | 决策 |
|------|------|
| ≥ 4.0 | 🟢 **优先放量** — 立即投入广告预算 |
| 3.0 - 3.9 | 🟡 **测试放量** — 小预算测试（$[待补充真实数据]/天） |
| 2.0 - 2.9 | 🟠 **优化后观察** — 修改标题/封面/标签后重新发布观察 |
| < 2.0 | 🔴 **不放量** — 节省预算，分析原因 |

## 五、不值得放量的视频类型

以下类型的视频 **不应投入付费放量**：

| 类型 | 原因 |
|------|------|
| **纯工厂环境展示（无产品/无讲解）** | 信息量低，无法传递产品价值 |
| **画质严重模糊/抖动/收音差** | 付费放大只会放大缺点，损害品牌形象 |
| **纯娱乐向/与产品无关** | 流量大但精准度为零，浪费预算 |
| **已过时的产品/停产型号** | 询盘无法转化为订单 |
| **有合规风险的内容** | 含未授权音乐、虚假宣传、侵权素材，付费推广可能触发封号 |
| **完播率极低（前 3 秒流失 > [待补充真实数据]%）** | 内容开头无法吸引注意力，放量也是浪费 |
| **评论区负面声音占比高** | 放量会让更多人看到负面评论 |
| **非目标市场流量为主** | 如做东南亚市场但视频主要吸引欧美观众 |

## 六、判断流程（Decision Tree）

```
开始
 │
 ├─ 1. 视频发布 48 小时后，数据是否达标？
 │   ├─ 否 → 标记"观察中"，72 小时后再评估
 │   └─ 是 ↓
 │
 ├─ 2. 完播率是否达到最低阈值？
 │   ├─ 否 → 🔴 不放量，分析原因（开头？节奏？内容？）
 │   └─ 是 ↓
 │
 ├─ 3. 互动率是否达标？
 │   ├─ 否 → 检查：观众是否为目标市场？
 │   │   ├─ 是目标市场但互动低 → 🟠 优化标题/封面后重发观察
 │   │   └─ 非目标市场 → 🔴 不放量，调整定向策略
 │   └─ 是 ↓
 │
 ├─ 4. 是否有自然询盘或商业意图评论？
 │   ├─ 否 → 评估视频内容是否有明确 CTA？
 │   │   ├─ 无 CTA → 🟠 添加 CTA 后重发
 │   │   └─ 有 CTA 但仍无询盘 → 🟡 小预算测试
 │   └─ 是 ↓
 │
 ├─ 5. 计算评分卡总分
 │   ├─ ≥ 4.0 → 🟢 优先放量（$[待补充真实数据]/天起）
 │   ├─ 3.0-3.9 → 🟡 测试放量（$[待补充真实数据]/天）
 │   ├─ 2.0-2.9 → 🟠 优化后观察
 │   └─ < 2.0 → 🔴 不放量
 │
 └─ 6. 放量后 7 天复盘
     ├─ CPC / CPL 达标 → 继续放量
     └─ CPC / CPL 不达标 → 暂停，分析原因
```

## 八、复盘与迭代

每次放量决策后，应在 7-14 天内复盘以下数据：

| 指标 | 说明 |
|------|------|
| 广告花费 | 总投入金额 |
| 播放量 | 付费带来的额外播放 |
| 互动量 | 付费带来的额外互动 |
| 询盘数 | 付费带来的新询盘 |
| CPL（单询盘成本） | 广告花费 / 询盘数 |
| ROI | 成交金额 / 广告花费 |

> **目标：** 通过多次放量决策积累数据，逐步校准阈值和评分权重，使模型更贴合自身业务。
