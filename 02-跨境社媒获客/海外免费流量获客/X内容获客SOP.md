# X 内容获客 SOP

> **适用行业**：B2B 食品加工机械 / 通用机械设备 / 成人用品（合规）出口
> **核心优势**：X 是唯一允许成人用品广告的主流社媒平台，同时是 B2B 行业讨论的聚集地。
> **定位**：短平快的行业观点 + 线程帖深度内容 + 视频短帖展示。

---

## 一、X 适合的品类

### 推荐品类

| 品类 | 适合度 | 说明 |
|------|--------|------|
| B2B 机械设备 | ★★★★ | 行业讨论活跃 |
| 食品机械 | ★★★ | 有食品行业社区 |
| 成人用品 | ★★★★ | X 允许成人用品内容和广告 |
| VR互动产品 | ★★★★ | 科技角度讨论活跃 |

### 不适合的品类

- 需要深度视觉展示的产品（更适合 IG/TikTok）
- 需要长篇技术说明的产品（更适合 Blog/YouTube）

---

## 二、内容节奏

### 发布频率

| 类型 | 频率 | 说明 |
|------|------|------|
| 短帖 | 每天 1-3 条 | 行业观点/产品片段/互动 |
| 线程帖 | 每周 1-2 条 | 深度内容/案例分析 |
| 视频帖 | 每周 2-3 条 | 机器运转/产品演示 |
| 回复他人 | 每天 5-10 条 | 行业讨论/专业解答 |

### 最佳时间

| 地区 | 时间 |
|------|------|
| 欧美 | 周二-周四 08:00-10:00 / 13:00-15:00 EST |
| 中东 | 周日-周四 10:00-14:00 AST |
| 东南亚 | 周一至周五 09:00-11:00 SGT |

## 三、线程帖（Thread）

### 线程帖结构

```
Tweet 1: Hook（吸引点击）
Tweet 2-7: Core content（核心内容）
Tweet 8: CTA（引导行动）
```

### B2B 线程帖示例

```
Tweet 1:
🏭 I've exported food machinery to 50+ countries in 15 years.
Here are the 7 biggest mistakes I see importers make (and how to avoid them):

Tweet 2:
1. Choosing based on price alone.
The cheapest machine often costs MORE in the long run due to:
- Poor quality parts
- No after-sales support
- Longer downtime
...

Tweet 8:
If you found this helpful, I write about food machinery export every week.
Follow for more insights. 📩 Free quote: www.yoursite.com
```

### 成人用品线程帖示例（合规）

```
Tweet 1:
The global intimate wellness market is projected to hit $52B by 2027.
Here's what manufacturers are seeing in terms of product trends:

Tweet 2:
1. Material shift: From PVC to medical-grade silicone
- Hypoallergenic
- More durable
- Easier to clean
...

Tweet 8:
We specialize in OEM manufacturing for this space.
For business inquiries: email@company.com 📧
```

## 四、视频短帖

### 视频规格

| 要素 | 标准 |
|------|------|
| 时长 | 15-60 秒 |
| 比例 | 16:9 或 9:16（都可） |
| 字幕 | 英文（必须） |
| 开头 | 3 秒钩子（最终成品/数字/对比） |

### 内容类型

| 类型 | 示例 |
|------|------|
| 产品演示 | "Watch this machine package 200 bags per minute" |
| 对比 | "Manual: 50 units/hr vs Auto: 500 units/hr" |
| 行业观点 | "Why automation is the future of food processing" |
| 展会现场 | "Live from [expo] — our new product launch" |

### 热门话题类型

| 话题 | 互动率 | 示例 |
|------|--------|------|
| 行业趋势 | 高 | "2026 food processing trends" |
| 成本分析 | 高 | "Real cost of importing machinery" |

| 使用行业标签 | #FoodProcessing #Manufacturing #Packaging |

## 六、成人用品合规边界

### X 的成人用品政策

| 方面 | 政策 |
|------|------|
| 有机内容 | 允许（需标注敏感内容） |
| 付费广告 | 允许（需遵守广告政策） |
| 敏感内容标注 | 必须在 Profile 中注明 |
| 年龄限制 | 必须标注 18+ |

### 合规做法

| 可做 | 不可做 |
|------|--------|
| 科技/材料角度讨论 | 露骨画面 |
| 行业趋势分析 | 面向未成年人的内容 |
| OEM 制造能力展示 | 违规内容（暴力/非法） |
| 独立站承接 | 直接私信推广 |

### 内容角度（安全）

| 角度 | 示例 |
|------|------|
| 科技 | "How medical-grade silicone changed the industry" |
| 市场 | "The $52B wellness market — what's driving growth" |
| 制造 | "Inside our ISO-certified production facility" |
| 趋势 | "2026 intimate wellness product trends" |

## 七、私信承接

### X 私信特点

- X 私信可以发链接
- 非互关用户消息进入"Message Requests"
- 可以设置自动回复

### 私信流程

```
用户互动（回复/点赞/关注）
  ↓
你回复/关注
  ↓
用户私信 or 你私信
  ↓
对话建立
  ↓
引导到独立站/邮件/WhatsApp
```

### 私信话术

```
Hi [Name], thanks for the engagement on my post!

I saw you're interested in [topic]. I'd love to learn more about your needs.

For business inquiries, the best ways to reach me:
📧 Email: sales@yoursite.com
🌐 Website: www.yoursite.com
📱 WhatsApp: wa.me/xxxxxxx
```

## 九、风险点

| 风险 | 预防 |
