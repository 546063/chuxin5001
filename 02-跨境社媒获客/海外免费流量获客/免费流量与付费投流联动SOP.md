# 免费流量与付费投流联动 SOP

> **适用行业**：B2B 食品加工机械 / 通用机械设备 / 成人用品（合规）出口
> **核心原则**：免费流量是测试场，付费投流是放大器。先用免费流量验证内容和受众，再用广告放大有效内容。

---

## 一、免费流量 + Facebook 投流怎么组合

### 组合策略

| 阶段 | 免费流量做什么 | 付费投流做什么 |
|------|--------------|--------------|
| 第 1-2 周 | 多发内容测试方向 | 不投广告 |
| 第 3-4 周 | 找出 Top 3 内容 | 用 Top 内容做 Spark Ads |
| 第 2 月 | 围绕 Top 做系列 | 扩展受众，测试新素材 |
| 第 3 月 | 稳定发布 + 收集询盘 | 再营销（看过视频的人） |

### 预算分配建议

| 预算范围 | 免费流量占比 | 付费投流占比 |
|---------|------------|------------|
| < $500/月 | 80% | 20% |
| $500-2000/月 | 50% | 50% |
| > $2000/月 | 30% | 70% |

## 二、自然视频如何筛选广告素材

### 筛选标准

| 指标 | 阈值 | 说明 |
|------|------|------|
| 播放量 | > 平均值的 2 倍 | 内容吸引力强 |
| 完播率 | > 30% | 内容留存好 |
| 互动率 | > 5% | 用户愿意参与 |
| 评论质量 | B 端询盘 > C 端闲聊 | 目标受众匹配 |
| 主页访问 | > 平均值的 3 倍 | 引导有效 |
| WhatsApp 点击 | > 5 次 | 有转化意向 |

### 筛选流程

```
发布 20 条自然内容（2-4 周）
  ↓
按播放量排序，筛选 Top 5
  ↓
按互动率排序，筛选 Top 3
  ↓
检查评论质量（B 端 vs C 端）
  ↓
Top 1-2 → 改造成广告素材
```

## 三、评论问题如何反哺广告文案

### 收集高频评论问题

| 评论问题 | 广告文案利用 |
|---------|------------|
| "What's the price?" | 广告中写 "Starting from $X" |
| "Does this ship to [country]?" | 广告中写 "Shipping to 50+ countries" |
| "How much can it produce?" | 广告标题用具体数字 |
| "Is it easy to operate?" | 广告强调 "One-button operation" |
| "What's the warranty?" | 广告中写 "X-year warranty included" |

### 操作方法

1. 每周导出评论区数据
2. 统计 Top 5 高频问题
3. 将问题转化为广告文案的"卖点"
4. A/B 测试不同卖点的广告效果

## 四、WhatsApp 询盘如何反哺内容选题

### 询盘分析

| 询盘类型 | 内容选题方向 |
|---------|------------|
| 价格询问最多 | 做价格透明类内容 |
| 技术规格询问 | 做技术参数详解类内容 |
| 售后问题 | 做售后保障类内容 |
| 物流问题 | 做发货/安装类内容 |
| 竞品对比 | 做对比评测类内容 |

### 操作方法

1. 每月统计 WhatsApp 询盘分类
2. 找出 Top 3 询盘类型
3. 每种类型做 3-5 条内容
4. 下月观察同类型内容的互动变化

## 五、TikTok 爆款如何改成 Facebook 广告

### 改造流程

```
TikTok 爆款视频（15-30 秒）
  ↓
去掉 TikTok 水印
  ↓
加 Facebook 广告 CTA 按钮
  ↓
加文字叠加（Facebook 用户更看文字）
  ↓
设置广告目标（WhatsApp/表单/主页访问）
  ↓
用 Spark Ads 或上传素材投放
```

### 适配差异

| 维度 | TikTok | Facebook 广告 |
|------|--------|-------------|
| 音乐 | 热门音乐 | 无版权音乐或无音乐 |
| 文字 | 少 | 可以多一些（信息密度） |
| CTA | Bio 引导 | 直接按钮 |
| 时长 | 15-30s | 15-60s（可长一点） |
| 格式 | 竖屏 | 竖屏/横屏都可 |

## 六、Facebook Group 问题如何转成短视频选题

### 转化流程

```
Group 中收集高频问题
  ↓
筛选 B2B 相关问题（非 C 端）
  ↓
将问题变成"解答型视频"
  ↓
发布到 TikTok / YouTube / IG
  ↓
在 Group 中分享视频（提供价值）
  ↓
形成内容飞轮
```

### 具体示例

| Group 问题 | 短视频选题 |
|-----------|-----------|
| "How much does a dumpling machine cost?" | "Dumpling Machine Price Guide 2026" |
| "Best equipment for small bakery?" | "Small Bakery Equipment Package — Under $10K" |
| "How to import machinery from China?" | "5 Steps to Import Food Machinery from China" |
| "Manual vs automatic — which is better?" | "Manual vs Automatic — Real Cost Comparison" |

## 七、什么时候从免费流量转付费

### 转付费的信号

| 信号 | 说明 |
|------|------|
| 自然内容有稳定询盘 | 证明产品-市场匹配 |
| 某个内容持续带来线索 | 可以放大这个内容 |
| Group/Page 互动稳定 | 受众基础建立 |
| 有明确的 CPL 目标 | 知道愿意花多少买线索 |
| 有承接能力（WhatsApp 及时回复） | 转化漏斗通畅 |

### 不建议投广告的信号

| 信号 | 说明 |
|------|------|
| 自然内容 0 询盘 | 产品/内容方向可能不对 |
| 主页/内容基础薄弱 | 广告来了也留不住 |
| WhatsApp 回复超过 2 小时 | 承接能力不足 |
| 不清楚目标客户是谁 | 广告定向不精准 |
| 预算 < $300/月 | 太小达不到学习阶段 |

| 渠道 | 角色 | 预算占比 |

| TikTok 自然流量 | 内容测试 + 爆发 | 免费 |

| Facebook 广告 | 放大爆款内容 | 50% |
| Facebook 再营销 | 看过内容的人 | 20% |

### 执行节奏

```
Month 1: 80% 自然 + 20% 广告测试
Month 2: 50% 自然 + 50% 广告放大
Month 3: 30% 自然 + 70% 广告优化
```
