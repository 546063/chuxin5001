# Facebook 个人号内容获客 SOP

> **适用行业**：B2B 食品加工机械 / 通用机械设备出口
> **核心原则**：个人号是"人"，不是"品牌"。建立专业信任比 Page 更快，但边界更敏感。
> **重要提醒**：个人号获客的核心是"先做朋友，再做业务"。硬广告会被举报。

---

## 一、个人号定位

### 1.1 账号形象

| 项目 | 建议 |
|------|------|
| 头像 | 专业个人照片（非工厂 Logo） |
| 封面 | 工作场景（展会/工厂/团队） |
| 简介 | "Food machinery export specialist | Helping businesses scale production" |
| 名字 | 真实英文名 + 姓氏（不要用品牌名） |
| 地区 | 目标市场语言（如客户主要在英语区，用英语资料） |

### 1.2 人设定位

选择以下一种：

| 人设 | 适用 | 内容方向 |
|------|------|---------|
| 行业专家 | 技术人员/工程师 | 技术解析、设备对比、维护技巧 |
| 贸易代表 | 销售/外贸人员 | 客户案例、发货现场、市场动态 |
| 工厂创始人 | 老板/创始人 | 创业故事、工厂文化、行业见解 |

## 二、加好友策略

### 2.1 加谁

| 目标 | 搜索方式 |
|------|---------|
| 食品加工厂老板 | 搜 Group 成员、行业活动参与者 |
| 餐饮设备经销商 | 搜相关 Group、竞品 Page 互动者 |
| 展会参与者 | 搜展会话题标签、活动参与者 |
| 同行推荐 | 通过共同好友推荐 |

### 2.2 加好友频率

| 阶段 | 每日上限 | 说明 |
|------|---------|------|
| 新号（<30 天） | 3-5 个 | 避免触发限制 |
| 成长期（30-90 天） | 5-10 个 | 逐步增加 |
| 成熟期（90 天+） | 10-20 个 | 保持稳定 |

### 2.3 加好友话术

```
Hi [Name], I noticed you're in the food processing industry.
I work in food machinery export and would love to connect
and share insights. Happy to be part of your network!
```

**注意**：不要一加好友就发广告。先建立连接，再谈业务。

## 三、发帖类型

### 3.1 内容比例

| 类型 | 占比 | 示例 |
|------|------|------|
| 个人工作日常 | 30% | "Just finished a factory tour — amazing to see the new production line!" |
| 行业见解 | 25% | "2026 food processing trend: automation is becoming affordable for small businesses" |
| 客户案例 | 20% | "Helped a client in UAE set up their production line. 3 weeks from order to installation!" |
| 产品演示 | 15% | 机器运转短视频（非硬广告） |
| 互动 | 10% | 行业讨论、投票、问答 |

### 3.2 发帖频率

| 阶段 | 频率 | 说明 |
|------|------|------|
| 起步期 | 每周 3-4 条 | 建立内容基础 |
| 稳定期 | 每周 2-3 条 | 保持活跃 |
| 不建议 | 每天多条 | 容易被当 spam |

### 3.3 B2B 设备适用内容

| 内容 | 为什么有效 |
|------|-----------|
| 工厂参观 | 展示真实性和规模 |
| 发货现场 | 证明实际出口能力 |
| 技术解析 | 建立专业信任 |
| 展会照片 | 行业参与度证明 |
| 客户合影 | 关系背书 |

## 四、私信边界

### 4.1 可以做什么

| 行为 | 说明 |
|------|------|
| 回复主动私信 | 必须回复，这是获客入口 |
| 在对方发帖后适度互动 | 点赞 + 专业评论 |
| 给好友发消息 | 先有对话基础，不发硬广告 |
| 邀请好友关注 Page | 自然引导，不强制 |

### 4.2 不能做什么

| 行为 | 风险 |
|------|------|
| 新加好友立即发广告 | 被举报 = 账号受限 |
| 群发相同消息 | Facebook 检测到 spam 行为 |
| 每天发超过 5 条消息 | 触发频率限制 |
| 给非好友发消息 | 对方可举报 |
| 持续骚扰不回复的人 | 严重违规 = 封号 |

### 4.3 私信模板（加好友后）

```
Hi [Name], thanks for connecting!

I work in food machinery export and help businesses
find the right equipment for their needs.

If you ever need advice on food processing equipment,
happy to chat! No pressure at all — just here if useful. 😊
```

## 五、避免骚扰 / 垃圾营销

### 5.1 垃圾营销的 5 个信号

1. 新号加多个商业 Group
2. 每天发相同内容
3. 只发产品广告
4. 不加区分地给所有人发消息
5. 评论全是"Contact me for price"

### 5.2 避免方法

1. 先养号 7-14 天（发日常内容）
2. 只加有共同点的人
3. 先给价值（回答问题/分享知识）
4. 个性化每条消息
5. 控制频率（每天 < 10 条私信）

## 六、从个人号导向 Page / WhatsApp

### 6.1 引导路径

| 方式 | 示例 |
|------|------|
| 个人简介 | "Food machinery export | WhatsApp: +86 XXX | FB Page: [link]" |
| 发帖末尾 | "For more content like this, follow our Page: [link]" |
| 私信引导 | "I share more product videos on our Page: [link]" |
| 评论引导 | "We have a detailed video on our Page: [link]" |

### 6.2 WhatsApp 引导

**不要在个人号直接发 WhatsApp 链接给陌生人。** 建议路径：

```
个人号发帖 → 用户感兴趣 → 私信对话建立 → 引导到 WhatsApp
```

**对话模板**：
```
[After 2-3 exchanges of value]
"I can send you more detailed specs and pricing. 
Is it OK if I share my WhatsApp? Easier for me to send files."
```

## 七、成人用品合规说明

**不建议使用 Facebook 个人号直接公开推广成人用品。**

| 原因 | 说明 |
|------|------|
| 个人审查严格 | 个人号更容易被举报和封禁 |
| 好友举报风险 | 好友可一键举报，触发账号审核 |
| 文化敏感 | 成人用品在大多数地区属于敏感话题 |

**替代方案**：
1. 使用独立站 + SEO/邮件营销
2. 在 Reddit/X 等允许成人用品内容的平台做合规内容
3. 通过合规社群（如行业论坛）建立联系
4. 用个人号做"行业观察"类内容，不直接推广产品

## 八、风险点

| 风险 | 触发条件 | 后果 | 预防 |
|------|---------|------|------|
| 账号受限 | 频繁加好友/发广告 | 发帖/评论受限 | 每天加 < 10 个好友 |
| 被举报 | 发硬广告/骚扰 | 账号降权或封禁 | 先给价值再引导 |
| 被封号 | 严重违规 | 永久封禁 | 不群发、不硬广 |
| 被当 fake | 新号加多人 | 被标记为假号 | 完善资料 + 先发内容 |
