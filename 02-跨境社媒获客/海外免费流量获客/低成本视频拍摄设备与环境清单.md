# 低成本视频拍摄设备与环境清单

> **声明：** 本文档中标注 `[待补充真实数据]` 的价格需根据实际市场情况查询填写。推荐产品为通用类别，具体品牌型号请根据预算和可用性选择。

## 一、最低可用配置（$0 起步）

如果你现在就想开始拍视频，以下是**最低可用配置**，不需要额外花钱。

| 组件 | 方案 | 说明 |
|------|------|------|
| **拍摄设备** | 智能手机（iPhone 12+ 或同级别安卓） | 几乎所有现代手机都支持 1080p/4K 录制 |
| **光线** | 自然光（靠窗/户外） | 面向光源，避免逆光 |
| **稳定** | 手持 + 双臂夹紧身体 | 适合短视频，长视频建议买三脚架 |
| **收音** | 手机自带麦克风 | 近距离拍摄（1-2 米内）可用 |
| **剪辑** | 剪映（CapCut）免费版 | 支持自动字幕、基础剪辑、转场 |
| **字幕** | CapCut 自动字幕（支持多语言） | 准确率约 [待补充真实数据]%，需人工校对 |
| **总成本** | **$0**（已有手机即可） | |

**适用场景：**
- 产品近距离演示（手机距设备 1-2 米）
- 短视频（< 60 秒）
- TikTok / Reels / Shorts 竖屏内容
- 初步测试内容方向

**注意事项：**
- 拍摄前擦拭手机镜头
- 关闭手机自动亮度/自动对焦锁定
- 使用后置摄像头（画质优于前置）
- 横屏拍摄（如需同时发布到多个平台）或竖屏（优先 TikTok/Reels）

### 2.2 各组件选购要点

**三脚架：**
- 高度可调（最低 0.5m，最高 1.5m+）
- 带手机夹具（支持手机宽度范围）
- 蓝牙遥控器（可远程开始/停止录制）
- 重量轻便（工厂移动拍摄需要）

**无线领夹麦克风：**
- 发射器 + 接收器一拖一即可
- 支持降噪功能（工厂环境噪音大）
- 续航 ≥ [待补充真实数据] 小时
- 手机直插（Lightning 或 USB-C）

**补光灯：**
- 可调色温（3200K-5600K）
- 可调亮度（至少 10 档）
- 电池供电或 USB 供电（工厂不一定有插座）
- 尺寸适中（易携带）

## 三、拍摄环境要求

### 3.1 不同环境的拍摄方案

| 环境 | 优势 | 挑战 | 解决方案 |
|------|------|------|----------|
| **工厂车间** | 真实感强、设备齐全 | 噪音大、光线不均、背景杂乱 | 降噪麦克风 + 补光灯 + 清理拍摄区域 |
| **展厅/样品间** | 环境整洁、光线好 | 空间可能较小 | 广角镜头 + 多角度拍摄 |
| **仓库** | 空间大、可展示库存 | 光线通常不足 | 补光灯 + 选择靠窗位置 |
| **户外** | 自然光好 | 天气不确定、噪音 | 选择晴天、使用防风罩麦克风 |
| **办公室** | 安静、可控 | 不适合展示设备 | 适合口播/讲解类内容 |

### 3.2 工厂拍摄注意事项

| 项目 | 说明 |
|------|------|
| 安全 | 穿戴安全帽/工作服，确保拍摄不影响生产安全 |
| 许可 | 获得工厂管理方拍摄许可 |
| 背景 | 清理拍摄区域内的杂物、垃圾、无关物品 |
| 品牌露出 | 确保公司 Logo/名称在画面中适当位置出现 |
| 光线 | 选择白天自然光充足时拍摄，或额外补光 |
| 噪音 | 工厂设备运行声音大，必须使用降噪麦克风 |

## 四、光线布置方案

### 4.1 最简单的光线方案（一灯法）

```
        补光灯（主光）
            │
            ▼
      [ 被拍摄设备 ]
            │
            ▼
         手机/相机
```

- 补光灯放在设备斜前方 45 度角
- 高度略高于设备
- 亮度调至中等（避免过曝）

### 4.2 标准光线方案（三点布光法）

```
         轮廓光（后方）
              │
              ▼
    主光 ← [ 设备 ] → 补光
    (45°)             (45°)
              │
              ▼
           手机/相机
```

| 灯光 | 位置 | 作用 | 亮度 |
|------|------|------|------|
| **主光（Key Light）** | 斜前方 45° | 主要照明 | 100% |
| **补光（Fill Light）** | 另一侧斜前方 45° | 减少阴影 | 50-70% |
| **轮廓光（Back Light）** | 后方 | 勾勒轮廓、分离背景 | 30-50% |

### 4.3 不同场景的光线建议

| 场景 | 方案 |
|------|------|
| 设备运行演示 | 一灯法（主光）即可，设备本身有金属反光 |
| 人物讲解 | 三点布光法 |
| 工厂全景 | 利用自然光 + 一盏补光灯 |
| 户外拍摄 | 自然光为主，必要时用反光板补光 |

## 五、收音方案

### 5.1 不同场景的收音方案

| 场景 | 方案 | 设备 |
|------|------|------|
| 安静环境（办公室/展厅） | 手机自带麦克风 | 无额外成本 |
| 轻度噪音（仓库） | 有线领夹麦克风 | [待补充真实数据] |
| 强噪音（工厂车间） | 无线降噪领夹麦克风 | [待补充真实数据] |
| 远距离收音 | 指向性麦克风/枪麦 | [待补充真实数据] |
| 纯设备运行（无人声） | 录制环境音 + 后期配音 | 后期加旁白 |

### 5.2 工厂环境收音技巧

1. **尽量靠近声源** — 麦克风距嘴部 15-20cm 最佳
2. **使用防风罩** — 工厂有风扇/排风时减少风噪
3. **关闭设备时录制讲解** — 先讲解，再单独录制设备运行声音
4. **后期降噪** — 使用 CapCut / Audacity 的降噪功能

## 八、拍摄 Checklist

### 8.1 拍摄前

- [ ] 手机电量充足（≥ 80%）或连接充电宝
- [ ] 存储空间充足（≥ 10GB）
- [ ] 镜头已擦拭干净
- [ ] 三脚架已架设并调平
- [ ] 麦克风已连接并测试录音
- [ ] 光线已布置（自然光或补光灯）
- [ ] 拍摄区域已清理
- [ ] 设备已就位并运行正常
- [ ] 已写好拍摄大纲/脚本（至少是关键词列表）
- [ ] 已测试录制 10 秒并回看效果

### 8.2 拍摄中

- [ ] 锁定曝光和对焦（长按屏幕锁定）
- [ ] 保持画面稳定（三脚架或稳定器）
- [ ] 注意构图（设备在画面中的位置）
- [ ] 口播时看着镜头（不是屏幕）
- [ ] 录制至少 2 遍（以备选择）
- [ ] 每段录制前后各留 3 秒空白（方便剪辑）

### 8.3 拍摄后

- [ ] 立即回看检查（画面/声音/光线）
- [ ] 备份到电脑/云盘
- [ ] 标注素材内容（文件名含日期+产品名）
- [ ] 记录需要补拍的镜头

| 视频类型 | 最低配置 | 推荐配置 | 备注 |
