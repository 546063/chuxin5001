# 海外免费流量 30 天执行检查表

> **适用行业**：B2B 食品加工机械 / 通用机械设备 / 成人用品（合规）出口
> **每日时间投入**：1-2 小时
> **预期效果**：30 天后积累内容基础、获得初始询盘、明确有效内容方向

---

## 第 1 周：账号与素材准备

### Day 1-2：账号定位与主页包装

- [ ] **确定主攻平台**（建议 2-3 个，参考 [[海外免费流量平台选择判断表]]）
  - 食品机械：TikTok + YouTube Shorts + Facebook Reels
  - 机械设备：YouTube Shorts + TikTok + Reddit
  - 成人用品：Reddit + X/Twitter
- [ ] **注册/优化账号**
  - [ ] 头像：工厂 Logo 或专业个人头像
  - [ ] 简介：一行说明（"Food machinery manufacturer | 20+ years | Global export"）
  - [ ] Bio 链接：wa.me 短链接 或 Linktree
- [ ] **联系方式准备**
  - [ ] WhatsApp Business 账号设置
  - [ ] WhatsApp 快捷回复模板（5 条以上）
  - [ ] 邮箱签名配置
  - [ ] 询盘表单（独立站或 Google Form）

### Day 3-5：选题与脚本

- [ ] **确定 30 个选题**（参考 [[食品机械海外短视频100选题库]] 或 [[B2B设备短视频选题库]]）
- [ ] **写 10 条脚本**
  - [ ] 每条 15-60 秒
  - [ ] 包含 3 秒钩子
  - [ ] 包含 CTA（WhatsApp 引导）
- [ ] **确定内容结构**（参考 [[免费短视频爆款打流量策略库]]）
  - [ ] 痛点型：2 条
  - [ ] 对比型：2 条
  - [ ] 过程展示型：2 条
  - [ ] 结果展示型：2 条
  - [ ] 工厂背书型：1 条
  - [ ] 问题解答型：1 条

### Day 6-7：拍摄与剪辑

- [ ] **拍摄 5 条样片**
  - [ ] 机器运转画面（核心）
  - [ ] 工厂环境（背书）
  - [ ] 成品展示（结果）
- [ ] **剪辑**
  - [ ] 竖屏 9:16
  - [ ] 加字幕（英语为主）
  - [ ] 加背景音乐（无版权）
  - [ ] 末尾加 CTA（WhatsApp 引导）
- [ ] **发布前检查**
  - [ ] 每个平台的标题格式
  - [ ] 标签（3-5 个相关标签）
  - [ ] Bio 链接是否正常

## 第 3 周：筛选有效内容

### Day 15-21：数据分析

#### 哪些内容有播放？

- [ ] 查看播放 Top 3 的内容
- [ ] 分析共同特征：
  - [ ] 什么钩子有效？
  - [ ] 什么时长有效？
  - [ ] 什么话题有效？
  - [ ] 什么时间发布有效？

#### 哪些内容有评论？

- [ ] 查看评论 Top 3 的内容
- [ ] 评论类型分析：
  - [ ] 价格询问（→ 高意向）
  - [ ] 技术讨论（→ 中意向）
  - [ ] 一般互动（→ 低意向）

#### 哪些内容有私信？

- [ ] 查看私信来源内容
- [ ] 私信内容分析：
  - [ ] 询盘 vs 闲聊
  - [ ] B2B vs B2C
  - [ ] 已转 WhatsApp 数

#### 哪些内容能转 WhatsApp？

- [ ] 统计 WhatsApp 新增联系人来源
- [ ] 分析从什么内容来的客户意向最高

#### 哪些内容应该停？

- [ ] 播放 < 100 且无互动的内容类型 → 停止
- [ ] 纯广告类内容 → 调整为价值型
- [ ] 与目标受众不匹配的内容 → 停止

### 内容优化

- [ ] 复制 Top 3 内容的结构（换产品/换钩子）
- [ ] 制作系列内容（"Part 1, Part 2..."）
- [ ] 尝试多语言（阿拉伯语/西班牙语字幕）

## 第 4 周：放大与承接

### Day 22-28：放大有效内容

- [ ] **复制 Top 内容**
  - [ ] 同一结构，不同产品 → 3 条
  - [ ] 同一产品，不同钩子 → 3 条
  - [ ] 同一内容，多语言 → 2 条
- [ ] **系列化**
  - [ ] 做 3 个系列（如 "Machine of the Week"）
  - [ ] 每周更新系列内容
- [ ] **评论区引导**
  - [ ] 所有热门评论置顶 CTA
  - [ ] 回复所有新评论
  - [ ] 主动在评论区引导 WhatsApp
- [ ] **私信 SOP 执行**
  - [ ] 按 [[免费流量转WhatsApp承接SOP]] 执行
  - [ ] 5 分钟内回复 WhatsApp
  - [ ] 24 小时内发送报价
