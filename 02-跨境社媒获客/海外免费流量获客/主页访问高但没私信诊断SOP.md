# 主页访问高但没私信诊断SOP

> **适用场景**：账号有稳定的视频播放和主页访问，但几乎没有私信或询盘
> **适用行业**：B2B 食品加工机械出口
> **适用平台**：TikTok / Facebook / Instagram / YouTube / 视频号 / 抖音

---

## 一、核心判断：什么情况算"高访问低转化"

### 1.1 量化标准

| 指标 | 正常范围 | 预警 | 需要行动 |
|------|---------|------|---------|
| 主页访问/私信比 | 100:1 ~ 300:1 | 300:1 ~ 500:1 | >500:1 |
| 链接点击率（主页链接） | >3% | 1-3% | <1% |
| 主页访问/播放比 | >5% | 2-5% | <2% |
| 私信回复率 | >50% | 20-50% | <20% |

> **举例**：如果每天主页访问 500 次，但一周只有 0-1 条私信，这就是典型的高访问低转化。

### 1.2 判断问题出在哪一层

```
视频播放 → 用户点击主页 → 浏览主页内容 → 点击链接/发私信 → WhatsApp/询盘
            ↑              ↑                  ↑
         第1层漏斗      第2层漏斗           第3层漏斗
```

- **视频到主页点击低**：内容好但用户没有进主页的动机（视频结尾无 CTA）
- **主页点击后无动作**：主页设计/信任度有问题（核心问题，本SOP重点）
- **发了私信但没回复**：不是主页问题，是私信承接问题（参考 [[05-海外短视频私信转化SOP]]）

## 二、诊断流程

### Step 1：检查主页基础信息（5分钟）

- [ ] Bio/简介是否清楚说明"你是谁、做什么、优势是什么"？
- [ ] 头像是否专业（工厂 Logo/真实人物/产品图）？
- [ ] 封面图/背景图是否展示了核心业务？
- [ ] 是否有明确的 CTA（Call to Action）？
- [ ] 是否有联系方式/链接？
- [ ] 链接是否有效（点击测试）？
- [ ] 置顶内容是否是最有吸引力的内容？

### Step 2：检查信任度（10分钟）

- [ ] 粉丝数是否过低（<100）？
- [ ] 内容数量是否过少（<10 条）？
- [ ] 是否有工厂/生产线/团队展示？
- [ ] 是否有任何形式的信任背书（认证/案例/客户评价）？
- [ ] 账号是否有认证标识？
- [ ] 主页内容是否看起来专业/可信？

### Step 3：检查转化路径（5分钟）

- [ ] 用户是否能方便地联系到你？
  - TikTok：Bio 链接 / 私信
  - Instagram：Bio 链接 / 私信 / WhatsApp 按钮
  - Facebook：Messenger / WhatsApp 按钮 / 主页链接
  - YouTube：频道"关于"页面链接 / 视频描述
- [ ] Bio 链接是否指向正确的落地页？
- [ ] WhatsApp 号码是否正确、是否在线？
- [ ] 链接加载速度是否快（特别是海外访问）？

### Step 4：检查内容一致性（10分钟）

- [ ] 视频内容和主页定位是否一致？
- [ ] 用户从视频进入主页后，是否看到"想看"的内容？
- [ ] 主页的视频列表是否杂乱/不垂直？
- [ ] 置顶内容是否能回答买家最关心的问题？

## 三、修复动作

### 3.1 Bio/简介优化

**公式**：我是谁 + 我做什么 + 我的优势 + 行动引导

**TikTok Bio 模板**：
```
🏭 Food Machinery Manufacturer since [年份]
✅ [数字]+ Countries Served | [数字]+ Machines Sold
📩 Get Quote → [链接]
📱 WhatsApp: [号码]
```

**Instagram Bio 模板**：
```
🏭 Professional Food Processing Equipment
🌍 Exported to [数字]+ Countries
📧 [邮箱] | 📱 WhatsApp: [号码]
👇 Get Free Quote
[链接]
```

**YouTube Channel Description 模板**：
```
We are a leading food processing machinery manufacturer based in [城市], [国家]. Since [年份], we have exported our machines to [数字]+ countries including [主要市场].

Our products: [产品列表]

Contact us:
📧 Email: [邮箱]
📱 WhatsApp: [号码]
🌐 Website: [网址]
```

### 3.2 主页链接优化

| 方案 | 优点 | 缺点 | 推荐场景 |
|------|------|------|---------|
| 直接放 WhatsApp 链接 | 路径最短 | 无法追踪数据 | 已有稳定询盘 |
| Linktree 类聚合页 | 多链接整合 | 多一次跳转 | 多渠道承接 |
| 自建落地页 | 完全可控 | 需要开发 | 长期运营 |
| WhatsApp 二维码 | 扫码直达 | 移动端不便 | 线下场景 |

**WhatsApp 链接格式**：
```
https://wa.me/[国家代码][手机号]?text=[预设消息]
```
例：`https://wa.me/8613800138000?text=Hi%2C%20I%27m%20interested%20in%20your%20food%20machinery.%20Can%20you%20send%20me%20a%20catalog%3F`

### 3.3 CTA 优化

**视频结尾 CTA**：
- "Click the link in my bio to get a free quote"
- "DM me 'CATALOG' for our latest price list"
- "Comment your country and I'll send you shipping info"

**主页 CTA**：
- Bio 中必须有明确的行动引导词
- 使用 emoji 增加视觉引导（👇📩📱）
- 链接文字要说明点击后得到什么

## 四、主页优化清单

### 4.1 必做项（影响最大）

| 项目 | 标准 | 检查方式 |
|------|------|---------|
| Bio 清晰 | 3 秒内能看懂你是做什么的 | 让非行业人士看，能否说出你的业务 |
| 头像专业 | Logo 清晰或真实人物 | 缩略图下是否可识别 |
| 有效链接 | 链接可点击、可打开、加载快 | 实际点击测试 |
| CTA 明确 | 有明确的行动引导 | 是否有"Get Quote""Contact Us"等词 |
| 联系方式 | 至少一种即时联系方式 | WhatsApp/邮箱/电话 |

### 4.2 加分项

| 项目 | 说明 |
|------|------|
| 封面图 | 展示工厂/产品/团队 |
| 置顶视频 | 最有说服力/最高质量的 1-3 条 |
| 精选合集 | 按产品分类整理内容 |
| 认证标识 | 平台认证或行业认证 |
| 客户案例 | 客户使用机器的视频 |
| 工厂展示 | 生产线/质检/发货实拍 |

### 4.3 信任度建设

| 信任元素 | 实现方式 | 优先级 |
|---------|---------|--------|
| 工厂实力 | 工厂实拍视频/图片 | ★★★★★ |
| 出口经验 | "已出口到 XX 国家" | ★★★★★ |
| 客户案例 | 客户反馈/使用视频 | ★★★★ |
| 资质证书 | CE/FDA/ISO 等认证展示 | ★★★★ |
| 团队展示 | 团队合影/工作场景 | ★★★ |
| 历史视频 | 展示公司发展历程 | ★★★ |
| 客户评价 | 文字/视频评价截图 | ★★★ |
| 发货记录 | 装箱/发货实拍 | ★★ |

## 五、内容-主页一致性检查

### 5.1 检查方法

1. **模拟用户路径**：用其他账号看你的视频 → 点击主页 → 判断
2. **核心问题**：用户进主页后，3 秒内能否确认"这就是我要找的供应商"？

### 5.2 一致性维度

| 维度 | 检查项 | 不一致的表现 |
|------|--------|-------------|
| 行业一致 | 视频和主页都聚焦食品机械 | 视频是食品机械，主页有其他行业内容 |
| 定位一致 | 角色定位统一 | 视频像工厂，主页像贸易商 |
| 质量一致 | 视频质量与主页形象匹配 | 视频专业，主页简陋 |
| 语言一致 | 视频语言和主页语言一致 | 视频英文，主页中文 |
| 品牌一致 | Logo/配色/风格统一 | 头像和封面图风格不统一 |

### 5.3 修复方案

- [ ] 清理不相关内容（删除/隐藏与食品机械无关的视频）
- [ ] 统一语言（面向海外买家，使用英文）
- [ ] 统一视觉风格（头像/封面/视频封面色调一致）
- [ ] 补充缺失的信任元素（工厂视频/认证展示）

## 六、各平台优化要点

### 6.1 TikTok

| 要点 | 说明 |
|------|------|
| Bio 字数 | 80 字符限制，必须精简 |
| 链接限制 | 需要 1000 粉丝或企业号才能放链接 |
| 私信 | 未关注用户也可发私信（需设置） |
| 置顶 | 可置顶 3 条视频 |
| 关键设置 | 设置 → 隐私 → 私信 → 所有人 |

**TikTok 链接问题解决**：
- 不满 1000 粉丝：用 Instagram 账号承接，Bio 放 IG 号
- 不满 1000 粉丝：用 WhatsApp 号码写在 Bio 文字中
- 申请企业号（Business Account）可解锁链接功能

### 6.2 Instagram

| 要点 | 说明 |
|------|------|
| Bio 字数 | 150 字符 |
| 链接 | 可放 1 个链接（或使用 Linktree） |
| 联系方式 | 可添加 Email/Phone/WhatsApp 按钮 |
| 精选动态 | Highlights 可分类展示产品/工厂/案例 |
| 账号类型 | 用 Creator 账号（比 Business 推荐权重高） |

### 6.3 Facebook

| 要点 | 说明 |
|------|------|
| 主页类型 | 用 Facebook Page（非个人号） |
| CTA 按钮 | 可设置 WhatsApp/Messenger/Website 按钮 |
| 置顶帖子 | 可置顶 1 条帖子 |
| 简介 | About 部分要填写完整 |
| 群组联动 | 可关联相关群组增加曝光 |

### 6.4 YouTube

| 要点 | 说明 |
|------|------|
| 频道描述 | About 页面写完整业务介绍 |
| 链接 | 可放多个链接（频道首页展示） |
| Banner | 频道封面图放核心业务信息 |
| 视频描述 | 每条视频描述放联系方式 |
| Community Tab | 可用 Community Tab 发布更新 |

### 6.5 视频号/抖音

| 要点 | 说明 |
|------|------|
| 简介 | 用中文，突出出口能力 |
| 链接 | 视频号可挂公众号链接 |
| 联系方式 | Bio 中可留微信号 |
| 认证 | 企业认证增加信任度 |

## 七、诊断清单（快速版）

发现"高访问低转化"时，用此清单快速定位：

- [ ] Bio 是否清楚说明业务？（3 秒测试）
- [ ] 链接是否有效？（点击测试）
- [ ] WhatsApp 是否可联系？（发送测试消息）
- [ ] 置顶内容是否有说服力？
- [ ] 主页是否有工厂/实力展示？
- [ ] 视频内容和主页是否一致？
- [ ] 是否有明确的 CTA？
- [ ] 语言是否面向目标市场？
- [ ] 联系方式是否方便找到？
- [ ] 整体视觉是否专业可信？

**得分 ≥8**：主页健康，问题可能在视频 CTA 或私信承接
**得分 5-7**：需要优化主页，优先修 Bio/链接/CTA
**得分 <5**：主页需要大改，按本 SOP 逐项修复
