# 新账号发10条没流量处理SOP

> **适用场景**：新注册账号连续发布 10+ 条视频，播放量远低于平台正常水平
> **适用行业**：B2B 食品加工机械出口
> **适用平台**：TikTok / Facebook / Instagram / YouTube / 视频号 / 抖音

---

## 一、核心判断：什么是"没流量"

### 1.1 播放量分级定义

| 级别 | 播放量范围 | 判断 | 应对级别 |
|------|-----------|------|----------|
| 0播放 | 0 | **严重异常** | 立即排查账号/技术问题 |
| 极低播放 | 1-50 | **高危** | 大概率账号或内容有问题 |
| 低播放 | 50-100 | **预警** | 需要优化内容质量 |
| 偏低播放 | 100-500 | **正常偏低** | 继续优化，保持发布频率 |
| 正常冷启动 | 500-2000 | **正常** | 新号合理水平，不需要焦虑 |

### 1.2 各平台新号前10条视频的正常播放范围

| 平台 | 正常范围 | 预警线 | 危险线 | 说明 |
|------|---------|--------|--------|------|
| TikTok | 200-2000 | <100 | 0 | TikTok 新号通常有 200-500 初始推送 |
| Instagram Reels | 100-1000 | <50 | 0 | IG 依赖粉丝+推荐混合 |
| Facebook Reels | 100-1500 | <50 | 0 | FB 新号冷启动较慢 |
| YouTube Shorts | 500-5000 | <100 | <50 | YouTube 给 Shorts 较大初始流量 |
| 视频号 | 50-500 | <20 | 0 | 依赖社交关系链，新号起步慢 |
| 抖音 | 200-3000 | <100 | 0 | 国内竞争激烈，新号有初始推送 |

> **注意**：以上数据基于 B2B 设备类账号的观察，实际值受内容质量、发布时间、标签使用等多因素影响。[待补充真实数据]

## 二、诊断流程（按优先级）

### Step 1：检查账号技术问题（5分钟）

- [ ] 账号是否完成注册/验证流程？
  - TikTok：手机号/邮箱验证、年龄验证
  - Facebook：手机号验证、个人号关联
  - Instagram：邮箱/手机验证
  - YouTube：Google 账号验证
- [ ] 账号是否被封禁/限制？
  - 尝试私信其他账号，看是否发送成功
  - 尝试评论，看是否可见
  - 检查是否有平台通知/警告
- [ ] 账号资料是否完整？
  - 头像、用户名、Bio 是否已填写
  - 是否使用真实/合理的账号信息
- [ ] 网络环境是否正常？
  - TikTok：手机卡/IP 是否匹配目标国家（非中国）
  - 其他平台：网络是否可正常访问目标平台
- [ ] 视频是否审核通过？
  - 发布后是否显示"审核中"超过 24 小时
  - 是否有违规/限流通知

### Step 2：检查内容基础问题（10分钟）

- [ ] 视频格式是否正确？
  - 时长：9:16 竖屏（TikTok/Reels/Shorts 推荐 9-60 秒）
  - 分辨率：至少 720p，推荐 1080p
  - 文件大小：未超过平台限制
- [ ] 视频是否有声音？
  - 静音视频在 TikTok 上推荐权重极低
  - 是否使用平台推荐的背景音乐
- [ ] 视频是否有文案/标题/描述？
  - 纯视频无文案，推荐权重降低
  - 是否有相关标签（3-5个）
- [ ] 内容是否原创？
  - 搬运/二次创作的内容可能被限流
  - 水印检测：是否带有其他平台水印

### Step 3：检查内容质量问题（15分钟）

- [ ] 前 3 秒是否有吸引力？
  - 画面是否有冲击力/好奇心/价值感
  - 是否有文字叠加说明内容主题
- [ ] 视频是否有明确的"看点"？
  - 展示机器运行效果？
  - 对比效果？
  - 客户案例？
- [ ] 视频是否过长/过短？
  - 内容空洞的长视频 = 完播率低 = 不推荐
  - 建议新号先从 15-30 秒开始
- [ ] 画面/声音质量是否合格？
  - 模糊、抖动、噪音会显著降低推荐

### Step 4：检查发布策略问题（5分钟）

- [ ] 发布时间是否合适？
  - TikTok：目标时区的 8-10AM / 12-2PM / 7-9PM
  - 其他平台参考各自活跃时间
- [ ] 标签使用是否正确？
  - 是否使用了相关标签（非热门标签堆砌）
  - 标签数量：3-8 个为宜
- [ ] 发布频率是否合理？
  - 一天发 10 条 vs 10 天发 10 条，效果不同
  - 建议：每天 1-3 条，持续 10 天

## 三、各平台专项诊断

### 3.1 TikTok 专项

| 检查项 | 说明 | 怎么查 |
|--------|------|--------|
| 影子 ban | 搜索自己的用户名，看是否能搜到账号 | 用其他账号搜索 |
| 环境检测 | TikTok 会检测 SIM 卡/IP/语言/时区 | 检查手机设置 |
| 音乐版权 | 使用了无版权音乐会被静音或限流 | 用 TikTok 音乐库 |
| 内容重复 | 连续发相似内容会被判定低质 | 增加内容多样性 |
| 新号考察期 | 前 3-7 天可能流量偏低 | 正常现象，坚持发 |

**TikTok 特殊排查**：
1. 清除 TikTok 缓存，重新登录
2. 检查手机是否插了中国 SIM 卡（必须拔除或使用目标国家 SIM）
3. 检查 IP 是否被 TikTok 标记（用目标国家住宅 IP）
4. 尝试用 TikTok 自带编辑器编辑后发布（排除第三方工具问题）

### 3.2 Facebook 专项

| 检查项 | 说明 |
|--------|------|
| 账号真实性 | 新注册 FB 号需要先完善资料、加一些好友 |
| 内容违规 | FB 对 B2B/商业内容审核严格 |
| Page vs 个人号 | 用 Page 发 Reels 初始流量较低 |

### 3.3 Instagram 专项

| 检查项 | 说明 |
|--------|------|
| 账号类型 | Business 账号 vs Creator 账号，Creator 推荐权重更高 |
| 标签策略 | IG 依赖标签发现，需要精准标签 |
| 互动要求 | 新号需要先互动（评论/点赞）再发布 |

### 3.4 YouTube 专项

| 检查项 | 说明 |
|--------|------|
| Shorts 标识 | 视频必须竖屏 + 60 秒内才走 Shorts 推荐流 |
| 标题关键词 | YouTube 是搜索引擎，标题关键词很重要 |
| 频道完善度 | 头像、Banner、频道描述需完善 |

### 3.5 视频号/抖音专项

| 检查项 | 说明 |
|--------|------|
| 实名认证 | 国内平台需实名认证才能获得推荐 |
| 内容审核 | 国内审核更严格，涉及出口设备内容注意措辞 |
| 社交推荐 | 视频号依赖社交分享，新号需要冷启动 |

## 五、修复动作清单

### 5.1 账号问题修复

- [ ] **补全注册信息**：完成所有验证步骤
- [ ] **更换网络环境**：TikTok 需确保 SIM/IP/语言/时区一致
- [ ] **清理缓存**：卸载重装或清除应用缓存
- [ ] **账号养号**：先浏览、点赞、评论 2-3 天再发布内容
- [ ] **联系平台客服**：确认账号是否被限制

### 5.2 内容问题修复

- [ ] **提升视频质量**：至少 720p，画面稳定，有声音
- [ ] **优化开头 3 秒**：使用强钩子（参考 [[TikTok开头3秒钩子与完播率提升SOP]]）
- [ ] **添加文字叠加**：在视频开头叠加文字说明主题
- [ ] **使用热门音乐**：使用平台推荐/热门背景音乐
- [ ] **缩短视频时长**：从 15-30 秒开始测试
- [ ] **增加内容多样性**：不要连续发相同类型的视频

### 5.3 标签问题修复

- [ ] **重新选标签**：
  - 2 个行业标签（如 #foodmachinery #packagingmachine）
  - 2 个产品标签（如 #sausage #bakery）
  - 1 个地域标签（如 #usa #indonesia）
  - 1 个热门标签（如 #fyp #viral）
- [ ] **避免热门标签堆砌**：#fyp #foryou #viral 最多用 1-2 个
- [ ] **使用平台推荐标签**：发布时看平台推荐的标签

### 5.4 时间问题修复

- [ ] **调整发布时间**：选择目标市场活跃时间段
- [ ] **控制发布频率**：每天 1-3 条，均匀分布
- [ ] **持续发布**：至少坚持 14-30 天再判断

### 5.5 技术问题修复

- [ ] **更换发布方式**：从手机 App 直接发布（非第三方工具）
- [ ] **检查视频参数**：格式 MP4/H.264、帧率 30fps、9:16
- [ ] **重新上传**：删除原视频，修改后重新上传

## 八、何时放弃重开新号

### 8.1 建议继续运营的情况

- 播放量在 100-500 之间，有波动上升趋势
- 偶尔有 1-2 条视频播放量 >1000
- 有少量粉丝增长和互动
- 发布时间 <30 天

### 8.2 建议放弃重开的情况

| 条件 | 说明 |
|------|------|
| 连续 30 天 0 播放 | 账号可能已被系统标记 |
| 账号被正式限流/封禁 | 收到平台通知 |
| 环境无法修复 | TikTok 环境无法稳定 |
| 连续 30 天 <50 播放 | 内容方向或账号质量有根本问题 |
| 养号 7 天后仍 0 播放 | 账号可能已被标记 |

### 8.3 重开新号的注意事项

1. **更换设备或恢复出厂设置**（尤其是 TikTok）
2. **更换网络环境**（新的 SIM 卡/IP）
3. **使用新的注册信息**（手机号/邮箱）
4. **注册后养号 3-7 天**再发布内容
5. **第一批内容要高质量**（前 5 条决定账号标签）
