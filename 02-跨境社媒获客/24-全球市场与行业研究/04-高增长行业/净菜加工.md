# 净菜加工

## 一句话定义

> **净菜加工 = 清洗+切菜+包装 = 直接供应餐饮企业的半成品蔬菜**

---

## 为什么增长

### 核心驱动因素

| 驱动因素 | 说明 | 影响 |
|---------|------|------|
| 连锁餐饮 | 需要标准化蔬菜 | 需求大 |
| 预制菜 | 净菜是基础 | 带动增长 |
| 人工成本 | 切菜人工贵 | 机器替代 |
| 食品安全 | 专业加工更安全 | 餐厅愿意买 |

### 数据

- 全球净菜市场：2023年约300亿美元
- 预计2030年达500亿美元
- 年复合增长率：8%

---

## 哪些国家增长最快

### 第一梯队

| 国家 | 增长率 | 说明 |
|------|--------|------|
| 中国 | 12-15%/年 | 最大市场 |
| 美国 | 8-10%/年 | 最大消费国 |
| 东南亚 | 10-14%/年 | 快速 |

### 第二梯队

| 国家 | 增长率 | 说明 |
|------|--------|------|
| 欧洲 | 5-6%/年 | 成熟 |
| 日本 | 3-4%/年 | 高度成熟 |

## 他们最需要什么设备

### 核心设备需求

| 设备 | 需求热度 | 原因 |
|------|---------|------|
| 洗菜机 | ⭐⭐⭐⭐⭐ | 核心设备 |
| 切菜机 | ⭐⭐⭐⭐⭐ | 核心设备 |
| 脱水机 | ⭐⭐⭐⭐ | 去除水分 |
| 真空包装机 | ⭐⭐⭐⭐⭐ | 保鲜包装 |
| 金属检测仪 | ⭐⭐⭐⭐ | 安全 |

### 生产线配置

```
蔬菜 → 清洗 → 切菜 → 脱水 → 包装 → 金属检测 → 冷藏
```

---

## 他们最怕什么

| 痛点 | 说明 | 解决方案 |
|------|------|---------|
| 农药残留 | 食品安全 | 超声波清洗 |
| 氧化变色 | 影响卖相 | 护色+速冻 |
| 保鲜期短 | 损耗大 | 真空包装 |
| 切割不均 | 影响卖相 | 高精度切菜机 |

---

## 如何开发他们

### 开发策略

| 步骤 | 内容 |
|------|------|
| 1. 了解产品 | 叶菜/根茎类 |
| 2. 展示设备 | 清洗+切菜效果 |
| 3. 强调保鲜 | 货架期 |
| 4. 提供案例 | 餐饮连锁案例 |
| 5. 服务承诺 | 维保 |

### 沟通重点

| 重点 | 话术方向 |
|------|---------|
| 卫生 | 农药残留去除99% |
| 保鲜 | 货架期延长50% |
| 效率 | 产能提升 |
| 损耗 | 减少浪费 |
