# 中央厨房

## 一句话定义

> **中央厨房 = 连锁餐饮标准化+成本控制+食品安全 = 餐饮企业必建基础设施**

---

## 为什么增长

### 核心驱动因素

| 驱动因素 | 说明 | 影响 |
|---------|------|------|
| 连锁餐饮爆发 | 麦肯系+本土连锁 | 需要标准化食物供应 |
| 人工成本上涨 | 厨师工资涨 | 中央厨房替代厨师 |
| 食品安全法规 | 政府监管加强 | 中央厨房更易监管 |
| 成本压力 | 餐饮利润率下降 | 集中采购制作降低成本 |
| 外卖平台 | 出餐时间要求更快 | 预处理好的食材加快出餐 |

### 数据

- 全球中央厨房市场规模：2023年约2800亿美元
- 预计2030年达4500亿美元
- 年复合增长率：7.1%

## 他们最需要什么设备

### 核心设备需求

| 设备 | 需求热度 | 原因 |
|------|---------|------|
| 洗菜机 | ⭐⭐⭐⭐⭐ | 批量清洗 |
| 切菜机 | ⭐⭐⭐⭐⭐ | 标准化切配 |
| 肉类加工设备 | ⭐⭐⭐⭐⭐ | 批量切肉 |
| 炒菜锅（自动） | ⭐⭐⭐⭐⭐ | 批量烹饪 |
| 真空包装机 | ⭐⭐⭐⭐ | 腌制+包装 |
| 速冻设备 | ⭐⭐⭐⭐ | 锁鲜储存 |
| 冷藏设备 | ⭐⭐⭐⭐⭐ | 冷链储存 |
| 油烟净化 | ⭐⭐⭐⭐ | 环保合规 |

### 完整解决方案

```
蔬菜线：清洗 → 切菜 → 炒制 → 速冻 → 包装
肉类线：解冻 → 切肉 → 腌制 → 炒制 → 包装
↓
配送：冷藏车 → 门店
```
