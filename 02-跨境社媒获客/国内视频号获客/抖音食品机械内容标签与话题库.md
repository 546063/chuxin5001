## 三、话题选择策略

### 3.1 大流量 vs 精准流量

| 策略 | 适用阶段 | 标签组合 | 目标 |
|------|---------|---------|------|
| 大流量优先 | 冷启动期（前10条） | 2个大流量 + 2个精准 + 1个场景 | 让算法认识你 |
| 精准流量优先 | 稳定运营期 | 1个大流量 + 3个精准 + 1个长尾 | 精准获客 |
| 搜索流量优先 | 长期积累 | 以B级/C级搜索词为主 | 搜索长尾流量 |

### 3.2 标签数量规范

| 平台规则 | 建议 |
|---------|------|
| 抖音最多可带多少个话题 | [待补充真实数据]（建议不超过10个） |
| 最佳数量 | 3-5个（信息密度高且不显堆砌） |
| 最少数量 | 至少1个（否则无话题信号） |

### 3.3 标签组合公式

**标准组合**：
```
#核心行业标签 + #产品标签 + #场景标签 + #地区标签(可选)
```

**示例**：
```
视频主题：香肠灌肠机产品演示
标签：#食品机械 #灌肠机 #香肠生产线 #肉制品加工 #广东
```

**搜索优先组合**：
```
#产品词 + #产品词+价格/厂家 + #场景词
```

**示例**：
```
视频主题：灌肠机价格介绍
```

| 热点类型 | 示例 | 内容角度 |
|---------|------|---------|
| 政策变化 | 食品安全新标准/SC认证新规 | "新标准下设备要如何选？" |
| 行业趋势 | 预制菜爆发/中央厨房热 | "预制菜热背后的设备需求" |
| 季节热点 | 年货季/中秋月饼季 | "月饼季到了，你的设备准备好了吗？" |
| 社会事件 | 食品安全事件 | "从XX事件看食品加工设备的卫生要求" |
| 展会活动 | 食品机械展/广交会 | "XX展会上的新设备趋势" |

### 5.3 话题热度评估方法

在决定是否追热点前，评估：

```
1. 搜索量：通过巨量算数查看话题搜索趋势
2. 相关度：与食品机械的关联程度（强相关/弱相关/不相关）
3. 竞争度：有多少同行在追这个话题
4. 持续性：是短期热点还是长期趋势
```

## 六、标签使用规范

### 6.1 数量规范

| 场景 | 建议数量 | 说明 |
|------|---------|------|
| 产品展示视频 | 4-5个 | 核心产品标签 + 行业标签 + 场景标签 |
| 知识科普视频 | 3-4个 | 行业标签 + 知识标签 + 场景标签 |
| 工厂实拍视频 | 3-4个 | 行业标签 + 工厂实拍 + 地区标签 |
| 客户案例视频 | 4-5个 | 行业标签 + 产品标签 + 案例标签 + 地区标签 |

### 6.2 位置规范

- 标签放在 **视频描述的末尾**
- 标签之间用空格分隔
- 不要在视频画面中堆砌标签
- 不要在评论区重复堆砌标签

### 6.3 格式规范

- 使用 `#标签名` 格式
- 标签名不含空格（如 `#食品机械` 而非 `#食品 机械`）
- 不使用特殊符号（除#外）
- 标签名使用中文（目标用户为中文用户时）

### 6.4 禁止行为

| 行为 | 风险 | 说明 |
|------|------|------|
| 堆砌无关标签 | 限流/降权 | 如食品机械视频带美妆标签 |
| 使用违规标签 | 限流/封号 | 涉及敏感/违规内容的标签 |
| 每个视频标签完全一样 | 算法判定模板化 | 建议每次微调 |
| 标签数量过多（>10个） | 可能被判定为营销号 | 保持精简 |
| 在评论区刷标签 | 可能被判定为垃圾信息 | 评论区正常互动即可 |

## 七、抖音话题规则与限制

### 7.1 平台规则

| 规则 | 说明 |
|------|------|
| 话题创建 | 输入 `#` + 关键词即可创建新话题 |
| 话题关联 | 视频带话题后，会出现在该话题页面 |
| 话题热度 | 基于该话题下所有视频的总播放量 |
| 话题推荐 | 用户点击话题可查看该话题下的所有视频 |

### 7.2 限制与注意事项

| 限制 | 说明 |
|------|------|
| 话题名称 | 不能包含违法违规/敏感词汇 |
| 话题关联内容 | 系统会审核话题下的内容，违规内容会被隐藏 |
| 话题合并 | 相似话题可能被系统自动合并 |
| 话题删除 | 违规话题可能被平台删除 |

### 7.3 搜索SEO补充

抖音除了推荐流量，还有 **搜索流量**：

| 优化点 | 方法 |
|--------|------|
| 视频标题 | 包含用户搜索词（如"灌肠机价格"） |
| 视频描述 | 自然融入关键词 |
| 话题标签 | 覆盖搜索词变体 |
| 字幕 | 语音识别会提取字幕关键词 |
| 封面文字 | 封面图上的文字也是搜索信号 |
