# 视频号私域承接 SOP

> **适用行业**：B2B 食品加工机械
> **核心优势**：视频号是微信生态内的平台，导私域无缝衔接。

---

## 一、获客链路

```
视频播放 → 评论/私信 → 加微信 → 朋友圈培育 → 私聊询盘 → 报价 → 成交
```

---

## 二、评论承接

### 2.1 评论回复话术

**询价评论**：
```
"您好！价格根据配置不同有差异，我发详细报价单给您。请加微信：XXX"
```

**咨询评论**：
```
"感谢关注！我们有完整的产品目录，加微信 XXX 发您参考。"
```

**简单点赞评论**：
```
"感谢支持！我们是食品机械源头工厂，有需要随时联系。"
```

### 2.2 评论区管理

- 及时回复（1 小时内）
- 置顶高价值评论
- 不要在评论区直接报价
- 将有意询盘的引导到微信

---

## 三、加微信后的承接流程

### 3.1 通过好友后首条消息

```
"您好，我是XX食品机械的[姓名]。
感谢您的关注！这是我的产品目录和报价参考：
📎 [产品目录PDF/链接]

请问您主要需要哪类设备？我给您推荐最合适的方案。"
```

### 3.2 标签体系

| 标签 | 说明 |
|------|------|
| 来源-视频号 | 从视频号添加的客户 |
| 需求-XX设备 | 感兴趣的设备类型 |
| 阶段-A/B/C | A=近期采购 B=考虑中 C=长期培育 |
| 地区 | 客户所在省市 |
| 行业 | 客户所属行业 |

---

## 四、朋友圈培育

### 4.1 朋友圈发布节奏

| 频率 | 每天 1-2 条 |
|------|-----------|
| 内容比例 | 产品展示 40% / 客户案例 20% / 工厂日常 20% / 行业知识 20% |

### 4.2 朋友圈内容模板

**产品展示**：
```
"XX切菜机发往山东客户工厂 🚛
200kg/h 产能，客户用后反馈：
'比人工快了10倍，人工成本省了不少'
需要同款设备的老板私信聊聊~"
```

**客户案例**：
```
"山东张总，去年买的饺子机
今天又追加了2台，准备扩大生产线 🏭
好的产品，客户会替我们说话 💪"
```

**工厂日常**：
```
"今天车间生产了20台切菜机 🔧
每一台出厂前都要经过5道质检
客户收到设备，我们才能放心 🤝"
```

---

## 五、私聊跟进节奏

| 天数 | 动作 |
|------|------|
| Day 0 | 通过好友 + 发产品目录 |
| Day 1 | 了解需求（采购什么/产能要求/预算） |
| Day 3 | 发送报价方案 + 推荐视频 |
| Day 5 | 发客户案例增加信任 |
| Day 7 | 跟进是否还有疑问 |
| Day 14 | 发新品/促销信息 |
| Day 30 | 标记沉默，定期唤醒 |

---

## 六、直播联动

### 6.1 直播获客

- 每周 1-2 次直播
- 直播内容：工厂参观/产品演示/答疑
- 直播中引导加微信
- 直播后对新增好友进行标签分类

### 6.2 直播话术

```
"欢迎各位！我是XX食品机械的[姓名]。
今天我们带大家参观我们的生产车间。
想了解具体价格的，可以加我微信：XXX
我给您发详细资料和报价。"
```
