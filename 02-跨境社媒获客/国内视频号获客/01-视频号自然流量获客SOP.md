# 视频号自然流量获客 SOP

> **适用行业**：B2B 食品加工机械（国内获客场景）
> **成本说明**：每日需 30min-1h 内容制作 + 私域运营。

---

## 一、视频号获客逻辑

### 1.1 获客链路

```
发布视频 → 社交推荐/推荐流 → 用户观看 → 关注/私信 → 加微信私聊 → 朋友圈培育 → 询盘 → 成交
```

### 1.2 与抖音的核心差异

| 差异 | 视频号 | 抖音 |
|------|--------|------|
| 推荐机制 | 社交推荐（朋友点赞）+ 算法推荐 | 纯算法推荐 |
| 私域能力 | 最强（直接导微信） | 弱 |
| 用户画像 | 25-55 岁为主 | 16-35 岁为主 |
| 内容调性 | 偏正式/专业 | 偏娱乐 |
| B2B 适配 | ★★★ | ★★★ |

---

## 二、账号设置

### 2.1 视频号配置

| 项目 | 建议 |
|------|------|
| 名称 | 品牌名 + 产品关键词 |
| 头像 | 工厂 Logo |
| 简介 | 一行描述 + 微信号引导 |
| 封面 | 统一风格 |
| 关联公众号 | 绑定企业公众号 |
| 企业微信 | 绑定企业微信客服 |

### 2.2 私域导流设计

| 方式 | 说明 |
|------|------|
| 简介区 | "微信号：XXX 获取产品目录" |
| 视频评论区 | 置顶评论放微信号 |
| 企业微信客服 | 视频底部客服按钮 |
| 公众号联动 | 视频号→公众号→个人微信 |

---

## 三、内容策略

### 3.1 适合视频号的内容

| 类型 | 占比 | 说明 |
|------|------|------|
| 产品演示 | 35% | 机器运转 |
| 工厂实力 | 20% | 车间/规模 |
| 知识科普 | 20% | 选购指南/行业知识 |
| 客户案例 | 15% | 使用场景 |
| 品牌故事 | 10% | 创始故事/工厂历史 |

### 3.2 与抖音的差异

| 维度 | 视频号 | 抖音 |
|------|--------|------|
| 时长 | 1-5 分钟更合适 | 15-60 秒 |
| 风格 | 偏专业/正式 | 偏娱乐/活泼 |
| 音乐 | 轻音乐/原声 | 热门 BGM |
| 标题 | 信息密度高 | 简洁吸引 |
| 社交传播 | 朋友点赞转发 | 算法推荐 |

### 3.3 标题模板

```
【产品名称】核心卖点一句话
Example: 【切菜机】5分钟切200公斤蔬菜，工厂老板都在用
```

---

## 四、发布策略

### 4.1 频率

| 阶段 | 频率 |
|------|------|
| 起步 | 每周 3-5 条 |
| 稳定 | 每周 5-7 条 |

### 4.2 发布时间

- 工作日：12:00-13:00, 18:00-20:00
- 周末：10:00-12:00, 20:00-22:00

### 4.3 联动发布

| 联动 | 说明 |
|------|------|
| 朋友圈 | 发布后转发朋友圈 |
| 微信群 | 分享到行业群 |
| 公众号 | 同步推送 |
| 个人号 | 老板/销售转发 |

---

## 五、线索承接

### 5.1 评论承接

- 及时回复评论
- 有意询盘的引导到微信私聊
- 评论区不要直接报价

### 5.2 私域承接流程

详见 [[视频号私域承接SOP]]

### 5.3 直播联动

- 每周 1-2 次直播
- 直播内容：工厂参观/产品演示/答疑
- 直播间引导加微信

---

## 六、数据指标

| 指标 | 目标（参考级） |
|------|--------------|
| 播放量 | 500-5000/条 |
| 点赞率 | 3%+ |
| 评论数 | 5-20/条 |
| 分享数 | 3-10/条 |
| 关注增长 | 50-200/周 |
| 私信询盘 | 3-10/周 |
| 微信添加率 | 私信→微信 50%+ |

---

## 七、风险点

| 风险 | 说明 | 应对 |
|------|------|------|
| 过度营销 | 频繁发广告被限流 | 保持内容价值 |
| 微信导流违规 | 过于直接的导流话术 | 用间接方式引导 |
| 内容同质化 | 工厂内容容易雷同 | 多角度/多话题 |
| 社交推荐依赖 | 过度依赖朋友圈转发 | 加强内容质量 |
