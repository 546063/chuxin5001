# TikTok自然流量获客 SOP

> **适用行业**：B2B 食品加工机械出口
> **成本说明**：自然流量 ≠ 免费。每日需 1-2 小时内容制作+发布+互动。建议至少 1 人兼职运营。

---

## 一、TikTok 自然流量获客逻辑

### 1.1 获客链路

```
发布视频 → TikTok 推荐 → 用户观看 → 查看主页 → 点击链接/发私信 → WhatsApp/询盘 → 跟进成交
```

### 1.2 核心要素

| 要素 | 说明 | 重要性 |
|------|------|--------|
| 内容质量 | 前 3 秒钩子 + 有价值的内容 | ★★★★★ |
| 发布频率 | 每日 1-3 条 | ★★★★ |
| 标签策略 | 行业 + 产品 + 地域标签 | ★★★★ |
| 主页设计 | Bio + 链接 + 联系方式 | ★★★★ |
| 私信回复 | 5 分钟内响应 | ★★★★★ |

---

## 二、平台规则与推荐机制

### 2.1 TikTok 推荐逻辑（基础了解）

1. **初始推送**：新视频推送给 200-500 人
2. **关键指标**：完播率 > 互动率 > 转发率
3. **逐级放大**：指标达标后推给更多人
4. **长尾效应**：好视频可能数周后突然爆发

### 2.2 B2B 账号注意事项

| 注意项 | 说明 |
|--------|------|
| 账号定位 | 明确是工厂/供应商，不是个人号 |
| 内容一致性 | 持续发布同类产品内容 |
| 不要频繁换号 | 新号需要 1-2 周建立标签 |
| 遵守社区规则 | 不要放硬广告/外部链接在视频中 |

---

## 三、账号定位

### 3.1 账号信息设置

| 项目 | 建议 |
|------|------|
| 用户名 | 品牌名 + 产品关键词（如 FoodMachine_Pro） |
| 头像 | 工厂 Logo 或产品图 |
| Bio | 一行描述 + 一个 CTA（如 "Food processing machinery manufacturer 🏭 Contact: wa.me/xxx"） |
| 链接 | WhatsApp 链接或独立落地页 |
| 邮箱 | 商务邮箱 |

### 3.2 主页链接设计

| 方式 | 说明 | 适合 |
|------|------|------|
| WhatsApp 直链 | wa.me/xxx | 即时沟通 |
| 独立落地页 | 产品目录 + 表单 | 多产品 |
| Linktree | 多链接聚合 | 多渠道 |
| 独立站 | 品牌信任 | 有网站的工厂 |

## 四、内容选题

参考 [[B2B设备短视频选题库]]

### 4.1 高频内容类型

| 类型 | 频率 | 说明 |
|------|------|------|
| 产品演示 | 40% | 机器运转画面 |
| 工厂日常 | 20% | 生产线/工人/发货 |

### 4.2 标签策略

| 标签类型 | 示例 |
|---------|------|
| 行业标签 | #foodprocessing #foodmachine #foodmachinery |
| 产品标签 | #dumplingmachine #vegetablecutter #meatgrinder |

| 通用标签 | #fyp #machinery #factory #manufacturing |

每个视频：3-5 个行业标签 + 1-2 个产品标签 + 1-2 个通用标签

## 五、视频脚本

### 5.1 通用结构

```
0-3s: 钩子（数字/对比/问题）
3-15s: 内容主体（展示价值）
15-30s: CTA（Contact us / Link in bio / WhatsApp）
```

### 5.2 5 个常用脚本模板

**模板 1：产品演示**
```
0-3s: "Watch this machine cut 200kg vegetables in 5 minutes"
3-20s: 机器运转画面（多角度）
20-30s: "Want this for your factory? WhatsApp link in bio"
```

**模板 2：对比型**
```
0-3s: "Manual cutting vs our machine — watch the difference"
3-25s: 左右对比画面
25-30s: "Save time & labor. Contact us for details."
```

**模板 3：工厂参观**
```
0-3s: "Welcome to our 5000sqm food machinery factory"
3-20s: 车间/生产线/质检
20-30s: "15 years experience, exported to 50+ countries"
```

**模板 4：FAQ 型**
```
0-3s: "How much does a dumpling machine cost?"
3-20s: 解释价格构成（材质/产能/配置）
20-30s: "Get a custom quote — link in bio"
```

**模板 5：客户案例**
```
0-3s: "How a Saudi bakery increased production 10x"
3-20s: 客户使用前后对比
20-30s: "Ready to scale? Contact us today."
```

## 八、数据指标

### 8.1 核心指标

| 指标 | 说明 | 目标（参考级） |
|------|------|--------------|
| 粉丝增长 | 每周新增粉丝 | 100-500/周（新号） |
| 播放量 | 单条视频播放 | 500-5000（新号） |
| 完播率 | 完整播放比例 | 20%+ |
| 互动率 | 赞+评+分享/播放 | 3%+ |
| 主页访问 | 视频→主页点击率 | 2%+ |
| 链接点击 | 主页链接点击 | 1%+ |
| 私信数 | 每周私信询盘 | 5-20/周 |
| 询盘→WhatsApp 转化率 | 转到 WhatsApp 的比例 | 30%+ |

### 8.2 B2B 特殊指标

| 指标 | 说明 |
|------|------|
| 有效询盘率 | B2B 询盘 / 总私信 |
| 询盘→报价率 | 发出报价数 / 有效询盘 |
| 报价→成交率 | 成交数 / 报价数 |
| 平均成交周期 | 询盘到成交的天数 |

## 九、合规风险

| 风险 | 说明 | 应对 |
|------|------|------|
| 音乐版权 | 使用 TikTok 音乐库 | 使用商业音乐 |
| 虚假宣传 | 夸大产能/效果 | 真实展示 |
| 竞品对比 | 贬低竞品 | 只做自身展示 |
| 价格透露 | 公开价格可能被竞品利用 | 引导私聊报价 |
| 客户信息 | 未经许可展示客户 | 先获得同意 |

## 十、短期/中期/长期打法

| 阶段 | 时间 | 目标 | 动作 |
|------|------|------|------|
| 短期 | 1-3月 | 建立存在感 | 日更 1-3 条，测试内容方向 |

| 长期 | 6-12月 | 品牌+线索 | 直播互动，矩阵号运营，Spark Ads 放大 |
