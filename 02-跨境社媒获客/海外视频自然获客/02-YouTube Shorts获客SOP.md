# YouTube Shorts获客 SOP

> **适用行业**：B2B 食品加工机械出口
> **成本说明**：每日需 30min-1h 内容制作+发布。可与 TikTok 共用素材。

---

## 一、YouTube Shorts vs TikTok 差异

| 维度 | YouTube Shorts | TikTok |
|------|---------------|--------|
| 推荐逻辑 | 搜索 + 推荐 | 纯推荐算法 |
| 内容寿命 | 长（数周-数月） | 短（数天） |
| 搜索流量 | 强（YouTube 是第二大搜索引擎） | 弱 |
| 用户意图 | 信息搜索 + 娱乐 | 娱乐为主 |
| B2B 适配 | ★★★★ | ★★★ |
| 外链能力 | 描述区可放链接 | 主页链接 |

### 关键优势：搜索长尾

YouTube Shorts 视频可能在发布数周后通过搜索获得持续流量，这是与 TikTok 最大的差异。

---

## 二、账号设置

### 2.1 频道配置

| 项目 | 建议 |
|------|------|
| 频道名 | 品牌名 + 产品关键词 |
| 频道描述 | 2-3 句描述工厂能力 + 关键词 |
| 头像 | 工厂 Logo |
| Banner | 产品线展示 + 联系方式 |
| 关于页 | 完整公司信息 + 邮箱 + 网站 |

### 2.2 链接设计

- 频道描述区放置 WhatsApp 链接
- 每条 Shorts 描述中添加联系链接
- 置顶评论放置 CTA

---

## 三、内容策略

### 3.1 适合 YouTube Shorts 的内容

| 类型 | 说明 | 示例标题 |
|------|------|---------|
| 产品演示 | 机器运转画面 | "This dumpling machine makes 1000/hour" |
| 对比测试 | 人工 vs 机器 | "Manual vs Machine vegetable cutting" |
| 知识科普 | 行业知识 | "How to choose a food processing line" |
| 工厂参观 | 展示实力 | "Inside our food machinery factory" |
| FAQ | 常见问题解答 | "How much does a meat grinder cost?" |
| 安装教程 | 使用/维护 | "How to install your new packaging machine" |

### 3.2 标题 SEO

| 原则 | 示例 |
|------|------|
| 包含关键词 | "Food Processing Machine - Dumpling Maker" |
| 前 60 字符关键信息 | "Automatic Vegetable Cutting Machine 200kg/h" |
| 加地域关键词 | "Food Machinery for Sale in Saudi Arabia" |
| 加行动词 | "How to Start a Dumpling Factory" |

### 3.3 标签策略

| 标签 | 数量 | 示例 |
|------|------|------|
| 行业标签 | 3-5 | #foodprocessing #foodmachine #foodmachinery |
| 产品标签 | 2-3 | #dumplingmachine #meatgrinder |
| SEO 标签 | 2-3 | #factory #manufacturing #industrial |
| 地域标签 | 1-2 | #saudiarabia #middleeast |

## 四、发布策略

### 4.1 频率

| 阶段 | 频率 |
|------|------|
| 起步期 | 每日 1 条 |
| 稳定期 | 每周 5-7 条 |

### 4.2 发布时间

- 目标市场当地时间 12:00-14:00 或 18:00-21:00
- YouTube Shorts 的发布时间比 TikTok 更灵活

## 七、与 TikTok 协同

### 7.1 素材复用

| 方向 | 说明 |
|------|------|
| TikTok → YouTube | TikTok 视频去水印后上传 YouTube Shorts |
| YouTube → TikTok | YouTube 原生视频裁剪后上传 TikTok |
| 统一制作 | 9:16 竖屏 30 秒，同时发两个平台 |

### 7.2 差异化

| 平台 | 侧重 |
|------|------|
| TikTok | 娱乐性、即时互动 |
| YouTube | 信息性、搜索长尾 |
