# Facebook Reels获客 SOP

> **适用行业**：B2B 食品加工机械出口
> **成本说明**：每日 30min。与 IG Reels 内容可完全复用。

---

## 一、FB Reels 特征

| 特征 | 说明 |
|------|------|
| 格式 | 9:16 竖屏，60秒内 |
| 分发 | Facebook Feed + Reels Tab |
| 与 IG 互通 | 可通过 Meta Business Suite 同时发 IG + FB Reels |
| 年龄覆盖 | 25-65+ 覆盖广 |
| B2B 适配 | 中高（与 Page 运营协同） |

---

## 二、账号设置

### 2.1 Page 配置

| 项目 | 建议 |
|------|------|
| Page 名称 | 品牌名 + 产品关键词 |
| 头像 | 工厂 Logo |
| 封面 | 产品展示/工厂全景 |
| About | 完整公司信息 + 邮箱 + WhatsApp |
| CTA 按钮 | "Send WhatsApp Message" 或 "Contact Us" |
| Services | 列出主要产品线 |

### 2.2 WhatsApp 按钮配置

1. Page Settings → WhatsApp → 绑定 WhatsApp Business 号码
2. 用户点击 CTA 按钮直接跳转 WhatsApp
3. 设置自动欢迎消息

---

## 三、内容策略

### 3.1 与 IG Reels 完全复用

| 方向 | 说明 |
|------|------|
| 素材 | 同一 9:16 竖屏视频 |
| 标题 | 可适当调整语气（FB 更正式） |
| 标签 | 3-5 个行业标签即可 |
| 发布 | 通过 Meta Business Suite 同时发布 |

### 3.2 标题模板

```
🏭 [产品名称] - [核心卖点]
Example: Automatic Dumpling Machine - 1000 pieces/hour output

Contact us:
📱 WhatsApp: +86 XXX XXXX XXXX
📧 Email: sales@xxx.com

#foodmachine #foodprocessing #factory
```

---

## 四、发布策略

| 频率 | 每周 3-5 条 |
|------|-----------|
| 时间 | 目标市场当地时间 10:00-12:00 或 19:00-21:00 |
| 工具 | Meta Business Suite 定时发布 |

---

## 五、线索承接

### 5.1 Messenger 承接

- Reels 观众可能通过 Messenger 联系
- 设置 Messenger 自动回复
- 引导到 WhatsApp（见 [[03-海外投流/Facebook投流知识库/12-Facebook Messenger获客链路SOP]]）

### 5.2 评论承接

- 及时回复评论
- 将有意询盘的评论引导到 Messenger/WhatsApp
- 不要在评论区直接报价

---

## 六、与付费投流协同

| 协同方式 | 说明 |
|---------|------|
| 有机 Reels 表现好 → 付费放大 | 将高互动 Reels 作为付费广告素材 |
| 付费广告观众 → 看到有机内容 | 增加品牌认知 |
| 有机积累信任 → 提高付费广告 CVR | Page 内容增加可信度 |

---

## 七、Checklist

- [ ] Facebook Page 完善信息
- [ ] CTA 按钮设置为 WhatsApp
- [ ] Messenger 自动回复配置
- [ ] 与 IG 账号关联
- [ ] Meta Business Suite 设置
- [ ] 准备 5-10 条 Reels
- [ ] 制定发布计划
