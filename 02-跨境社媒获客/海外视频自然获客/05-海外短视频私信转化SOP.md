# 海外短视频私信转化 SOP

> **适用行业**：B2B 食品加工机械出口
> **适用范围**：TikTok / Instagram / YouTube / Facebook 私信

---

## 一、私信承接核心逻辑

### 1.1 转化链路

```
视频播放 → 用户发私信 → 5分钟内回复 → 引导到 WhatsApp → 发目录/报价 → 跟进成交
```

### 1.2 关键原则

| 原则 | 说明 |
|------|------|
| 速度 | 5 分钟内首响 |
| 引导 | 引导到 WhatsApp 深度沟通 |
| 分类 | 判断 B2B 还是 C 端 |
| 跟进 | 标准化 7 天跟进节奏 |

---

## 二、各平台私信特点

| 平台 | 私信入口 | 自动回复 | 适合度 |
|------|---------|---------|--------|
| TikTok | Inbox → Messages | 有（Business Account） | ★★★ |
| Instagram | DM | 有（Business Account FAQ） | ★★★★ |
| YouTube | 评论为主，私信弱 | 无 | ★★ |
| Facebook | Messenger | 有（自动回复） | ★★★★ |

---

## 三、首响话术

### 3.1 通用首响模板

```
"Hi! Thank you for your interest in our food processing machinery. 
I'm [Name] from [Company]. How can I help you?

For faster response and to send you our catalog, please message us on WhatsApp:
📱 [WhatsApp链接]"
```

### 3.2 针对不同询盘类型的回复

**询价型**：
```
"Thanks for asking about pricing! Our machines range from $X,XXX to $XX,XXX depending on configuration.
To give you an accurate quote, I need to know your requirements.
Could you share on WhatsApp? [link]"
```

**产品咨询型**：
```
"Great question about our [product]! Let me send you detailed specs and a demo video.
Please message us on WhatsApp for the full catalog: [link]"
```

**简单打招呼型**：
```
"Hi there! 👋 Are you looking for food processing machinery?
I'd love to help. What kind of products do you need to process?
📱 WhatsApp: [link]"
```

## 四、B2B vs C 端判断

### 4.1 判断标准

| 线索 | B2B 信号 | C 端信号 |
|------|---------|---------|
| 身份 | 工厂/经销商/食品厂 | 个人/家庭使用 |
| 需求 | 批量/产线/定制 | 单台/家用 |
| 问题 | 技术参数/MOQ/交期 | 价格/怎么用 |
| 国家 | 目标市场 | 非目标市场 |

### 4.2 处理策略

| 类型 | 处理 |
|------|------|
| B2B 高价值 | 立即引导 WhatsApp + 专属销售跟进 |
| B2B 中等 | 引导 WhatsApp + 标准跟进 |
| C 端 | 礼貌回复，提供基础信息 |
| 同行 | 不提供敏感信息 |

---

## 五、WhatsApp 引导策略

### 5.1 引导话术

| 阶段 | 话术 |
|------|------|
| 第一次 | "Let's continue on WhatsApp for faster response: [link]" |
| 第二次 | "I'll send you our catalog with prices on WhatsApp. Here's our number: [number]" |
| 第三次 | "For a detailed quote tailored to your needs, please WhatsApp us: [link]" |

### 5.2 WhatsApp 承接准备

- [ ] WhatsApp Business 配置完成
- [ ] 欢迎消息设置
- [ ] 产品目录准备
- [ ] 报价模板准备
- [ ] 标签体系建立（见 [[03-海外投流/Facebook投流知识库/13-Facebook WhatsApp获客链路SOP]]）

## 六、7 天跟进节奏

| 天数 | 动作 | 说明 |
|------|------|------|
| Day 0 | 首响 + 引导 WhatsApp | 5 分钟内 |
| Day 1 | 发目录 + 了解需求 | WhatsApp |
| Day 3 | 发报价 + 确认需求 | WhatsApp |
| Day 5 | 发案例/视频 | 增加信任 |
| Day 7 | 最后跟进 | "Any questions?" |
| Day 14 | 沉默唤醒 | 发新内容/促销 |
| Day 30 | 标记为沉默 | 定期回顾 |

## 七、数据指标

| 指标 | 目标（参考级） |
|------|--------------|
| 私信→回复率 | > 90% |
| 回复→WhatsApp 转移率 | > 50% |
| WhatsApp→有效询盘率 | > 30% |
| 询盘→报价率 | > 60% |
| 报价→成交率 | 5-15%（B2B） |
