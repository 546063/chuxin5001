# 连锁餐饮供应链

## 一句话定义

> **连锁餐饮供应链 = 标准化+集中采购+中央厨房 = 食品设备最大的采购群体**

---

## 为什么增长

### 核心驱动因素

| 驱动因素 | 说明 | 影响 |
|---------|------|------|
| 连锁餐饮爆发 | 麦肯系+本土连锁 | 设备需求大 |
| 标准化要求 | 所有门店一致 | 中央厨房 |
| 集中采购 | 降低成本 | 量大 |
| 外卖平台 | 出餐快 | 中央厨房 |

### 数据

- 全球连锁餐饮市场规模：2023年约2.5万亿美元
- 预计2030年达3.5万亿美元
- 年复合增长率：5%

---

## 哪些国家增长最快

### 第一梯队

| 国家 | 增长率 | 说明 |
|------|--------|------|
| 中国 | 10-15%/年 | 最大市场 |
| 东南亚 | 8-12%/年 | 连锁化 |
| 印度 | 10-15%/年 | 人口红利 |

### 第二梯队

| 国家 | 增长率 | 说明 |
|------|--------|------|
| 美国 | 3-4%/年 | 成熟 |
| 欧洲 | 2-3%/年 | 稳定 |

## 他们最需要什么设备

### 核心设备需求

| 设备 | 需求热度 | 客户类型 |
|------|---------|---------|
| 炒菜锅（自动） | ⭐⭐⭐⭐⭐ | 中餐连锁 |
| 炸炉 | ⭐⭐⭐⭐⭐ | 快餐连锁 |
| 烤箱 | ⭐⭐⭐⭐⭐ | 烘焙连锁 |
| 冷藏设备 | ⭐⭐⭐⭐⭐ | 所有连锁 |
| 包装设备 | ⭐⭐⭐⭐⭐ | 外卖 |
| 清洗设备 | ⭐⭐⭐⭐ | 中央厨房 |

### 连锁餐饮设备配置

```
中央厨房：洗菜机 + 切菜机 + 炒菜锅 + 包装机
门店：炸炉 + 烤箱 + 冷藏柜 + 保温设备
```

## 他们最怕什么

| 痛点 | 说明 | 解决方案 |
|------|------|---------|
| 口味不一致 | 各店不同 | 标准化设备+配方 |
| 出餐慢 | 外卖投诉 | 高产能设备 |
| 设备故障 | 影响营业 | 可靠设备+快速维修 |
| 人工成本 | 招工难 | 自动化设备 |
