## 默认上下文预算

| 任务级别 | 知识包上限 | Top K 上限 | 最大总行数 |
|---|---|---|---|
| 简单（单段素材剪辑/加字幕） | 1 个 | 2 篇 | ≤ 200 行 |
| 普通（多段拼接/转场/BGM） | 1 个 | 3 篇 | ≤ 300 行 |
| 复杂（完整后期/特效/多版本） | 2 个 | 3 篇 | ≤ 500 行 |

**硬规则**：
- 禁止默认全库 grep
- 禁止默认全文读取 8 篇以上
- C/D 级文稿不得作为主依据
- 视频剪辑任务不得调用任何文案课、人物蒸馏、投流知识包

## Claude Code 执行方式

```
1. 识别任务级别（简单/普通/复杂）
2. 根据预算加载主知识包 Top K 文稿（仅读取 SOP + 关键参数片段）
3. 如需品牌文件，按需读取（不超额）
4. 生成剪辑操作清单 → generate_only 模式输出，不修改文件
5. 输出验收报告 → 标注命中验收标准哪些项
6. 用户按清单手动执行剪辑工具
```
