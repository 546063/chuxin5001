# 01-全库工作总览MOC

## 中文总结
本MOC是全库18个MOC的导航总入口。用户或AI想要进入任意工作域时，先查这里找到对应MOC的入口路径。全库分为三大层：治理层（AI读写、工具自动化）、业务层（食品机械8大领域）、支撑层（决策系统、AI执行）。每个MOC都链接到其核心主文稿，告诉你去哪里找答案、做决策。

## English Summary
Master navigation hub for all 18 vault MOCs. Routes any task to the correct domain MOC before diving into specific content.

## 一句话结论
全库MOC总入口，找哪个工作域，先从这里出发。

## 适用场景
- 进入任意工作域之前先查这里
- AI任务路由，判断该走哪个MOC
- 新用户理解全库结构
- 跨MOC协调任务时确认边界

## 不适用场景
- 不包含具体业务内容（去具体MOC）
- 不包含碎片文稿（去具体工作目录）
- 不包含AI执行日志（去AI执行系统）

## 工作路径

1. **识别任务类型** → 判断是业务执行还是治理支持
2. **进入对应MOC** → 导航到对应工作域的MOC
3. **找到核心文稿** → 进入该MOC的核心主文稿
4. **执行任务** → 使用文稿中的决策系统、模板、话术
