# 内容素材调用路由

## 核心目标

内容不是表达欲，而是业务动作。

## 内容生产顺序

```text
业务目标
→ 目标客户
→ 选题角度
→ 开头钩子
→ 主体结构
→ 信任证据
→ 咨询动作
→ 平台适配
```

## 常用结构

```text
痛点开头
→ 问题解释
→ 真实案例/现场/细节
→ 专业判断
→ 解决方案
→ 咨询动作
```

## 平台适配

- 抖音/视频号：开头强、场景真、咨询动作明确。
- Facebook/TikTok：画面直观、问题通俗、CTA 到 WhatsApp/Messenger。
- LinkedIn：专业判断、B2B采购逻辑、案例证据。
- YouTube：产品演示、工厂介绍、长教程。

## 输出标准

若用户只要文案，只输出文案；若用户要方案，输出选题、开头、脚本、画面、CTA、复盘指标。
