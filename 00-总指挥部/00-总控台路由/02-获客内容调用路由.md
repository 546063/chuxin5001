# 获客内容调用路由

## 核心目标

不是追求播放量，而是筛选好成交客户。

## 优先读取目录

- [[02-跨境社媒获客]]
- [[04-内容素材弹药库]]
- [[05-客户开发与成交]]
- [[13-国内业务与代运营]]
- [[25-成交案例战报库]]

## 调用顺序

```text
客户是谁
→ 痛点是什么
→ 为什么现在要解决
→ 为什么相信你
→ 如何引导咨询
→ 咨询后怎么承接
```

## 适用任务

- 写获客文案
- 设计询盘脚本
- 提高私信率
- 设计评论区关键词
- 设计加微信 / WhatsApp 动作
- 诊断为什么内容有播放没咨询

## 输出标准

AI 输出获客建议时必须包含：

1. 目标客户。
2. 核心痛点。
3. 信任证据。
4. 咨询动作。
5. 后续承接。
6. 可直接复制的文案。
