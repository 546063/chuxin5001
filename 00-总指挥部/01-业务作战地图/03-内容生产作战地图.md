# 内容生产作战地图

## 一、内容生产核心目标

内容不是单纯表达，而是完成业务动作。

每条内容至少服务一个目标：获取关注、筛选客户、建立信任、引导咨询、解释产品、展示案例、促进成交、沉淀品牌资产。

## 二、内容生产调用顺序

```text
业务目标
→ 目标客户
→ 内容角度
→ 开头钩子
→ 主体结构
→ 信任证据
→ 咨询动作
→ 平台适配
→ 发布后复盘
```

选题类型：痛点型、避坑型、对比型、案例型、价格型、流程型、结果型、误区型、决策型、复盘型。

### 2. 开头模块

开头要求：

1. 直接点名目标客户。
2. 直接说具体问题。
3. 不要空泛铺垫。
4. 10 秒内让客户知道和自己有关。
5. 包含反常识、风险、利益、损失、判断标准之一。

### 3. 脚本结构模块

```text
痛点开头
→ 问题解释
→ 真实案例/现场/细节
→ 专业判断
→ 解决方案
→ 咨询动作
```

### 4. 镜头素材模块

优先调用：

素材类型：现场过程、产品细节、客户反馈、服务流程、前后对比、失败案例、成功案例、真实团队/工厂/门店画面。

### 5. 平台适配模块

- 抖音 / 视频号：开头强、场景真实、信息密度高、本地服务强调地名和服务范围。
- Facebook / TikTok：问题通俗、画面能无声理解、CTA 到 WhatsApp/Messenger/表单。
- LinkedIn：专业判断、行业洞察、B2B采购逻辑、案例数据。
- YouTube：产品演示、工厂介绍、客户案例、教程内容。
