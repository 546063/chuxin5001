# 业务作战总地图

## 一、总原则

本库不是资料仓库，而是业务作战系统。

```text
业务问题
→ 判断业务阶段
→ 进入对应作战地图
→ 调用高价值文档
→ 输出执行动作
→ 复盘沉淀
```

## 三、业务阶段流转

```text
产品/行业研究
→ 获客定位
→ 内容生产
→ 投流放大
→ 私信/WhatsApp承接
→ 客户成交
→ 交付/售后
→ 复盘案例
→ 方法论沉淀
```

## 四、新文档接入规则

新文档必须回答：

1. 它服务哪个业务动作？
2. 它应该进入哪个作战地图？
3. 它能解决什么真实问题？
4. 它适合哪个行业、平台、客户类型？
5. 它是方法论、案例、SOP、话术、素材，还是复盘？
6. 它是否值得 AI 优先调用？
