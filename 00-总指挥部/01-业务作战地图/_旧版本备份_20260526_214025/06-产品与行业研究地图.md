# 产品与行业研究地图

## 一、产品研究核心目标

回答：

1. 这个产品能不能卖？
2. 谁会买？
3. 客户为什么买？
4. 客户买之前担心什么？
5. 内容怎么拍？
6. 销售怎么问？
7. 投流怎么测？

## 二、调用顺序

```text
产品定义
→ 目标客户
→ 使用场景
→ 采购动机
→ 决策因素
→ 竞品对比
→ 内容卖点
→ 销售问题
→ 投放测试
```

每个行业必须回答：客户是谁、谁付钱、谁使用、谁决策、最关心什么、采购阻碍、内容优先讲什么、销售必须问什么。
