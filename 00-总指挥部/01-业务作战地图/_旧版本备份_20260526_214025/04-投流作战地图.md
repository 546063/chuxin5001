# 投流作战地图

## 一、核心原则

投流不是直接买成交，而是放大已经验证过的获客信号。

投流前判断：

1. 内容是否已经能带来咨询。
2. 目标客户是否清楚。
3. 咨询动作是否明确。
4. 承接链路是否有人负责。
5. 数据不好时是否知道改什么。

## 二、调用顺序

```text
业务目标
→ 平台选择
→ 素材选择
→ 受众/地域/兴趣
→ 落地页/私信/WhatsApp承接
→ 数据监控
→ 素材迭代
→ 成交复盘
```

## 四、投流诊断框架

```text
曝光低 → 账户/出价/受众/素材初始质量
点击低 → 标题/封面/前三秒/利益点
咨询低 → 内容不够痛/CTA不明确/客户不精准
线索差 → 表单太宽/承接慢/问题筛选不足
成交差 → 客户质量/报价/信任/跟进/产品匹配
```
