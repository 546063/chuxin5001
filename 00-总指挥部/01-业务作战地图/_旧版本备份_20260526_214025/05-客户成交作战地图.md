# 客户成交作战地图

## 一、成交核心原则

客户咨询不是成交，真正的成交来自：需求判断、信任建立、问题确认、方案匹配、报价理由、持续跟进、风险消除。

## 二、调用顺序

```text
客户来源
→ 客户需求
→ 客户预算
→ 客户决策权
→ 产品/服务匹配
→ 信任证据
→ 报价
→ 跟进
→ 成交/流失复盘
```

## 三、客户沟通模块

### 咨询接待

优先目录：

- [[05-客户开发与成交]]
- [[06-客户沟通复盘]]
- [[09-销售过程复盘]]

首轮判断：客户是谁、来自哪个渠道、问什么、是否有明确需求、是否有预算和时间。

### 需求挖掘

食品机械：原料、产能、整线、电压国家、现有设备、预算、采购时间、安装售后。

装修：小区、面积、新旧房、风格、预算、量房、是否对比其他公司。

教育培训：年龄、基础、目标、时间、家长关注点、是否体验。

美业/抗衰：主要困扰、做过项目、预算、风险接受度、审美目标、是否先评估。

### 信任建立

案例、过程、专业判断、客户反馈、交付能力、售后能力、风险说明。

### 报价逻辑

报价前确认：需求、方案、单位、包含/不包含、付款、周期、售后、备选方案。

### 跟进逻辑

不要只问“考虑怎么样了”。优先补案例、风险提醒、选型建议、预算方案、对比标准、下一步动作。
