# 获客作战地图

## 一、获客核心判断

获客不是追求播放量，而是筛选能成交的人。

判断一条内容是否有获客价值，看五个信号：

1. 是否吸引目标客户停留。
2. 是否让客户觉得“这说的就是我”。
3. 是否筛掉低质量客户。
4. 是否建立专业信任。
5. 是否促成咨询动作。

## 二、获客知识调用顺序

```text
客户是谁
→ 客户有什么痛点
→ 客户为什么现在要解决
→ 客户为什么相信你
→ 客户为什么愿意咨询
→ 咨询后怎么承接
```
