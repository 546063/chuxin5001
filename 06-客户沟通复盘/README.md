# 06-客户沟通复盘

## 这个目录放什么

- WhatsApp 沟通复盘
- LinkedIn 沟通复盘
- Facebook 线索复盘
- 客户异议复盘
- 客户沉默复盘
- 客户成交复盘

## 不放什么

- 具体客户跟进内容（05-客户开发与成交）
- 销售过程复盘（09-销售过程复盘）
- 工厂沟通记录（07-工厂沟通与探厂复盘）

## 适合什么时候查

- 客户沉默不知道原因
- 客户有异议不知道怎么解决
- 沟通效果不好需要复盘
- 学习如何更好沟通

## 与其他目录的关系

- 基于05-客户开发与成交的实践记录
- 与09-销售过程复盘关联（销售结果分析）
- 为下一次沟通提供改进方向
