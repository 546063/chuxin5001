# 诊断SOP 食品加工机械社媒获客诊断

> [!info]
> 本SOP用于诊断食品加工机械外贸社媒获客的现状、问题与机会。适用于已有社媒账号但询盘少的团队，或准备启动社媒获客的新团队。

### 1.2 资料完整性检查

- [ ] 头像：工厂logo或专业头像
- [ ] Bio/简介：包含行业关键词（food processing machine / vegetable washing machine等）
- [ ] WhatsApp链接：bio中包含WhatsApp直接链接

## Step 2：内容质量诊断

### 2.1 内容类型分布

统计最近30条视频的内容类型：

| 内容类型 | 数量 | 占比 | 目标占比 |
|---------|------|------|---------|
| 设备运行实拍 | | | 35% |
| 整线方案展示 | | | 25% |
| 客户案例 | | | 15% |
| 工厂实力展示 | | | 10% |
| 行业知识 | | | 10% |
| 互动/促销 | | | 5% |

### 2.2 内容质量评分（每条视频1-5分）

评估维度：
- **视觉冲击力：** 前3秒是否有吸引力？
- **信息密度：** 是否包含关键信息（产能/效果/卖点）？
- **CTA明确性：** 是否引导WhatsApp询价？
- **画面质量：** 清晰度、光线、稳定性
- **标题/标签：** 是否使用英文关键词？

### 2.3 内容缺口识别

- [ ] 是否展示过整线方案？
- [ ] 是否拍摄过Before/After对比？
- [ ] 是否展示过客户工厂/案例？
- [ ] 是否做过ASMR风格内容？
- [ ] 是否展示过装柜发货？
- [ ] 是否做过教程类内容（选型/维护）？

## Step 3：投放诊断

### 3.1 投放现状

| 项目 | 当前状态 | 建议状态 |
|------|---------|---------|
| 投放平台 | | Facebook+TikTok（P0区域） |
| 日预算 | | $10-$50/天（按区域） |
| 定向国家 | | 东南亚+中东优先 |
| 定向标签 | | 食品加工/小生意/清真 |
| 素材语言 | | 英语+目标市场本地语 |
| CPL（单条线索成本） | | 目标<$30 |

### 3.2 素材诊断

- [ ] 素材是否为设备运行实拍？（非静态图轮播）
- [ ] 素材前3秒是否有视觉冲击？
- [ ] 素材是否包含英文字幕？
- [ ] 素材时长是否适合平台？（TikTok 15-30s，Facebook 30-60s，YouTube 1-5min）
- [ ] 是否有CTA引导WhatsApp？
