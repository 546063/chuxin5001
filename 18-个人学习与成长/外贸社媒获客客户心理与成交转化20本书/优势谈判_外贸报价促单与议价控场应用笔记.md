## 二、核心模型/原则：5条

## 1. 先设高价值锚点，再进入价格谈判

客户第一次看到价格前，必须先看到价值。

如果我直接发：

```text
Price: USD 4800
```

客户会立刻拿价格去比。

更好的报价前铺垫：

```text
Based on your material and capacity, this model is suitable because it can keep stable cutting size, support continuous production, and reduce manual cutting work. It includes testing before delivery, wooden case packing, spare parts support and after-sales service.
```

然后再报价。

价格谈判前要先锚定：

- 机器适配
- 真实产能
- 稳定性
- 售后
- 测试
- 包装
- 长期成本

客户先看到价值，价格才更容易被接受。

---

## 2. 不要第一次就给到底价

很多外贸销售怕丢单，第一次报价就给最低价。  
结果客户还会继续砍价。

这是因为客户通常会默认：

```text
你报出来的价格还有空间
```

所以报价要保留谈判空间，但不能虚高到失去信任。

正确做法：

- 报合理价格
- 说明包含内容
- 预留小幅让步空间
- 让步必须换条件
- 不要无理由降价

例如：

```text
This is the price based on the current configuration and support. If you have a target budget, I can check whether we can adjust the configuration or order terms.
```

这样不是直接降价，而是保留谈判结构。

---

## 3. 每一次让步都要显得有成本

如果客户说便宜一点，我马上说可以，客户会觉得：

- 还能再便宜
- 原价不真实
- 你急着卖
- 你利润很高

所以让步不能太快、不能太轻、不能无条件。

正确让步表达：

```text
Let me check what we can do. The price is already based on the current configuration, but if you can confirm the order this week, I can try to apply for a small discount.
```

注意关键词：

- let me check
- current configuration
- if you can confirm
- apply for
- small discount

这会让客户知道让步是有限的、有条件的。

---

## 4. 用“条件交换”代替“单方面让价”

客户要求：

- lower price
- free spare parts
- free shipping
- credit terms
- longer warranty
- faster delivery

我不能直接答应。  
我要把它变成交换。

例如：

```text
If you can increase the order quantity, we can check better unit price.
```

```text
If you confirm the order this week, we can keep the current production schedule.
```

```text
If you choose the standard configuration, we can reduce the cost.
```

```text
If you can arrange deposit earlier, we can start production earlier.
```

谈判本质是交换，不是让步。

---

## 5. 促单要制造“合理紧迫感”

客户拖延时，不能只说：

```text
Please order now.
```

要给真实理由：

- 当前生产排期
- 原材料价格变化
- 海运周期
- 节假日
- 旺季
- 交货时间
- 定制周期
- 客户业务启动时间

示例：

```text
If you want to receive the machine before your production season, we need to reserve time for production, testing, packing and shipping. Confirming earlier will reduce delivery risk.
```

这不是压迫客户，而是帮助客户看见拖延成本。

---

## 三、这本书解决我哪个成交卡点？

《优势谈判》主要解决：

```text
客户进入报价阶段后，我容易被压价、拖延和额外要求牵着走
```

具体卡点包括：

- 客户总问最低价
- 客户要求一降再降
- 客户说别人更便宜
- 客户要免费配件
- 客户要求账期
- 客户大单迟迟不决策
- 客户不断提条件但不下单
- 我不知道什么时候该让步
- 我不知道怎么促单不显得急
- 我一降价客户还继续砍

这本书提醒我：谈判中最危险的是“无条件让步”和“没有下一步”。

## 四、直接可用话术模板

## 1. 客户砍价：“能不能再便宜？”

## WhatsApp模板1：有限让步

```text
I understand you want a better price.

This price is based on the current machine configuration, testing before delivery, packing and after-sales support.

Let me check what we can do. If you can confirm the order this week, I can try to apply for a small discount for you.
```

## WhatsApp模板2：换配置降价

```text
If the budget is tight, we can check a lower-cost configuration.

But I need to explain the difference clearly, because reducing cost may affect capacity, motor, blade or long-term stability.
```

## 邮件模板

```text
Hi [Name],

Thank you for your feedback.

The current price is based on the recommended configuration for your production need. If you need a lower price, we can check whether to adjust the configuration or prepare another model.

However, I suggest not reducing the key parts that affect capacity, machine stability and long-term use.

Best regards,
[Your Name]
```

---

## 2. 客户对比竞争对手价格

## WhatsApp模板1：不攻击对手，拉回配置

```text
I understand.

Some suppliers may offer lower prices, but the machine may be different in motor, blade, real capacity, material, packing or after-sales support.

If we compare the same configuration, the price difference will be clearer.
```

## WhatsApp模板2：测试客户认真程度

```text
Can you confirm whether the other offer includes testing video before delivery, spare parts support, warranty and wooden case packing?

These points are important for long-term use.
```

## 邮件模板

```text
Hi [Name],

It is normal to compare prices.

However, for machinery, I suggest comparing the full offer, not only the price. Please check whether both suppliers provide the same machine configuration, testing process, packing, spare parts and after-sales support.

A lower first price may not always mean lower total cost.

Best regards,
[Your Name]
```

## 3. 客户要求账期/赊销

## WhatsApp模板1：坚守首单规则

```text
For first cooperation, we cannot offer credit terms.

The normal term is deposit before production and balance before shipment. This protects both sides and allows us to arrange production, testing and packing properly.
```

## WhatsApp模板2：给客户安全感

```text
To reduce your risk, we can provide production updates, testing video before delivery, packing photos and shipment documents.

After several successful orders, we can discuss more flexible terms.
```

## 邮件模板

```text

Our standard payment term is deposit before production and balance before shipment. Before the balance payment, we can provide testing video, packing photos and shipment documents for your confirmation.

After stable cooperation is built, we can discuss more flexible payment terms for future orders.

Best regards,
[Your Name]
```

## 五、一个可以立刻用的谈判/定价策略

## 策略：让步三步法

当客户要求降价时，按这三步走：

```text
第一步：不要马上答应
第二步：说明当前价格包含什么价值
第三步：如果让步，必须换条件
```

## 模板

```text
I understand you want a better price.

This price includes the recommended configuration, testing before delivery, wooden case packing, spare parts support and after-sales service.

If you can confirm the order this week and arrange the deposit, I can try to apply for a small discount for you.
```

## 为什么有效

它同时完成了：

- 不让客户觉得原价虚高
- 不马上陷入价格战
- 强调当前价格价值
- 把折扣和行动绑定
- 推动客户进入确认订单

## 六、一句话总结这本书对我成交的最大帮助

《优势谈判》最大的帮助是：让我在客户砍价、比价、拖延和索要条件时掌握节奏，用锚点、条件交换和合理紧迫感推动客户下单。

# 《优势谈判》外贸报价促单与议价控场应用笔记

## 一、这本书为什么适合外贸报价谈判

《优势谈判》非常适合外贸销售中的报价、议价、促单和让步管理。它的重点是：谈判中不要被客户情绪和要求带着走，要掌握节奏、设置锚点、控制让步、推动成交。

我做食品机械和工业设备外贸，客户来自中东、东南亚、非洲，在 WhatsApp、邮件、评论区里经常会出现这些情况：

- 客户直接问最低价
- 客户说别人便宜
- 客户要求再降一点
- 客户说下次再买
- 客户要账期
- 客户大单拖很久
- 客户要求很多赠品
- 客户要求免费配件
- 客户不断压价但不确认订单

如果我没有谈判框架，就会变成被动销售。  
客户问什么，我回什么；客户压价，我解释；客户拖延，我等；客户消失，我焦虑。

《优势谈判》对我的价值是：让我在客户议价和拖延时，仍然能掌握主动权。
