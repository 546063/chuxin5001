### 1. Advantage+ Sales 取代 Advantage+ Shopping（Andromeda 核心变化）

- **关键点**: Advantage+ Sales 是 2025 年初 Meta 推出的新广告系列，取代了原 Advantage+ Shopping
- **关键点**: Advantage+ Sales 现在可以使用 Advantage+ Audience，提供受众建议和自定义受众排除功能，解决了原版最大痛点
- **关键点**: Ad Sets 重回 Advantage+ Sales，可以添加多个广告组，不再有数量限制
- **关键点**: 广告结构从"无Ad Set"变为支持多 Ad Set，为优化提供更多灵活性
- **判断标准**: 当广告主需要销售/转化类广告且希望保留一定受众控制时，优先选择 Advantage+ Sales
- **操作步骤**: Ads Manager → 创建销售广告 → 选择 Advantage+ Sales → 设置受众建议和排除 → 上传创意
- **来源**: [Advantage+ Sales Replaces Advantage+ Shopping](https://www.jonloomer.com/qvt/advantage-sales-replaces-advantage-shopping/)
- **发布日期**: 2026-04-26

### 3. Meta Andromeda 算法详解：算法如何决定广告投放

- **关键点**: Andromeda 是 Meta 2024年末推出的全新算法，取代旧的定向系统
- **关键点**: 旧模式：广告主手动选择受众（兴趣/行为）；Andromeda 新模式：Meta 使用广告创意本身来决定谁应该看到内容
- **关键点**: Andromeda 使用深度学习和序列学习，将创意与用户意图及购买旅程阶段匹配
- **关键点**: 算法可检测用户最近行为（如：预订滑雪度假村）并智能推送相关广告（滑雪装备）
- **关键点**: 广告创意的质量比定向设置对效果的影响更大
- **判断标准**: 当 CPA 突然上升或下降时，首先检查创意质量而非定向设置
- **操作步骤**: 优化创意质量 → 确保前3秒有强钩子 → 测试多种变体 → 让算法有足够学习数据
- **来源**: [Facebook Ad Algorithm Changes for 2026](https://www.socialmediaexaminer.com/facebook-ad-algorithm-changes-for-2026-what-marketers-need-to-know/)

### 8. Meta 转化 API（CAPI）2026 必要性

- **关键点**: 仅依赖 Meta Pixel 在 2026 年几乎肯定会丢失转化数据（iOS 隐私限制 + 广告拦截器）

- **关键点**: 典型设置方式：Google Tag Manager Server Container + CAPI 集成
- **关键点**: 关键事件：PageView、ViewItem、AddToCart、BeginCheckout、Purchase

- **判断标准**: 当 Pixel 数据与实际转化差异超过 20% 时，优先排查 CAPI 配置
- **操作步骤**: 创建 GTM 服务器容器 → 配置 CAPI 端点 → 设置事件去重和用户数据哈希 → 验证数据流

### 9. 视频广告前3秒钩子策略（2025）

- **关键点**: 前3秒是决定用户是否继续观看的关键窗口
- **关键点**: 高效钩子类型：反转预期、情绪共鸣、直接承诺、数据震惊
- **关键点**: 避免在前3秒使用 LOGO 或品牌介绍——用户在此阶段尚未决定是否值得看下去
- **关键点**: UGC（用户生成内容）风格的视频在前3秒通常表现优于品牌制作内容
- **关键点**: 字幕必须在无声状态下也能传达核心信息
- **判断标准**: 当视频完成率低于 30% 时，首先检查前3秒是否足够有吸引力
- **操作步骤**: 开头3秒使用最强钩子 → 避免品牌 LOGO 前置 → 添加无声字幕 → 测试多种钩子类型
- **来源**: [Why The First 3 Seconds of Video Matter More Than the Next 30](https://animoto.com/blog/video-marketing/why-first-3-seconds-matter)
- **发布日期**: 2026-04-26

### 10. Meta 广告创意疲劳（Ad Fatigue）识别与应对

- **关键点**: 创意疲劳典型症状：CPM 上升 + CPA 上升 + 频次 > 4
- **关键点**: 预防为主：建议每 3-5 天轮换新创意变体
- **关键点**: 当 CPM 相比基线上升 15%+ 且频次 > 3 时，创意疲劳是主要原因
- **关键点**: 应对策略：增加新创意 + 提高出价 + 缩小定向范围（降低匹配速度）
- **关键点**: 建议保持至少 5-8 个活跃创意变体长期轮换
- **判断标准**: 当广告组出现"高频次+高CPM+低CTR"组合时，判定为创意疲劳
- **操作步骤**: 监测频次和 CPM 趋势 → 识别疲劳临界点（频次3-4） → 立即上线新创意 → 暂停表现最差的旧创意
- **来源**: [How to Beat Facebook Ad Fatigue: 12 Proven Strategies](https://bir.ch/blog/facebook-ad-fatigue)
- **发布日期**: 2026-04-26

### 13. Meta 广告系列结构推荐（2025）

- **关键点**: 1-1-N 结构（1个预算 · 1个广告组 · N条创意）仍是电商广告主流
- **关键点**: CBO（Campaign Budget Optimization）在 Advantage+ 系列中效果最佳
- **关键点**: ABO（Ad Set Budget）适合需要精细控制各广告组预算的场景
- **关键点**: 建议每个广告系列不超过 50 个 Ad Sets，避免学习阶段过度分散
- **关键点**: 多广告组结构适用于分层测试（受众分层 vs 创意分层）
- **判断标准**: 新品推广优先使用 Advantage+ Shopping（简化结构）；稳定产品可使用 ABO 做精细优化
- **操作步骤**: 新品 → Advantage+ Shopping（1个广告系列）→ 数据积累后 → 切换为 ABO 结构做精细化管理
- **来源**: [The BEST Facebook Ads Campaign Structure for 2025](https://themoonlighters.com/the-best-facebook-ads-campaign-structure-for-2025/)

### 14. 学习阶段优化认知（2025更新）

- **关键点**: 学习阶段不应被视为需要"克服"的问题，而是算法正常学习的过程
- **关键点**: 退出学习阶段不是必要的优先事项——广告在学习阶段也可以产生效果
- **关键点**: 每周约50次优化操作是经验法则，不是硬性要求——过度优化反而延长学习时间
- **关键点**: 给广告足够时间（通常 3-7 天）再判断效果，避免过早关停
- **关键点**: 当广告长时间卡在学习阶段时，检查是否预算过低（建议单广告组日预算 ≥ 20 倍目标 CPA）
- **判断标准**: 广告运行 < 3 天且预算充足时，优先等待而非优化；3-5 天后仍有学习警告才考虑调整
- **操作步骤**: 设置足够预算 → 等待 5-7 天 → 如持续卡在学习阶段 → 检查预算/受众/创意是否过窄
- **来源**: [About the Learning Phase](https://www.facebook.com/business/help/112167992830700), [Meta Advertising Learning Phase Issues](https://www.adstellar.ai/blog/meta-advertising-learning-phase-issues)
- **发布日期**: 2026-04-26

### 15. 创意测试策略（2025）

- **关键点**: 创意测试是 2025 年 Facebook 广告成功的必备条件，而非可选项
- **关键点**: 建议同时测试 3-5 个核心变量：钩子类型、产品展示方式、社会证明、问题-解决方案结构
- **关键点**: 每个变量至少 2 个变体，使用 Statistically Significant 测试工具判断胜出者
- **关键点**: 建议至少 50,000 次触达或 100+ 转化后才判定创意胜负
- **关键点**: 创意疲劳后立即补充新变体，保持"永远有新素材在测试"的状态
- **判断标准**: 当广告 CPA 在 2 周内上升超过 15%，优先检查是否有新创意储备
- **操作步骤**: 建立创意测试矩阵 → 每周上线新变体 → 2-4 周判定胜负 → 胜出者扩大预算 + 快速补充新测试
- **来源**: [The Ultimate Guide to Facebook Ad Creative Testing 2025](https://motionapp.com/blog/ultimate-guide-creative-testing-2025)
- **发布日期**: 2026-04-26

- **关键点**: Advantage+ Opportunity Score 是 Meta 2025 年 2 月推出的新工具，帮助广告主识别优化机会
- **关键点**: 评分范围 1-10，分数越高表示广告表现潜力越大
- **关键点**: 系统会给出具体建议（如"添加更多创意"或"提高预算"）并预估提升效果
- **关键点**: 通过 Ads Manager 广告系列详情页可直接查看 Opportunity Score
- **关键点**: Opportunity Score 低于 5 时，通常存在明显的结构性优化空间
- **判断标准**: 定期检查 Opportunity Score；分数低且有具体建议时，优先按建议优化
- **操作步骤**: Ads Manager → 选择广告系列 → 查看 Opportunity Score → 采纳系统建议 → 重新评估分数变化
