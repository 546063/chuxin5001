## 一、账户结构与预算策略

### 1.1 账户结构设计

#### 1-1-N 结构（推荐起步）
```
广告系列 (Campaign) → 广告组 (Ad Set) → 广告 (Ad)
- 1个广告系列
- 1-3个广告组
- 每个广告组2-3条广告
```

#### 1-3-1 结构（进阶）
```
广告系列 → 广告组 → 广告
- 1个广告系列
- 3个广告组（不同受众/版位）
- 每个广告组1条广告
```

### 1.2 预算分配策略

#### CBO (Campaign Budget Optimization)
- 优点：系统自动优化，简化管理
- 适用：成熟账户、预算充足
- 建议：50%预算跑ASC，50%测试

#### ABO (Ad Set Budget)
- 优点：精细控制，可观测性强
- 适用：测试阶段、需要分受众测试
- 建议：新账户、小预算测试

#### 预算测试起步
- 最低30-50美金/天起步
- 测试出winner后单独扩量

### 1.3 自动规则设置
- 特定时段增加20-30%预算
- 设置预算上限防止超支

## 二、受众定位策略

### 2.1 核心受众定位方式

| 定位方式 | 使用比例 | 说明 |
|----------|----------|------|
| 基本人口统计 | 35.5% | 年龄、性别、地区 |
| 购物习惯 | 34.6% | 购买行为、频次 |
| 过往购买记录 | 32% | 已有客户数据 |
| 兴趣爱好 | 29% | 兴趣标签 |
| 痛点需求 | 19.7% | 深层需求（竞争少） |

### 2.2 自定义受众 (Custom Audience)

#### 数据来源
- 网站访客 (Pixel数据)
- 互动用户 (点赞/评论/分享)
- 客服名单/表单提交
- 购买客户

#### 匹配方式
- Pixel代码追踪
- 客户文件上传(邮箱/电话)
- 主页粉丝
- 视频观看/直播互动

### 2.3 相似受众 (Lookalike Audience)

#### 相似度层级
- 1%：最相似，质量最高，量级小
- 2-5%：平衡型
- 6-10%：量级大，相似度降低

#### 种子用户选择
- 购买客户（首选）
- 高互动用户
- 高价值用户

### 2.4 自动受众 (Advantage+ Audience)
- AI自动扩展优质受众
- 多素材同时测试
- 适合素材丰富的情况

### 2.5 受众优化技巧
- 排除已购买客户（拉新时）
- 排除网站访问者（重定向时）
- 细分人群包测试

## 三、创意素材优化

### 3.1 素材类型优先级

| 类型 | 点击率 | 适用场景 |
|------|--------|----------|
| 视频 (Reels) | 最高 | 品牌认知、产品展示 |
| 图片 | 中等 | 测试不同角度 |
| 轮播 | 较高 | 多产品展示 |
| 收藏集 | 较高 | 电商展示 |

### 3.2 视频广告最佳实践

#### 规格要求
- 时长：15-30秒最佳
- 格式：9:16 (Reels) / 16:9 (信息流)
- 文件：MP4, MOV
- 大小：<4GB

#### 内容技巧
- 前3秒抓住注意力
- 使用字幕（无声观看）
- 讲好一个故事
- 幕后花絮更真实
- 每个视频一个核心卖点

#### 数据参考
- 视频广告点击率比图片高50%+
- 短视频完播率是长视频2倍

### 3.3 图片广告设计

#### 元素要点
- 突出痛点/解决方案
- 高质量产品图
- 清晰的行动号召
- 品牌元素（Logo）

#### 避免
- 文字过多（20%限制）
- 模糊/低质量图
- 过度PS
- 纯文字海报

### 3.4 创意疲劳管理

#### 原因
- 同一受众重复看到相同素材
- 效果下降

#### 解决
- 每周更新2-3套素材
- A/B测试不同角度
- 多版本轮换

### 3.5 KOL/KOC 合作

#### 效果数据
- 63%消费者信任网红推荐
- 88%品牌计划增加网红投入

#### 选择标准
- 品牌调性匹配
- 真实互动率
- 受众重合度

### 3.6 AI文案应用

#### 技巧
- 利用帖子评论生成文案
- 用户真实评价做广告语
- 测试不同文案角度
- 批量生成测试

## 四、出价与优化

### 4.1 出价方式

| 方式 | 优点 | 适用场景 |
|------|------|----------|
| 自动出价 | 简单、系统优化 | 新手、测试期 |
| 最低成本 | 最大化流量 | 预算有限 |
| 最高价值 | 优化转化 | 成熟账户 |
| 手动出价 | 精确控制 | 稳定账户 |

### 4.2 转化漏斗优化

| 阶段 | 核心指标 | 优化方向 |
|------|----------|----------|
| 曝光 | CPM | 受众精准度 |
| 点击 | CTR | 创意吸引力 |
| 访问 | 跳出率 | 落地页体验 |
| 加购 | CVR | 产品吸引力 |
| 购买 | 转化率 | 信任/价格 |

### 4.3 优化技巧

#### 转化优化
- 告诉Facebook"谁更可能下单"
- 建立受众池细分（访问/加购/购买）
- Pixel事件追踪完整

#### 购物广告 (DPA)
- 动态产品广告提升20%转化
- 重定向已浏览用户
- 产品目录质量优化

## 五、广告与转化漏斗

### 5.1 Offer设计

#### 优惠类型
- 折扣码
- 满减/买赠
- 免费配送
- 限时促销
- 赠品/抽奖

#### Facebook工具
- 信息标签（徽章）
- 优惠码追踪
- 灵活广告测试多版本

### 5.2 漏斗设计

#### 曝光阶段
- 目标：品牌认知
- 指标：CPM、触达

#### 互动阶段
- 目标：点击/互动
- 指标：CTR、CPC

#### 转化阶段
- 目标：购买/询盘
- 指标：转化率、CPA

### 5.3 广告序列 (Sequential Ads)

#### 应用场景
- 用户点击后 → 展示详情
- 用户加购后 → 促销优惠
- 用户未购买 → 追加展示

#### 设置方法
- 广告组层级设置
- 触发条件定义
- 创意顺序排列

### 5.4 社媒配合

#### 内容策略
- 投放不是全部转化的
- 部分预算投内容/品牌
- 与社媒活动配合

#### 赠品/竞赛
- 提升品牌知名度
- 带来更多潜在客户
- 长期回报

## 六、数据分析与优化

### 6.1 核心指标

| 指标 | 含义 | 优化方向 |
|------|------|----------|
| CPM | 千次展示成本 | 受众/版位 |
| CTR | 点击率 | 创意/文案 |
| CPC | 单次点击成本 | 整体优化 |
| CVR | 转化率 | 落地页/产品 |
| CPA | 单次转化成本 | 全链路 |
| ROAS | 广告支出回报 | 盈利判断 |

### 6.2 归因模型

- 最后一次点击
- 首次点击
- 多触点归因
- 查看转化

## 九、常见问题

### Q1: 广告跑不出去怎么办？
- 检查受众是否过窄
- 提高出价测试
- 检查素材合规性
- 检查账户状态

### Q2: CPM太高怎么办？
- 扩大受众
- 调整版位
- 优化素材相关性
- 检查竞争程度

### Q3: CTR低怎么办？
- 优化素材吸引力
- 测试不同文案
- 调整受众定位
- 检查版位适配

### Q4: 转化率低怎么办？
- 检查落地页
- 检查产品吸引力
- 检查信任元素
- 优化转化流程

### Q5: 如何扩量？
- 复制高效广告
- 扩大相似受众
- 增加预算
- 多版位投放
