# 法律获客课程系统整理｜可迁移到跨境社媒与B2B获客

> 一句话总结：这套课的核心不是“律师怎么做个人IP”，而是把法律服务当成一门可成交的生意：先定义优质用户，再用精准内容找到人，用信任内容降低成交阻力，用产品化让客户带着预算来咨询，最后用复盘把内容、产品、成交链路持续调准。

## 1. 结构化总纲

### 1.1 课程主线

```text
不做泛IP → 做生意
做生意 → 产品前置
产品前置 → 让客户知道你卖什么、适合谁、解决什么问题、多少钱/预算区间
内容获客 → 不追泛流量，只筛优质用户
信任成交 → 用真实人设、专业证据、客户证言、服务流程降低解释成本
复盘增长 → 5条视频内必须看到有效咨询或产品/内容就要调整
```

### 1.2 最值得保留的核心模型

| 模型 | 课程含义 | 与你工作的连接 |
|---|---|---|
| **优质用户筛选模型** | 短视频不是找所有人，而是在沙子里筛金银钻；拒绝低预算、低意愿、低配合度的无效用户。 | 用于食品机械：不要追所有采购商，只筛有工厂、有产线、有预算、有交期、有明确设备需求的海外客户。 |
| **产品前置模型** | 每条内容都要让用户知道你卖的不是泛知识，而是一套服务/产品。 | 用于跨境社媒：不要只讲“食品机械知识”，要讲“某类产线解决方案”“某设备选型标准”“工厂验厂陪跑服务”。 |
| **私域激活模型** | 短视频第一批要激活私域和已有渠道，而不是只等公域爆发。 | 用于你：朋友圈、LinkedIn、WhatsApp、老客户、工厂联系人都要被持续内容激活。 |

### 1.3 主用模型与备选模型

**主用模型：**
1. [[优质用户筛选模型]]：先排除低价值人群。
2. [[产品前置模型]]：内容不是单纯讲知识，而是为成交服务。
3. [[精准获客内容模型]]：用痛点、预算、场景、身份筛客户。
4. [[五条视频复盘模型]]：用小批量视频快速验证方向。

**备选模型：**
1. [[热点落专业模型]]：热点可用，但必须落回专业和产品。
2. [[名片Plus模型]]：适合账号早期没有流量时增强信任。
3. [[服务命名模型]]：把服务起名，变成客户可询问、可比较、可购买的产品。
4. [[私域激活模型]]：适合你已有客户、工厂、朋友、老资源唤醒。

## 2. 跟你工作的直接连接

### 2.1 连接到跨境社媒获客

这套课最适合迁移到你的 [[02-跨境社媒获客]]：你不是要做泛娱乐账号，而是要做“精准线索账号”。内容目标不是播放量最大，而是让正确客户主动暴露需求。

可执行方向：
- 每条视频前置一个明确服务或产品：例如“食品机械海外获客冷启动包”“LinkedIn工厂客户开发包”“Facebook询盘内容测试包”。
- 内容中加入筛选条件：国家、设备、日产量、预算、交期、认证、是否需要整线方案。
- 私信承接不要免费咨询泛问题，而是引导填写需求：设备类型、物料、产能、国家、预算、联系方式。

### 2.2 连接到食品机械工厂账号

法律服务产品化可以迁移成食品机械解决方案产品化：

| 法律课概念 | 食品机械迁移 | 示例 |
|---|---|---|
| 法律服务产品化 | 设备/产线方案产品化 | “中央厨房洗切配产线选型方案” |
| 客户带预算咨询 | 客户带产能和预算咨询 | “日产5吨蔬菜清洗切割线怎么配？” |
| 精准获客内容 | 只吸引采购/老板/工程负责人 | “买切菜机前必须问工厂的5个参数” |
| 信任内容 | 工厂能力证据 | 车间、库存、焊接、试机、发货、客户案例 |
| 复盘高于一切 | 每5条视频看询盘质量 | 是否进入WhatsApp、是否要报价、是否有真实产线需求 |

### 2.3 连接到你的销售成交

你现在最需要的不是“更多人看见”，而是“对的人看见之后知道怎么问你”。所以你的内容要从“我懂很多”改成“我有一套能帮你解决问题的服务/方案”。

建议你把服务命名成：
- 食品机械海外询盘冷启动包
- 工厂短视频获客陪跑包
- 30天LinkedIn客户开发包
- Facebook广告素材测试包
- 食品机械工厂出海内容诊断包

## 4. 课程逐课结构化总结

### 4.1 先导课：所有律师必走的弯路，每人都能落地的方法

**结构化总结：** 核心是“不做IP做生意”：法律短视频不应追求泛播放，而要先定义优质用户，用产品前置提高广告性和成交率。法律服务没有强种草属性，泛流量容易带来鸡毛蒜皮的无效咨询。

**与你工作的连接：** 迁移到你：不要做泛“跨境知识IP”，要做可成交的“食品机械海外获客服务/工厂出海内容服务”。

**可沉淀为知识库模型：** [[优质用户筛选模型]]

### 4.2 第一课：用优质用户思维定位好成交的人群

**结构化总结：** 核心是“优质用户思维”：传统渠道已经有信任背书和预算预期，短视频公域却是沙子里筛金银钻。互联网渠道不缺客资，缺的是优质客资，所以要敢于拒绝低价值用户。

**与你工作的连接：** 迁移到你：海外客户也要分金银钻，不要把所有询盘都当好客户；先定义国家、行业、产能、预算、采购周期。

**可沉淀为知识库模型：** [[产品前置模型]]

### 4.3 第二课：用精准获客内容找到好成交的人群

**结构化总结：** 核心是“先激活私域，再做公域精准获客”：短视频不仅是陌生人获客，也能成为名片Plus，增强现有渠道转化；精准内容要像积木孔，只让目标客户进来。

**与你工作的连接：** 迁移到你：LinkedIn、WhatsApp、朋友圈、老客户、工厂老板都要持续被内容激活；账号也是你的数字名片。

**可沉淀为知识库模型：** [[精准获客内容模型]]

### 4.4 第三课：建立信任促进成交的内容方法论

**结构化总结：** 核心是“信任内容”：账号要让用户感觉你是真人、有专业、有态度、有证据。热点可以蹭，但必须目标受众不冲突，并落回专业和法律服务产品。

**与你工作的连接：** 迁移到你：工厂探访、设备试机、客户问答、交付记录、报价透明、售后流程都是信任内容。

**可沉淀为知识库模型：** [[信任内容模型]]

### 4.5 第四课：法律服务产品化让客户带着预算咨询

**结构化总结：** 核心是“法律服务产品化”：行业机会不在于更卷，而在于多数律师缺用户思维和产品思维。服务必须被命名、被包装、被解释成客户能理解的具体权益和结果。

**与你工作的连接：** 迁移到你：把你的服务包和工厂设备方案产品化，不要只说“我会做海外社媒”。

**可沉淀为知识库模型：** [[服务产品化模型]]

### 4.6 第五课：能够做大做强的律师自媒体成长规划

**结构化总结：** 核心是“成长路径”：第一阶段去IP化和模仿，找对标、打标签、看懂好坏，尽早变现；第二阶段再做差异化和IP化，热点不超过20%，浅层普法要少做。

**与你工作的连接：** 迁移到你：先模仿已验证的工厂号/律师号/外贸号结构，拿到线索后再做独特风格。

**可沉淀为知识库模型：** [[五条视频复盘模型]]

### 4.7 加餐课：法律服务产品化落地方法

**结构化总结：** 核心是“产品化落地”：最简单的方法是给服务起名，把抽象服务变成客户能询问的产品，例如办案法、办案标准、几步法、几项权益。

**与你工作的连接：** 迁移到你：给你的服务起名，例如“食品机械海外询盘冷启动包”“30天LinkedIn开发包”。

**可沉淀为知识库模型：** [[五条视频复盘模型]]

### 4.8 加餐课：律师做短视频复盘高于一切

**结构化总结：** 核心是“复盘高于一切”：做对比坚持更重要。5条视频没有有效线索，就要判断内容或产品出了问题；有精准咨询但没成交，多半是产品吸引力不足；没有精准咨询，多半是内容筛选错。

**与你工作的连接：** 迁移到你：每5条内容复盘一次，不要靠蛮干；没有询盘就改内容，有询盘没成交就改产品和承接。

**可沉淀为知识库模型：** [[五条视频复盘模型]]
