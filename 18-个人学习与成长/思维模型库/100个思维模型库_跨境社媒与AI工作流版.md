# 100个思维模型库｜跨境社媒与AI工作流版

> 一句话总结：**你不需要平均学习100个模型，而要把其中30-50个高频模型变成自己的销售、投流、内容、工厂筛选、知识库治理和AI工作流决策工具。**

## 0. 使用原则

- **核心模型**：直接放入你的模型库，高频使用。
- **备选模型**：保留在模型库，可在特定场景调用。
- **暂存模型**：知道即可，不优先占用工作流注意力。
- **图片策略**：默认不保留全部图片；如果要保留图片，建议只保留下方“精选图片页”30张以内，避免Obsidian膨胀。

## 1. 根据你的工作，最值得优先使用的模型

### A. 跨境社媒/外贸获客核心模型

| 模型 | 用法 |
|---|---|
| [[黄金圈思维模型]] | 文案先讲为什么，再讲怎么做，最后讲卖什么。 |
| [[损失规避思维模型]] | B2B客户怕买错、怕停产、怕售后没人管，成交话术要先兜风险。 |
| [[冰山理论思维模型]] | 客户问价格只是表层，底层是信任、交付、安全感和责任归属。 |
| [[长尾理论思维模型]] | 食品机械细分设备、长尾关键词、小众国家市场都可能带来高意向询盘。 |
| [[波特五力分析思维模型]] | 判断某个设备类目是否适合做海外内容和投流。 |
| [[护城河理论思维模型]] | 把案例、标准、知识库、客户关系做成长期壁垒。 |

### B. 投流/内容/素材核心模型

| 模型 | 用法 |
|---|---|
| [[机会成本模型]] | 判断投一个渠道意味着放弃另一个渠道，避免“什么都想做”。 |
| [[决策树模型]] | 广告测试、素材测试、预算加减、停投条件全部树状化。 |
| [[优先排序思维模型]] | 先处理高意向客户、高潜产品、高价值素材。 |
| [[HOOK上瘾思维模型]] | 做短视频开头、广告钩子、私域关注路径。 |
| [[峰终定律思维模型]] | 客户记住的是高峰体验和结束体验，报价和交付收尾要设计。 |
| [[福格行为模型]] | 让客户行动=动机×能力×触发，CTA不能只喊“联系我们”。 |

### C. AI工作流/Obsidian知识库治理核心模型

| 模型 | 用法 |
|---|---|
| [[MECE原则思维模型]] | 目录、客户分类、产品分类必须不重不漏。 |
| [[金字塔原则思维模型]] | AI输出、飞书文档、客户方案都先结论后论据。 |
| [[反熵增思维模型]] | 知识库不治理必然混乱，必须周期性压缩、去重、补summary。 |
| [[冗余备份思维模型]] | 主库、配置、客户资料、项目文件必须有快照和备份。 |
| [[PDCA思维模型]] | 每一批整理：计划、执行、检查、调整，不要一次性乱改。 |
| [[路径依赖思维模型]] | 防止旧目录、旧工具、旧流程继续拖累新系统。 |

### D. 销售成交/客户沟通核心模型

| 模型 | 用法 |
|---|---|
| [[第一性原理思维]] | 回到客户真正购买原因：省钱、降风险、增产能、有人负责。 |
| [[逆向思维模型]] | 从客户为什么不买、为什么不信、为什么拖延倒推话术。 |
| [[替身决策思维模型]] | 站在采购、老板、技术、财务、工厂五种角色看方案。 |
| [[多维视角思维模型]] | 不只看客户，还看竞争对手、物流、售后、采购流程、预算周期。 |
| [[博弈论思维模型]] | 报价、议价、合作分成、工厂资源都要有博弈视角。 |
| [[前景理论思维模型]] | 客户对损失更敏感，表达收益时要同时表达不做的损失。 |

## 6. 每周调用方法

1. 当前问题属于：客户、产品、内容、投流、知识库、AI工作流，哪一类？
2. 先用哪个模型判断？
3. 有没有反证？有没有确认性偏差？
4. 不做这件事的机会成本是什么？
5. 这件事能否沉淀成模板、SOP、案例库或Obsidian文稿？
