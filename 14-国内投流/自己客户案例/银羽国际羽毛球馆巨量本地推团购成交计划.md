# 银羽国际羽毛球馆巨量本地推团购成交计划

## 一句话结论

这条计划的核心不是“投门店”，而是**投商品成交**：只投 **49元羽毛球体验课**，不投 **45元场地费**，不投 **325元一对一私教课**。

这样做的目的：让系统学习“谁会买体验课”，而不是学习“谁会买场地费”。

---

## 1. 当前投放入口

正确入口：

```text
巨量本地推
→ 投放
→ 新建项目和单元
→ 营销场景：短视频/图文
→ 投放目标：团购成交
→ 单元类型：通投
→ 下一步
→ 投放内容与目标
→ 投放商品
```

不要再走：

```text
门店全域投放
```

因为门店全域投放偏向整体门店成交，容易把场地费、课程、一对一等商品混在一起，让系统学习方向变乱。

---

## 2. 商品选择

### 只选择

```text
羽毛球体验课90分钟，小班课程，需要提前预约
价格：49元
```

### 暂时不选择

```text
工作日场地使用，限时9点到16点
价格：45元
```

原因：这是场地费，不是课程体验。投它会让系统偏向找订场地的人，而不是找新课学员。

### 暂时不选择

```text
羽毛球资深教练一对一次卡
价格：325元
```

原因：高客单一对一课程和低价体验课的转化人群不同，第一条计划不要混投。

| 模块 | 选择 |
|---|---|
| 营销场景 | 短视频/图文 |
| 投放目标 | 团购成交 |
| 单元类型 | 通投 |

| 商品 | 只选49元体验课 |
| 优化目标 | 商品购买 / 团购成交 |

## 4. 用户定向配置

### 地域

```text
门店附近：8km
```

说明：8km 比 10km 更稳，适合当前预算不大的冷启动阶段。  
如果后续没量，再从 8km 扩到 10km。

### 人群模式

```text
自定义
```

不要第一版直接用“智能推荐”。羽毛球课程有明确到店半径和年龄特征，先用自定义把基本盘框住。

### 性别

```text
不限
```

原因：成人羽毛球体验课男女都可能成交，前期不要主观砍掉一半流量。

### 年龄

推荐第一版：

```text
24-30
31-40
41-49
```

也就是：

```text
24-49岁
```

说明：

- 18-23：学生多，价格敏感，到店稳定性可能弱。
- 24-40：主力上班族，有运动、社交、减脂、兴趣需求。
- 41-49：有消费能力，也可能为自己或孩子咨询。
- 50+：后续可以单独做“中老年健康运动/低强度羽毛球”素材，不和成人小班课混投。

### 人群包

```text
不限
```

前期不要乱加自定义人群包，避免限量。

### 抖音达人

```text
不限
```

本地课程前期更依赖“门店半径 + 商品 + 素材”，不要先限制达人兴趣。

### 过滤已转化用户

建议：

```text
项目
```

不建议第一版选“本地推账户”。  
原因：买过场地费的人，未来也可能买课程。如果选“本地推账户”，可能误排除潜在课程学员。

### 智能定向拓展

```text
启用
```

前面已经控制了商品、半径、年龄，可以让系统在可控范围内拓展。

## 5. 排期与预算

### 投放日期

```text
从今天起长期投放
```

如果只是测试，也可以设置 3-5 天结束日期，但第一版建议长期投放，方便系统学习。

### 投放时段

推荐：

```text
周一至周五：
11:00-15:00
17:00-23:30

周六至周日：
09:00-24:00
```

不建议周末 00:00-09:00 也投。凌晨流量可能便宜，但对羽毛球体验课的下单质量不一定好。

### 出价方式

```text
手动出价
```

### 日预算

保守版：

```text
200元/天
```

快速拿数据版：

```text
300元/天
```

当前建议：如果账户余额不足1000元，优先用 200 元；如果想快速拿样本，可以用 300 元跑一天看数据。

### 出价

```text
20元 / 商品购买
```

暂时不要压到 15 元，容易没量。等跑出稳定成交后，再逐步压到 18 元或 16 元。

### 搜索出价系数

```text
先不填
```

第一版不要加太多变量，先看通投效果。

## 7. 素材方向

素材必须围绕“课程体验”，不要出现“场地预订”“自由打”“订场”等表达。

### 推荐标题

```text
49元羽毛球90分钟体验课｜成人小班｜零基础可学
```

### 推荐卖点

- 90分钟体验课
- 成人小班
- 教练带练
- 零基础可学
- 下班后可约
- 石家庄本地羽毛球馆
- 适合想运动、减脂、社交、提升球技的人
- 不是场地预订，是课程体验

### 避免词

- 场地费
- 订场
- 自由打
- 工作日场地
- 仅场地使用

## 8. 24小时复盘标准

计划上线后不要频繁修改，至少跑满 24 小时再看。

### 必看指标

| 指标 | 判断标准 |
|---|---|
| 消耗 | 是否正常消耗 |
| 成交商品 | 是否都是49元体验课 |
| 成交成本 | 是否低于25元 |
| ROI | 是否稳定高于1 |
| 咨询量 | 是否有人私信/电话/到店问课 |
| 核销率 | 买课后是否预约到店 |
| 素材点击 | 哪条视频/图文带来成交 |

### 处理规则

| 情况 | 动作 |
|---|---|
| 成本 < 20元 | 第二天预算可从200加到300 |
| 成本 20-30元 | 继续跑一天，不急着改 |
| 成本 > 35元 | 先换素材，不要先大改人群 |
| 没量 | 半径从8km扩到10km，或年龄扩到24-49 |
| 成交仍跑偏 | 检查是否误选场地费，或商品页是否引导到了场地商品 |

## 11. 关键提醒

不要在第一条计划里混投：

```text
49元体验课 + 45元场地费 + 325元一对一
```

这会让系统不知道你到底想要哪种成交。

第一阶段只解决一个问题：

```text
让系统找到愿意买49元体验课的新客。
```

等49元体验课稳定成交后，再单独开35元体验课、一对一私教课、场地费计划。
