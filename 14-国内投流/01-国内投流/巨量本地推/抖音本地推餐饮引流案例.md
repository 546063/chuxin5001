# 抖音本地推餐饮引流案例

## 一句话总结

成都5家连锁火锅门店30天抖音本地生活投放，3万预算产出25.5万GMV，ROAS 8.5x，核销率70%。

## 核心结论

- 达人探店+企业号自产+团购直播三管齐下效果最佳
- 企业号内容带来的用户核销率和复购率最高（75%/18%）
- 团购有效期缩至15天+一键预约可显著提升核销率
- 达人选择看近10条数据不看粉丝量

## 适用场景

- 餐饮门店抖音本地生活从0启动
- 达人探店矩阵搭建
- 团购套餐设计与优化
- 内容节奏规划

## 具体内容

### 案例拆解

#### 案例背景

| 字段 | 值 |
|------|-----|
| 行业 | 餐饮（火锅/烤肉/日料） |
| 平台 | 抖音本地生活（POI 自然+付费混合） |
| 广告类型 | POI 短视频 + 达人探店 + 团购直播 |
| 投放周期 | 30天 |
| 投放市场 | 成都（单城市） |
| 广告主 | 连锁餐饮品牌（成都 5 家门店） |
| 总消耗 | ¥30,000（¥1,000/天） |
| 总团购销量 | 2,850 份 |
| 总核销 | 1,995 份（70%） |
| 客单价 | ¥128 |
| 总GMV | ¥255,360 |
| ROAS | 8.5x |
| CAC（单团购） | ¥10.5 |
| 核销成本 | ¥15 |

#### 投放目标

- **主目标**：团购券售卖量（到店核销）
- **次目标**：POI 页面曝光（品牌认知 + 自然流量带动）

> 抖音本地生活 vs 巨量本地推：本地生活侧重POI页面+团购（内容驱动），巨量本地推侧重广告计划（投放驱动），两者互补。

#### 目标人群

| 维度 | 设置 |
|------|------|
| 地域 | 门店周边 5 公里（POI 自然流量）+ 全成都（达人内容） |
| 年龄 | 18-35（火锅主力消费人群） |
| 兴趣 | "美食" / "探店" / "成都本地" |
| 消费水平 | 人均 ¥80-¥150 |

#### 素材策略

**内容构成（30天内产出）**：

| 内容类型 | 数量 | 单条成本 | 贡献团购销量 | ROI |
|---------|------|---------|------------|-----|
| **达人探店视频** | 15条（达人） | ¥800/条（置换+费用） | 1,420份 | 10.2x |
| **企业号自产内容** | 20条 | ¥100/条（内部拍摄） | 850份 | 12.8x |
| **团购直播** | 4场（每场2小时） | ¥2,000/场 | 580份 | 4.1x |

**达人探店素材特征**：
1. **前 3 秒**：菜品特写（沸腾的火锅/滋滋作响的烤肉）
2. **中段**：达人吃播反应（真实表情 > 夸张表演）
3. **结尾**：团购引导（"点击左下角，双人餐只要 ¥168"）
4. **POI 挂载**：必须挂载门店 POI，引导到店

**企业号自产内容公式**：

```
"后厨揭秘"（信任）: 食材进货 → 洗菜 → 切菜 → 上桌全过程
"老板讲故事"（情感）: 创业故事/配方研发/顾客故事
"菜品上新"（刺激）: 新菜品 + 限时优惠
"顾客反应"（社交证明）: 真实顾客的吃播反应
```

#### 文案结构

**达人探店文案**：
```
Hook: "成都这家火锅店，我吃了3年还吃不腻"
Body: "今天带你们看看他们家的隐藏吃法——毛肚涮8秒+鸭肠涮3秒，绝了！
现在抖音团购双人餐只要168，原价298，冲！"
CTA: "点击左下角抢购"
```

**企业号文案**：
```
Hook: "开了12年的老火锅店，后厨长这样"
Body: "每天早上6点去市场选菜，毛肚必须是当天的。不做预制菜，不卖隔夜菜。新客首单立减50，团购已上。"
CTA: "点击左下角"
```

#### 投放结构

```
内容策略（非付费）:
├── 达人探店: 15个达人（5000-5万粉丝），置换+少量费用
├── 企业号日更: 每天发布 1 条短视频
└── 每周直播: 每周 1 场，2小时，推团购套餐

付费策略（巨量本地推辅助）:
├── 达人视频加热: ¥300/天 × 5条优质达人视频
├── 企业号视频加热: ¥200/天 × 3条数据好的视频
└── 团购推广: ¥500/天（目标：团购成交）

总预算: ¥1,000/天 × 30天 = ¥30,000
```

#### 转化链路

```
抖音短视频/直播
  ↓ (刷到/推荐页/POI 页面）
团购详情页
  ↓ (冲动购买，3秒决策）
2,850 份团购券售出
  ↓ (7天内到店）
1,995 份核销（70%）
  ↓ (客单 ¥128）
GMV ¥255,360
```

| 环节 | 数据 | 行业基准 |
|------|------|---------|
| 视频播放量 | 180万 | 50万-300万 |
| 团购点击率 | 4.5% | 2%-6% |
| 购买转化率 | 8% | 5%-12% |
| 核销率 | 70% | 50%-80% |

#### 不同来源的核销率对比

| 来源 | 售出份数 | 核销率 | 复购率（30天） |
|------|---------|--------|--------------|
| 达人探店 | 1,420 | 68% | 12% |
| 企业号内容 | 850 | 75% | 18% |
| 直播 | 580 | 65% | 8% |

> 发现：企业号内容带来的用户核销率和复购率最高，因为粉丝黏性更强。

**影响核销率的关键因素**：

| 因素 | 影响 | 优化建议 |
|------|------|---------|
| 有效期太长（>30天） | 核销率下降 20% | 缩至 15 天 |
| 需要预约 | 核销率下降 15% | 免预约直接到店 |
| 工作日限定 | 核销率下降 10% | 全时段可用 |
| 门店位置偏僻 | 核销率下降 25% | 提供停车/地铁指引 |

### 可复用方法

#### 方法1：达人矩阵策略

```
达人选择标准:
- 粉丝量: 5,000 - 50,000（本地达人）
- 互动率: > 3%
- 内容质量: 真实探店风格（非硬广）

合作模式:
- 免费置换（粉丝 < 1万）
- ¥300-800/条（粉丝 1-5万）
- ¥1,000-3,000/条（粉丝 5-20万）
```

#### 方法2：团购套餐设计

```
引流款（不赚钱，纯拉新）:
  双人餐 ¥128（原价 ¥220），毛利 15%
利润款（赚钱）:
  四人餐 ¥368（原价 ¥580），毛利 35%
加价款（提升客单）:
  +¥38 升级和牛拼盘，+¥28 升级甜品套餐
```

#### 方法3：内容节奏

```
周一/三/五: 企业号发布（菜品/后厨/故事）
周二/四: 达人探店内容发布
周末: 直播（2小时）
每天: 回复评论/私信（提升互动率）
```

### 风险和限制

| 风险 | 影响 | 缓解 |
|------|------|------|
| 达人内容质量不稳定 | 部分视频播放 < 1,000 | 选人时看近10条数据，不看粉丝量 |
| 核销率低 | 卖了很多但收入低 | 缩短有效期 + 短信提醒 |
| 差评反噬 | 差评导致 POI 评分下降 | 及时回复差评，引导好评 |
| 团购价格过低 | 损害品牌形象 | 区分"引流款"和"利润款" |

## 对我的业务有什么价值

- **对巨量本地推**：餐饮行业的标准投放模板，可直接套用达人矩阵+企业号+直播三合一策略
- **对国内业务**：餐饮类本地客户可以直接参考团购设计和内容节奏
- **对教育培训/装修/医美获客**：虽然行业不同，但达人合作模式、内容公式、核销率优化方法可以迁移
