# 巨量本地推投流知识库

## 一句话总结

基于2026年4月公开资料整理的巨量本地推全链路操作指南，覆盖账户设置、出价策略、定向优化、素材创作、起量放量、问题排查等全部环节。

## 核心结论

- 每个营业执照仅支持开通一个本地推账号
- 门店信息需与抖音来客后台营业执照信息完全一致
- 优先采用「自动控制」模式释放算法优势
- 预算需达到「目标成本×20」阈值
- 初始定向范围3-5公里，切忌盲目扩大

## 适用场景

- 新开户商家从零开始搭建投放体系
- 已有投放经验的商家系统梳理操作规范
- 团队内部培训和SOP制定

## 具体内容

### 1. 账户结构与基础设置（必会操作清单）

**开户流程**：
- PC端入口：抖音来客电脑版 → 左侧导航栏 → 推广中心 → 本地推广
- 独立登录页或巨量引擎方舟平台（代理商）
- 确认3个信息：账户名称、行业主体名称

**签约方式**：
- 自助客户：勾选《巨量本地推推广服务协议》《巨量本地推隐私政策》
- 直客：线下销售签约或线上自主签约（需CA签章）

**基础配置检查**：
- [ ] 门店信息与营业执照一致
- [ ] 抖音来客门店已认领
- [ ] 带货口碑分≥4.0（低于4分禁止投放）
- [ ] 店铺体验分达标

### 2. 出价逻辑与预算分配（决策框架）

**预算设置原则**：
- 新计划：目标成本×20倍作为日预算
- 成熟计划：根据历史消耗动态调整
- 避免预算过低影响系统学习

**出价策略**：
- **新计划初期**：高于同行平均水平5-10个点出价
- **出价上限**：不超过目标价的40%
- **计算公式**：商品价格的1/3-1/2，结合佣金、运费、运营成本调整
- **智能放量**：配合自动出价探索人群

**预算分配**：
- 成单计划：分配主要预算（如1万预算中80%给成单）
- 点击计划：100元以内预算足够
- 进入计划：300-500元预算用于人群探索

### 3. 定向策略（人群/地域/兴趣/行为组合）

**地域定向**：
- 优先选择门店周边3-5公里
- 餐饮类：添加「美食探店」「网红打卡」标签
- 教育类：添加「亲子教育」「少儿培训」标签

**人群定向**：
- **系统推荐定向**：初期适合人群模型探索
- **自定义定向**：待账户有转化模型后使用
- **DMP人群包**：圈选类目精准人群（如茶叶类目+高消费人群）

**智能放量配置**：
- 成熟计划开启智能放量，探索已选择定向之外人群
- 配合提高出价增强精准人群流量竞争力

### 4. 素材策略（格式/文案/测试方法）

**素材创作要点**：
- **视频格式**：竖版9:16（15-60秒），横版16:9
- **内容风格**：vlog形式，聚焦"产品种草"而非"产品销售"
- **视觉要求**：画质清晰明亮，色彩明艳丰富
- **音频要求**：配乐曲风温和欢快

**标题模板（6种高转化）**：
1. 热点法："同城爆火的XX店，刷屏了！"
2. 数字法："3公里内300+人打卡的宝藏店"
3. 共鸣法："本地人才知道的隐藏福利"
4. 利益法："新店开业5折套餐，限时抢"
5. 疑问法："这家店凭什么让老饕们排队2小时？"
6. 对比法："比网红店更便宜，比街边摊更精致"

**AB测试方法**：
- 同一计划下创建不同创意对比测试
- 测试维度：封面、标题、剪辑风格
- 数据对比筛选转化效果最佳版本

### 5. 起量与放量SOP（步骤化）

**首投冷启动期（7天）**：
- Day1-3：保持定向、出价等核心参数不变
- Day4-7：观察数据，轻微调整（<10%）
- 重点关注：曝光>点击>到店漏斗数据

**续投放量期**：
- 复制跑量计划，修改定向或创意分类
- 开启智能放量拓宽人群范围
- 按5%-10%幅度微调预算和出价

**计划生命周期管理**：
- **新计划**：每天固定时间段开播，建立粉丝心智
- **成熟计划**：设置固定时段投放（半小时为维度）
- **衰退计划**：及时补充新计划替代，避免青黄不接

### 6. 优化排查手册（问题→原因→解法）

| 问题 | 原因 | 解决方案 |
|------|------|---------|
| 计划完全没消耗 | 审核未通过/口碑分<4/定向过窄/素材差 | 确认审核状态→检查口碑分→提高出价扩大人群→优化素材 |
| 消耗不稳定 | 大盘流量波动/直播间数据变化/参数频繁更改 | 拉长至天维度观察→避免频繁更改→结合直播间节奏调整 |
| 转化成本高于出价 | 素材质量/人货场不佳/定向不精准 | 优化定向商品素材→适当调低出价→联系销售排查 |
| 计划突然爆量 | 出价过高/内容指标提升/系统波动 | 降低出价→设置预算限制→选择控成本投放+精准定向 |

### 7. 数据指标体系

**漏斗指标**：
- 曝光量 → 点击率（CTR）→ 转化率（CVR）
- 健康标准：CTR>3%，CVR>1%

**成本指标**：
- CPM / CPC / CPA
- ROI = GMV/消耗

**质量指标**：
- 带货口碑分（需≥4.6无限制）
- 店铺体验分
- 48小时发货率（需≥70%）

**关键判断标准**：
- 计划健康度：学习成功≠优质计划，需观察衰减速度
- 素材效果：前5秒完播率>30%为合格
- 人群精准度：搜索核销率70%为优秀水平

### 8. 常见踩坑与避坑清单

| 踩坑 | 错误做法 | 正确做法 |
|------|---------|---------|
| 冷启动频繁调整 | 频繁修改定向、出价 | 保持3天观察期 |
| 预算设置过低 | 日预算<目标成本×10 | 日预算≥目标成本×20 |
| 定向范围过宽 | 初始投放选择全城或全省 | 优先3-5公里+行业兴趣标签 |
| 素材营销属性过强 | 视频中出现价格/促销/二维码 | 采用种草风格，弱化价格利益点 |
| 跨计划AB测试 | 不同计划测试不同创意 | 同一计划下创建不同创意对比 |
| 忽视口碑分限制 | 口碑分<4分仍持续投放 | <4分禁止投放，4-4.6分每日限制单量 |

### 9. 本地推 vs Facebook 投放对比

| 维度 | 巨量本地推 | Facebook 本地广告 |
|------|----------|----------------|
| 地域精度 | 3-20 公里 | 1-50 英里 |
| 转化目标 | 到店核销 | 到店/表单/消息 |
| 内容生态 | 短视频/直播为主 | 图文/视频混合 |
| 学习期 | 7 天 | 50 次转化/7 天 |
| 预算公式 | 目标成本 × 20 | 目标 CPA × 3（日预算） |
| 素材风格 | 种草 > 硬广 | 直接转化型更有效 |
| 核心指标 | 核销率 | ROAS/CPA |

**经验迁移**：

| Facebook 经验 | 本地推对应 |
|-------------|----------|
| CBO 自动分配 | 智能放量 |
| 素材疲劳（Frequency > 2.5）| CTR 下降 > 25% |
| ABO 测试 → CBO 放量 | 同计划 AB 测试 → 复制计划放量 |
| Pixel 归因 | 团购核销追踪 |
| 学习期 50 转化 | 冷启动 7 天 |
| 20% 扩量规则 | 每次调整 < 10% |

### 10. 本地推进阶策略

**短视频 + 直播联动**：

```
短视频引流（日常投放）
  ↓
用户进入主页/直播间
  ↓
直播集中转化（限时优惠）
  ↓
沉淀粉丝 → 再营销
```

预算分配建议：
- 短视频日常投放：60-70%
- 直播集中投放：20-30%
- 搜索直投：10%

**多门店连锁投放策略**：

| 策略 | 做法 | 适用场景 |
|------|------|---------|
| **集中投放** | 所有门店共用账户，按门店分计划 | 5 家店以内 |
| **分散投放** | 每个门店独立账户 | 5 家店以上/跨城市 |
| **混合模式** | 品牌账户 + 门店子账户 | 连锁品牌 + 加盟店 |

**搜索直投（SEO + SEM 结合）**：

| 关键词类型 | 示例 | 出价 |
|-----------|------|------|
| 品牌词 | "XX 咖啡" | 低（自有品牌） |
| 品类词 | "附近咖啡" "美甲推荐" | 中 |
| 竞品词 | "XX 咖啡店 替代" | 高（拦截流量） |
| 场景词 | "约会餐厅" "亲子乐园" | 中 |

**季节性与节假日投放节奏**：

| 节日 | 提前准备 | 投放策略 |
|------|---------|---------|
| 春节 | 提前 3 周 | 年夜饭/聚会套餐 |
| 情人节 | 提前 2 周 | 双人套餐/浪漫场景 |
| 五一/十一 | 提前 2 周 | 旅游/酒店/餐饮 |
| 暑期 | 提前 1 月 | 亲子/教培/旅游 |
| 双11 | 提前 3 周 | 囤券/充值优惠 |
