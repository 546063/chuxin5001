# 家装装修行业巨量本地推投放案例

> 类型：模拟案例/可复用模板
> 适用：整装公司、半包施工队、全案设计工作室、全屋定制品牌、局部改造（厨卫翻新）
> 最后更新：2026-05-22

---

## 一、行业场景

### 家装装修行业核心特征

- **超高客单价**：整装 10-50 万元，半包 5-15 万元，局部改造 1-5 万元，全屋定制 3-20 万元
- **极长决策周期**：从首次接触到签约通常 2-8 周，期间需要多次沟通、看方案、比价格
- **强信任依赖**：客户需要看到真实案例、工地实景、材料品质才会放心签约
- **地域性强**：服务半径通常在一个城市内，跨城市接单成本高
- **信息不对称严重**：客户不懂装修，怕被坑、怕增项、怕质量差
- **口碑驱动**：熟人推荐仍然是最大获客来源，但线上获客占比逐年提升到 30-50%
- **决策者**：通常是夫妻两人共同决策，男性关注预算和材料，女性关注设计和风格

### 高客单价长决策周期的应对策略

```
认知阶段（第1周）：短视频/直播吸引注意 → 关注账号 → 私信咨询
了解阶段（第2-3周）：设计师1对1沟通 → 量房 → 出初步方案 → 报价
信任建立（第3-5周）：参观工地/展厅 → 看材料 → 确认设计细节
决策阶段（第5-8周）：对比3-5家 → 最终选择 → 签约 → 开工
```

本地推在每个阶段的作用不同：
- **认知阶段**：靠素材吸引（案例/知识/避坑）→ 目标：私信咨询
- **了解阶段**：靠设计师专业能力 → 目标：预约量房
- **信任建立**：靠工地实景和口碑 → 目标：确认方案
- **决策阶段**：靠限时优惠和对比优势 → 目标：签约

| 阶段 | 目标 | 优化方向 |

| 冷启动（1-7天） | 积累私信咨询量，打人群标签 | 私信优化目标，高出价抢量 |

## 七、短视频脚本方向

### 脚本模板 1：完工实景一镜到底（30秒）

```
0-3s 钩子（开门瞬间）："100平现代简约，硬装+软装一共花了18万，进来看看效果"
3-8s 客厅全景：一镜推进，展示客厅整体效果（沙发背景墙+电视柜+灯光）
8-13s 厨房：展示橱柜+台面+集成灶，"全屋定制橱柜，收纳空间翻倍"
13-18s 主卧：展示衣柜+床头背景墙+飘窗改造，"主卧做了整面墙衣柜"
18-23s 卫生间：干湿分离+智能马桶+浴室柜，"卫生间做了三分离"
23-27s 关键信息："这套是在XX小区，施工周期75天，半包8.5万"
27-30s 引导："想知道你家装修要花多少钱？点下方免费获取报价方案"
```

### 脚本模板 2：装修避坑型（25秒）

```
0-3s 钩子（数字冲击）："装修这6个地方省钱，入住后你一定会后悔"
3-8s 第一点：防水—"卫生间防水只做一遍？两年后楼下找上门"
8-13s 第二点：电线—"电线用便宜的？以后换要砸墙，成本翻10倍"
13-18s 第三点：五金—"铰链滑轨用杂牌？半年后柜门关不上"
18-22s 总结："装修不是越便宜越好，该花的地方一分不能省"
22-25s 引导："关注我，每天分享一个装修干货。需要报价方案点下方"
```

### 脚本模板 3：工地直播预告/精彩片段（20秒）

```
0-3s 钩子（冲突感）："水电验收发现这3个问题，直接让工人返工"
3-8s 问题1：电线没有套管，裸露在墙里
8-12s 问题2：水管走顶没有保温棉
12-16s 问题3：线盒间距不规范
16-20s 引导："明天下午2点，XX小区工地直播水电验收全过程，点预约提醒"
```

### 脚本模板 4：设计前后对比（30秒）

```
0-3s 钩子："这个89平的户型，改造前VS改造后，简直像两套房"
3-10s 改造前：原始户型平面图，标注问题（厨房小/卫生间暗/收纳不足）
10-15s 改造方案动画：平面图标注改造点（打通厨房/干湿分离/增加收纳）
15-22s 改造后实景：依次展示改造后的厨房、卫生间、客厅
22-27s 数据："改造后使用面积增加15平，收纳空间翻3倍"
27-30s 引导："你家户型有什么问题？发给我免费帮你出改造方案"
```

### 脚本模板 5：业主真实评价（30秒）

```
0-3s 钩子："在XX装修完之后，我把这套房子推荐给了3个朋友"
3-10s 业主出镜："我是XX小区的业主，去年10月找他们装的，硬装花了12万"
10-18s 谈体验："最满意的是两点：一是报价透明，中途没有增项；二是项目经理很负责，每周发进度"
18-24s 展示实景：跟着业主逛一遍家里，"这是我最喜欢的开放式厨房"
24-28s 补充："唯一建议是一定要提前选好材料，不然工期会拖"
28-30s 引导："想了解他们家价格的，点下方免费报价"
```

### 脚本模板 6：报价解析型（25秒）

```
0-3s 钩子："100平的房子，半包报价8万和15万，差在哪里？"
3-10s 对比材料：电线、水管、防水涂料、瓷砖胶的品牌和价格对比
10-17s 对比工艺：墙面处理（2遍腻子vs3遍+打磨）、水电走线（点对点vs横平竖直）
17-22s 结论："便宜的不一定差，贵的不一定好，关键看钱花在哪"
22-25s 引导："免费帮你审核报价单，找出虚报和漏项，点下方预约"
```

## 十一、复盘优化动作

1. **案例素材持续产出**：每完工一个工地，拍摄至少 5 条短视频素材（全景/细节/对比/业主采访/工地过程）。建立"案例素材库"，按风格/户型/预算分类管理，按数据表现轮换投放。
2. **线索分级管理**：将线索按意向度分级（A级：3个月内要装修+预算明确；B级：半年内+预算模糊；C级：纯了解），A级线索优先由资深设计师跟进，B/C级由销售助理培育。
3. **楼盘定向精准获客**：监控城市新楼盘交付信息，在交付前 1 个月开始针对该楼盘周边投放"XX小区专属设计方案"素材，精准度极高，线索成本可降低 40%。
4. **工地直播常态化**：每周 2-3 场工地直播（水电/泥瓦/木工节点），展示真实施工质量和工艺。直播期间留资的客户，量房到场率比短视频线索高 25%。
5. **私信响应速度优化**：目标 5 分钟内首次回复，超时自动提醒。响应速度每快 10 分钟，量房预约率提升 8%。非工作时间设置自动回复+次日优先跟进。
6. **报价工具引流**：开发简易"在线报价计算器"（面积×类型→估算价格），用户输入信息后显示估算价并引导留资获取详细方案。此类工具留资转化率比纯表单高 50%。
7. **素材合规审查**：避免出现"最低价""全城最便宜""零增项"等绝对化用语。用"参考价格""平均报价""透明报价"替代。材料品牌展示需获得授权。
8. **同行屏蔽**：在定向中排除"装修行业从业者"标签，在搜索词中排除"加盟""招聘""培训"等词。定期检查搜索报告，将无关搜索词设为否定词。
9. **口碑转介绍体系**：签约客户在装修完成后邀请填写评价，评价后赠送小家电/购物卡。转介绍客户签约率是投流线索的 3-5 倍，获客成本接近零。
10. **季节节奏调整**：3-5月和9-11月是装修旺季，预算提高 50-100%，素材围绕"春季家装节""金九银十"主题。6-8月（梅雨/高温）和12-1月（冬季/年底）预算降低 30%，素材转向"年底囤优惠""跨年装修"。
11. **设计师个人IP**：让公司设计师在抖音建立个人账号，发布设计知识和案例。个人IP素材的 CTR 比公司号高 30-50%，信任度也更高。
12. **竞品对比素材**：制作"选择装修公司要看这5点"类内容，暗含自身优势但不直接点名竞品。这类内容筛选出的线索意向度更高。

## 十二、常见坑

| 坑 | 表现 | 解决方案 |
|---|---|---|
| 线索量多但签约少 | 大量"随便问问"的线索 | 表单增加"预计装修时间"字段，过滤纯了解人群 |
| 设计师跟进不及时 | 线索留资后2天才联系，客户已找别家 | 建立线索分配机制+超时自动升级提醒 |
| 素材同质化严重 | 所有素材都是完工效果，用户审美疲劳 | 增加避坑/报价/工地/对比/采访等多元素材 |
| 报价虚低引流 | 广告报低价吸引客户，到店后涨价导致信任崩塌 | 广告报价必须真实可执行，宁可报高不报低 |
| 跨城市接单成本高 | 接到外地线索但无法服务 | 严格按城市定向，或明确标注服务区域 |
| 直播工地效果差 | 工地噪音大/画面抖动/没有讲解 | 配备专业直播设备，安排专人讲解 |
| 负面评价影响 | 抖音上出现客户投诉 | 建立售后快速响应机制，公开解决问题 |
| 预算花不出去 | 高客单价导致广告系统跑不出量 | 降低优化目标门槛（先跑私信→再跑留资），先积累数据 |

## 十三、可复制 SOP

### 家装行业本地推 30 天启动 SOP

| 天数 | 动作 | 检查标准 |
|---|---|---|
| D1 | 完成企业号装修，上传 10 条案例/知识视频 | 主页完整可展示 |
| D2 | 创建 5 条计划（案例/避坑/活动/直播/口碑），各 100 元 | 计划审核通过开始消耗 |
| D3-5 | 观察消耗和私信量，调整出价和定向 | 日私信 ≥ 5 条 |
| D7 | 首场工地直播，投放引流 | 直播留资 ≥ 3 条 |
| D10 | 替换 CTR < 1.5% 的素材，新增 5 条素材 | 累计私信 ≥ 50 条 |
| D14 | 汇总两周数据，计算各渠道线索成本和量房率 | 量房预约率 ≥ 15% |
| D21 | 开启楼盘定向，针对新交付楼盘投放 | 楼盘线索占比 ≥ 20% |
| D30 | 全面复盘，调整下月预算分配和素材策略 | 月签约 ≥ 2 单，ROI ≥ 1:5 |

## 十四、适不适合投流

### 适合投流的情况
- **整装/全案公司**：客单价 10-50 万，单笔利润足以支撑投流成本，**强烈推荐**
- **全屋定制品牌**：标准化程度高，客单价 3-20 万，展示效果直观，**强烈推荐**
- **局部改造/翻新**（厨卫/墙面/阳台）：客单价 1-5 万，需求量大决策快，**推荐**
- **设计工作室**：注重设计感的客户愿意为设计付费，客单价高，**推荐**
- **新房集中交付城市**：楼盘定向精准获客，线索质量极高，**强烈推荐**

### 不适合投流的情况
- **纯施工队（无设计能力）**：客户需要的是设计+施工一体化，纯施工在抖音上难以建立信任，**建议先提升展示能力**
- **没有案例积累的初创公司**：没有完工案例可展示，素材质量差，**建议先做3-5个标杆案例**
- **超低价游击队**：客单价 2-3 万，利润空间薄，获客成本占比过高，**投流ROI难做正**
- **没有销售跟进团队**：线索留资后无人及时联系，**必须先建立跟进机制**
- **只做高端定制（100万+）**：目标客户极小众，本地推难以精准触达，**更适合私域/圈层营销**

### 投流前自检清单
- [ ] 是否有 5 个以上完整完工案例可拍摄？
- [ ] 是否有专业的量房+设计团队？
- [ ] 是否能在 30 分钟内响应线索？
- [ ] 是否有 CRM 系统管理线索跟进流程？
- [ ] 是否能承受 30 天的冷启动期？
- [ ] 是否有合规的营业执照和施工资质？

全部勾选"是"即可开始投流。家装是本地推 ROI 最高的行业之一，但前提是有完善的承接和转化能力。
