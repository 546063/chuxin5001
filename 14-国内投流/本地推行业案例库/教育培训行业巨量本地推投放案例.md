# 教育培训行业巨量本地推投放案例

> 类型：模拟案例/可复用模板
> 适用：素质教育（美术/音乐/体育/编程）、职业技能培训、成人教育、托管辅导、驾校
> 最后更新：2026-05-22

---

## 一、行业场景与AD下线影响

### 教培行业核心特征

- **强地域性**：服务半径 3-8 公里，超出范围家长/学员不愿来
- **决策周期长**：素质教育 1-4 周，职业技能 1-3 周，成人教育 3-7 天
- **决策者多元**：素质教育由家长决策（25-45 岁），成人/职教由本人决策（18-40 岁）
- **试听课是核心转化节点**：免费体验课 → 正价课，转化率 15-30%
- **政策敏感**：双减政策影响 K12 学科培训，素质教育和职教合规可投

### AD 下线后的迁移策略

巨量 AD 线索广告逐步下线，教培行业必须迁移至本地推，关键差异如下：

| 维度 | AD 线索广告（已下线） | 巨量本地推（当前） |
|---|---|---|
| 优化目标 | 表单提交/私信 | 私信留资/表单提交/团购成单 |
| 流量来源 | 信息流为主 | 信息流+POI+搜索+同城页 |
| 转化链路 | 广告→表单→跟进 | 短视频/直播→私信/表单→体验课→正价课 |
| 定向精度 | 人群包+Lookalike | LBS+兴趣+搜索词+达人粉丝 |
| 素材形式 | 图文/短视频 | 短视频为主，图文辅助 |
| 合规要求 | 较宽松 | 更严格，禁用"保过""包就业"等承诺 |

### 迁移时间线建议

```
第1周：搭建本地推账户，完成 POI 认领，上线企业号装修
第2周：上传 5-10 条短视频素材，创建 3-5 条计划，冷启动跑量
第3周：跑通"短视频→私信→体验课预约"链路，积累首批线索
第4周：对比 AD 历史数据，调整出价和定向，优化线索质量
```

## 七、短视频脚本方向

### 脚本模板 1：素质教育——学员成长型（30秒）

```
0-3s 钩子（视觉冲击）："学画画6个月，从涂鸦到画出这幅画，她只用了一个方法"
3-8s 对比展示：左边放6个月前的涂鸦，右边放现在完成的画作（BGM渐强）
8-15s 过程快剪：孩子上课时专注的画面、老师指导的画面（3-4个快剪镜头）
15-22s 家长证言："刚开始只是抱着试试的心态，没想到孩子真的爱上了画画，每周都盼着来上课"
22-27s 机构展示：明亮的教室、专业的画具、其他学员作品墙
27-30s 引导行动："XX美术，现在免费体验课开放预约中，点左下角报名，名额有限"
```

### 脚本模板 2：职业技能——就业结果型（25秒）

```
0-3s 钩子（数字冲击）："28岁从工厂辞职学UI设计，现在月薪翻了三倍"
3-8s 人物介绍：小张，之前在某工厂做质检，月薪4000，每天重复劳动
8-15s 学习过程：课堂实录+软件操作画面+老师一对一指导（快剪3个镜头）
15-20s 结果展示：现在在某互联网公司做UI设计师，月薪12000+五险一金
20-25s 引导行动："XX职业培训，0基础也能学，现在报名享8折，点下方咨询"
```

### 脚本模板 3：成人教育——痛点解决型（20秒）

```
0-3s 痛点共鸣："工作5年还没升上去？可能差的就是这个证"
3-8s 数据支撑："XX行业80%的管理岗要求具备XX证书，你有吗？"
8-13s 解决方案：展示线上+线下混合学习模式，"上班也能学，周末上课+平时线上复习"
13-17s 通过率："去年学员通过率92%，高于全国平均25个百分点"
17-20s 引导："现在报名立减500，点下方获取免费备考资料"
```

### 脚本模板 4：驾校——对比体验型（30秒）

```
0-3s 钩子："在XX学车最怕遇到什么？暴躁教练+破旧车辆+排队两小时"
3-10s 痛点展示：行业乱象快剪（可用素材/动画演示，不拍其他驾校实景）
10-20s 差异化：XX驾校的 1对1教学+全新教练车+随到随学+空调车
20-25s 承诺："不过免费重学，教练不满意随时换"
25-30s 引导："现在报名立减800，还送模拟考试，点左下角抢名额"
```

### 脚本模板 5：亲子体验活动型（25秒）

```
0-3s 钩子："周末带孩子去哪？XX机构免费亲子体验日来了"
3-10s 活动展示：上周末体验日现场，家长和孩子一起动手做手工/画画/做实验
10-18s 价值传递："不只是玩，更是发现孩子的兴趣和天赋"
18-22s 师资展示：专业教师团队介绍（姓名+资质+教龄）
22-25s 引导："本周末最后一场，仅剩15个名额，点下方免费预约"
```

## 十一、复盘优化动作

1. **素材疲劳检测**：每条素材连续投放 3 天后 CTR 下降超过 30%，立即替换新素材。建立素材库，每周产出 3-5 条新素材。
2. **线索质量追踪**：标记每条线索的最终状态（未接通/到场/未转化/转化），反推高质量线索的素材和定向特征，加大投入。
3. **时段优化**：教育类线索高峰在 10:00-12:00 和 20:00-22:00，这两个时段提高出价 20%，其他时段降低出价或暂停。
4. **表单字段精简**：字段从 6 个减少到 4 个，留资率提升 35%。非必要字段（如收入、职业）放到顾问电话跟进时再收集。
5. **私信话术A/B测试**：测试"发送资料型"vs"直接邀约型"自动回复，通常邀约型到场率高 15-20%。
6. **搜索词扩量**：每周分析搜索定向报告，新增 CTR > 2% 的搜索词，剔除 CTR < 0.5% 的词。
7. **体验课到场提升**：增加"体验课前2小时微信提醒+当天上午电话确认"环节，到场率从 35% 提升到 55%。
8. **老客转介绍机制**：正价课学员推荐新学员报名，双方各得 200 元优惠券。转介绍线索成本比投流低 60%，转化率高三倍。
9. **地域扩缩测试**：将定向半径从 5km 逐步扩大到 8km、10km，观察线索成本变化。超过 8km 后线索成本通常上涨 40-60%，到场率下降 25%。
10. **直播+短视频组合**：每周 2-3 场直播（公开课/答疑），同时投放短视频引流到直播间。直播线索质量通常比短视频高 20-30%。
11. **竞品截流策略**：定向投放到竞品机构 POI 周边 1km，素材强调自身差异化优势（师资/价格/距离/环境）。
12. **节点营销节奏**：开学季（2月/9月）、寒暑假前（1月/6月）提前 15 天开始投放，预算提高 50-100%，素材围绕"新学期新开始"主题。

## 十二、常见坑

| 坑 | 表现 | 解决方案 |
|---|---|---|
| 线索多但到场率低 | 留资后不接电话/放鸽子 | 表单增加"确认愿意接听电话"勾选，留资后5分钟内首次联系 |
| AD迁移后量级下降 | 本地推线索量只有AD的50% | 增加直播渠道+搜索定向+达人定向，多渠道互补 |
| 素材审核不过 | "包过""包就业""保过"等承诺词 | 改用"帮助""提升""通过率参考数据"，避免绝对化用语 |
| K12学科培训违规 | 违反双减政策，广告被拒 | 严格只做素质教育/职业教育/成人教育，不涉及学科辅导 |
| 线索重复跟进 | 多个顾问同时联系同一线索 | 建立 CRM 系统，线索自动去重分配，24小时保护期 |
| 预算花不完 | 定向过窄/出价过低/素材不吸睛 | 先放宽定向、提高出价、检查素材质量 |
| 线索成本低但质量差 | 大量无效号码/非目标人群 | 增加投资预算字段、设置最低消费门槛、排除学生标签 |

## 十四、适不适合投流

### 适合投流的情况
- **素质教育类**（美术/音乐/舞蹈/编程/体育）：合规可投，家长决策明确，体验课模式成熟，**强烈推荐**
- **职业技能培训**（IT设计/电商运营/烘焙/汽修/美业）：高客单价支撑投流成本，就业结果可展示，**强烈推荐**
- **成人教育**（学历提升/考证培训/语言培训）：需求刚性，决策快，线索量大，**推荐**
- **驾校**：地域性强、需求稳定、竞争激励，适合本地推精准获客，**推荐**
- **托管/早教/托育**：家长关注距离和安全性，适合 POI 定向投放，**推荐**

### 不适合投流的情况
- **K12 学科培训**：双减政策限制，广告审核严格，有合规风险，**不建议投流**
- **纯线上教育**：本地推的 LBS 定向优势无法发挥，更适合信息流广告或其他渠道，**不建议用本地推**
- **超低价课程**（9.9元以下）：获客成本可能超过课程本身价值，ROI 难以做正，**需谨慎评估**
- **没有承接能力的机构**：留资后无法及时跟进（5分钟内首次联系）、没有体验课排期、没有销售团队转化，投流=浪费钱，**先建设承接能力再投流**

- [ ] 是否有合规的营业执照和办学许可？
- [ ] 是否有完整的 POI 门店信息（地址、电话、营业时间）？
- [ ] 是否能提供有价值的体验课/试听内容？
- [ ] 是否安排了专人跟进线索（5分钟内首次联系）？
- [ ] 是否有体验课→正价课的转化 SOP？
- [ ] 是否能承受至少 14 天的冷启动期？

如果以上全部勾选"是"，可以立即开始本地推投放。如果有一项以上为"否"，建议先补齐再投流。
