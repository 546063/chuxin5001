# 除甲醛行业巨量本地推投放案例

> 类型：模拟案例/可复用模板
> 适用：除甲醛公司、室内空气治理、新房除醛、汽车除醛
> 最后更新：2026-05-22

---

## 一、行业场景

除甲醛行业核心特征：
- **强季节性**：春季（3-5月）和秋季（9-11月）是装修旺季，需求翻倍
- **焦虑驱动**：有孕妇/婴幼儿的家庭焦虑度极高，决策快
- **信息不对称**：客户不懂检测标准/治理方法，容易被忽悠
- **中等客单**：家庭治理 1500-8000 元，办公室 3000-20000 元
- **效果可量化**：治理前后甲醛检测数据对比是最好的转化素材
- **口碑传播**：效果好的客户会推荐给同小区邻居

## 十、转化链路

```
短视频曝光（焦虑/科普/案例）
  → 点击免费检测券
    → 预约上门时间
      → 工程师上门检测
        → 出具检测报告+治理方案
          → 现场签约治理
            → 施工+复检
              → 好评/转介绍
```

**关键指标**：
- 曝光→点击率：≥ 2.5%
- 点击→领券率：≥ 10%
- 领券→预约率：≥ 60%
- 上门→签约率：≥ 30%
- 转介绍率：≥ 15%

## 十一、常见问题

| 问题 | 原因 | 解决方案 |
|---|---|---|
| 旺季跑量不够 | 预算不足/素材少 | 提前1个月准备素材，旺季预算翻倍 |
| 领券不预约 | 无紧迫感 | 券有效期缩短至7天，到期前短信提醒 |
| 上门不签约 | 报价过高/信任不足 | 现场出CMA检测报告，用数据说话 |
| 审核不过 | "100%去除""零甲醛"等绝对化用语 | 改用"达标""降至国标以下" |
| 复购率低 | 一次性服务 | 推年度会员（每年免费复检2次） |
