# 口腔门诊行业巨量本地推投放案例

> 类型：模拟案例/可复用模板
> 适用：口腔门诊/牙科诊所/口腔医院，面向同城用户分项目（种植/正畸/儿牙/综合）收集线索
> 最后更新：2026-05-22

---

## 一句话总结
口腔门诊通过巨量本地推的核心打法是：分项目独立投放（种植/正畸/儿牙/综合），高客单价项目用深度咨询表单承接，低门槛项目用体验团购引流，严格把控医疗广告合规，单条线索成本 50-300 元不等（因项目差异大），到院成交 ROI 可达 1:5-1:20。

## 行业场景

**行业特点：**
- 项目差异极大：洗牙/补牙（50-500 元）到种植/全口重建（5000-50000 元）
- 决策周期分层：洗牙即时决策，种植牙决策周期 1-3 个月
- 高度依赖信任：用户需要信任医生技术、诊所资质
- 强地域属性：服务半径 5-15 公里，跨区意愿低

**获客难点：**
- 医疗广告审核严格，资质不全无法投放
- 种植牙/正畸等高客单价项目线索成本高（150-300 元/条）
- 用户决策顾虑多：价格不透明、担心疼痛、对效果不确定
- 到院率偏低（15-30%），大量线索流失在电话沟通环节

**决策链路（分项目）：**
- **洗牙/补牙**：刷到视频 -> 点击团购 -> 购买 -> 预约到店
- **种植牙**：刷到科普/案例 -> 留资咨询 -> 电话沟通 -> 到院拍片检查 -> 制定方案 -> 成交
- **正畸**：刷到案例/对比 -> 留资咨询 -> 到院面诊 -> 方案设计 -> 成交
- **儿牙**：家长刷到科普 -> 留资/团购 -> 预约到店

## 投放目标

| 阶段 | 目标 | 优化方向 | 关键指标 |
|------|------|---------|---------|
| 第 1-2 周（测试期） | 测试各项目素材 + 定向 | 线索收集/团购购买 | CPM ≤ 35，CTR ≥ 1.5%，CPL ≤ 150 |
| 第 3-4 周（放量期） | 聚焦高 ROI 项目扩量 | 深度线索/到院预约 | CPM ≤ 30，CTR ≥ 2%，CPL ≤ 120，到院率 ≥ 20% |
| 第 5 周+（稳定期） | ROI 优化 + 项目组合 | 到院成交/升单 | 到院率 ≥ 25%，成交率 ≥ 20%，ROI ≥ 1:5 |

**结构要点：**
- 低门槛引流占预算 20-30%，用于扩大品牌曝光和积累基础客流
- 种植/正畸占预算 50-60%，是高 ROI 核心项目
- 医生 IP 占 10-15%，用于长期信任建设
- 各项目独立计划，不可混投（受众、素材、承接方式完全不同）

## 数据指标

| 指标 | 优秀 | 合格 | 预警 |
|------|------|------|------|
| CPM（千次曝光成本） | ≤ 25 元 | 25-40 元 | > 40 元 |
| CTR（点击率） | ≥ 2.5% | 1.5-2.5% | < 1.5% |
| CPL（单条线索成本） | ≤ 80 元 | 80-150 元 | > 150 元 |
| 线索有效率 | ≥ 65% | 45-65% | < 45% |
| 到院率 | ≥ 25% | 15-25% | < 15% |
| 成交转化率 | ≥ 20% | 12-20% | < 12% |
| ROI | ≥ 1:8 | 1:5-1:8 | < 1:5 |

**关键指标说明：**
- 口腔行业 CPL 因项目差异极大，不可用一个 CPL 标准衡量所有项目
- 种植/正畸等高价项目 CPL 可接受 150-300 元，但需关注成交后 ROI
- 线索有效率 = 真实有效线索 / 总线索，排除空号/打错/无意向
- ROI 计算：成交总金额（含后续复诊/维护消费）/ 广告总花费

## 复盘优化动作

1. **分项目独立优化**：种植/正畸/洗牙/儿牙分开建计划、分开看数据，不可混为一谈
2. **素材合规自查**：每次上新素材前核对医疗广告审查标准，避免"最有效""根治""保证"等违规用语
3. **线索跟进提速**：表单提交后 10 分钟内未致电，线索有效率下降 30%+，设置专人值守
4. **到院率优化**：到院率低于 20% 时，排查：电话话术问题/预约时间不灵活/地址不清晰
5. **定向扩展**：跑量计划中，逐步扩展定向半径（5km -> 8km -> 10km），观察 CPL 变化
6. **医生 IP 建设**：每周产出 2-3 条医生科普视频，长期积累信任资产
7. **落地页 A/B 测试**：测试"价格透明" vs "医生资质"哪个首屏转化更高
8. **团购品迭代**：洗牙体验价测试不同价格带（39.9/59.9/79.9），找到最佳转化点
9. **深度回传**：接入到院/成交数据回传巨量引擎，让系统学习高价值人群特征

## 常见坑

1. **资质不全被拒审**：缺少《医疗机构执业许可证》或医生资质不全，账户无法过审
2. **素材违规被罚**：使用患者治疗前后对比过度展示、使用绝对化用语，导致限流/封户
3. **线索跟进不及时**：口腔线索时效性极强，1 小时内未联系基本流失
4. **价格不透明导致信任崩塌**：线上报低价到院后加价，差评传播影响后续投放
5. **混投不同项目**：种植和洗牙放在同一个计划里投，定向和素材完全不匹配
6. **忽略子女代决策**：种植牙大量是子女帮父母咨询，需单独建计划定向 30-50 岁人群
7. **到院后体验差**：排队时间长/环境差/医生敷衍，导致差评和低复购
8. **未设置表单去重**：同一用户重复提交，浪费线索成本
9. **只看 CPL 不看 ROI**：洗牙线索便宜但客单低，种植线索贵但客单高，需综合评估

## 适不适合投流

**适合投流的口腔门诊：**
- 具备完整医疗资质（《医疗机构执业许可证》+ 医师执业证 + 广告审查批准文件）
- 有专职客服/咨询团队能快速响应线索（10 分钟内致电）
- 医生有丰富经验且愿意出镜做科普（建立信任的核心）
- 价格体系透明（线上报价与到院一致，不存在隐形消费）
- 日均接待能力 15-30 人，不会因爆单导致体验下降
- 位于交通便利区域（地铁/公交/停车方便）

**不适合投流的口腔门诊：**
- 资质不全或无法提供医疗广告审查批准文件
- 无线索承接能力（表单提交后无人跟进或隔天回复）
- 价格不透明（线上低价引流，到院后大幅加价）
- 医生不出镜、不做科普（缺乏信任建设能力）
- 周边 3 公里内竞品超过 8 家且自身无明显差异化优势
- 预算极低（日预算 < 200 元，难以在高竞争的口腔品类中跑出有效数据）
- 以低价洗牙为唯一盈利模式（客单太低无法覆盖获客成本）

**合规特别提醒：**
- 口腔医疗广告需取得《医疗广告审查证明》后方可投放
- 不得使用患者治疗前后对比的过度展示（尤其种植牙、正畸）
- 不得使用"最先进技术""100%成功""无痛""根治"等绝对化或保证性用语
- 涉及药品/器械需标注批准文号和适用范围
- 所有素材上线前必须由法务或合规人员审核签字
- 建议在落地页显著位置标注"广告"字样和医疗广告审查批准文号
