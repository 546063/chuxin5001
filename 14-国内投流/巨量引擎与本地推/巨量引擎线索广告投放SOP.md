# 巨量引擎线索广告投放SOP

> 适用范围：巨量引擎/巨量本地推 线索类广告
> 最后更新：2026-05-22

---

## 一、三种线索获取方式对比

| 方式 | 优势 | 劣势 | 适用行业 |
|---|---|---|---|
| 表单留资 | 信息结构化、方便CRM管理 | 填表门槛高、量偏少 | 消防/教培/家装/B2B |
| 私信咨询 | 互动感强、可实时沟通 | 需要人工回复、时效要求高 | 医美/美业/餐饮/零售 |
| 电话拨打 | 即时沟通、意向度高 | 无人接听率高、需要客服在线 | 消防急单/除虫/汽修 |

### 选择决策树
```
客户预算 > 500元/单？ → 是 → 表单留资（精准）
                    → 否 → 私信咨询（量大）

客户需要即时响应？ → 是 → 电话拨打/私信
                    → 否 → 表单留资

客户行业有资质审核？ → 是 → 表单留资（过滤+留档）
                      → 否 → 三种都可以
```

## 二、表单线索投放

### 表单字段设计

| 阶段 | 字段数 | 字段内容 | 预期转化率 |
|---|---|---|---|
| 测试期 | 2-3个 | 姓名+电话+（可选城市） | 8-15% |
| 成熟期 | 4-5个 | 姓名+电话+项目类型+面积+预算 | 5-10% |
| 放量期 | 3-4个 | 姓名+电话+意向+时间 | 6-12% |

### 字段设计原则
1. **必填字段不超过3个** — 每多一个字段，转化率下降 20-30%
2. **下拉选项代替文本输入** — 降低填写难度
3. **隐私声明必须包含** — "我们不会将您的信息分享给第三方"
4. **提交按钮要有吸引力** — 用"免费获取方案"而非"提交"

### 表单页面优化
- 首屏展示：标题+表单+行动按钮（不要滚动就能看到）
- 信任元素：资质证书/客户数量/合作品牌 logo
- 社会证明：客户评价/案例数量/好评截图
- 紧迫感：限时优惠/每日名额限制

## 三、私信线索投放

### 自动回复话术模板

**首次触发（用户点击广告进入私信）：**
```
您好！欢迎来到[品牌名]~ 👋
我们是[城市]专业的[服务]公司，已服务[数字]+客户。

请问您想了解哪方面？
回复数字获取：
1️⃣ 免费上门检测
2️⃣ 服务报价参考
3️⃣ 客户案例展示
4️⃣ 直接人工咨询
```

**用户回复后的承接话术：**
```
好的！为了给您精准报价，需要了解：
📍 您的位置（区域即可）
📐 面积/规模大概是多少？
⏰ 您希望什么时候处理？

回复后我们会在5分钟内给您详细方案~
```

### 私信运营要点
| 指标 | 标准 | 说明 |
|---|---|---|
| 首响时间 | ≤ 5分钟 | 超过30分钟流失率 70%+ |
| 工作时间 | 8:00-22:00 | 建议覆盖12小时 |
| 非工作时间 | AI自动回复+次日跟进 | 不放过夜间线索 |
| 私信→留资率 | ≥ 30% | 目标获取电话/微信 |
| 留资→接通率 | ≥ 80% | 当场回电最佳 |

## 四、电话线索投放

### 适用条件
- 行业需求紧急（除虫/消防整改/管道疏通）
- 客单价低、决策快（< 500 元）
- 客服团队能即时接听

### 接听SOP
```
响铃 → 3声内接听
开场 → "[公司名]您好，请问有什么可以帮您？"
记录 → 需求类型+位置+紧急程度
报价 → 给出价格范围（不要一口价）
预约 → 确认上门时间+发送确认短信
跟进 → 未成交客户24小时内回访
```

## 五、微信承接

### 导流路径
```
广告 → 表单/私信 → 获取微信号/手机号
  → 添加微信 → 发送品牌介绍+案例
    → 语音/视频沟通 → 发送报价方案
      → 成交
```

### 微信承接话术
**添加好友通过后：**
```
[客户称呼]您好！我是[品牌名]的[姓名]~
感谢您通过抖音联系我们！

这是我们整理的[行业]方案参考，您先看一下👇
[发送PDF/图片方案]

有什么疑问随时问我，我看到会第一时间回复~
```

### 微信运营要点
| 动作 | 时间 | 内容 |
|---|---|---|
| 通过后第1天 | 即时 | 发送方案+自我介绍 |
| 第3天 | 跟进 | "方案看了吗？有什么疑问？" |
| 第7天 | 价值 | 发送案例/客户好评 |
| 第14天 | 促单 | "本月优惠最后X天" |
| 第30天 | 唤醒 | "最近有[行业]新活动" |

## 六、有效线索判断

### 有效线索标准（BANT模型）

| 维度 | 有效标准 | 无效标准 |
|---|---|---|
| Budget 预算 | 明确预算范围或能接受报价 | 只问价格、对比多家 |
| Authority 决策权 | 本人是决策人或能影响决策 | 只是帮别人问问 |
| Need 需求 | 有明确需求和痛点 | 随便看看、无明确需求 |
| Timeline 时间 | 有明确时间计划（1个月内） | "先了解着，以后再说" |

### 线索分级
| 等级 | 特征 | 跟进方式 |
|---|---|---|
| A级（高意向） | 4项全满足 | 30分钟内回电，当天出方案 |
| B级（中意向） | 满足3项 | 2小时内回电，3天内跟进 |
| C级（低意向） | 满足1-2项 | 纳入培育池，每周触达 |
| D级（无效） | 0项满足 | 标记无效，不再跟进 |

## 七、无效线索过滤

### 源头过滤（表单层面）
1. **增加"预算范围"字段** — 过滤只看价格的人
2. **增加"项目紧急程度"** — 过滤不急的人
3. **设置最小面积/规模** — 过滤太小的客户
4. **添加验证码** — 防止机器人提交

### 过程过滤（跟进层面）
1. **首次回电时快速判断** — 3个问题确认意向
2. **标记无效原因** — 方便后续分析
3. **设置"冷却期"** — 无效线索30天内不再触达
4. **黑名单机制** — 恶意/同行线索永久排除

### 常见无效线索类型
| 类型 | 占比 | 过滤方式 |
|---|---|---|
| 同行询价 | 10-20% | 增加公司名验证字段 |
| 学生/无预算 | 5-10% | 增加预算字段 |
| 重复提交 | 5-15% | 去重机制（手机号唯一） |
| 机器人/误触 | 3-8% | 验证码+人工确认 |
| 只问不买 | 15-25% | 跟进时效+意向判断 |

## 八、线索成本优化

### 成本过高的排查路径
```
线索成本高
├── CPC高 → 检查素材CTR（低于1.5%换素材）
│   ├── CTR正常 → 检查CVR（低于5%优化表单/私信）
│   └── CTR低 → 换素材/换定向
└── CPC正常 → CVR低
    ├── 表单转化低 → 减少字段/优化页面
    ├── 私信转化低 → 优化自动回复/提高首响
    └── 电话转化低 → 检查接听率/话术
```

### 优化动作优先级
| 优先级 | 动作 | 预期降幅 |
|---|---|---|
| P0 | 替换低CTR素材 | 20-40% |
| P0 | 优化表单字段数量 | 15-30% |
| P1 | 调整定向范围 | 10-25% |
| P1 | 优化私信自动回复 | 10-20% |
| P2 | 调整出价策略 | 5-15% |
| P2 | 优化落地页内容 | 10-20% |
