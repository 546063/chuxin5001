# 巨量本地推通用方法论

## 一句话总结

巨量本地推的底层逻辑是以"核销"为最终目标，通过精准地域定向和算法优化，实现从曝光到到店核销的短链路高效转化。

## 核心结论

- 本地推链路：曝光 → 团购下单 → 到店核销（短链路，可追踪）
- 传统广告链路：曝光 → 点击 → 跳转 → 购买（链路长，流失大）
- 三大优势：精准地域、闭环追踪、算法优化
- 四大工具选择：根据业务类型选择本地推/千川/DOU+/巨量引擎AD

## 适用场景

- 选择正确的巨量投放工具
- 制定投放全链路SOP
- 管理计划生命周期
- 优化ROI和核销率

## 具体内容

### 模块1：四大投流工具对照表

| 维度 | 巨量本地推 | 巨量千川 | DOU+ | 巨量引擎 AD |
|------|----------|---------|------|-----------|
| **核心目标** | 门店到店/团购核销 | 电商成交（商品购买） | 视频加热/账号涨粉 | 品牌曝光/线索获取 |
| **适用主体** | 实体门店（需抖音来客） | 电商卖家（抖店） | 所有抖音用户 | 企业/品牌 |
| **地域限制** | 有（3-20公里） | 无（全国） | 无 | 可选 |
| **转化链路** | 短视频/直播 → 团购 → 核销 | 短视频/直播 → 下单 → 发货 | 无转化要求 | 表单/私信/下载 |
| **计费方式** | OCPM | OCPM | 按展示/按播放 | CPC/OCPM |
| **最低门槛** | 口碑分≥4.0 | 抖店已开通 | 无 | 营业执照 |
| **智能程度** | 高 | 最高（电商全链路） | 低 | 中高 |
| **适合行业** | 餐饮/美业/教培/医疗/本地生活 | 电商全品类 | 内容创作者 | 全行业 |

### 工具选择决策树

```
你要投的是什么？
├── 实体门店到店消费
│     ├── 有抖音来客门店 + 口碑分≥4.0
│     │     → 巨量本地推 ✅
│     └── 没有门店
│           → 先开抖音来客
│
├── 电商卖货（发快递）
│     └── 巨量千川 ✅
│
├── 视频加热/账号涨粉
│     └── DOU+ ✅
│
└── 品牌全国曝光/APP下载
      └── 巨量引擎 AD ✅
```

### 模块2：投放全链路 SOP

**从0到投放的7步流程**：

1. **资质准备（1-3天）**：营业执照 + 行业许可证 → 开通抖音来客 + 认领门店 → 确认口碑分≥4.0
2. **团购商品设计（1-2天）**：设计引流款（低价体验）+ 利润款（正价套餐）+ 确认核销流程
3. **素材制作（2-3天）**：拍摄门店实景 → 制作 5-10 条短视频 → 准备 3 套标题文案
4. **建计划（当天）**：选择推广目标 → 设置定向（3-5公里+兴趣标签）→ 设置预算（目标成本×20）→ 上传素材
5. **冷启动观察（7天）**：Day1-3完全不调整 → Day4-7轻微优化（<10%）→ 重点看曝光→点击→团购→核销漏斗
6. **放量（7天后）**：复制跑量计划 → 开启智能放量 → 预算+20%逐步扩展
7. **持续优化（日常）**：每日监控核心指标 → 每周素材更新3-5条 → 每月策略复盘

**计划结构最佳实践**：

| 账户规模 | 在投计划数 | 日更新计划 | 素材储备 |
|---------|-----------|-----------|---------|
| 单店小预算 | 5-10 条 | 1-2 条 | 5-10 条 |
| 单店中预算 | 15-25 条 | 3-5 条 | 15-20 条 |
| 多店/连锁 | 30-50 条 | 5-10 条 | 30-50 条 |

### 模块3：投放生命周期管理

**计划生命周期四阶段**：

```
新建 → 冷启动 → 稳定跑量 → 衰退
  │       │          │          │
  当天    7天       14-30天    30天+
  建计划  观察      放量       替换
```

| 阶段 | 特征 | 核心动作 | 禁忌 |
|------|------|---------|------|
| **新建** | 刚创建 | 确保审核通过、素材正常 | 不要建完就改参数 |
| **冷启动** | 7天内 | 保持稳定，每天看数据 | 不要调出价/定向/预算 |
| **稳定跑量** | CPA 稳定 | 复制计划 + 智能放量 | 不要一次改多个参数 |
| **衰退** | 消耗下降30%+ | 建新计划接力 | 不要无限制加价抢救 |

**计划衰退预警信号**：

| 信号 | 阈值 | 行动 |
|------|------|------|
| 日消耗连续3天下降 | 降幅 > 30% | 建 2-3 条新计划 |
| CPA 连续3天上升 | 涨幅 > 20% | 换素材或调整定向 |
| CTR 下降 | 降幅 > 25% | 换封面/标题 |
| 团购下单量下降 | 降幅 > 30% | 检查竞品是否有新活动 |

**计划接力 SOP**：

当主力计划出现衰退信号时：
1. 不要关停原计划（还能跑一些量）
2. 新建 2-3 条计划：
   - 计划A：相同定向 + 新素材
   - 计划B：微调定向（扩大 2 公里）+ 相同素材
   - 计划C：相同定向 + 相同素材 + 提高出价 10%
3. 新计划预算 = 原计划的 30-50%
4. 新计划跑出量后，逐步降低原计划预算
5. 原计划消耗 < 新计划 20% 时可关停

### 模块4：ROI 优化体系

**ROI 拆解公式**：

```
ROI = GMV / 消耗
    = (订单量 × 客单价) / 消耗
    = 客单价 / CPA

提升 ROI 的 3 个杠杆：
1. 提高客单价（套餐升级/组合销售）
2. 降低 CPA（优化素材/定向/出价）
3. 提高核销率（优化团购商品/到店体验）
```

**核销率优化清单**：

| 环节 | 问题 | 解法 |
|------|------|------|
| **购买前** | 团购描述不清晰 | 详细描述套餐内容 + 使用规则 |
| **购买后** | 忘记使用 | 短信提醒 + 到期前 3 天推送 |
| **预约时** | 预约流程复杂 | 一键预约 + 电话直拨 |
| **到店时** | 排队时间长 | 分时段预约 + 到店即享 |
| **体验后** | 体验差导致差评 | 服务培训 + 评价引导 |

**不同行业的健康 ROI 标准**：

| 行业 | 健康 ROI | 优秀 ROI | 说明 |
|------|---------|---------|------|
| 餐饮 | 1.5-2.5 | > 3.0 | 低客单靠复购和口碑 |
| 美业 | 2.0-4.0 | > 5.0 | 高毛利可承受高 CPA |
| 教培 | 3.0-8.0 | > 10.0 | 看长期 LTV（续课率） |
| 家装 | 5.0-15.0 | > 20.0 | 高客单，ROI 自然高 |
| 医疗 | 3.0-10.0 | > 15.0 | 高客单但合规成本高 |
| 健身 | 2.0-5.0 | > 6.0 | 看会员卡续费 |

### 模块5：素材策略框架

**本地推素材 vs 电商素材的差异**：

| 维度 | 本地推素材 | 电商素材 |
|------|----------|---------|
| **核心目标** | 引导到店 | 引导下单 |
| **内容风格** | 种草 > 硬广 | 直接转化 |
| **关键信息** | 门店位置 + 环境 + 套餐 | 产品功能 + 价格 |
| **时长偏好** | 15-30 秒 | 15-60 秒 |
| **信任元素** | 门店实拍 + 真实顾客 | 用户评价 + 销量 |

**素材生产流水线**：

```
选题（竞品分析/热点/门店特色）
  ↓
脚本（Hook + 展示 + CTA 三段式）
  ↓
拍摄（门店实景/顾客体验/产品展示）
  ↓
剪辑（字幕 + BGM + 封面）
  ↓
上传 + A/B 测试
  ↓
筛选 WINNER → 裂变（改封面/标题/BGM）
```

**素材数据标准**：

| 指标 | 合格 | 优秀 | 差 |
|------|------|------|-----|
| 前3秒留存 | > 40% | > 60% | < 30% |
| 完播率 | > 30% | > 50% | < 20% |
| CTR | > 3% | > 5% | < 2% |
| 团购点击率 | > 5% | > 10% | < 3% |

### 模块6：数据监控体系

**三层数据看板**：

| 层级 | 指标 | 查看频率 | 异常阈值 |
|------|------|---------|---------|
| **实时层** | 曝光量/消耗/CTR | 每小时 | 消耗突降 50%+ |
| **日度层** | CPA/ROI/核销率 | 每天 | CPA > 目标 × 1.5 |
| **周度层** | 素材效果/人群画像/计划生命周期 | 每周 | Top 计划衰退 |

**每日5分钟晨检流程**：

1. 看总消耗 → 是否正常花费（70-100% 预算）
2. 看 ROI → 是否在健康范围
3. 看核销率 → 是否 < 50%（预警）
4. 看计划状态 → 是否有计划审核不通过
5. 看竞品动态 → 是否有新活动
