# 巨量本地推高频问题清单

## 一句话总结

120个巨量本地推高频问题速查，覆盖从开户到投放到优化到问题排查的全链路，按14个分类组织。

## 核心结论

- 培训新人重点掌握前50个问题
- 日常参考按分类快速查找解决方案
- 每月补充新的实战问题，每季度更新答案和策略

## 适用场景

- 新人培训教材
- 日常投放问题速查
- 团队内部知识库
- 客户答疑

## 具体内容

### 一、开户与资质类（10个）

| 问题 | 核心答案 |
|------|---------|
| Q1：开户需要准备哪些材料？ | 营业执照、法人身份证正反面、法人手机号和邮箱、对公账户信息。特殊行业需附加资质（教育需办学许可证，医疗需医疗机构执业许可证+广审表） |
| Q2：一个营业执照可以开几个账号？ | 每个营业执照仅支持开通一个本地推账号 |
| Q3：个人可以开通吗？ | 需要通过个体工商户营业执照开通 |
| Q4：审核需要多长时间？ | 正常情况1-3个工作日 |
| Q5：开户被拒常见原因？ | 行业与内容不符、资质材料不清晰、信息填写错误、特殊行业缺少必要许可证 |
| Q6：代理商和直客有什么区别？ | 直客直接与官方合作；代理商提供代投服务，适合缺乏经验的商家 |
| Q7：开户后如何充值？ | PC端首页点击"去充值"，或财务模块→资金钱包→充值 |
| Q8：最低充值金额？ | 通常最低1000元 |
| Q9：充值后可以退款吗？ | 一般不支持退款，建议根据实际消耗分批充值 |

### 二、账户设置与基础操作类（10个）

| 问题 | 核心答案 |
|------|---------|
| Q11：如何绑定抖音来客门店？ | 在抖音来客APP中认领或创建门店，确保门店信息与营业执照一致 |
| Q12：带货口碑分要求？ | ≥4.6分无限制；4-4.6分每日限制投放单量；<4分禁止投放 |
| Q13：48小时发货率要求？ | 需≥70%，低于该指标当日广告权限会被关闭 |
| Q14：如何设置自动投放模式？ | 创建计划时选择「自动控制」模式 |
| Q15：一个账户可以投多个门店吗？ | 可以，需要每个门店都完成抖音来客认领 |

### 三、出价与预算类（10个）

| 问题 | 核心答案 |
|------|---------|
| Q21：新计划预算设置多少？ | 目标成本×20，例如目标100元/单→日预算≥2000元 |
| Q22：出价应该参考系统建议吗？ | 新账户前期不建议完全参考，积累数据后凭经验估价更精确 |
| Q23：智能放量和控成本投放哪个好？ | 智能放量适合拓展人群，控成本适合成本控制严格场景，两者可配合使用 |
| Q24：出价调整幅度多少？ | 每次不超过10% |
| Q25：出价超过目标价40%会怎样？ | 可能导致成本难以压制 |
| Q26：如何设置分时段投放？ | 计划设置中选择「固定时段投放」，以半小时为维度 |
| Q29：如何分配不同转化目标的预算？ | 核心目标80%，辅助目标少量预算 |

### 四、定向策略类（10个）

| 问题 | 核心答案 |
|------|---------|
| Q31：初始定向范围设置多大？ | 优先3-5公里 |
| Q32：如何选择兴趣标签？ | 叠加行业相关标签，如餐饮加「美食探店」「网红打卡」 |
| Q33：DMP人群包怎么用？ | 冷启动期圈选类目精准人群，配合智能放量拓展相似人群 |
| Q34：系统推荐和自定义定向哪个好？ | 系统推荐适合初期探索，自定义更精准但需要数据积累 |
| Q36：如何排除无效人群？ | 基于负向数据设置排除条件，避免过度排除影响人群规模 |

### 五、素材创作与优化类（10个）

| 问题 | 核心答案 |
|------|---------|
| Q41：素材前5秒怎么设计？ | 设计悬念或钩子内容，快速抓住用户注意力 |
| Q42：视频时长多长合适？ | 竖版15-60秒，关键在前5-10秒 |
| Q43：种草风格和硬广哪个好？ | 本地推更青睐真实、生动的种草类内容 |
| Q44：标题有哪些高转化模板？ | 六大模板：热点法、数字法、共鸣法、利益法、疑问法、对比法 |
| Q45：如何做AB测试？ | 同一计划下创建不同创意对比测试 |
| Q46：素材审核不通过常见原因？ | 虚假宣传、价格描述不规范、联系方式违规、营销属性过强 |
| Q48：素材多久需要更新？ | 通常7-15天，避免素材疲劳 |
| Q49：如何判断素材质量？ | 前5秒完播率>30%、整体完播率>50%、CTR>3% |

### 六、起量与放量类（10个）

| 问题 | 核心答案 |
|------|---------|
| Q51：冷启动期需要多久？ | 通常7天，前3天保持参数稳定 |
| Q52：冷启动期可以调整出价吗？ | 不建议频繁调整，如需调整幅度<10% |
| Q54：计划突然爆量怎么办？ | 检查是否出价过高，设置预算限制，选择控成本投放 |
| Q55：计划衰退有什么征兆？ | 消耗持续下降、成本逐渐上升、ROI持续走低 |
| Q56：衰退计划应该关停还是优化？ | 及时补充新计划承接，原计划可尝试优化但效果有限 |
| Q57：如何复制成功计划？ | 一键复制跑量计划，修改定向或创意分类 |
| Q58：每天应该建多少条计划？ | 跑量较好的账户在投20-30条以上，平均每天更新10-15条 |

### 七、数据监控与优化类（10个）

| 问题 | 核心答案 |
|------|---------|
| Q61：核心监控指标有哪些？ | 曝光量、CTR、CVR、CPM、CPC、CPA、ROI、核销率、口碑分 |
| Q62：CTR多少算合格？ | >3%合格，>5%优秀，<2%需要优化素材 |
| Q63：CVR多少算合格？ | >1%合格，>3%优秀，<0.5%需要优化承接环节 |
| Q64：如何计算ROI？ | ROI = GMV / 消耗，本地生活行业目标ROI在1.5-3之间 |
| Q65：核销率低怎么办？ | 优化团购商品吸引力，加强到店引导，提升门店服务质量 |
| Q69：多久做一次数据复盘？ | 每日查看核心指标，每周全面复盘，每月趋势分析 |

### 八、问题排查与解决类（10个）

| 问题 | 核心答案 |
|------|---------|
| Q71：计划完全没消耗怎么办？ | 检查审核状态、口碑分、定向覆盖、素材质量、出价水平 |
| Q72：消耗不稳定如何解决？ | 拉长观察周期，避免频繁操作，结合直播间节奏优化 |
| Q73：成本高于出价怎么办？ | 优化素材质量和人货场，调整定向策略，适当调低出价 |
| Q75：自然流量下降是付费影响吗？ | 付费流量不影响自然流量，需关注人货场是否兜不住流量 |
| Q76：账户被限制投放怎么办？ | 检查口碑分、体验分、发货率等指标，提升后联系客服解限 |
| Q79：多个代理投同一直播间有影响吗？ | 会有影响，可能互相抢量，建议统一由一家代理投放 |

### 九、进阶策略类（10个）

| 问题 | 核心答案 |
|------|---------|
| Q81：如何利用DMP提升冷启动效果？ | 圈选类目精准人群，在冷启动期提高转化率，加速模型学习 |
| Q83：如何做搜索广告投放？ | 在本地推平台开启搜索直投，配置关键词，通过团购卡承接转化 |
| Q86：短视频和直播如何配合？ | 短视频引流直播间，直播转化沉淀用户，形成流量闭环 |
| Q87：如何做多门店联动投放？ | 统一品牌形象，分门店设置定向，数据集中分析，策略协同优化 |

### 十、行业特定类（10个）

| 问题 | 核心答案 |
|------|---------|
| Q91：餐饮行业投放重点？ | 突出菜品特色和用餐环境，强化地域属性，配合优惠活动提升到店率 |
| Q92：教育机构如何投放？ | 展示教学成果和师资力量，精准定向家长人群，提供试听体验转化 |
| Q93：美容美发行业策略？ | 强调技术水平和效果对比，利用案例见证，预约制避免排队 |
| Q94：健身房如何做本地推？ | 展示健身环境和课程特色，定向运动爱好人群，推出体验课转化 |
| Q98：医疗健康类注意事项？ | 严格遵守广告法规，突出专业资质，避免疗效承诺，引导到店咨询 |

### 十一、实战案例类（10个精选）

| 问题 | 核心答案 |
|------|---------|
| Q101：餐饮店日预算500元怎么分配？ | 70%给团购成交计划，20%给短视频进店流量，10%给搜索直投 |
| Q102：新店开业第一周怎么投？ | 核心是"曝光+测试"，预算设高（目标成本×25），用3-5条不同素材测试 |
| Q103：老店投了3个月效果越来越差？ | 素材疲劳是主因，检查CTR、核销率、竞品活动，建3-5条新计划接力 |
| Q104：核销率30%问题出在哪？ | 团购有效期太长/预约不方便/描述与实物不符/门店位置难找 |
| Q106：口碑分从4.8降到4.3影响大吗？ | 影响显著，4.0-4.6会受到单量限制，需立即排查差评来源 |
| Q107：直播间和短视频投哪个好？ | 短视频60-70%日常引流 + 直播30-40%集中转化 |
| Q108：怎么判断计划该关停还是继续？ | CPA超目标2倍持续3天/连续3天零消耗/核销率<20%→关停 |
| Q109：多店连锁用一个账户还是分散？ | 5家店以内一个账户，5家以上或跨城市分散账户 |
| Q110：付费流量会挤压自然流量吗？ | 不会，两个独立推荐池 |

### 十二、竞品分析与差异化（4个精选）

| 问题 | 核心答案 |
|------|---------|
| Q111：怎么查看竞品在投什么广告？ | 巨量创意工具查看同行业热门素材，抖音搜索竞品品牌词 |
| Q112：竞品价格比我低怎么应对？ | 不打价格战，突出品质差异、增加附加值、改变套餐结构 |
| Q114：同行都在投本地推我不投会怎样？ | 同行加大投放会推高区域CPM，建议至少保持"防守型投放" |

### 十三、团队管理与效率（3个精选）

| 问题 | 核心答案 |
|------|---------|
| Q115：一个人管几个账户合适？ | 单人建议3-5个账户（每个日预算500-2000元） |
| Q116：怎么给优化师设定KPI？ | ROI占40%、核销率占20%、消耗完成率占20%、素材产出占20% |
| Q117：素材生产跟不上怎么办？ | 建立模板库、批量拍摄、引入AI剪辑、UGC激励 |

### 十四、大促与节点运营（3个精选）

| 问题 | 核心答案 |
|------|---------|
| Q118：五一/十一假期怎么提前准备？ | 提前2-3周设计节日套餐、拍摄素材、提高预算50-100% |
| Q119：双11本地推怎么玩？ | 囤券逻辑：超值套餐+提前3周预热+当天加大预算2-3倍+直播集中转化 |
| Q120：淡季怎么控制预算不浪费？ | 降低预算30-50%、保留核心跑量计划、用淡季测试新素材和新定向 |
