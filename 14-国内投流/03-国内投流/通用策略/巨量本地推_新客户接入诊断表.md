# 巨量本地推｜新客户接入诊断表（完整版）

> 适用对象：多行业本地推操盘手｜任何新客户进来，填完5个变量即可定方案

### 变量1：行业类目（对应红线等级和模板）

| ☐ **A** | 餐饮/咖啡/美业/健身/亲子游乐/便利店 | ⭐ 低 | 打法卡4 | 即时消费，低客单，冲动决策 |
| ☐ **B1** | 素质类教培（羽毛球/舞蹈/音乐/书法/篮球） | ⭐⭐ 中低 | 打法卡1 | 家长决策，子女使用，季节性强 |
| ☐ **B2** | 学历类教培（中专/职校/技校/成人学历） | ⭐⭐⭐⭐ 高 | 打法卡2 | 合规最严，电销转化，长周期 |
| ☐ **C1** | 家装（整装/局装/旧房翻新） | ⭐⭐⭐ 中高 | 打法卡3 | 超长决策，高客单，信任难建立 |
| ☐ **C2** | 建材（地板/瓷砖/门窗/橱柜） | ⭐⭐⭐ 中高 | 打法卡3 | B端为主，工程批量 |
| ☐ **D1** | 医疗健康（口腔/眼科/体检） | ⭐⭐⭐⭐⭐ 极高 | 打法卡5 | 合规最严，专业信任建立难 |

| ☐ **E1** | 汽车4S店/汽贸/二手车 | ⭐⭐⭐ 中高 | 打法卡6 | 超长决策，高客单，需多次跟进 |
| ☐ **E2** | 家电（空调/厨电/全屋定制） | ⭐⭐ 中低 | 打法卡6 | 季节性强，家装链条关联 |

| ☐ **F** | 其他本地生活行业 | 待定 | 通用方法论 | 参考通用框架，根据行业特征调整 |


### 变量3：转化路径（选择主目标和转化链路）

| 选项 | 路径 | 适用场景 | 适配工具 | 核心指标 |
|------|------|---------|---------|---------|
| ☐ **A** | **到店/核销** | 餐饮/美业/健身等实体门店 | 全域推广（团购成交） | 到店率/核销率 |
| ☐ **B** | **留资表单** | 家装/汽车/教培/医疗等高客单 | 标准推广（表单目标） | 表单量/表单成本 |
| ☐ **C** | **私信咨询** | 高客单需深度沟通/婚庆/婚纱摄影 | 标准推广（私信目标） | 私信量/套电成本 |
| ☐ **D** | **电话拨打** | 汽车/家装/医美/婚纱摄影 | 标准推广（电话目标） | 接通率/有效通话率 |
| ☐ **E** | **直播引导** | 餐饮团购直播/教培公开课/医美科普直播 | 标准推广（直播间） | 进场率/停留时长/下单率 |
| ☐ **F** | **多目标组合** | 需同时追求留资+到店+私信 | 主目标+辅目标组合 | 综合CPA/综合ROI |

**全域推广 vs 标准推广选择标准**：
- 选A（到店/核销）且客单<5000元 → 全域推广，1条计划就够
- 选B/C/D/E/F → 标准推广，按目标分别建计划


### 变量5：合规等级

| 选项 | 合规要求 | 审核重点 | 违规风险 | 投放注意事项 |
|------|----------|----------|---------|--------------|
| ☐ **A** | **低** | 基本广审，无特殊限制 | 低 | 无需特殊资质 |
| ☐ **B** | **中** | 行业特殊要求+基础广审 | 中 | 需行业资质（教培许可证/卫生证等） |
| ☐ **C** | **高** | 严格审查+特殊资质+广告审批 | 高 | 需办学许可证/施工资质等+广告审查表 |
| ☐ **D** | **极高** | 极严监管+随时抽查+专项审批 | 极高 | 医疗/学历需许可证+广告审查表+随时应对监管 |

**高合规行业前置动作**：
- 医疗（口腔/眼科/医美）：先确认医疗机构执业许可证+广告审查表，物料过审再投
- 学历（中专/职校）：先确认办学许可证+招生简章审批，物料过审再投
- 家装（施工）：先确认施工资质+安全生产许可


### B类组合（留资表单为主，标准推广）

适用变量3选B的客户，核心目标是收集表单留资进行电销。

| 变量组合 | 推荐打法 | 核心定向 | 素材重点 | 表单CPA基准 |
|----------|----------|---------|---------|-------------|
| B1 + 一致 + B + B + B | **标准推广·素质教培打法** | 父母定向（28-42岁）+教育/体育兴趣 | 效果展示+专业背书+成长故事 | 60-120元 |
| B2 + 一致 + B + C/D + C/D | **标准推广·学历教培打法** | 16-25岁（学生）+25-45岁（家长）+教育兴趣 | 校园环境+就业保障+政策解读 | 80-200元 |
| C1 + 一致 + B + C/D + C | **标准推广·家装打法** | 28-48岁+房产/家居/装修兴趣+DMP新小区 | 工地实拍+透明报价+口碑背书 | 150-400元 |
| D1/D2 + 一致 + B + C/D + D | **标准推广·医疗打法** | 20-45岁（医美）/25-55岁（医疗）+健康/美业兴趣 | 专业背书+科普+真实案例 | 100-300元 |
| E1 + 一致 + B/D + C/D + C | **标准推广·汽车打法** | 28-50岁+汽车兴趣+DMP高消费人群 | 车型展示+价格透明+口碑背书 | 200-500元 |
| E4 + 不一致-C + B/C + C/D + C | **标准推广·婚庆打法** | 22-35岁（新人）+婚庆兴趣+DMP婚庆人群 | 案例展示+服务保障+情感共鸣 | 200-400元 |


### D类组合（多目标组合）

适用变量3选F的客户，需要同时实现多个转化目标。

| 变量组合 | 推荐打法 | 主目标 | 辅目标 | 预算分配 |
|----------|----------|--------|--------|---------|
| B1 + 不一致-A + F + B + B | **标准推广·教培组合打法** | 表单（主） | 私信+电话（辅） | 主60%/辅40% |
| C1 + 一致 + F + C/D + C | **标准推广·家装组合打法** | 表单（主） | 私信+到店（辅） | 主70%/辅30% |
| A + 一致 + F + A + A | **标准推广·餐饮组合打法** | 团购（主） | 进店+私信（辅） | 主50%/辅50% |
| D2 + 一致 + F + B/D + D | **标准推广·医美组合打法** | 表单（主） | 私信+电话（辅） | 主50%/辅50% |

### Step 4：调整定向公式（1分钟）

在对应打法卡的「定向公式」基础上，根据变量2（决策人是否分离）调整：

| 变量2选项 | 定向调整方式 |
|-----------|-------------|
| **一致** | 直接使用打法卡默认定向 |
| **不一致-A（父母→子女）** | 在基础定向上改为「父母年龄段（28-42岁）+教育/亲子兴趣」，素材改为家长视角 |
| **不一致-B（子女→父母）** | 在基础定向上改为「子女年龄段（25-40岁）+医疗/健康兴趣」，素材改为孝心角度 |
| **不一致-C（多人决策）** | 定向前期决策人，素材兼顾多方关注点（案例展示+服务保障） |

## 六、诊断表使用说明

### 适用场景
- 新客户首次对接（5 分钟出方案）
- 老客户策略调整（重新评估变量变化）
- 团队内部方案评审（标准化输出格式）

### 注意事项
1. 变量1（行业类目）是最重要的变量，决定了后续所有策略方向
2. 变量2（决策一致性）经常被忽略，但会显著影响定向和素材策略
3. 变量4（口碑分）是"一票否决"项，< 4.0 必须先解决再投
4. 每次诊断结果需保存，3 个月后复盘对比变量变化
