# TikTok投流前准备清单

> **当前状态**：待真实投流后验证。本清单为参考级准备方案。
> **参考来源**：TikTok Business 官方文档（https://www.tiktok.com/business/en/learn）

---

## 一、TikTok Ads 账户

| 检查项 | 操作说明 | 状态 |
|--------|---------|------|
| 注册 Ads 账号 | business.tiktok.com | ☐ |
| 企业验证 | 提交营业执照 | ☐ |
| 支付方式 | 添加信用卡 | ☐ |
| 时区设置 | 客户当地时间 | ☐ |
| 账户限额 | 新账户有日限额，需逐步提升 | ☐ |

---

## 二、Pixel 安装

| 检查项 | 操作说明 | 状态 |
|--------|---------|------|
| Pixel 创建 | Events Manager 中创建 | ☐ |
| Pixel 安装 | 代码嵌入网站 `<head>` | ☐ |
| 事件配置 | PageView, Lead, CompleteRegistration | ☐ |
| 事件测试 | Test Events 验证 | ☐ |

---

## 三、TikTok 主页

| 检查项 | 操作说明 | 状态 |
|--------|---------|------|
| 企业号注册 | TikTok 企业号 | ☐ |
| Profile 完善 | 头像、Bio、链接 | ☐ |
| Bio 链接 | WhatsApp 或落地页 | ☐ |
| 首发内容 | 至少 5-10 条 organic 视频 | ☐ |

---

## 四、素材准备

| 素材 | 规格 | 数量 | 状态 |
|------|------|------|------|
| 产品运转视频 | 9:16, 15-30s | 5+ | ☐ |
| 工厂视频 | 9:16, 30-60s | 3+ | ☐ |
| 对比视频 | 9:16, 15-30s | 2+ | ☐ |
| 客户案例视频 | 9:16, 30-60s | 1+ | ☐ |

> TikTok 素材可以手机拍摄，但需保证画面清晰、稳定。

---

## 五、Creative Center 使用

| 检查项 | 操作说明 | 状态 |
|--------|---------|------|
| 注册 | creative.tiktok.com | ☐ |
| 竞品观察 | 搜索行业关键词 | ☐ |
| 热门素材 | 记录高互动素材方向 | ☐ |
| 趋势音乐 | 检查 Commercial Music Library | ☐ |

---

## 六、测试计划

| 项目 | 设置（参考级） |
|------|---------------|
| 首选目标 | Lead Generation 或 Conversions |
| 日预算 | $20-40/日 |
| 测试天数 | 14 天 |
| 素材数量 | 3-5 条 |
| 市场 | 东南亚 1-2 个（菲律宾/马来西亚） |

---

## 七、落地页/Lead Form

| 检查项 | 操作说明 | 状态 |
|--------|---------|------|
| Lead Form 设计 | 内置表单字段 | ☐ |
| 过滤问题 | 加入行业/产量等过滤 | ☐ |
| 落地页（如用 Conversions） | 移动端适配 | ☐ |
| 隐私条款 | 合规勾选 | ☐ |

---

## 八、总 Checklist

- [ ] TikTok Ads 账户注册并验证
- [ ] Pixel 安装并验证
- [ ] 企业号完善
- [ ] 首发 organic 内容 5+ 条
- [ ] 素材 10+ 条
- [ ] Creative Center 竞品观察
- [ ] Lead Form / 落地页就绪
- [ ] 测试计划就绪

> 全部勾选后方可开始投放。
