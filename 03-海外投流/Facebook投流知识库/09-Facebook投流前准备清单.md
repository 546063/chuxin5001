# Facebook投流前准备清单

> **当前状态**：待真实投流后验证。本清单为参考级准备方案，非真实投放经验。
> **参考来源**：Meta for Business 官方文档（https://www.facebook.com/business/help）

---

## 一、Business Manager（BM）准备

| 检查项 | 操作说明 | 状态 |
|--------|---------|------|
| 创建 BM | 访问 business.facebook.com，用企业邮箱注册 | ☐ |
| 验证企业身份 | 提交营业执照/公司注册文件 | ☐ |
| 添加主页 | 在 BM 中添加公司 Facebook 主页 | ☐ |
| 添加 Instagram 账号 | 关联 IG 商业账号 | ☐ |
| 添加 Pixel | 在 BM 中创建 Pixel | ☐ |
| 设置支付方式 | 添加信用卡/PayPal | ☐ |
| 域名验证 | 在 Brand Safety → Domains 中验证域名 | ☐ |
| 添加团队成员 | 按角色分配权限（管理员/广告运营/分析师） | ☐ |

> 参考：https://www.facebook.com/business/help/1710077379203657

---

## 二、广告账户准备

| 检查项 | 操作说明 | 状态 |
|--------|---------|------|
| 广告账户创建 | 在 BM 中创建广告账户 | ☐ |
| 账户限额确认 | 新账户日限额默认 $50，需逐步提升 | ☐ |
| 时区设置 | 设为客户当地时间（如 Asia/Riyadh） | ☐ |
| 货币设置 | 设为投放地区常用货币（USD 或 SAR） | ☐ |

> 新账户有日限额限制，需通过正常投放逐步解锁更高限额。

---

## 三、Pixel 安装与事件配置

| 检查项 | 操作说明 | 状态 |
|--------|---------|------|
| Pixel 创建 | 在 Events Manager 中创建 | ☐ |
| Pixel 安装 | 代码嵌入网站 `<head>`，或通过 GTM 安装 | ☐ |
| Pixel 助手 | 安装 Chrome 扩展验证 | ☐ |
| 标准事件配置 | PageView, Lead, CompleteRegistration, Contact | ☐ |
| 自定义事件 | 根据业务需求添加（如 ViewContent, AddToCart） | ☐ |
| Conversions API | 配置服务器端事件追踪（推荐） | ☐ |
| 事件测试 | 用 Test Events 验证事件正常触发 | ☐ |

> 参考：https://developers.facebook.com/docs/meta-pixel/

---

## 四、主页与 Messenger 配置

| 检查项 | 操作说明 | 状态 |
|--------|---------|------|
| 主页完善信息 | 头像、封面、关于、联系方式、营业时间 | ☐ |
| 欢迎消息设置 | Messenger → 自动回复设置 | ☐ |
| 常见问题配置 | 预设 3-5 个 FAQ 快捷按钮 | ☐ |
| 响应时间展示 | 确认主页显示"通常在几分钟内回复" | ☐ |
| CTA 按钮 | 设为"Send Message"或"Contact Us" | ☐ |

---

## 五、WhatsApp Business 配置

| 检查项 | 操作说明 | 状态 |
|--------|---------|------|
| WhatsApp Business App | 下载并设置商业资料 | ☐ |
| 自动回复 | 设置欢迎消息、离开消息 | ☐ |
| 快捷回复 | 预设 5-10 个常用话术模板 | ☐ |
| 产品目录 | 在 WA Business 中添加产品/服务 | ☐ |
| 与 Facebook 关联 | 在 BM 中关联 WhatsApp Business 账号 | ☐ |
| Click to WhatsApp 测试 | 确认广告点击后能正常打开 WA 对话 | ☐ |

> 参考：https://www.facebook.com/business/help/1064080798372304

## 六、素材准备

### 6.1 图片素材（至少 5 张）

| 素材 | 规格 | 说明 | 状态 |
|------|------|------|------|
| 工厂外观 | 1080x1080 / 1200x628 | 展示工厂规模 | ☐ |
| 产品特写 | 1080x1080 | 重点产品细节 | ☐ |
| 客户使用场景 | 1080x1080 | 海外客户工厂实拍 | ☐ |
| 对比图 | 1080x1080 | 人工 vs 机器 | ☐ |
| 证书/资质 | 1080x1080 | CE/ISO 等证书 | ☐ |

### 6.2 视频素材（至少 3 条）

| 素材 | 时长 | 比例 | 说明 | 状态 |
|------|------|------|------|------|
| 工厂全景 | 30-60s | 9:16 + 16:9 | 车间/设备/出货 | ☐ |
| 产品运转 | 15-30s | 9:16 | 机器工作画面 | ☐ |
| 客户案例 | 30-60s | 9:16 | 海外客户使用场景 | ☐ |

> 参考：https://www.facebook.com/business/help/972184699548742

### 6.3 广告文案（至少 3 套）

| 版本 | 方向 | 状态 |
|------|------|------|
| 文案 A | 痛点切入（人工成本高/产能不足） | ☐ |
| 文案 B | 产品展示（产能/材质/优势） | ☐ |
| 文案 C | 客户案例（信任背书） | ☐ |

## 七、落地页/Lead Form 准备

| 检查项 | 操作说明 | 状态 |
|--------|---------|------|
| 落地页加载速度 | 移动端 3 秒内加载 | ☐ |
| 移动端适配 | 手机浏览正常 | ☐ |
| 表单字段 | 姓名、邮箱、电话、公司名、需求描述 | ☐ |
| 隐私条款 | GDPR 合规勾选（欧洲市场） | ☐ |
| 感谢页 | 提交后显示感谢信息 + 下一步说明 | ☐ |
| Lead Form 内嵌 | 如用 Lead Ads，设计 Instant Form | ☐ |
| 过滤问题 | 加入"您的年产量""主要产品类型"等过滤问题 | ☐ |

## 八、产品目录准备

| 检查项 | 操作说明 | 状态 |
|--------|---------|------|
| 产品清单 | 列出主推 5-10 款产品 | ☐ |
| 产品图片 | 每款产品至少 1 张高清图 | ☐ |
| 产品描述 | 中/英/阿拉伯语描述 | ☐ |
| 价格范围 | 参考价或"Contact for price" | ☐ |
| 规格参数 | 产能/功率/尺寸/重量 | ☐ |

## 九、客服话术准备

| 场景 | 话术模板 | 状态 |
|------|---------|------|
| 首次响应 | "Thank you for your interest in our food processing machinery..." | ☐ |
| 发送产品目录 | "Here's our full catalog. Which products are you most interested in?" | ☐ |
| 报价引导 | "Could you share your production requirements so I can give you an accurate quote?" | ☐ |
| 跟进 | "Just following up on your inquiry about [product]. Do you have any questions?" | ☐ |
| 售后 | "Our machine comes with [X] year warranty. Here's our after-sales process..." | ☐ |

## 十、预算规划

| 项目 | 金额（参考级，待验证） | 说明 |
|------|----------------------|------|
| 测试期日预算 | $30-50/日 | 参考 Meta 官方 B2B 案例 |
| 测试期天数 | 14 天 | 至少跑完学习期 |
| 测试期总预算 | $420-700 | |
| 素材制作成本 | 人力成本为主 | 拍摄/剪辑 |
| 落地页开发 | 视情况 | 如需独立建站 |
| 工具费用 | $0-50/月 | 设计工具、去水印等 |

> 以上预算为参考级估算，需根据实际资金调整。
> 无历史数据时，建议先用小预算测试 14 天，再决定是否放量。
