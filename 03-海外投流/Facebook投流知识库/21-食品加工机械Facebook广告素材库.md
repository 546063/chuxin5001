## 附录 A：素材制作 Checklist

### 图片类素材

| 检查项 | 标准 |
|--------|------|

### 视频类素材

| 检查项 | 标准 |
|--------|------|
| 时长 | 主力投放 30s；再营销 15s；品牌曝光 45-60s |
| 前 3 秒 | 必须有视觉冲击或痛点提问（黄金 3 秒法则） |
| 字幕 | 全程英文字幕，无旁白时字幕即为文案 |
| BGM | 选择节奏感强但不喧宾夺主的音乐 |
| 竖版比例 | 9:16（Reels/Stories）+ 1:1（Feed）两个版本 |
| 结尾 CTA | 最后 5 秒必须有明确的 WhatsApp / Messenger 引导 |
| 产品展示 | 机器必须出现在画面中，最好有铭牌/Logo 特写 |

### 文案类素材

| 检查项 | 标准 |
|--------|------|
| Headline | 不超过 40 字符，包含核心卖点或数字 |
| Primary Text | 100-200 词，分段落，使用 emoji 分隔 |

| 痛点切入 | 至少 1/3 的文案从痛点出发 |
| 信任元素 | 提及认证（CE/ISO）、客户数量、国家数量等 |

> [!tip] 使用指南
> 1. **新品类上线**：从本文档对应品类直接复制素材方向 → 安排拍摄 → 复制文案 → 创建广告。
> 2. **A/B 测试**：同一品类用 3 条不同文案 + 2 种素材方向，共 6 个广告同时跑，3-5 天后保留最优组合。
> 3. **再营销**：将 30s 视频剪成 15s 版本，文案改为信任型（认证/客户数/售后保障），投放给看过主视频但未询盘的用户。
> 4. **WhatsApp vs Messenger**：东南亚优先 WhatsApp，欧美优先 Messenger，中东两者皆可。
> 5. **本地化**：投放阿拉伯语市场时，将文案翻译为阿拉伯语（参考 [[24-阿拉伯语Facebook广告文案模板]]）。
