# Facebook投流复盘与优化SOP

## 一句话总结

诊断前先拿三个数字：CPM → CTR → CVR，三步定位问题环节，7棵决策树覆盖所有常见问题。

## 核心结论

- 80% 的问题出在素材疲劳或 CPM 变化，先查这两个
- CPA 偏高 = CPM 高 或 CTR 低 或 CVR 低，先定位是哪个环节
- 素材疲劳判断：Frequency > 2.5 或 CTR 连续 3 天下降 > 20% 或跑了 2-4 周以上

---

## Step 0：先问 3 个数字

```
CPM = ?
CTR = ?
CVR = ?

CPM > 正常值 30%+  → 问题在展示层（先解决 CPM）
CPM 正常，CTR 低   → 问题在点击层（换素材）
CPM 和 CTR 正常    → 问题在转化层（换落地页/受众）
```

核心公式：`CPA = CPM ÷ (CTR × CVR)`

## 决策树 A：不起量（消耗 < 预期 30%）

```
广告不消耗 / 消耗极慢
├── 1. 查审核状态 → Disapproved → 修复 → 申诉
├── 2. 查出价 → Bid Cap/Cost Cap 太低 → 调高或切 Lowest Cost
├── 3. 查受众规模 → 受众 < 10 万人 → 扩到 Broad
├── 4. 查预算 → 日预算 < 目标 CPA × 3 → 提高预算
├── 5. 查优化事件 → 选了 Purchase 但数据极少 → 降目标到 AddToCart
└── 6. 查支付 → 付款方式过期 → 更新
```

**第一优先级动作**：确认审核状态。其他都可以慢慢调，审核不过一切白投。

## 决策树 B：成本高（CPA > 目标 30%+）

```
CPA 偏高
├── Step 1：查 CPM → 涨 30%+ → 排查展示层（季节性/受众窄/版位/质量分）
├── Step 2：查 CTR → < 1% → 排查点击层（素材不吸引/受众不匹配/频次>3）
└── Step 3：查 CVR → < 2% → 排查转化层（落地页速度/价格/信任元素/库存）
```

## 决策树 C：CTR 低（< 1%）

```
CTR 偏低
├── 检查运行时长 → 跑了 2 周+ → 素材疲劳 → 换新素材
├── 检查频次 → Frequency > 3 → 扩展受众
├── 检查素材 → Hook 留存率 < 40% → 重做 Hook
└── 检查版位 → Reels 版位 CTR 天然低 → 用 ThruPlay 评估
```

CTR 基准：优秀 > 2%，正常 1-2%，偏低 0.5-1%，危险 < 0.5%

## 决策树 D：学习期失败（Learning Limited > 7 天）

```
学习期过不了
├── 事件数不足 → 降目标到 AddToCart 或提高预算
├── 预算太低 → 日预算 < 目标 CPA × 3 → 提高预算
├── 受众太窄 → 受众 < 10 万人 → 扩到 Broad
├── 素材问题 → CTR < 0.5% 或质量分 Below Average → 换素材
└── 落地页问题 → 跳出率 > 80% → 优化落地页
```

## 决策树 E：素材疲劳

**疲劳判断（满足任一即需换）**：
- Frequency > 2.5
- CTR 连续 3 天下降 > 20%
- CPA 连续 3 天上涨 > 20%
- 跑了 2-4 周以上

```
素材疲劳
├── 立即动作：关掉疲劳广告（不要暂停），保留预算，新建广告组
├── 换素材策略：保留赢的框架，只改一个维度，一次上 3-5 条新素材
└── 预防措施：高预算+小受众 3-4 天换一轮，中等预算 7-10 天，低预算+大受众 14-21 天
```

## 决策树 F：广告被拒登

```
广告 Disapproved
├── 查看具体原因 → 图片/文案/落地页/受众定向违规
├── 修复后：点击"申请复审"，通常 24-48 小时内恢复
└── 申诉模板：说明已修改的内容 + 提供支撑材料
```

## 决策树 G：ROAS 下滑

```
ROAS 下滑
├── CPM 涨 > 30% → 季节性竞争 / 受众耗尽 / 竞品加投
├── CTR 降 > 20% → 素材疲劳，换素材
├── CVR 降 > 20% → 落地页问题 / 价格问题 / 库存问题
└── 三者都正常但 ROAS 降 → 客单价下降 / 归因窗口变化
```

## 30 秒快速诊断卡

| 顺序 | 指标 | 正常范围 | 异常动作 |
|------|------|---------|---------|
| 1 | 审核状态 | Active/Approved | Disapproved → 立即申诉 |
| 2 | 预算消耗率 | 80-100% | < 70% → 查出价/受众 |
| 3 | CPA | < 目标值 | > 目标 30% → 走决策树 B |
| 4 | Frequency | 1-2.5 | > 2.5 → 准备换素材 |
| 5 | ROAS 趋势 | 平稳或上升 | 连降 3 天 → 走决策树 G |

**原则：前 2 个数字正常 + 后 3 个数字正常 = 今天不需要做任何操作。**

## 关停判断标准

满足以下**任意 2 条**即关停：
- 花费 > 目标 CPA × 2 且 0 转化
- CTR < 0.5% 持续 3 天
- CPA > 目标 × 3 持续 3 天
- Frequency > 4 且 CTR 下降 > 30%
- 跑了 21 天 + ROAS < 目标 × 0.5

## 行业基准对照

| 品类 | CPM（美国） | CTR | CVR | 目标 CPA |
|------|------------|-----|-----|---------|
| 服装配饰 | $8-15 | 1.5-2.5% | 2-4% | $8-15 |
| 美妆护肤 | $10-20 | 1.2-2.0% | 1.5-3% | $12-25 |
| 家居日用 | $6-12 | 1.0-1.8% | 2-3.5% | $6-12 |
| 电子数码 | $12-25 | 0.8-1.5% | 1-2% | $15-30 |
| 食品保健 | $8-16 | 1.0-1.8% | 1.5-2.5% | $10-20 |
| **机械/B2B** | **$15-35** | **0.5-1.2%** | **0.5-1.5%** | **$30-80** |

## 操作节奏

| 频率 | 可以做的操作 |
|------|-------------|
| 每天 | 看 CPM/审核状态/预算消耗率（不动根） |
| 每周 | 关疲劳广告、上 2-3 条新素材、调整预算 ±20%、检查受众重叠 |
| 每两周 | 关 CPA 超目标 2 倍的组、复制成功的组、评估是否切 ASC |

### 绝对不要做的
- 一天内多次调整广告（会重置学习期）
- 预算一次加超过 50%（触发重新学习）
- 学习期内做大改（等 7 天再评估）
- 用 CTR 低于 0.5% 的素材继续跑

## 复盘模板核心结构

### 六维评分卡

| 维度 | 5分（优秀） | 3分（及格） | 1分（危险） |
|------|------------|------------|------------|
| ROAS | > 目标 120% | 目标 80-120% | < 目标 50% |
| CPA | < 盈利线 60% | 盈利线 60-100% | > 盈利线 |
| CTR | > 2% | 1-2% | < 0.8% |
| CVR | > 3% | 1.5-3% | < 1% |
| 学习期 | Active + 稳定 7天+ | Active 或 Learning 正常 | Learning Limited > 7天 |
| 预算消耗 | 90-100% | 70-90% | < 70% |

**总分 24-30**：优秀 → 加预算扩量
**总分 18-23**：及格 → 找失分维度，针对性优化
**总分 12-17**：危险 → 按诊断 SOP 排查
**总分 < 12**：紧急 → 建议关停重做
