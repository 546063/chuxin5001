# Facebook目标市场广告合规清单

## 一句话总结
投放前过一遍这份清单，避免因政策违规导致广告被拒或账户受限。

## 禁止内容（绝对不可投放）

| 类别 | 具体说明 | B2B 食品机械相关风险 |
|------|---------|-------------------|
| 非法产品或服务 | 推广违反当地法律的产品 | 一般不涉及 |
| 歧视性内容 | 基于种族、性别、宗教等歧视 | 文案避免"只卖给XX国家"等表达 |
| 误导性功能 | 广告落地页无法实现的功能 | 不要承诺不存在的认证或服务 |
| 仿冒品 | 假冒品牌产品 | 使用自有品牌，不要贴牌他人商标 |
| 恶意软件 | 含病毒或恶意代码的链接 | 确保网站安全，无恶意跳转 |
| 成人内容 | 色情、性暗示内容 | 不涉及 |
| 武器 | 枪支、弹药、爆炸物 | **锯骨机等涉及刀具的设备需注意：不要在素材中展示刀具作为武器的使用方式** |

## 受限内容（需特别注意）

### 1. 前后对比类素材

| 风险点 | 说明 | 规避建议 |
|--------|------|---------|
| 效果对比 | "使用前/使用后"类素材可能被拒 | 用数据对比替代视觉对比（如"效率提升 300%" 而非 "用前用后对比图"） |
| 体重/身体变化 | 不适用于机械行业 | - |

### 2. 夸大承诺风险

| 风险点 | 说明 | 规避建议 |
|--------|------|---------|
| 收入承诺 | "月赚10万"类表述 | 用设备参数说话："产能 500kg/h"，不要说"用了就发财" |
| 效果保证 | "100% 无故障" | 改用"提供 12 个月质保"等可验证承诺 |
| 免费但需付费 | "免费获取"实际需付费 | 明确标注价格范围或"获取报价" |

### 3. 图片文字比例

| 风险点 | 说明 | 规避建议 |
|--------|------|---------|
| 图片文字过多 | 文字覆盖超过图片 20% 可能被限制展示 | 使用 Facebook Text Overlay Tool 检查（已取消严格限制，但文字过多仍影响展示） |
| 文字质量 | 拼写错误、语法错误、过多大写 | 检查拼写，避免"FREE!!!"、"BUY NOW!!!" |

### 4. 落地页合规

| 风险点 | 说明 | 规避建议 |
|--------|------|---------|
| 落地页与广告不相关 | 广告展示 A 产品，落地页是 B 产品 | 确保落地页与广告内容一致 |
| 弹窗过多 | 过多弹窗或强制订阅 | 保持干净的落地页体验 |
| 缺少联系信息 | 无公司信息、无隐私政策 | B2B 广告落地页必须包含公司信息 |

## B2B 机械广告特别注意事项

### 可正常投放的内容

- 工业设备产品展示
- 工厂环境和车间实拍
- 设备运行效果演示
- 技术参数和规格
- 认证证书展示（CE/ISO/SGS）
- 客户案例和评价
- 报价/询盘引导

### 需要谨慎的内容

| 内容类型 | 风险等级 | 说明 |
|---------|---------|------|
| 涉及刀具/刀片展示 | 低 | 展示正常工业用途即可，不要突出锋利、切割人体等 |
| "免费"字眼 | 中 | "免费样品"可以，但不要单独用"FREE"作为钩子 |
| 价格展示 | 低 | 展示 FOB/CIF 价格范围可以，但避免"最低价""最便宜" |
| 客户工厂内部拍摄 | 低 | 确保获得客户授权 |
| 竞品对比 | 中 | 可以对比"人工 vs 机器"，不要点名对比具体竞品品牌 |

### 食品机械素材特别注意

- **卫生安全**：展示食品接触部分时，强调食品级材质（304不锈钢等），避免暗示不卫生
- **噪音/安全**：不要在素材中展示安全隐患（如工人不戴防护操作）
- **认证展示**：CE/ISO/SGS 等认证可以展示，但必须是真实有效的

## 中东市场敏感表达

### 沙特 / 阿联酋 / 卡塔尔 / 科威特

| 风险类别 | 需避免的表达 | 替代方案 |
|---------|-------------|---------|
| 宗教敏感 | 避免在斋月期间使用"庆祝"类词汇 | 用"斋月特惠"而非"斋月庆祝" |
| 饮食禁忌 | 猪肉加工设备广告在穆斯林国家可能被排斥 | 注意目标国家的宗教背景 |
| 文化符号 | 避免使用十字架、大卫之星等宗教符号 | 用中性工业符号 |
| 国家标识 | 某些政治敏感的国家组合 | 投放前检查当前地缘政治状况 |
| 女性形象 | 过于暴露的穿着 | 使用专业工作装束，或只展示设备 |
| 以色列相关 | 涉及以色列的内容在部分国家可能被抵制 | 注意目标受众国家 |

### 中东市场广告语言建议

- 使用正式阿拉伯语（Fusha），不要用方言
- 避免幽默和双关——在阿拉伯文化中可能不合适
- 使用 RTL（从右到左）排版的落地页
- 避免展示酒精、猪肉相关产品

## 东南亚市场敏感表达

### 马来西亚 / 印尼

| 风险类别 | 注意事项 |
|---------|---------|
| 清真认证 | 食品加工设备可以强调"适合清真食品生产" |
| 宗教节日 | 开斋节期间广告预算竞争加剧 |
| 多语言 | 马来语/英语双语广告效果更好 |

### 泰国 / 越南

| 风险类别 | 注意事项 |
|---------|---------|
| 本地语言 | 英语广告可接受，但本地语言效果更好 |
| 文化尊重 | 避免涉及政治和王室（泰国） |

## 合规自查清单

### 投放前检查（每条广告）

- [ ] 广告文案无夸大承诺（如"100%"、"保证"、"一定"）
- [ ] 广告图片文字覆盖率 < 20%
- [ ] 广告展示的设备和落地页一致
- [ ] 无拼写和语法错误
- [ ] 无竞品品牌名称
- [ ] 无宗教/政治/敏感文化内容
- [ ] 无违规武器展示（工业刀具正常使用场景除外）
- [ ] 落地页加载速度 < 3 秒
- [ ] 落地页包含公司联系信息
- [ ] 落地页无过多弹窗
- [ ] 广告目标受众不包含未成年人
- [ ] 素材无色情/暴力/歧视内容
- [ ] 价格信息准确（如有展示）
- [ ] 认证信息真实有效（如有展示）

### 中东市场额外检查

- [ ] 阿拉伯语文经由母语者校对（或使用已验证模板）
- [ ] 无宗教敏感内容
- [ ] 无饮食禁忌相关内容
- [ ] 女性形象穿着得体（如有出现）
- [ ] RTL 排版正确
- [ ] 无政治敏感国家/地区组合

### 投放后监控

- [ ] 每天检查广告状态（Active / Disapproved / Limited）
- [ ] 检查账户质量分（Account Quality）
- [ ] 收到违规通知后 24 小时内处理
- [ ] 定期查看 [Meta 广告政策更新](https://www.facebook.com/policies/ads)

## 广告被拒登处理流程

```
广告 Disapproved
├── 1. 查看拒登原因 → 广告管理器 → 账户质量
├── 2. 判断是否可以修改
│   ├── 可以修改 → 修改后提交复审（24-48小时）
│   └── 不可修改 → 删除并创建新广告
├── 3. 如果认为误判
│   ├── 点击"申请复审"
│   ├── 说明修改内容和合规依据
│   └── 等待审核结果（通常 24-48 小时）
└── 4. 多次拒登
    ├── 检查是否有账户级限制
    └── 联系 Meta 广告支持
```
