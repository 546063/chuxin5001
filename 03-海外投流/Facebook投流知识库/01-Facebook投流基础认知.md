# Facebook投流基础认知

## 一句话总结

2025-2026年Facebook投流的核心变化：Andromeda算法让**创意=定向**，广泛定向优于精细定向，账户结构要极度简化。

## 核心结论

- Andromeda算法用创意内容本身来匹配用户意图，而非传统手动兴趣定向
- 广泛定向（不限年龄/性别/兴趣）优于精细手动定向，因为给算法最大数据空间
- Meta视觉识别模型会将"相似文本覆盖的图片"视为同一创意，缺乏多样性会被惩罚
- 创意相似性高 = 平台判定为重复内容 = CPM升高
- 当你觉得广告在"烧钱"时，问题大概率出在创意而非设置

## 算法演变

### Andromeda 算法：创意 = 定向

**底层逻辑**：
- 广告创意成为主要定向信号——创意决定你能触达谁
- 给够学习时间，不要过早关闭 campaign
- 账户结构要极度简化：甚至可以把冷受众和再营销合并到一个 ASC campaign

### 创意多样性是算法新要求

**为什么需要多样性**：
- Meta视觉识别模型会将"相似文本覆盖的图片"视为同一创意
- 缺乏多样性 → 账户被惩罚 → CPM升高
- 创意相似性（Creative Similarity）高 = 平台判定为重复内容 = 涨成本

**执行方向**：
- **格式多样**：静图、短视频、创始人自拍、精致制作视频、GIF、轮播
  - 静图仍占Meta转化量60-70%，不要只做视频
- **文案长度多样**：超短/中/长/博客长度都要测试
- **心理角度多样**：痛点/乐趣/证言/好奇/直接提问
- **刷新频率**：小账户（几千/月）月度刷新；大账户周度刷新

### 原创内容判定

- Facebook用"视频指纹"识别内容结构和模式，而非面部识别
- 追踪哪个页面首次上传该内容，有原创历史记录的页面更安全
- 低质量转发（简单剪辑、无实质评论）会被降权

## 七步闭环流程

Facebook投流的核心框架遵循：**账户结构 → 受众定向 → 素材策略 → 出价优化 → 学习期管理 → 再营销 → 数据复盘**

### 第一步：账户结构
合理的账户结构是高效投放的基础：
- **账户层级**: 单账户多页面 vs 多账户分离管理策略
- **营销目标**: 认知、考虑、转化三大类目标选择
- **计划设置**: 每日预算、投放时间、频率控制

### 第二步：受众定向
- **核心受众**: 年龄、性别、位置等基础属性
- **兴趣受众**: 行为兴趣、购买意向、生活方式
- **自定义受众**: 网站访客、邮件列表、现有客户
- **相似受众**: 基于优质客户扩展的Lookalike人群

### 第三步：素材策略
- **静态图片**: 产品展示、场景应用、促销信息
- **视频素材**: 产品演示、客户见证、品牌故事
- **轮播广告**: 多产品展示、功能对比、使用教程
- **动态创意**: 自动组合不同元素的智能广告
