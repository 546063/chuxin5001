## 转化记录模板

### 记录格式

每条记录代表一个从广告线索到成交的完整客户转化链路。

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| conversion_id | string | ✅ | 格式：CONV-YYYYMMDD-001 |
| customer_id | string | ✅ | 格式：Customer-A, Customer-B... |
| lead_id | string | ✅ | 关联询盘编号：LEAD-YYYYMMDD-001 |
| case_id | string | ⬜ | 关联案例编号：CASE-YYYYMMDD-001 |
| source_platform | string | ✅ | 来源平台：Facebook / TikTok / LinkedIn |
| source_country | string | ✅ | 来源国家 |
| source_industry | string | ⬜ | 来源行业 |
| source_product | string | ⬜ | 来源产品 |
| first_contact_date | date | ✅ | 首次接触日期 |
| contact_method | string | ✅ | WhatsApp / Email / Phone / LinkedIn DM |
| lead_grade | string | ✅ | A / B / C / D（详见字段规范） |
| follow_up_count | number | ⬜ | 跟进次数 |
| quote_sent_date | date | ⬜ | 报价发送日期 |
| quote_amount | number | ⬜ | 报价金额 (USD) |
| closed_date | date | ⬜ | 成交日期 |
| closed_amount | number | ⬜ | 成交金额 (USD) |
| conversion_days | number | ⬜ | 从首触到成交天数 |
| conversion_status | string | ✅ | pending / quoted / won / lost / dormant |
| loss_reason | string | ⬜ | 丢单原因（lost 时填写） |
| data_source | string | ✅ | manual_entry / crm_export |
| verified_status | string | ✅ | pending / verified |
| notes | string | ⬜ | 备注 |
