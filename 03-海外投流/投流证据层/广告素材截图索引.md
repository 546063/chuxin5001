```

```

### 字段取值

| 字段 | 取值示例 | 说明 |
|------|----------|------|
| 平台 | facebook / tiktok / linkedin / google | 小写英文 |
| 国家 | sa / us / ae / ph / my | ISO 3166-1 alpha-2 |
| 行业 | food-machinery / packaging / textile | 小写连字符 |
| 产品 | dumpling-machine / packaging-line | 小写连字符 |
| 素材类型 | image-ad / video-ad / carousel / reel / story / landing-page | 小写连字符 |

| 编号 | 001, 002, 003 | 三位序号 |
