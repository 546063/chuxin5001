# 巨量引擎B2B素材方向扩展

> **当前状态**：参考级素材指南，基于平台规则和行业经验，待真实投放后验证。

---

## 一、视频素材方向

| 方向 | 占比 | 时长 | 说明 |
|------|------|------|------|
| 产品运转 | 40% | 15-30s | 机器运转 + 成品展示 |
| 工厂实力 | 25% | 30-60s | 车间全景 + 质检 + 出货 |
| 客户案例 | 15% | 30-60s | 客户工厂使用场景 |
| 对比测试 | 10% | 15-30s | 人工 vs 机器效率对比 |
| 科普教育 | 10% | 30-60s | 选型指南/行业知识 |

---

## 二、视频规格要求

| 项目 | 要求 |
|------|------|
| 比例 | 9:16 竖屏（推荐）或 16:9 |
| 时长 | 15-60 秒（推荐 30 秒） |
| 格式 | MP4 |
| 文件大小 | ≤ 500MB |
| 分辨率 | ≥ 720p |
| 字幕 | 必须（中文） |
| BGM | 可用抖音音效库 |

---

## 三、图文素材方向

| 类型 | 规格 | 用途 |
|------|------|------|
| 产品图 | 1080x1080 | 信息流广告 |
| 工厂图 | 1080x1920 | 竖屏信息流 |
| 客户案例图 | 1080x1080 | 社会证明 |
| Banner | 1080x1920 | 落地页头图 |

### 图文设计要点
- 标题醒目（大字报风格有效）
- 产品清晰展示
- 加入信任元素（证书/数字/案例）
- CTA 明确（立即咨询/免费报价）

---

## 四、落地页设计

### 核心模块
1. **头图**：产品大图 + 标题 + CTA
2. **产品优势**：3-4 个卖点
3. **参数规格**：表格形式
4. **客户案例**：1-2 个案例
5. **资质证书**：CE/ISO 等
6. **表单**：底部表单（姓名/电话/需求）

### 移动端适配
- 首屏加载 < 3 秒
- 按钮大且醒目
- 表单字段 < 5 个
- 底部悬浮 CTA
