# LinkedIn食品机械客户开发SOP

> **当前状态**：参考级 SOP，基于行业经验，待真实运营后验证。

---

## 一、账号包装

### 个人 Profile 设置
| 项目 | 设置建议 |
|------|---------|
| 头像 | 专业商务照 |
| Banner | 工厂/产品展示 + 联系方式 |
| 标题 | "Food Processing Machine Manufacturer | Export to 50+ Countries | CE/ISO Certified" |
| About | 200 字公司介绍 + 产品 + CTA |
| Experience | 公司主页 + 职位 |
| Featured | 产品目录/案例/视频链接 |

### 公司 Page 设置
| 项目 | 设置建议 |
|------|---------|
| Logo | 品牌 Logo |
| Banner | 产品展示 |
| 简介 | 公司介绍 + 关键词 |
| 产品页 | 添加核心产品 |
| 员工关联 | 员工关联到公司页 |

## 二、搜索策略

### 目标客户画像
| 条件 | 设置 |
|------|------|
| 行业 | Food & Beverages, Food Production |
| 职位 | Purchasing Manager, Operations Director, Factory Owner |
| 公司规模 | 50-500 人 |
| 地域 | 东南亚/中东/欧美 |

### 搜索关键词
```
"food processing manager"
"production manager" "food"
"purchasing" "food machinery"
"operations director" "food factory"
"food manufacturing"
```

### 搜索流程
1. 用 LinkedIn 搜索找到目标客户
2. 查看 Profile 确认匹配度
3. 发送连接请求（带个性化消息）
4. 通过后发送 DM

---

## 三、DM 话术模板

### 连接请求
```
Hi [Name], I noticed you're in the food processing industry.
We manufacture food processing machines and export to [country].
Would love to connect and share industry insights.
```

### 首次 DM（连接通过后）
```
Hi [Name], thanks for connecting!

We specialize in food processing machines (dumpling makers, 
vegetable cutters, meat processors) with 15+ years experience.

If you're ever looking to upgrade your production line, 
I'd be happy to send you our catalog and a free quote.

Best regards,
[Name]
[Company]
```

### 跟进 DM（7 天后）
```
Hi [Name], hope you're doing well!

We just published a case study on how a client in [country] 
increased their production 8x with our equipment. 
Thought it might be relevant to your work.

[Link to case study]

Best,
[Name]
```

---

## 四、跟进流程

```
Day 1: 发送连接请求
Day 3: 通过后发送首次 DM
Day 10: 跟进 DM（分享有价值内容）
Day 20: 再次跟进（产品更新/促销）
Day 30: 标记为长线跟进
```

### 线索分类
| 级别 | 条件 | 跟进频率 |
|------|------|---------|
| A 级 | 有明确需求 | 每周 |
| B 级 | 有兴趣但未明确 | 每 2 周 |
| C 级 | 无明确需求 | 每月 |
| D 级 | 无回复 | 每季度 |
