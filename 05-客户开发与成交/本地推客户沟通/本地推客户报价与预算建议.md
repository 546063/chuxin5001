## 一、服务费报价结构

### 三档服务报价

| 档位 | 月服务费 | 包含服务 | 适合客户 |
|---|---|---|---|
| 基础版 | 3000-5000 元/月 | 账户搭建+日常优化+周报 | 有素材能力、只需优化的客户 |
| 标准版 | 5000-10000 元/月 | 基础版+素材制作指导+月报+月度复盘 | 需要素材指导的客户 |
| 旗舰版 | 10000-20000 元/月 | 标准版+素材代制作+全链路承接+周会 | 需要全托管的客户 |

|---|---|---|---|
| 测试档 | 1500-3000 元 | 30-100 条 | D1-D30 测试期 |
| 起步档 | 3000-6000 元 | 60-200 条 | D30-D90 成长期 |
| 放量档 | 6000-15000 元 | 120-500 条 | D90+ 稳定期 |
| 品牌档 | 15000-50000 元 | 300-1500 条 | 多店/连锁品牌 |

## 二、报价计算逻辑

### 服务费定价因素
| 因素 | 加价 | 说明 |
|---|---|---|
| 基础服务费 | - | 账户搭建、优化、报告 |
| +素材制作 | +30-50% | 拍摄、剪辑、文案 |
| +全链路承接 | +20-30% | 私信回复、电话接听 |
| +多店管理 | +10-20%/店 | 每个门店单独管理 |
| +紧急项目 | +10-20% | 需要在1周内出效果 |

### 报价公式
```
月总费用 = 服务费 + 广告费

月预期线索 = 广告费 / 目标线索成本
月预期成交 = 月预期线索 × 有效率 × 到店率 × 成交率
月预期收入 = 月预期成交 × 客单价
月ROI = 月预期收入 / 月总费用
```

## 四、ROI测算示例

### 美容美发店 ROI 测算
```
月广告费：5000 元
月服务费：5000 元
月总费用：10000 元

预期线索：5000 / 30 = 167 条
有效率：50% → 有效线索 83 条
到店率：40% → 到店 33 人
成交率：30% → 成交 10 人
客单价：800 元
月收入：10 × 800 = 8000 元

首月 ROI：8000 / 10000 = 1:0.8（亏损，正常）

注意：
- 首月是测试期，通常亏损
- 第2-3月优化后 ROI 可提升到 1:2-1:3
- 加上复购客户，月度 ROI 会更健康
```

## 五、报价沟通话术

### 客户问"你们收多少钱？"
```
"我们的服务费分为三个档位，看您需要哪些服务：

基础版 XXX 元/月 — 我们帮您管理账户和优化
标准版 XXX 元/月 — 加上素材指导和内容建议
旗舰版 XXX 元/月 — 全托管，包括素材制作和线索承接

广告费是直接充值到巨量引擎账户的，我们不经手。
建议先从 [测试档] 开始，月广告费 XXX 元。

您觉得哪个档位比较适合您？"
```

### 客户说"太贵了"
```
"我理解您的想法。不过您算一笔账：

如果我们每月帮您获 [X] 条有效线索，按您的成交率能成交 [X] 单，
每单利润 [X] 元，月收入就是 [X] 元。

对比您现在每月花在 [美团/百度/地推] 上的 [X] 元，
抖音的获客成本其实更低，而且客户质量更高。

要不我们先做1个月试试？效果不好可以不续。"
```

### 客户问"效果能保证吗？"
```
"坦白说，投放效果受很多因素影响：
- 素材质量
- 市场竞争
- 承接效率
- 销售转化

我们能保证的是：
1. 每周给您详细的数据报告
2. 发现效果不好立即调整
3. 如果连续1个月没效果，我们免费延长服务

但我们不能保证具体的线索数量，因为平台算法和市场是动态的。
不过根据我们的经验，[行业] 的月线索量通常在 [X-X] 条之间。"
```
