你这个需求本质上不是“查单个机器参数”，而是要搭建一套适合外贸社媒的“产业带 + 工厂能力 + 客户需求 + 出口卖点”研究框架。  
这些设备非常多，建议不要一台一台零散查，而是先按产品族做研究，再补重点机型。

**先分组**
建议先按这 6 类去整理，不然信息会很碎：

1. 清洗类  
`鼓泡喷淋清洗机、震动式涡流清洗机`

2. 去皮类  
`连续式削皮机、毛刷去皮机`

3. 脱水类  
`变频式离心甩干机`

4. 果蔬切割类  
`分半机、切丝机、切片机、切丁机、切块机、切菜机、剁块机`

5. 肉类加工类  
`冻肉切丁机、冻肉切片机、鲜肉切片机、分层机、分条机、熟肉切片机、砍排机、锯骨机、绞肉机、去皮机、肉饼成型机`

6. 面食/配套/整线类  
`包子机、饺子机、推条机、提升输送机、挑选工作台、整颗菜综合类生产线、根茎类加工生产线`

**你需要了解的核心信息**
做外贸社媒，至少要整理这 8 类信息：

1. 产业带信息  
`河北哪些城市/县区在做这类设备，是否有明显产业集群，核心配套在哪里，和山东/河南/广东相比优势是什么`

2. 工厂能力信息  
`工厂做单机还是整线，标准机还是定制机，是否支持OEM/ODM，交期，月产能，工程师人数，售后方式`

3. 产品技术信息  
`适用原料、处理能力、切割规格、材质、刀具方案、电压、功率、是否变频、是否可定制、易损件、清洗维护难度`

4. 客户应用场景  
`餐饮中央厨房、净菜工厂、预制菜工厂、肉类加工厂、果蔬加工厂、冷冻食品厂、学校食堂、商超配送中心`

5. 出口市场信息  
`主要出口国家、目标市场偏好、中东/东南亚/俄罗斯/非洲/拉美分别更看重什么、常见认证要求`

6. 贸易信息  
`海关编码、出口是否常见、运输方式、整机/木箱/拼柜要求、是否涉及食品接触材料要求`

7. 社媒内容卖点  
`效率提升、人工替代、损耗降低、切割一致性、卫生标准、整线方案、投资回报周期、客户案例`

8. 竞品信息  
`河北本地竞品、山东诸城/博兴/潍坊等竞品、阿里国际站/中国制造网/独立站上同类厂家怎么写卖点`

**提示词 1：做产业带总研究**
```text
请帮我研究“河北食品加工机械/果蔬加工机械/肉类加工机械”的产业带情况，重点围绕以下设备类别：
1. 清洗类：鼓泡喷淋清洗机、震动式涡流清洗机
2. 去皮类：连续式削皮机、毛刷去皮机
3. 脱水类：变频式离心甩干机
4. 果蔬切割类：分半机、切丝机、切片机、切丁机、切块机、切菜机
5. 肉类加工类：冻肉切丁机、冻肉切片机、鲜肉切片机、分层机、分条机、砍排机、锯骨机、绞肉机、肉饼成型机
6. 面食与配套类：包子机、饺子机、推条机、提升输送机、挑选台、整线

请输出：
- 河北相关产业带城市/县区分布
- 每个区域主要做哪些机器
- 当地产业链配套特点
- 与山东、河南、广东相比的优势和短板
- 适合做外贸社媒的切入点
- 如果信息不足，请给出“高概率产业带判断”和依据
- 最后整理成表格
```

请输出以下内容：
- 产品定义：它是什么，解决什么问题
- 适用行业和客户类型

- 关键技术参数
- 不同型号差异

- 客户采购时最关心的问题
- 海外客户常问问题
- 主要出口市场和市场需求特点
- 与竞品相比的卖点
- 社媒可传播的内容角度
- 适合拍摄的视频素材点
- 跟工厂沟通时必须确认的问题清单
- 跟客户沟通时必须了解的需求清单

最后请输出：
1. 一段适合销售快速理解的中文总结
2. 一份表格版信息卡

```

- 工厂通常有哪些生产能力
- 单机厂、整线厂、贸易型工厂的区别
- 真实工厂的核心竞争力体现在哪些地方
- 哪些参数最能体现专业度
- 哪些证书/案例/出口记录最值得展示
- 如何判断一家工厂是否真的有实力
- 哪些内容适合做成社媒素材增强信任感

最后请给我：
- 工厂沟通提问清单
- 判断工厂实力的评分表
- 社媒内容素材采集清单
```

【贴产品分类或产品名】

目标平台：
- TikTok

请输出：
- 每类产品的核心卖点

- 每个选题说明适合吸引哪类客户

**跟工厂沟通，你必须搞清楚的问题**
这些是最关键的，不然社媒内容会很空：

1. `这台机器主要卖给谁`
2. `最常出口到哪些国家`
3. `客户最常买的型号是什么`
4. `客户为什么选你，不选别家`
5. `这台机器最核心的3个卖点是什么`
6. `哪些参数是客户最在意的`
7. `哪些部分可以定制`
8. `有没有整线方案`
9. `有没有现场运行视频、案例、客户反馈`
10. `交期、售后、备件、安装支持怎么做`
11. `有没有CE或其他认证`
12. `哪些同行最常跟你竞争`

**跟客户沟通，你要提前知道的问题**
1. `客户加工什么原料`
2. `每小时产量要求多少`
3. `切割/清洗/去皮效果要求是什么`
4. `原料尺寸、硬度、温度状态是什么`
5. `单机还是整线`
6. `预算区间`
7. `电压要求`
8. `场地大小`
9. `是否需要定制`
10. `是否需要安装调试和售后支持`

**你后续整理时的建议方向**
最省力的方式不是“全产品平铺”，而是这样做：

1. 先做 `河北产业带总览`
2. 再做 `6大产品族研究`
3. 再从每类里挑 `2-3个主推机型` 深挖
4. 最后输出 `工厂沟通清单 + 客户沟通清单 + 社媒选题库`
