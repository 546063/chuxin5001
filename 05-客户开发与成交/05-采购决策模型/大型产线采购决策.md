# 大型产线采购决策（$50,000-$200,000+）

## 一句话总结
> 大型产线（完整肉类加工线、中央厨房整线等）决策周期5-9个月，5-10人委员会决策，需要工厂审核、现场考察、多轮谈判。

---

## 决策时间线

| 阶段 | 时长 | 关键行为 |
|------|------|----------|
| 战略规划 | 2-4周 | 管理层决议升级产线 |
| 需求评估 | 2-4周 | 内部需求调研、规格定义 |
| 供应商预审 | 4-6周 | 工厂资质、认证、参考客户 |
| RFQ/RFP | 4-6周 | 向3-5家发详细技术需求 |
| 技术评估 | 4-8周 | 工厂考察、试机、参考客户拜访 |
| 商务谈判 | 2-4周 | 价格、付款、售后SLA |
| 管理层审批 | 2-4周 | 董事会/CFO审批 |
| 合同签署 | 2-4周 | 法律审查、签约 |
| **总计** | **5-9个月** | |

---

| 角色 | 职责 | 影响力 |
|------|------|--------|
| CEO/老板 | 最终审批 | ★★★★★ |
| 生产总监 | 技术选型主导 | ★★★★★ |

| 外部顾问 | 第三方评估 | ★★ |

## 关键决策要素

### 工厂审核（必须过）
- [ ] 现场参观（客户派人到中国工厂）
- [ ] 生产能力评估
- [ ] 质量管理体系审核
- [ ] 同类型产线运行案例参观

### 技术评估
- [ ] 试机（客户自带原料到工厂测试）
- [ ] 产线设计方案评审
- [ ] 产能/得率保证条款
- [ ] 自动化程度和控制系统

### 商务评估
- [ ] 多方案报价（基本/标准/高端）
- [ ] 付款条件（L/C常见）
- [ ] 售后SLA（响应时间、备件供应）
- [ ] 安装和培训方案

### 风险控制
- [ ] 银行保函/LC保护
- [ ] 性能保证条款
- [ ] 延迟交付违约金
- [ ] 保留金（5-10%到成功调试后支付）

---

## 成交加速因素

| 因素 | 影响 |
|------|------|
| 邀请到工厂参观 | 信任度提升10倍 |
| 提供完整设计方案（非单纯报价） | 技术评分领先 |
| 安排参考客户现场拜访 | 消除最后疑虑 |
| 灵活的付款方案（L/C+里程碑） | 财务审批更容易 |
