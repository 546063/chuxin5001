## WhatsApp发送规则

| 规则 | 说明 |
|------|------|
| 发送时间 | 对方工作时间（考虑时差） |
| 频率 | 首次→2天后跟进→1周后再次跟进 |
| 语气 | 简洁、专业、友好 |
| 表情 | 适度使用，不要过多 |
| 附件 | 先发文字，确认兴趣后再发图片/PDF |
| 跟进 | 24-48小时内回复 |
