# 中型设备采购决策（$10,000-$50,000）

## 一句话总结
> 中型设备（自动切割线、清洗去皮线、中型灌肠生产线等）决策周期2-4个月，2-4人参与决策，需要技术验证和商务谈判并行。

---

## 决策时间线

| 阶段 | 时长 | 关键行为 |
|------|------|----------|
| 需求触发 | 1-2周 | 生产经理发现问题，向老板汇报 |
| 内部讨论 | 1-2周 | 确定规格、预算、时间表 |
| 供应商调研 | 2-3周 | 联系5-8家供应商，获取报价 |
| 技术评估 | 2-3周 | 视频验证、寄样测试、认证检查 |
| 商务谈判 | 1-2周 | 价格、付款条件、售后 |
| 内部审批 | 1周 | 老板/CFO确认 |
| 付款下单 | 1-2周 | 合同签订 |
| **总计** | **2-4个月** | |

---

## 决策参与者

| 角色 | 职责 | 影响力 |
|------|------|--------|
| 生产经理 | 技术规格定义、设备验证 | ★★★★★ |
| 老板 | 预算审批、最终拍板 | ★★★★★ |
| 采购经理 | 供应商比较、价格谈判 | ★★★ |
| QA（大公司） | 合规审核 | ⚠️ 否决权 |

---

## 关键决策要素

### 技术验证（必须过）
- [ ] 寄样测试：客户寄产品到工厂，录制测试视频
- [ ] 视频通话：实时看设备运行
- [ ] 技术参数确认：产能、功率、材质
- [ ] 认证检查：CE/Halal/SASO等

### 商务验证
- [ ] 3家以上报价对比
- [ ] TCO分析（含运费、关税、安装、5年备件）
- [ ] 付款条件协商
- [ ] 售后方案确认

### 风险控制
- [ ] 供应商资质确认（营业执照、出口经验）
- [ ] 同国客户参考
- [ ] 保修条款确认
- [ ] 备件清单和价格

---

## 成交加速因素

| 因素 | 影响 |
|------|------|
| 主动提供寄样测试视频 | 缩短技术评估阶段50% |
| TCO对比报告 | 减少价格谈判轮次 |
| 同国客户可视频通话 | 信任度大幅提升 |
| 提供安装方案（远程/上门） | 消除售后顾虑 |
