# 小型设备采购决策（$2,000-$10,000）

## 一句话总结
> 小型设备（单机：切肉机、小型切菜机、小型灌肠机等）决策周期3-5周，通常由老板一人拍板，Alibaba+YouTube是主要信息渠道，WhatsApp是主要成交工具。

---

## 决策时间线

| 阶段 | 时长 | 关键行为 |
|------|------|----------|
| 需求触发 | 1-3天 | "我需要一台XX机器" |
| 搜索调研 | 1-2周 | Alibaba搜索、YouTube看视频、联系2-3家 |
| 技术确认 | 3-5天 | 参数对比、视频通话 |
| 价格谈判 | 2-3天 | WhatsApp议价、要折扣 |
| 付款下单 | 2-5天 | T/T预付、确认PI |
| **总计** | **3-5周** | |

---

## 典型买家画像

| 特征 | 描述 |
|------|------|
| 客户类型 | 烧烤供应链、小餐厅、小型食品店、创业型 |
| 决策人 | 老板/创始人（一人说了算） |
| 预算敏感度 | 高（价格是第一考虑因素） |
| 技术深度 | 浅（不需要复杂技术参数，能看懂视频就行） |
| 信息渠道 | Alibaba（60%）、YouTube（25%）、同行推荐（15%） |

---

## 采购行为特征

### 发现渠道
| 渠道 | 占比 | 说明 |
|------|------|------|
| Alibaba搜索 | 60% | "meat slicer"、"vegetable cutter" 等关键词 |
| YouTube | 25% | 看设备演示视频 |
| 同行推荐 | 10% | WhatsApp群组、行业群 |
| Facebook | 5% | 刷到广告 |

### 验证方式
- YouTube视频（最核心）
- 阿里巴巴评价和交易记录
- WhatsApp视频通话看设备
- 不需要工厂审核

### 付款偏好
- T/T 30%预付 + 70%发货前
- Alibaba Trade Assurance
- 不接受L/C（金额太小）

---

## 成交加速因素

| 因素 | 影响 |
|------|------|
| YouTube视频质量高 | 缩短调研阶段50% |
| WhatsApp响应<1小时 | 成交概率翻倍 |
| 提供同国客户参考 | 信任度大幅提升 |
| 现货可立即发货 | 加速决策（等待耐心有限） |
| 包邮/含配件 | 减少谈判轮次 |

## 社媒内容策略

### Facebook 广告
- 钩子："$X,XXX起，一台机器替代3个工人"
- 素材：设备运行短视频
- CTA：WhatsApp联系获取报价

### TikTok 视频
- 主题：设备运行展示（越简单越好）
- 风格：实拍、无剪辑特效
- 时长：15-30秒

### WhatsApp 转化
- 响应时间：<30分钟
- 首次回复内容：报价 + 视频链接 + 1个客户案例
