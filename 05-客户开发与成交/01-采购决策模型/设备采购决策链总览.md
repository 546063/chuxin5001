# 设备采购决策链总览

## 一句话总结
> 设备采购经历七步流程：需求触发→内部讨论→供应商筛选→技术评估→商务谈判→内部审批→签约执行。不同阶段由不同角色主导，销售需在正确的时间接触正确的人并提供正确的信息。

---

## 七步决策流程

### Step 1: 需求触发（Awareness）
**触发人**: 通常是生产经理或老板

| 触发类型 | 具体场景 |
|----------|----------|
| 产能瓶颈 | 订单增加，现有设备/人工无法处理 |
| 品质问题 | 客户投诉出品不一致 |
| 合规压力 | 食安审核不合格、新规要求 |
| 成本压力 | 人工成本持续上涨 |
| 竞争压力 | 竞争对手自动化后降价 |
| 新业务 | 签订新客户/进入新市场 |

**销售机会**: 此时客户开始搜索。确保你的内容（Facebook广告、YouTube视频、Alibaba listing）能被客户找到。

---

### Step 2: 内部讨论（Internal Alignment）
**参与人**: 老板 + 生产经理 +（采购经理）

| 讨论内容 | 主导角色 |
|----------|----------|
| "我们到底需要什么设备？" | 生产经理 |
| "预算多少？" | 老板 |
| "什么时候要？" | 生产经理 |
| "自己进口还是找本地代理？" | 老板/采购 |

**销售机会**: 客户可能在Alibaba/Google搜索。关键词广告很重要。

---

### Step 3: 供应商筛选（Supplier Research）
**参与人**: 采购经理（大公司）/ 老板（小公司）

| 行为 | 说明 |
|------|------|
| Alibaba/Made-in-China搜索 | 80%以上的起点 |
| YouTube看设备演示 | 最关键的验证手段 |
| 联系3-8家供应商 | 获取报价和方案 |
| 工厂资质检查 | 营业执照、认证、网站 |
| 同行推荐 | 信任度最高 |

**销售机会**: Alibaba排名、YouTube视频质量、WhatsApp响应速度是关键。

---

### Step 4: 技术评估（Technical Evaluation）
**参与人**: 生产经理 + QA（大公司）

| 评估维度 | 内容 |
|----------|------|
| 技术参数 | 产能/h、功率、材质（SS304/316L） |
| 认证 | CE/FDA/Halal/SASO/SNI |
| 案例 | 同类型/同国家客户案例 |
| 验证 | 视频通话看设备、寄样测试、工厂审核 |

**销售机会**: 提供高质量演示视频、技术参数表、客户案例。

---

### Step 5: 商务谈判（Commercial Negotiation）
**参与人**: 采购经理 + 老板

| 谈判要素 | 内容 |
|----------|------|
| 价格 | FOB/CIF/DDP报价 |
| 付款条件 | T/T（30%预付+70%发货前）、L/C |
| 保修 | 1-2年、备件包 |
| 交期 | 生产周期+运输时间 |
| 安装培训 | 是否含工程师上门 |

**销售机会**: 灵活的付款方案、打包报价（含安装+培训+备件）能加速成交。

---

### Step 6: 内部审批（Internal Approval）
**参与人**: 老板/CFO/委员会

| 审批要素 | 说明 |
|----------|------|
| ROI分析 | 投资回报期 |
| 风险评估 | 供应商可靠性 |
| 预算确认 | 是否在预算内 |
| 多方比价 | 大公司需3家以上报价 |

**销售机会**: 提供ROI计算帮助、风险评估文件、客户参考名单。

---

### Step 7: 签约执行（Contract & Execution）
**参与人**: 采购经理 + 法务 + 财务

| 执行事项 | 说明 |
|----------|------|
| 合同签署 | PI/合同确认 |
| 付款 | 预付款到账 |
| 生产排期 | 工厂开始生产 |
| 进度跟进 | 定期更新生产进度 |
| 发货 | 装箱、报关、海运 |
| 安装调试 | 工程师上门或远程指导 |

---

## 决策角色权责分工

| 角色 | 发起需求 | 技术评估 | 商务谈判 | 最终决策 |
|------|:-------:|:-------:|:-------:|:-------:|
| 生产经理 | ✅ 常见 | ✅ 主导 | ⚪ 参与 | ❌ 无 |
| 老板 | ✅ 常见 | ⚪ 了解 | ⚪ 了解 | ✅ 必须 |
| 采购经理 | ❌ 少见 | ⚪ 协助 | ✅ 主导 | ❌ 无 |
| QA | ❌ 少见 | ✅ 关键 | ❌ 不参与 | ⚠️ 否决权 |
| 财务 | ❌ 少见 | ❌ 不参与 | ⚪ 付款审核 | ⚠️ 预算否决 |

## 典型异议处理地图

| 阶段 | 常见异议 | 应对策略 |
|------|----------|----------|
| 筛选 | "中国设备质量行吗" | 发认证+客户案例+视频 |
| 筛选 | "没见过这个牌子" | 发工厂视频+成立年份+出口国家 |
| 评估 | "能处理我们的产品吗" | 寄样测试/视频验证 |
| 评估 | "没有CE/目标国认证" | 立即补发或说明申请中 |
| 谈判 | "太贵了" | TCO对比+ROI计算 |
| 谈判 | "付款条件太严" | 灵活方案（分期/LC） |
| 谈判 | "交期太长" | 库存现货/加急方案 |
| 审批 | "售后怎么办" | 本地备件库+远程支持方案 |
