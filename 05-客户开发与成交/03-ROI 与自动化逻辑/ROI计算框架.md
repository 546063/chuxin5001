# ROI计算框架

## 一句话总结
> 食品加工设备ROI = (人工节省 + 得率/损耗改善 + 产能扩张收入 + 品质溢价) ÷ 设备总投入。不同行业/设备的回收期从1个月到36个月不等。

---

## 通用公式

### 基础公式
```
ROI（年化）= 年度净收益 ÷ 设备总投入 × 100%
回收期（月）= 设备总投入 ÷ 月度净收益
```

### 设备总投入包含
| 项目 | 说明 | 占比 |
|------|------|------|
| 设备出厂价 | FOB价格 | 60-70% |
| 国际运输 | 海运/陆运 | 5-15% |
| 关税/进口税 | 根据HS Code和国家 | 5-15% |
| 安装调试 | 工程师差旅+人工 | 3-10% |
| 培训 | 操作员培训 | 1-3% |
| 备件包 | 首年备件 | 2-5% |

---

## 四大收益来源

### 1. 人工替代 💰（最直接、最好算）
```
人工节省 = 替代人数 × 人均月薪（含社保福利）
```

| 客户类型 | 典型替代人数 | 人工成本参考 |
|----------|-------------|-------------|
| 中央厨房 | 5-15人 | 中东$400-800/人/月，东南亚$200-500 |
| 肉类加工厂 | 3-10人 | 东南亚$250-600，东欧$500-1,000 |
| 烧烤供应链 | 5-8人（穿串环节） | 中东$300-600 |
| 预制菜工厂 | 3-8人 | 东南亚$250-500 |

### 2. 得率/损耗改善 📈（ROI最高但最难证明）
```
得率收益 = 得率提升% × 年度原料支出
损耗降低 = 损耗降低% × 年度损耗金额
```

| 改善项目 | 典型改善幅度 | 收益量级 |
|----------|-------------|----------|
| 切割精度提升（得率+1-3%） | 月处理50吨的工厂 | $15,000-$45,000/月 |
| 自动灌装（损耗降低） | 香肠厂 | $3,000-$10,000/月 |
| 包装升级（保质期延长→损耗降低） | 预制菜厂 | $2,000-$8,000/月 |

### 3. 产能扩张收入 📊（战略价值最大）
```
新增收入 = 新增产能 × 产品单价 × 产能利用率
```

| 场景 | 产能提升 | 收入影响 |
|------|----------|----------|
| 中央厨房接新连锁合同 | 产能翻倍 | 营收可翻倍 |
| 烧烤供应链旺季交付 | 从拒绝订单到全部接单 | 季节性收入+$50K/月 |
| 肉类加工厂深加工升级 | 从原料到调理品 | 单价翻倍 |

### 4. 品质溢价 ⭐（最难量化但最持久）
- 切面一致性好 → 客户满意度高 → 复购率提升
- 品牌升级（散装→零售）→ 单价翻倍
- 出口认证 → 进入新市场

## ROI沟通话术

### 对老板说
> "王总，您现在有8个工人做切菜，每人月成本$500，合计$4,000。这台切菜机$15,000，替代8人，每月直接省$4,000。**不到4个月就回本**。第5个月开始每个月净赚$4,000。一年就是$48,000。"

### 对生产经理说
> "李经理，这台机器每小时处理500kg，相当于8个工人的量。工人只要看着机器就行，培训2天上手。切面一致性比人工好30%，您的客户投诉会减少一半。"

### 对采购说
> "这是完整报价：设备$15,000 + 海运$2,000 + 关税$1,500 = $18,500全包。保修2年，送价值$500的备件包。比A家的$20,000便宜$1,500，而且产能多20%。"
